# P1: 퀵서치 + 제목/썸네일 — 소금

## 핵심 팩트 리포트 (퀵서치)

1. [사실] Salary(급여), soldier(군인), salad(샐러드) 어원이 모두 라틴어 sal(소금)에서 유래 (출처: 위키백과, 한국민족문화대백과사전)
2. [사실] 14세기 서아프리카 팀북투에서 소금 90kg은 금 450g과 교환 — 같은 무게 교환 기록 (출처: World History Encyclopedia, 이븐 바투타 1352년 기록)
3. [사실] 프랑스 가벨(소금세)은 소금 원가의 20배 세금, 매년 3만 명 투옥·500명 처형 → 프랑스 대혁명 기폭제 중 하나 (출처: 오마이뉴스)
4. [사실] 간디의 소금 행진(1930) — 6만 명이 386km 행진, 영국 소금법 저항 → 인도 독립 운동 전환점 (출처: 위키백과)
5. [사실] 인류 역사상 가장 오래된 도시 예리코(Jericho)는 소금을 구할 수 있는 사해 근처에서 탄생 (출처: 한국민족문화대백과사전)

## 제목/썸네일 선정

**제목:** 이걸로 월급 받고 전쟁까지 했다;; | 소금
- 이문호공: 상식파괴(5) + 호기심(3)
- 동사형 종결 O / 금지어 없음 O

**썸네일:** "금이랑 같은 무게로 바꿨다" (강화형)
- 이문호공: 상식파괴(5) + 호기심(3)
- 답 유추 불가 O / 동사형 종결 O
- 배경 이미지: 강화 (소금 결정 클로즈업 + 금괴 병렬)

**핵심 내용:** 지금은 천 원짜리 조미료인 소금이, 한때 금과 동일한 가치를 지닌 전략 자원이었다. 왜 그랬고, 왜 지금은 싸졌는가.
**메커니즘:** "이걸로 월급?" → 상식파괴 클릭 → "금이랑 바꿨다" → 규모 충격으로 시청 시작
