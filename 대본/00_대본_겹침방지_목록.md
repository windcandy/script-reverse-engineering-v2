## 2단: 소재·개념 교차 색인

주제 선정 전, 다룰 소재가 이미 등장한 대본이 있는지 여기서 먼저 확인한다.

```
[이란]             → 003(주), 007(이란 전쟁 배경), 011(에너지 패권 구조)
[두바이·UAE]       → 007(탈출러시), 008(경제구조)
[에너지·원유]      → 003(석유 수출), 007(호르무즈), 008(석유 1%), 011(에너지 패권), 013(원전)
[제재·경제봉쇄]    → 003(이란), 010(북한), 011(미-이란)
[중국(에너지·산업)] → 011(에너지 패권), 012(조선업 경쟁)
[한국 산업]        → 012(조선업), 013(원전 수출-체코)
[역사적 상품·문명] → 002(소금), 005(설탕)
[유럽 역사]        → 004(잉글-스코), 005(설탕-노예제), 009(스위스)
[독재·생존 구조]   → 010(북한)
[부동산·금융]      → 001(전세)
[전통문화]         → 006(판소리)
[달러 패권]        → (미등장 — 신규 주제 가능)
[인도·남아시아]    → 002(간디 소금 행진만) — 독립적 대본 미작성
[중동(이란 제외)]  → (미등장 — 신규 주제 가능)
[남미]             → (미등장 — 신규 주제 가능)
```

## 체크 방법

**P0 주제 선정 시:**
1. 표에서 유사 주제·프레임이 있는지 확인
2. 색인에서 다루려는 소재·지역·개념이 기등장했는지 확인
3. 기등장 시 → 새 대본이 기존 대본과 **다른 역설**을 갖는지 검증 후 진행

**P4 집필 시:**
1. 사용하려는 사례(인물·사건·수치)가 다른 대본에서 이미 주요 소재로 쓰였는지 확인
2. 기등장 시 → 해당 대본 번호·맥락을 파악해 표현 방식 차별화
