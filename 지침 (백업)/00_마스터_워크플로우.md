# 마스터 워크플로우 (전체 제작 컨트롤러)

> **이 파일의 역할:** 대본 제작의 전체 흐름을 제어한다. 각 Phase에서 해당 Phase의 지침 파일만 로드하여 작업한다. 이전 Phase의 지침은 참조하지 않는다.

---

## 지침 망각 방지 원칙

1. **한 번에 한 Phase만 로드한다.** 전체 지침을 동시에 읽지 않는다.
2. **각 Phase 완료 시 GATE CHECK를 통과해야 다음 Phase로 진행한다.**
3. **Phase 파일 내부의 규칙만 따른다.** 다른 Phase 파일의 규칙을 기억해서 적용하지 않는다.

## 할루시네이션 방지 원칙

1. **지침의 예시 문구를 대본에 그대로 사용하지 않는다.** 원리를 이해한 뒤 새로운 표현을 생성한다.
2. **확인되지 않은 수치/사실을 단정하지 않는다.** 웹 검색으로 팩트를 확인한 뒤 사용한다.
3. **지침 내부 용어(역설엔진, A극/B극, SL, BR, ML 등)를 대본에 노출하지 않는다.**
4. **같은 비유/밈/감탄사를 한 편 내에서 반복 사용하지 않는다.**

---

## Phase 흐름

```text
[키워드 입력]
    ↓
Phase 1: 퀵서치 → 핵심 팩트 5개 + 제목/썸네일 후보 5개
    ↓ 유저 선택
Phase 2: 딥리서치 → 팩트 리포트
    ↓ GATE 2 통과
Phase 3: 구조설계 → 구조 보고 (전체 조감도)
    ↓ 유저 승인
Phase 4: 대본 집필
    ↓ GATE 4 통과
Phase 5: 퇴고 + 최종 검증
    ↓
Phase 6: 장면 소스 제작 (방식 A 애니메이션 / 방식 B 실사 혼합)
    ↓
[7단계] 영상 제작 (촬영/편집) — 유저 담당
    ↓
[8단계] 썸네일/제목 최종화
    ↓
[9단계] 업로드 + SEO
    ↓
[10단계] 배포/프로모션
```

## Phase별 로드 파일

| Phase | 로드할 파일 | 핵심 임무 |
|---|---|---|
| 1 | `P1_퀵서치_제목썸네일.md` | 퀵서치 → 핵심 팩트 5개 + 제목/썸네일 후보 5개 생성 |
| 2 | `P2_딥리서치.md` | 선택된 제목/썸네일 방향으로 팩트 리포트 작성 |
| 3 | `P3_구조설계.md` | 서사 유형 확정 + 전체 구조 설계 + 조감도 보고 |
| 4 | `P4_집필.md` | 도입부→본론→엔딩 순서로 집필 |
| 5 | `P5_퇴고.md` | 체크리스트 통과, 맥락 검증, 시뮬레이션 |

---

## 핵심 설계 원리: 제목이 약속하고, 팩트가 증명한다

```text
키워드 입력
→ 퀵서치 기반 제목/썸네일 확정 (시청자와의 약속)
→ 딥리서치로 팩트 확보 (약속 이행 재료)
→ 팩트 기반 구조설계 → 집필
```

제목/썸네일이 약속한 내용을 팩트로 증명할 수 없으면 제목을 바꾼다. 팩트에 맞지 않는 제목을 유지하지 않는다.

## GATE CHECK 상세

### GATE 1 (퀵서치 → 딥리서치 진입 조건)

- [ ] 유저가 제목/썸네일 후보 중 하나를 선택했는가?
- [ ] 선택된 제목이 동사형 종결인가?
- [ ] 선택된 썸네일 문구가 답을 유추할 수 없는가?
- [ ] 금지어가 없는가?

### GATE 2 (딥리서치 → 구조설계 진입 조건)

- [ ] 선택된 제목/썸네일이 약속하는 내용을 팩트로 뒷받침할 수 있는가?
- [ ] Aha-moment 후보가 실제 데이터에 기반하는가?
- [ ] 결론/지적 전리품을 뒷받침하는 근거가 확보되었는가?
- [ ] 모든 팩트에 출처가 기록되었는가?
- [ ] 서사 유형 가능성 메모가 완료되었는가?

### GATE 3 (구조설계 → 집필 진입 조건)

- [ ] 유저가 구조 보고를 승인했는가?
- [ ] SL 배열이 전제 건설 체인을 따르는가?
- [ ] 엔딩 유형이 선택되었는가?
- [ ] 서사 비유 프레임이 주제 메커니즘과 일치하는가?
- [ ] 영상 목표 길이가 13~17분인가?
- [ ] 각 SL의 보상이 유지 시간에 비례하는가?
- [ ] ML 해소가 "타인에게 설명하고 싶은" 수준의 통찰인가?

### GATE 4 (집필 → 퇴고 진입 조건)

- [ ] 대본 전문이 완성되었는가?
- [ ] 글자 수 약 4,500~6,500자 (13~17분)인가?
- [ ] 도입부 2분 내 ML이 열려 있는가?
- [ ] 박스 이미지(강) + 뒷받침(약)이 최소 3세트 있는가?
- [ ] SL 열림 유형이 교차 배치되었는가?
- [ ] 반전 질문이 5회+ 있는가?
- [ ] 2분+ SL 없는 구간이 없는가?
- [ ] "우리/한국/여러분" 배치가 후반부에 있는가?

## 7~10단계 (영상 제작 → 배포)

> Phase 5 완료 후 진입. 7단계는 유저 주도, 8~10단계는 Claude + 유저 협업.

| 단계 | 담당 | Claude 역할 |
|---|---|---|
| 7. 영상 제작 | **유저** (촬영/녹음/편집/자막/BGM) | 연출 가이드 기반 시각 자료 리스트, 자막 텍스트 정리 |
| 8. 썸네일/제목 최종화 | Claude + **유저** 최종 선택 | P1 후보 기반 제목 변형안 3~5개 |
| 9. 업로드 + SEO | Claude + **유저** 업로드 | 영상 설명문, 태그, 챕터(타임스탬프), 고정 댓글 문구 생성 |
| 10. 배포/프로모션 | **유저** 게시 | 커뮤니티 요약 텍스트, 쇼츠 하이라이트 구간, SNS 홍보 문구 생성 |
