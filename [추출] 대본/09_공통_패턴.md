## PART 1. 도입부 공통 패턴

### 1-1. 썸네일 약속의 간접 확인 원칙

**원칙:** 썸네일 문구를 대본에서 직접 말하지 않는다. 시청자가 "아, 이게 그거구나"를 스스로 깨닫게 만든다.

**3유형 공통 구현 방식:**

| 방식 | 설명 | 예시 출처 |
|---|---|---|
| 수치로 역설 발견하게 하기 | 결론은 말하지 않고 전제 수치만 보여줌 | 01 감자: "3억8천만 톤"으로 역설 전제 세팅 |
| 동의어 치환 | 썸네일 의미를 다른 표현으로 전달 | 11 에어컨: "이전과 이후로 나뉜다" → "향방을 바꿨다" |
| 공감 진입 후 "사실은…" | 시청자 경험에서 출발 → 이면 암시 | 02 입식vs좌식: "미국 영화 보면 이해 안 됨" |
| 역방향 진입 | 썸네일과 정반대 방향에서 시작 → 반전 암시 | 08 서울: "뚫릴 수밖에 없는 위치" → "요새 도시" |
| 일화로 체감 | 수치 없이 에피소드로 약속을 체감시킴 | 03 한반도: "지평선이 특산물" |

**공통 원리:**
```
썸네일 = 결론
도입부 = 결론이 성립하는 전제 조건을 체감시키는 구간
→ 시청자가 전제를 보고 "왜?"를 자발적으로 생성 → ML 진입
```

### 1-2. ML(메인 루프) 구조

> **용어 정의:**
> - **ML(메인 루프):** 도입부에서 열고 본문 후반부에서 닫히는 영상 전체의 축. 생애주기 10~20분.
> - **서브 OL(서브 오픈루프):** 본문 각 레이어/SL 전환에서 작은 갈증을 유발하는 ML의 도구. 생애주기 1~2분.

**25편 공통 구조:**

```
[질문 1~2개] + [채널 시그니처 선언 1개]
```

**3유형 공통 ML 기능:**

| 기능 | 설명 | 실제 예시 |
|---|---|---|
| **본문 로드맵 예고** | 질문 1·2가 본문 파트를 예고 | 01 감자: "언제부터?" → 역사 / "장단점?" → 분석 |
| **채널 시그니처로 전환 마무리** | "~알아봅니다/알아보았습니다"로 종결 | 7편 100% 동일 패턴 |

**허용 변주:**
- 선언형 ML: 질문 없이 "답을 먼저 제시"하는 방식 허용 (이미지 강화형에서 출현)
- 단일 질문 관통: 하나의 질문이 영상 전체를 관통 (과정 추적형에서 출현)

## PART 2. 본문 공통 패턴

### 2-1. 오픈루프 4단계 (ML 생애주기)

**정의:** ML이 도입부에서 열려 본문 후반부에서 닫히기까지의 생애주기.

| 단계 | 역할 |
|---|---|
| **열기 (Opening)** | 도입부에서 질문/예기치 못한 상황 → 인지적 공백 생성 |
| **유지 (Holding)** | 답변을 뒤로 미루고 기초 맥락 설명 → 긴장감 유지 |
| **강화 (Reinforcing)** | 결정적 단서 제공 or "왜 필요한가" 재상기 → 완주 동력 재충전 |
| **닫기 (Closing)** | 궁금증 명확히 해소 → 지적 카타르시스 |

**3유형 공통 "유지" 전략:**
- Q1을 먼저 분산 해소하면서 Q2는 보류 → 공백 유지
- 더 강한 보상을 나중에 배치 (ML 순서 역전 허용)

**3유형 공통 "강화" 2방식:**
- **방식 1 — ML 직접 재환기:** "그렇다면 대체 왜?" 식 표현으로 메인 질문을 명시적으로 재상기
- **방식 2 — 서브 루프로 ML 지지:** 서브 OL을 배치하여 ML의 긴장을 간접 유지
- **배치 지점:** 영상 전체의 40~50% 지점에서 공백 재생성이 3유형 공통으로 관찰

**유형별 강화·닫기 변주는 다름** → `10~12_유형별_*.md` 참조

> **모순형 특수 패턴 — '암시적 루프'(v2 신설):** 명시적 ML 질문 없이도 A극/B극의 대칭 구조만으로 시청자가 "B극에는 뭐가 나올까?"를 자동 예측하여 ML로 기능하는 패턴. 이미지 강화형·과정 추적형에서는 대칭 구조가 약해 작동 어려움 → `10_유형별_모순형.md` 참조

### 2-2. 접착제 툴킷 — 공통 체계

**3유형 공통 비율 (25편 평균):**
- **[A-1] 서브 오픈루프: 52%**
- **[A-2] 심화 브릿지: 27%**
- **[B] 논리적 브릿지: 21%**

> **비율 해설:** "서브 오픈루프 약 80%"가 아님. 심화 브릿지 27%가 Type A에 포함되어 합산 79%가 도출된다.

#### [A-1] 몰입 유지형 — 서브 오픈루프 (Sub-Open Loop)

시청자가 '다 알았다'고 자만하는 순간 정보를 차단하여 몰입을 강제합니다.

| 하위 유형 | 정의 | 예시 문구 |
|---|---|---|
| **종결 부정형** | "그거면 충분하지?" → "그건 또 아닙니다" | "역병만 조심하면 되냐? 그건 또 아닙니다" |
| **인과관계 추적형** | "이쯤 되면 의문이 하나 생기죠. 왜?" | "이쯤 되면 의문. 마약이 맞냐?" |
| **완벽함 뒤 함정형** | 완벽함을 확인시킨 뒤 "근데 이건 양반이었다" | "뇌절 아닌가 싶으실 수도 있는데 이건 양반이었다" |
| **사회 통념 반전형** | 시청자가 당연하게 여기는 것을 뒤집음 | "다들 아시죠? 석면 이거 발암 물질입니다" |
| **스케일업형** | 예상보다 훨씬 큰 규모로 전환 | "감자만 기르다가 전염병 크리로 멸망할 뻔한 곳이 있다" |

> **주의:** "서사적 초필살기"는 **과정 추적형 전용** (모순형·이미지 강화형 0건). 공통 툴킷에 해당하지 않음. → `12_유형별_과정추적형.md` 참조

---

#### [A-2] 몰입 유지형 — 심화 브릿지 (Intensification Bridge) ★ v2 신설

서브 오픈루프가 질문으로 몰입을 유지한다면, 심화 브릿지는 **선언과 예고로 몰입을 유지**합니다. 25편 평균 27%로 실전 비중이 큽니다.

| 전환 문구 패턴 | 용도 | 실전 예시 |
|---|---|---|
| **"진짜는 지금부터" 형** | 텐션 고조 예고 | `진짜 무서운 건 지금부터입니다.` (04 석면) |
| **"결정적 사건" 형** | 전환점 예고 | `결정적인 사건은 역시 18세기 말에 일어난 산업 혁명이었습니다.` (04 석면) |
| **"예상치 못한 또 다른" 형** | Scale Jump 진입 | `예상치 못한 또 다른 효과가 하나 더 있네. 냄새가 없어지더라.` (05 락스) |
| **"마침내" 형** | 스케일/변화의 도달 | `마침내 1890년대 클로르 알칼리 공정이 상용화됩니다.` (05 락스) |
| **"사실은 이미" 형** | 반전 폭로 | `간접적으로는 매일 옥수수를 먹고 있습니다. 액상과당의 풀네임이 고과당 옥수수 시럽입니다.` (07 옥수수) |
| **"양반이었다" 형** | 레이어 계단식 확장 | `이쯤 되면 뇌절 아닌가 싶으실 수도 있는데 이건 양반이었다는 거.` (04 석면) |
| **"시즌/블록 전환" 형** | 서사 단위 교체 | `시즌 2 주인공은 대가야.` (12 가야) / `한편... 아리아인들이` (10 종교와 고기) |
| **"시간 역행" 형** | 과정 추적 중 메커니즘 삽입 | `근본적인 원인을 따지자면 지구의 역사를 한 2억년 정도 거슬러 올라가야 합니다.` (13 호르무즈) |
| **"아 맞다" 형** | 추가 Scale 열기 (비공식 톤) | `아 맞다 중요한 걸 하나 까먹었는데` (11 에어컨) |

**심화 브릿지 주 사용 조건:**
- A극/How 파트가 **시간순 역사 서사**일 때 증가 (모순형 04 석면 44%, 25 소주 57%)
- Scale Jump 간 전환이 많을수록 증가 (이미지 강화형 08 서울 38%)
- 시즌/블록이 교체될 때 (과정 추적형 12 가야 40%)

### 2-4. 유형별 주력 접착제 가이드 ★ v2 신설

서사 유형별로 주력 서브 오픈루프 하위 유형과 심화 브릿지 패턴이 다릅니다.

| 유형 | 주력 서브 OL | 주력 심화 BR | 피벗/정점 유형 |
|---|---|---|---|
| **모순형** | **완벽함 뒤 함정** (30%) + 종결 부정형 (17%) | "진짜는 지금부터" / "결정적 사건" / "양반이었다" | 피벗 = **완벽함 뒤 함정 57% + 사회통념 반전 43%** |
| **이미지 강화형** | **인과관계추적** (29%) + 스케일업 (20%) | "예상치 못한 또 다른" / "마침내" / "아 맞다" | Scale Jump 진입 = **"예상치 못한 또 다른"** |
| **과정 추적형** | **사회적 통념 반전** (30%) + 인과관계추적 (20%) + **서사적 초필살기** (10%, **과정 추적형 전용**) | "시즌 전환" / "시간 역행" / "블록 전환" | 전환점 = **사회적 통념 반전** |

> **※ 통계 구분 주의:** 모순형 "완벽함 뒤 함정 30%"는 §2-3의 **서브 OL 전체** 중 비율이고, "피벗 57%"는 **피벗 장면 7건 한정** 집계(완벽함 뒤 함정 4건+사회통념 반전 3건). 서로 다른 모집단의 수치이므로 혼용 금지.

### 2-5. A극/How 성격에 따른 접착제 조합 변화

### 2-6. 레이어 전환 비율 규정 ★ v2 전면 개정

#### 기본 3분할 규정

1. **[A-1] 서브 오픈루프:** 45~60% (25편 평균 52%)

→ 합산: Type A 몰입 유지형 75~85% / Type B 체계 전달형 15~25%

#### 루프 중첩 — "순차 교체" 메커니즘 (v2 신설)

- **루프 중첩 원칙:** 서브 오픈루프의 중첩은 1개 유지를 원칙으로 하되, 서사 전개상 필요한 경우에만 예외적으로 최대 2개까지 허용.
- **실전 메커니즘 — "순차 교체":** 새 서브 OL을 열기 전에 기존 서브 OL을 해소하여 동시 중첩보다 교체를 우선. 25편 전편에서 3개 이상 동시 열림 0건.
- **교체 및 확장 조건:** 이미 2개의 서브 오픈 루프가 열려 있다면, 새 서브 OL을 생성하기 전에 반드시 기존 OL 중 최소 하나를 폐쇄(해소)해야 한다.
- **중첩 한계:** 어떠한 경우에도 3개 이상의 서브 오픈루프가 동시에 열려 있는 상태(Triple Overlap)는 절대 불가.

### 2-8. 해소 비유 공식 ★ v2 신설

오픈루프의 **닫기(Closing)** 단계에서 지적 카타르시스를 한 문장으로 각인하는 공식입니다. 25편 분석 결과 **"SL 열림에서의 비유 0건 / SL 해소에서의 비유 16건"**으로, 해소 비유는 **"열림"이 아니라 "해소"에서만 작동**합니다.

#### 공식

```
해소 비유 = (핵심 대상 + 비유/은유) + 강화문구(이문호공 2요인 중첩)
```

- **배치 방식:** 1문장에 2요인 **중첩** 고정 (25편 16건 전부 중첩, 복합배치 0건)
- **필수 요인:** **상식파괴(5)** — 13/16건(81%)에서 사용

#### 유형별 보조 요인 궁합

| 유형 | 보조 요인 최다 | 조합 | 대표 사례 |
|---|---|---|---|
| 모순형 | **경고(7)** | 상식파괴 + 경고 | `무적이지만 몸에 들어오면 저주` (04 석면) / `순도 100% 무쓸모 작물` (14 까트) |
| 이미지 강화형 | **비교(8)** | 상식파괴 + 비교 | `사실상 상륙 작전` (08 서울) / `자연 극복 vs 이용을 고민` (17 서안해양성) |
| 과정 추적형 | **공감(4)** | 상식파괴/경고 + 공감 | `작고 가난한 나라를 위해` (21 유엔군) / `사실상 기록 말살형` (12 가야) |

#### 작동 지점

해소 비유는 서사의 **전환·정점·완성 지점**에서 자연 발생합니다.

- **피벗/반전 지점 (모순형):** A극→B극 전환 시 역설을 한 줄로 압축
- **Scale Jump 정점 (이미지 강화형):** 대상의 파급력이 최고조에 달하는 순간
- **인과 완성 (과정 추적형):** "이래서 이렇게 됐구나"의 핵심 고리
- **누적 충격 정점:** 문제/가치가 쌓인 후 한 줄로 각인
- **메커니즘 완성:** How의 원리가 결과로 증명되는 순간

#### ML 비유(열림)와 해소 비유(해소) 비교

| | ML 비유 (열림) | 해소 비유 (해소) |
|---|---|---|
| 필수 요인 | **호기심(3)** 100% | **상식파괴(5)** 81% |
| 배치 방식 | 복합(각 질문에 분산) 위주 | **중첩(1문장 밀집) 100%** |
| 기능 | "궁금하게 만들기" → 루프 열기 | **"각인시키기" → 루프 닫기** |
| 25편 사용 | 비유+2요인 중첩 **0건** | **16건** |

> **집필 지침:** 해소 비유 공식 적용 우선순위는 **"해소(SL 해소)"가 "열림(SL 열림)"보다 높습니다.** 열림에서는 팩트/질문으로 공백을 생성하고, 해소에서 비유+이문호공 2요인 중첩으로 각인하는 것이 지식해적단의 실전 패턴입니다.

### 2-11. 시청자 반박 선제 흡수

**3유형 공통 패턴.** SL 1~2개마다 1회 등장.

**문구 패턴:**
```

**역할:** 시청자의 "과장 아니냐?" "논리 이상하다" 반응을 선점하여 신뢰 유지. 단순 신뢰 유지에 그치지 않고, **이완 장치 + SL 전환 브릿지를 자체적으로 겸용**하는 구조적 기능을 한다. 특히 모순형 A극에서 확신 쌓기 과정 중 발생하는 의심을 차단하는 용도로 집중 사용.

### 3-2. 엔딩 P3 5유형 — 공통 체계

| 유형 | 명칭 | 설명 |
|---|---|---|
| ① | 일상행위연결 | 시청 후 일상에서 재현할 행위 제안 |
| ② | 감정적여운 | 역사적·감정적 무게를 남기는 마무리 |
| ③ | 미래희망 | "앞으로 ~이 되길 바라며" |
| ④ | 시의성마무리 | 현재 진행 중인 이슈와 연결 |
| ⑤ | 위트반전 | 가벼운 반전·웃음으로 착지 |

**유형별 자연 궁합:**
- 모순형 → 미래희망 + 위트 혼합
- 이미지 강화형 → 미래희망 (60%)
- 과정 추적형 → 감정적여운 + 시의성

**엔딩 유형 혼합 허용:** 3유형 모두에서 단일 유형 택1보다 **1주+1보조 혼합**이 실전에서 다수 관찰.
- 예: 01 감자(미래희망+위트), 08 서울(시의성+미래희망), 06 루프탑(감정여운+통찰)

> **유형별 추가 착지 유형:** 5유형 외에 유형별 고유 추가 유형이 존재한다 (모순형: 경고형·통찰형·광고연착형 / 과정 추적형: 질문열기형·시리즈연결형·경고형). → `10~12_유형별_*.md` 참조

### 3-3. ML 해소 완료 원칙

- ML 질문에 대한 답은 본문에서 완료
- 엔딩은 해소 이후의 착지 구간 — 새로운 핵심 정보 원칙적 금지
- 실전에서는 부가 정보·후일담이 추가되는 경우 있음 (유형별 충족률 — 모순형 57%, 이미지 강화형 30%, 과정 추적형 38%)

**실전 허용 단서:** '신규 팩트'와 '현재 상태 확인/후일담'을 구분한다. ML 해소와 무관한 부가 정보(보너스 정보)는 전체의 20% 이내로 제한하며, 본문 주제의 현재 상태 확인이나 후일담은 허용 범주에 포함한다.

```

## PART 4. 오픈루프 설계 원칙 (§2 공통)

### 4-1. 8가지 원칙

**① 보상의 비례 원칙**
루프 유지 시간만큼 보상 가치(정보 밀도)가 커야 함. 기다림에 비해 결론이 평범하면 이탈.

- *v2 단서:* 모순형에서 A극이 역사적 서사나 공감 소재로 "읽는 재미"를 제공하는 경우, A극/B극 비율이 2:1~3:1까지 불균형해도 허용됩니다.

**② 지적 보상의 품질**
루프를 닫을 때 시청자가 충분히 납득할 수준의 통찰 제공 → "낚시"가 아닌 "학습"으로 인지.

**③ 어조의 전략적 변주**

| 구간 | 어조 |
|---|---|
| 유지 단계 | 부드러움: "~라고 하실 수도 있는데요", "~않을까요?" |
| 전환·강화 단계 | 유형별 변주 (모순형=단호함 / 이미지강화형=경외·감탄 / 과정추적형=감정이입) |
| 엔딩 단계 | 위트 / 감정 여운 / 미래희망 중 선택 |

**④ 빌드업 농도 조절**
피벗/Scale Jump/전환점 직전 빌드업은 **1~3문장**으로 극히 짧게 처리. 호흡을 끌면 보상 시점에 시청자가 지침.

**⑤ 새로운 루프의 생성**
본문 전체에 걸쳐 서브 OL을 연쇄 배치. 유형별 연쇄 횟수:
- 모순형: 서브루프 3~4회
- 이미지 강화형: Scale Jump 2~5회
- 과정 추적형: 전환점 3~6회

**⑥ 지적 유능감 자극**
"나만 이걸 알게 됐다" / "이제 이해가 간다"는 느낌. 시청자가 수동적 수신자가 아닌 능동적 발견자로 포지셔닝.

**⑦ 착지 방식의 선택 (v2 재정의 — 필수→권장 하향)**
ML 해소 후 아래 4가지 중 **주제와 맥락에 맞게 택1** (필수가 아닌 권장):

- **공감·공유 명분:** "이 내용을 타인에게 공유하고 싶게 만드는 한 줄 요약"이나 "사회적 가치"
- **감정 여운:** 역사적·정서적 무게를 남기는 마무리 ("억울했던 마음도 같이 생각해 보면")
- **위트 반전:** 가벼운 반전과 웃음 ("삼겹살은 불판에 구워 드시길")
- **통찰 선언:** 본문을 관통하는 관점 제시 ("자연을 극복할지 vs 이용할지를 고민했다")

*v2 유형별 자연 궁합:* 모순형=위트/감정여운, 이미지 강화형=미래희망/통찰, 과정 추적형=감정여운/시의성

> **필수→권장 하향 배경:** 25편 분석 결과 "공감·공유 명분" 충족률이 3유형 중 가장 낮음 (모순형 1/7, 이미지강화형 4~5/10, 과정추적형 3~4/8). 실전에서는 위트·감정·경고·통찰로 대체되는 경우가 다수.

**⑧ 보상의 분할 제공 (v2)**
가장 큰 보상을 한 번에 던지기보다, 서사 유형에 맞춰 **구조적으로 분할**합니다.

- **모순형 (평균 3.3단계):** B극에서 문제/원인/메커니즘을 순차 제공
  - 예: `현상 → 원인 → 메커니즘 → 현재 영향`
  - 또는: `문제 1 → 문제 2 → 문제 3 → 문제 4` (동질 문제 누적)
- **이미지 강화형 (평균 3.9단계):** Scale Jump를 단계적으로 확대
  - 예: `What → How → Scale 1 → Scale 2 → Scale 3 → Why`
- **과정 추적형 (평균 5.9단계 — 3유형 중 최다):** 시간순 전환점을 세밀하게 분할
  - 예: `발단(배경+촉발) → 전개 1 → 전개 2 → ... → 절정 → 결말(현재 의미)`

**3유형 공통 — SL 내부 3단 상승 구조:**
```
기초 SL (수치/규모 제시) → 심화 SL (메커니즘/역사 해설) → 정점 SL (충격 사례 투하)
```
각 단계마다 보상 가치가 상승하여, 이탈 위험 지점에서 "아직 핵심이 안 나왔다"는 유지 동력을 제공한다. ⑧ 분할 제공 원칙의 파트 내부 구현 패턴.

#### 25편 실전 충족률

| 원칙 | 충족률 | 비고 |
|---|---|---|
| ① 보상의 비례 | 높음 | — |
| ② 지적 보상의 품질 | **25/25 (100%)** | 3유형 완전 충족 |
| ③ 어조의 전략적 변주 | 높음 | — |
| ④ 빌드업 농도 조절 | **25/25 (100%)** | 3유형 완전 충족 |
| ⑤ 새로운 루프의 생성 | **25/25 (100%)** | 3유형 완전 충족 |
| ⑥ 지적 유능감 자극 | **25/25 (100%)** | 3유형 완전 충족 |
| ⑦ 착지 방식의 선택 | **약 36%** | 모순형 14%(1/7), 이미지강화형 40~50%(4~5/10), 과정추적형 38~50%(3~4/8) |
