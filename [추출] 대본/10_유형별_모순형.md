### 2-5. 썸네일 약속의 간접 확인 기법 — 6가지 (모순형)

**기법 1: 수치로 역설을 스스로 발견하게 만들기**
- **01 감자** — 썸네일: "감자를 주식으로 먹지 못하는 결정적인 이유"
- 도입부: "3억8천만 톤, 주식 작물 4위. 오지게 많이 먹습니다"
- "주식으로 못 먹는다"를 말하지 않음. "이렇게 많이 먹는다"를 보여줘서 시청자가 **"이렇게 많이 먹는데 왜 주식이 안 되지?"**를 스스로 떠올리게 만듦

**기법 2: 일화로 우회 진입하여 체감시키기**
- **03 한반도** — 썸네일: "한반도는 얼마나 저주받은 땅일까?"
- 도입부: "지역 축제 중에 특이한 게 있습니다. 지평선 축제. 지역 특산물이 무려 지평선이다"
- "저주받은 땅"이라고 말하지 않음. "지평선이 특산물"이라는 일화로 시청자가 **"진짜 불리한 땅이긴 하네"**를 체감하게 만듦

**기법 3: 속성을 쌓아서 결론을 시청자에게 맡기기**
- **04 석면** — 썸네일: "인류에게는 아직 이른 신의 물질(진짜임)"
- 도입부: "불에 타지도 않고. 부식되지도 않고. 끊어지지도 않고. 열과 전기마저 차단. 엄청나게 저렴."
- "신의 물질"이라고 말하지 않음. 속성 5개를 연속 투하해서 시청자가 **"이거 진짜 사기 아이템이네"**를 스스로 결론 내리게 만듦

**기법 4: 익숙한 것과 대비시켜 이질감 유발**
- **14 까트** — 썸네일: "지금도 아프리카를 좀먹고 있는 악마의 풀"
- 도입부: "커피는 전 세계인에게 사랑받는 기호식품... 사실 에티오피아에는 커피보다 오래된 더 강력한 각성제가 있습니다"
- "악마의 풀"이라고 말하지 않음. 모두가 아는 커피를 먼저 꺼내서 **"커피보다 강한 게 있는데 왜 안 퍼졌지?"**라는 이질감을 유발

**기법 5: 시청자의 상식을 먼저 확인시킨 뒤 뒤집기**
- **19 대통령제** — "굴리기 어렵다"를 말하지 않음. "우리한테는 당연한데 세계적으로는 아니다"를 보여줘서 **"왜 우리는 대통령제를 하고 있지?"**를 유발
- **25 소주** — "희석식/증류식"을 설명하지 않음. "지금 소주가 원래 모습이 아니다"만 던져서 **"그럼 원래는 뭐였는데?"**를 유발

**기법 6: 공감 유도 후 반전 암시**
- **23 아보카도** — "멸종됐어야 했다"를 말하지 않음. "착한 과일"로 공감시킨 뒤 "사람이 죽는다"로 반전

**모순형 ML 비유 사용 조건:**

- **04 석면 "금단의 물질":** A극(신의 물질) + B극(1군 발암물질). "금단의"가 담는 것: **"대단하지만 손대면 안 된다"는 모순을 한 단어로 압축**
- **23 아보카도 "뭔놈의 과일이 사람을 죽이고":** A극(착한 과일) + B극(사람을 죽이고 환경 파괴). "뭔놈의 과일이 죽인다"가 담는 것: **"과일(무해해야 할 것)이 사람을 죽인다(있을 수 없는 일)"라는 모순을 의인화로 충돌시킴**

**사용 조건:**
1. 대상이 단일 사물/개체일 때 (3유형 공통)
2. **A극과 B극의 낙차가 클 때 (모순형 고유)** — 낙차가 작은 모순형(01 감자, 19 대통령제)에서는 비유 미사용

**사용률:** 2/7편(29%) — 비유가 담는 것 = **역설 자체** ("~인 동시에 ~이다")

### 2-7. ML의 실제 기능

**기능 A: 본문 로드맵 제시 (7/7편)** — 질문 2개가 본문 2파트를 예고하는 구조

| # | 질문 1 → 파트 1 | 질문 2 → 파트 2 |
|---|---|---|
| 01 | "언제부터?" → 역사 | "장단점?" → 비교분석 |
| 03 | "어떤 시련?" → A극(불리한 땅) | "어떻게 생존?" → B극(적응) |
| 04 | "왜 암 유발?" → 메커니즘 | "왜 못 알았나?" → 역사 |
| 14 | (단일 질문) | "보급 불가 이유 + 금지 이유" → 2파트 |
| 23 | "언제부터?" → 기원 | "왜 죽이나?" → 해악 |
| 25 | "원래 모습?" → 과거 | "어쩌다?" → 변천사 |

**기능 B: 채널 시그니처로 전환 마무리 (7/7편)** — "~알아봅니다" / "~알아보았습니다"로 종결. 이문호공 요인이 아니라 채널 브랜딩 + 본문 진입 신호.

**기능 C: 도입부 역설의 방향 설정 (모순형 고유)** — A극과 B극의 방향을 질문으로 미리 설정:
- 04: A극(신의 물질) → ML("왜 암?") → B극(살인자)
- 25: A극(갑산 술) → ML("원래 모습?") → B극(고급 증류주) ← 역방향 모순
- 23: A극(착한 과일) → ML("왜 죽이나?") → B극(환경 파괴)

#### ML 지침 대비 종합

| 항목 | 이거봐봐 지침 | 지식해적단 실제 |
|---|---|---|
| 문장 수 | 2문장 이내 | 질문 2 + 선언 1 = 3문장 (선언은 채널 시그니처) |
| 이문호공 | 3요인 이상 필수 | 평균 2요인, 3요인 충족은 29% |
| 호기심(3)/상식파괴(5) 필수 | 호기심 필수 | 호기심(3)은 7/7편 100% — 상식파괴는 3/7편 |
| 중첩/복합 | 중첩 + 복합 혼합 | 복합만 사용 — 요인이 각 질문에 분산 |

| ML 형태 | 질문형 고정 | 질문형 맞지만 로드맵 예고 기능이 핵심 |

## PART 3. 본문 — 모순형 고유 패턴

### 3-1. 지침 구조 템플릿

```
인트로 → [신뢰 상환 + ML] → A극(기초SL→심화SL→정점SL) → 피벗BR → B극(What→How→Why→현재의미) → 엔딩
```

**지침과의 주요 차이점:**

1. **신뢰 상환 명시성:** 실제 대본에서는 신뢰 상환이 A극 첫 번째 SL 안에 흡수된 형태가 대부분.
2. **B극 Why 배치 시점:** 실제 대본에서는 B극 중반부(**전체 길이의 63~80%**)에 Aha-moment와 결합하여 등장.
3. **A극의 기초SL 생략:** 실제 대본에서는 기초SL과 심화SL의 경계가 불분명한 경우가 많고 A극을 **2단(기초+정점)으로 처리**하는 편이 더 많다.
4. **A극/B극 비율 불균형:** A극이 역사적 맥락이나 시간순 서사를 포함하는 경우(03, 19, 23) A극/B극 비율이 2:1~3:1로 불균형.

---

### 3-2. 피벗 브릿지 패턴 — 7가지 변주

**공통 구조:**
```
[A극 확신 최고점] → [직접 전환 (선언/질문/유보)] → [반전 신호] → [B극 첫 What 투입]
```

| 유형 | 대본 | 피벗 문장 | 특징 |
|---|---|---|---|
| 직접 선언형 | 01 감자 | "결론부터 말하면 이랬다가 진짜 큰일 날 수도 있습니다" | 반전 질문 없이 선언만으로 전환 |
| 직접 선언형 | 23 아보카도 | "여기서 문제는 이 아보카도 때문에 환경이 아장나고 있다는 겁니다" | 위와 동일 |
| 직접 선언형 | 25 소주 | "여러 국란을 거치며 점차 쇠퇴할 수밖에 없었습니다" | 위와 동일 |
| 반어적 질문형 | 04 석면 | "그런데 다들 아시죠? 석면 이거 발암 물질입니다" | 시청자가 이미 아는 사실을 반어적으로 확인 |
| 단호 선언형 | 14 까트 | "이렇게 보니까 누가 봐도 마약 맞죠" | A극 "기호식품" 인식을 단호하게 부정 |
| 대비 선언형 | 19 대통령제 | "하지만 우리는 다 알고 있죠. 결코 장점만 있는 제도는 아닙니다" | 시청자 공감을 활용한 전환 |
| 유보형 | 03 한반도 | "어 그래서 장점은 없냐? 있긴 있죠. 잠시만요" | 장점 소개를 극적으로 지연하여 기대감 생성 |

**피벗 접착제 유형:** 피벗 = **100% 서브 오픈루프**. 하위 = "완벽함 뒤 함정" 57%(4편) + "사회적 통념 반전" 43%(3편)

**피벗 구체 예시:**

- "뭐 더 이상의 자세한 설명은 생략해도 될 정도죠. 근데 사실 석면의 전성기는 2차대전 이후부터라는 거. (...) 그런데 다들 아시죠? 석면 이거 발암 물질입니다." (04 석면 — A극 극적 확장 직후 반어적 질문으로 피벗)
- "어 그래서 장점은 없냐? 아 있긴 있죠. 어, 있긴 있어요. 잠시만요. 아, 아, 그래." (03 한반도 — 유보형 피벗, 장점 소개를 극적으로 지연)

**Type A(서브 오픈 루프) vs Type B(논리적 브릿지) 전체 비율:**
7편 전체를 통틀어 Type A(서브 오픈 루프)가 약 80~85%, Type B(논리적 브릿지)가 약 15~20%를 차지하며, 이는 지침(80~100% Type A 사용)과 부합한다.

**지침 브릿지 개념과 실제 사용 방식 간 차이:**
지침(P3 §9)은 BR-SL 동시 작동 원칙("절반 가까이를 동시 작동으로 설계")을 명시하는데, 실제 대본에서는 **피벗 BR에서만** BR-SL 동시 작동이 뚜렷하게 나타나고, 일반 섹션 전환 브릿지는 브릿지 단독으로 작동하는 비율이 더 높다. → P4 모순형 집필 섹션에 "피벗 브릿지의 직접 선언+반전 신호 복합 구조"를 예시 패턴으로 명시하면 피벗이 너무 약하거나 갑작스럽게 처리되는 문제를 방지할 수 있다.

#### 암시적 루프 (v2 신설) — 모순형 고유

**정의:** 명시적 ML 질문 없이도 A극/B극의 대칭 구조만으로 시청자가 반대쪽을 자동 예측하여 ML로 기능하는 패턴.

**대표 사례 — 19 대통령제:**
- ML: "어떤 장점? 왜 채택?"
- A극 해소: Q1(장점 5가지) + Q2(신생국에 유리, 냉전 영향)
- B극 진입: "결코 장점만 있는 제도는 아닙니다" → **단점** 해소 시작
- 시청자 뇌: "장점을 이만큼 보여줬으면 당연히 단점이 나올 차례"라는 **암묵적 기대**가 형성되어 있음

**원리:** 모순형은 장점/단점, 성공/실패 같은 **대칭 구조**이므로, 한쪽만 보여줘도 시청자는 반대쪽을 자동으로 예측. ML이 명시하지 않아도 B극 진입이 자연스러움.

**사용 조건:**
- 서사가 대칭 구조를 전제할 때 (모순형 피벗 구조)
- Q1·Q2가 A극에서 모두 해소되어 명시적 루프가 닫힌 상태

**주의:** 이미지 강화형·과정 추적형에서는 대칭 구조가 약하므로 암시적 루프 작동 어려움. **모순형 고유 패턴.**

#### 해소 패턴 3가지

**패턴 1: Q1=A극, Q2=B극 분할 해소 (3편: 03, 23, 25)**
```
ML Q1 → A극에서 해소 → 피벗 → ML Q2 → B극에서 해소
```
가장 정석적인 모순형 구조. 25 소주가 가장 깔끔한 사례: "원래 모습?"=A극 전체, "어쩌다?"=B극 전체로 1:1 매핑.

**패턴 2: Q1+Q2 모두 A극에서 해소, B극은 암시적 루프 (2편: 01, 19)**
```
ML Q1 → A극에서 해소 → ML Q2 → A극 중후반에서 해소 → 피벗 → B극(질문하지 않은 영역)
```
이 패턴의 핵심: B극이 "질문에 대한 답"이 아니라 **"예상치 못한 반전"**으로 작동하여 충격이 더 큼.

**패턴 3: 두 질문 모두 B극에서 해소, ML 순서 역전 (2편: 04, 14)**
```
ML Q1 → (해소 보류) → ML Q2 → B극 초반에서 Q2 해소 → B극 중반에서 Q1 해소
```
역전이 발생하는 조건: 두 질문이 같은 극(B극)에서 해소될 때. A극/B극에 분산되면 항상 ML 순서대로.
역전의 효과: 더 강한 보상(근본 메커니즘, 핵심 원인)을 나중에 배치하여 **클라이맥스 강화**.

#### ML 질문 유형별 해소 위치

| 질문 유형 | 해소 위치 | 빈도 |
|---|---|---|
| "언제부터?"/"원래 모습?" (역사·기원) | A극 시간순 분산 | 4회 |
| "어떤 장단점?"/"어떤 시련?" (속성·현상) | A극 전체 또는 A극+B극 | 3회 |
| "왜?"/"어떻게?" (메커니즘·원인) | B극 중반~후반 | 4회 |
| "어쩌다?"/"뭔놈의?" (반전·문제) | B극 전체 | 3회 |

- **SL 해소("닫는 박스")에서 이문호공 2요인 강화문구 = 최소 6건.** 아래 상세 참조. 비유+이문호공이 결합된 한 줄 각인으로 기능.

**시사점:** P4 박스이미지 공식의 실전 적용 위치는 "열림/해소 양쪽"이 아니라 **"해소(닫는 박스) 중심"**. 열림에서는 팩트/질문으로 공백을 생성하고, 해소에서 비유+이문호공으로 각인하는 것이 지식해적단의 실제 패턴.

#### 모순형 보조 요인 패턴

- 상식파괴(5) + **경고(7)**: 3건 — B극의 위험성/문제점을 각인하는 역할
- 상식파괴(5) + 공감(4): 1건
- 상식파괴(5) + 호기심(3): 1건
- 경고(7) + 공감(4): 1건

→ **모순형에서 보조 요인 최다 = 경고(7)** — B극의 부정적 결과를 경고로 각인

**ML 이문호공과의 차이:**

| | ML (도입부) | 해소 강화문구 (본문) |
|---|---|---|
| 필수 요인 | 호기심(3) 100% | **상식파괴(5) 83%** |
| 배치 방식 | 복합(각 질문에 분산) 위주 | **중첩(1문장 밀집) 100%** |
| 기능 | "궁금하게 만들기" → 루프 열기 | **"각인시키기" → 루프 닫기** |

→ ML(열림)=호기심으로 궁금하게, 해소(닫기)=상식파괴로 각인. 도입부와 본문의 이문호공 활용 목적이 구조적으로 분리된다.

### 3-11a. 이완 장치 2분 규칙 — 모순형 충족률

**충족률:** 7/7편 충족 — 2분 이상 이완 없는 구간 거의 없음.

**원칙별 상세:**

**① 보상의 비례:** A극이 길수록(감자 8분, 석면 7분) B극 보상도 구체적 메커니즘 해설과 현재 임팩트를 충분히 제공한다. 03 한반도(A:B=3:1), 19 대통령제(2:1), 23 아보카도(2.8:1)는 A극 자체가 역사적 서사·공감 소재로 "읽는 재미"가 있어 비례 원칙이 약간 느슨해지는 것이 허용된다.

**② 지적 보상의 품질:** B극 핵심 해소는 단순한 "나쁜 점" 나열이 아니라 메커니즘 해설을 통해 "그래서 그렇구나" 반응을 유발한다. 석면 편은 면역 세포 싸움 과정을 단계별로 설명하여 이해를 유발: "아무리 공격을 해도 딜이 거의 안 들어가요. 그렇게 면역 세포들이 와서 싸우다 죽고 다시 핑 찍히고 또 와서 죽고 하는 과정이 무한히 반복됩니다." (04 석면)

**④ 빌드업 농도 조절:** A극이 일방적 장점 나열로 텐션이 하락할 시점에 반전 질문을 삽입하여 긴장을 재점화한다. 까트 편은 장점 나열이 길어질 위험을 "이렇게 보니까 누가 봐도 마약 맞죠"라는 단호한 전환으로 차단.

**⑤ 새로운 루프의 생성:** 피벗 직전·직후에 새로운 루프를 열어 B극으로 관성을 이어간다. B극 진입 후 Why 질문("왜 이게 가능한 건데?")을 통해 시청자 궁금증을 재점화.

**⑥ 지적 유능감 자극:** 한반도 지형 편 엔딩은 "결국 이런 지형이 우리 역사와 문화를 깎아냈다"는 통찰로 마무리 — 시청자가 역사를 다르게 볼 수 있다는 인식 전환을 제공.

**⑦ 공감·공유 명분 (가장 빈번하게 구현 부족):** 7편 중 5편은 엔딩에서 "공유하고 싶은 한 줄 요약"이나 "사회적 가치"를 명시적으로 제공하지 않는다. → P4 엔딩 집필 체크리스트에 "타인에게 공유하고 싶은 한 줄이 있는가?" 항목을 추가하면 보완 가능.

**⑧ 보상의 분할 제공:** 가장 큰 보상을 한 번에 투하하지 않고 분할. 감자 편: 전염병 취약성(1차)→아일랜드 사례(2차, 수치 강화)→보관 한계(3차)→환경별 재배 불가(4차) 순서로 분할 제공.

**⑦ 충족률 14% 상세:**

- 실전 대체: 위트반전(01, 04), 경고(14), 통찰(19), 화자감정(23), 광고연착(25)

### 3-13. 지침에 없는 추가 기술 — 모순형 발견

#### 시청자 반박 선제 흡수 (Preemptive Objection Handling)

A극과 B극 모두에서 "~라고 하실 수도 있는데요"라는 구문이 일관되게 등장한다. 지침(§B)에서는 언급되지만 빈도 규정이 없으나, 실제 대본에서는 **SL 1~2개마다 1회씩** 등장하여 A극의 확신 쌓기 과정에서 발생하는 의심을 선제로 차단한다.

> "엥? 척박하고 추운 데서도 잘 자란다며 라고 하실 수도 있는데요. 감자는 그런만큼 사람의 노동력이 상당히 많이 들어가는 작물입니다." (01 감자)

**집필 시사점:** SL당 0~1회가 적정. 무분별한 남용을 방지하기 위해 P4 지침에 최적 빈도를 명시하면 유용. C(부정 선행형) SL 열림과 결합하여 자연스러운 섹션 전환 장치로도 기능.

#### 자기언급형 서사 (Self-Referential Narrative)

채널 자체를 등장시켜 친밀감을 강화하는 기법.

> "지식해적단 팀도 어찌됐든 밀레니얼 런저리 세대라서 또 아보카도를 워낙 좋아하는데요" (23 아보카도)

화자가 시청자와 같은 입장임을 보여줘 정보 채널과 감성 채널의 경계를 허문다.

**집필 시사점:** B극 이후 엔딩 진입 전에 1회 허용하되, A극에서는 자제하는 방향이 실전적. P4 13개 언어 장치에 없는 기법이므로 "채널-시청자 동일시" 기법으로 별도 규정 가능.

### 3-14. 지적 공백(Curiosity Gap) 활용 — 모순형

#### ML의 도입부 생성과 본문 유지 방식

7편 모두 인트로 말미에 ML을 질문 형태로 열고, 본문에서 두 가지 방식으로 유지한다.

**방식 1 — ML 직접 재환기:** 새 챕터 진입 전에 "그렇다면 대체 왜?"식 표현으로 메인 질문을 재상기시킨다.

> "그렇다면 대체 석면은 왜 암을 유발하는 걸까요? 인류는 어떻게 수천 년 동안 이런 위험성을 진작에 눈치채지 못했을까요?" (04 석면 — 도입부 ML이 본문 전체를 두 갈래로 구조화)

**방식 2 — 서브 루프로 ML 지지:** ML을 직접 언급하지 않지만 서브 루프가 ML 방향으로 수렴하여 긴장을 유지한다. 까트 편은 "왜 금지 안 하냐?"라는 서브 루프를 반복적으로 열고 닫으며 "왜 전 세계로 퍼지지 않았냐"는 ML을 지지한다.

#### 중간 지점에서 공백 재생성

A극 중반부(전체 길이의 40~50% 구간)에서 공백을 재생성하는 패턴이 공통적이다. A극이 단순 장점 나열이 아닌 "그런데 이것도 문제가 있지 않냐?"식 의문을 던지고 답하는 내부 SL 구조를 갖는다.

> "심지어 아보카도가 원주민들 사이에서는 일종의 정력제로 여겨졌다는 점도 영향이 있었는데요." (23 아보카도 — A극 내에서 예상치 못한 새 정보로 공백 재생성)

#### A극 정점→피벗에서 공백 작동 방식

A극 정점에서 확신이 극대화된 직후 피벗이 투입되면, 시청자는 순간적으로 "그런데 왜 이게 문제야?"라는 새로운 공백 상태에 진입한다. 이 순간 B극 첫 번째 What(현상 제시)이 투입되어 공백을 즉시 부분 충족하면서 How/Why는 여전히 열린 상태로 유지된다.

**집필 시사점:** A극 정점-피벗 사이의 공백 생성 메커니즘을 P3 구조설계 지침에서 "피벗 공백 설계"로 명시하고, B극 첫 SL이 공백을 부분 충족(What)한 뒤 나머지 공백(How/Why)을 유지하는 방식을 구체적으로 기술하면 집필 시 피벗 처리가 더 정교해진다.

### 3-15. 정보를 조각으로 흩뿌리기 (지적 유추 유발) — 모순형

#### 정답형 서술 회피 패턴

7편 전체에서 B극의 결론(근본 원인, Why)을 최대한 늦게 공개하고, 그 이전에 단서를 흩뿌리는 방식이 일관되게 나타난다. 시청자가 스스로 "아, 이게 그래서인가?"를 유추하게 만드는 구조다.

**구체 기법 3가지:**

**1. 질문 연쇄 배치** — 한 질문에 대한 답이 나오기 전에 다음 질문을 던져 시청자의 추론 공간을 확장한다.

> "왜일까요? 가트가 절대 전 세계로 보급될 수 없는 이유에 대해 또 절대 그래서는 안 되는 이유에 대해 알아봅니다." (14 까트 — 두 질문을 동시에 열어 시청자가 둘을 연결하게 유도)

**2. 부분 공개 + 의도적 유보** — 메커니즘의 일부만 공개하고 핵심을 뒤로 미룬다. 감자 편은 전염병 취약성을 제시하기 전에 "아일랜드가 유럽에서 가장 먼저 감자를 수용한 나라"라는 단서를 먼저 배치하여 시청자가 결말을 예감하게 만든다.

**3. 대조 쌍을 통한 유추 자극** — A극과 B극의 대조가 명확할수록 시청자는 스스로 "그러니까 A는 양날의 검이었구나"를 조립하게 된다.

> "쌀이 수확하기까지 최소 3개월에서 길면 5개월. 감자는 종에 따라 고작 두 달만 넘기면 수확할 수가 있고" (01 감자) → B극에서 "보관 기한은 길어야 1년"과 대비되어 장점이 동시에 단점임을 유추하게 만든다.

#### A극-B극 연결을 시청자가 스스로 조립하게 만드는 방식

A극에서 쌓인 확신이 B극에서 역전될 때, 그 역전의 원리를 B극이 설명하기 전에 시청자가 A극 정보를 재조합하여 추론하도록 유도한다.

> "그나마 그 특성상 다른 나라나 지역으로 퍼질 일은 잘 없을 것 같다는게 다행이겠죠. 그러나 인간의 욕심과 의지력은 끝이 없고" (14 까트 — B극 결론 전에 시청자가 "그러면 결국 퍼질 수도 있겠네"를 유추하게 만드는 구조)

**집필 시사점:** 부분 공개+유보 기법은 P4 지침의 "정보 단순 나열 금지" 원칙과 직접 연결된다. "단서 배치 → 시청자 유추 → 확인" 3단 패턴으로 명시하면 집필 시 구체적 지침으로 활용 가능하다.

- 착지1(미래희망): "먼훗날 화성식민지에서도 감자부터 재배해 보고 시작하지 않을까요?"
- 착지2(위트): "저희는 햄버거의 감자 튀김 먹으러 이만"
→ 착지가 **2단계**(미래희망→위트). **1주+1보조 혼합**.

**03 한반도 — 통찰+정리→일상행위→아웃트로:**
- 정리+통찰: "울퉁불퉁한 땅이라도 최적화 전략을 깎아내면... 이런 걸 **애증**이라고 하는 건가"
- 착지(일상행위): "아무 때나 등산 한 번씩 조을 수 있는 나라... 가족이나 친구들과 함께"
→ 정리 단계에 **통찰("애증")이 결합**. 신규 팩트는 아니지만 **본문 전체의 관점 재구성**.

**04 석면 — [신규 수치]→정리→위트→아웃트로:**
- (본문 연장): "WH오피셜 1억 2500만 명... 대한민국은 걱정 안 해도"
- 정리: "삼겹살 구워 먹던게 별미로 취급됐다는 걸 생각하면 수습된게 기적"
- 착지(위트): "삼겹살은 **불판에** 구워 드시길"
→ 정리 직전에 **현재 규모 수치** 삽입. **본문 소재("삼겹살 일화") 회수**로 수미상관 기능. ※ 정리 부분에 신규 수치(1억2500만) 포함 — "정리=새 정보 없이 요약" 원칙과 충돌하나 실전에서는 허용.

**14 까트 — [한국 사례]→정리→경고→아웃트로:**
- (본문 연장): "인천공항 세관 밀수 까트 3.6톤", "한국 세관이 꼼꼼해서 경유 목적"
- 정리: "다문화 사회... 미리미리 알아두고 더 철저히 단속해야"
- 착지(경고): "문화 상대주의도 마약에까지 적용할 수는 없는 일"
→ P4 "교훈/경고로 끝내지 말 것" 원칙 위반. 다만 사회적 문제 주제에서 경고 착지가 자연스러운 경우 존재.

**19 대통령제 — 통찰→아웃트로 (정리 생략):**
- 통찰+착지: "뭐든간에 결국은 국민들이 이성적이고 합리적인 판단을 해야만 잘 돌아가는게 아닐까"
→ **정리 단계 생략**, 통찰이 정리+착지를 겸함.

**23 아보카도 — 정리→화자감정→미래희망→아웃트로:**
- 정리: "먹지 말자는 얘기는 아닙니다. 극복되고 있고 1차적 원인 아니니까"
- 화자 감정: "저희 팀도 밀레니얼이라 좋아하는데 가격 부담"
- 착지(미래희망): "더 싸게 부담없이 먹을 수 있게 됐으면"
→ **자기언급형 서사**의 엔딩 적용. "채널-시청자 동일시"로 친밀감 강화.

**25 소주 — B극 결론→광고 전환→아웃트로:**
- B극 결론: "전통주 판매 늘어나고 증류식 소주 관심 증가"
- 광고: 여울 제품 소개 (서사 "전통 복원"의 논리적 결론으로 연결)
→ **서사-광고 일체형**. 광고가 서사의 결론이므로 "끝났는데 왜 안 끝나지" 느낌 없음.

### 4-5. 결론부 추가 발견 5건

1. **정리 단계에 "통찰" 결합** (03, 19): 인과 요약이 아닌 관점 재구성.
2. **본문 소재 회수** (04): 도입부 소재뿐 아니라 본문 소재("삼겹살 일화")도 수미상관으로 기능.
3. **엔딩 유형 혼합** (01, 03, 23): 단일 유형이 아니라 **1주+1보조 혼합**.
4. **자기언급형 서사의 엔딩 적용** (23): "채널-시청자 동일시"로 별도 규정 가능.
5. **결론부 신규 정보의 실전적 허용 범위** (01, 04, 14): "본문 주제의 현재 규모/한국 사례 확인"은 ML 해소가 아니므로 허용.
