# 분석 완료 대본 목록

> **이 파일의 역할:** `대본분석.md` 에이전트(이거봐봐 채널 전용)가 누적 분석 결과를 추적하기 위한 인덱스.
>
> **신규 대본 판정 방식:** `대본/NNN_*` 폴더에 있으나 본 목록에 없는 번호 = 신규.
>
> **갱신 주체:** 본 파일은 메인 에이전트가 유저 합의 후 갱신한다. `대본분석.md` 에이전트는 직접 수정하지 않는다.
