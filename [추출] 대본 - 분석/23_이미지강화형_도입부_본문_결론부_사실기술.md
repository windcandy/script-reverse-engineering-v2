## 2. 10편 3축 사실 기술

### 2-1. 대본 02 — 신발 (조각 제시형)

**도입부 (00:00~01:22):** "관념 자체가 다르다" 부분 답 + 입식 vs 좌식 개념 호명. 본격 답(기후·역사·문화)은 본문

**본문 (01:22~):** 한·중·일 가옥 구조·좌식 군대 내무실·기후 요인(습한 동아시아·사막 페르시아·유럽 입식)·입식 가축 동거 본격 풀이

**결론부 (10:34~):** "미국 영화 신발 = 자본의 논리"·"실내·실외 개념·청결 개념부터가 다르다"·"제발 침대에 올라갈 때만큼은 신발 벗는게 상식"
- **유형: 개념 통찰 회수형**

**관찰:** 도입부 호명 표현 본문 동일 재호명 없음

### 2-2. 대본 05 — 락스 (결과 제시형)

**도입부 (00:00~01:37):** "오히려 사람을 살리는 물질"·"세균 죽이는 이만한 물질 없음"·"현대 도시 위생 공공 보건은 락스가 없으면 유지 불가능" 결론 직접 진술

**본문 (01:37~):** 18세기 표백 필요성 → 염소 발견 → 자벨수 → 분말 표백 → 작동 원리 → 도시 위생·미아즈마 → 라바라크 액체 → 클로르 알칼리 → 클로락스 본격 풀이

**결론부 (12:34~):** "락스는 무조건 차가운 물에 단독으로 타서 써야 된다"·"현대인들의 위생과 보건을 떠바치고 있습니다"·"락스 온더락은 유머로만 사용하자"
- **유형: 드립 회수 + 통찰 확장형**

**관찰:** 도입부 호명 표현 본문 동일 재호명 없음

**본문 (02:06~):** 돼지 효율성 vs 사막 환경 → 사막화 + 특권 자원 → 종교 금지 → 기독교 vs 이슬람 분기 → 인도 불교/힌두교 → 소 분기 본격 풀이

## 3. 3축 영역 종합 (10편 누적 사실)

### 3-1. 도입부 영역 (10편 공통)

- 10편 모두 도입부에서 약속을 100% 해소하지 않음
- 도입부 정보량 분류:
  - **결론 직접 진술형 (3편):** 05·08·11 — 모두 결과 제시형
  - **부분 답 호명형 (4편):** 02·07·10·17 — 모두 조각 제시형
  - **결과 사실 호명형 (1편):** 09 — 조각 제시형
  - **사전 정보 호명형 (2편):** 20·24 — 모두 조각 제시형
- 본격 답·구체 메커니즘·정점은 모두 본문 진입

### 3-2. 본문 영역 (10편 공통)

- 10편 모두 도입부 호명 표현·수치 본문 동일 재호명 발견되지 않음
- 도입부 정보는 본문에서 시기별·세부·맥락·구체 등 다른 각도로 풀이

### 3-3. 결론부 영역 (10편 분기)

| - | - |
