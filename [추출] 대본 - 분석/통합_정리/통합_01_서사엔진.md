## 0. 서사 엔진이란?

서사 엔진 = 본문이 시청자를 끌어당기는 **동력 메커니즘**.

각 엔진은 시청자에게 어떤 인지·갈증·통찰을 유발하여 본문을 따라오게 하는지를 정의한다.

**엔진 본질 = 시청자 경험을 만드는 작동 원리**. 형식이나 구조 갯수가 아닌, **시청자가 어떤 갈증·통찰·발견을 경험하는가**가 엔진의 핵심.

## 1. 서사 엔진 7종 — 작동 원리

### 1.1 역설 엔진 (모순형)

**작동 원리**: 양극(A극·B극) 공존·역설·모순 발견. A극과 B극이 같은 대상의 두 측면으로 동시 존재.

**시청자 경험**: 본 대상에 대한 한쪽 인지(A극 장점)가 확신될 때쯤 반대편(B극 단점·역설)이 드러나며 통찰 도달.

**핵심 어법**:

- A극 누적 → 메가 SL(피벗) → B극 분할 (모순형 표준)
- A극 스펙이 곧 B극 약점 원인 (Aha 발견)
- 피벗 3축 어법: 확신 극대화 + 시청자 예상 흡수 + 선언

**변주 4종 (실제 사례 기반)**:

| 변주 | 작동 | 본 채널 사례 |
|---|---|---|
| 정방향 | A극 강점 → B극 약점 폭로 | 01 감자 |
| 역방향 | A극 약점 → B극 강점 재발견 | 03 한반도 |
| 관점 전환 메타 | A극 본질이 시청자 관점에 따라 양극 교차 | 19 |
| 단일 속성 양면 작동 | 한 속성이 동시에 강점·약점 작동 | 14 |

→ **핵심**: 양극 공존이 본문 척추라면 역설 엔진. 단순 반전·대조가 아닌 같은 대상의 두 측면 동시 존재.

### 1.2 영역 확장 엔진 (이미지 강화형)

**작동 원리**: 단일 대상이 시청자가 인식 못한 다층 영역까지 영향. 영역 확장 자체가 체감 격상.

**시청자 경험**: "내가 알던 것보다 훨씬 광범위하다"는 영역 확장 체감.

**핵심 어법**:

- "이제 끝났냐? 아니죠" 단계 추가 반복
- 영역 확장 정점 = 도입부 약속 회수
- 시청자 본인 일상 도달

**변주**:

| 변주 | 작동 | 본 채널 사례 |
|---|---|---|
| 비교 통한 일상 재인식 | A 비교 사례로 본 대상 일상 재해석 | 02 소금 |
| 추상 영역 확장 | 본 대상이 추상 영역(법·종교·정치)까지 영향 | 07·17·30 |
| 공간 영역 확장 | 본 대상이 다지역 공간 영향 | 08 |
| 시간 분리 영역 확장 | 본 대상이 다시대 영역 영향 | 11 |

→ **핵심**: 단일 대상의 다층 영역이 본문 척추라면 영역 확장 엔진. 단순 규모 확장이 아닌 다층 영향 발견.

### 1.3 인과 추적 엔진 (과정 추적형)

**작동 원리**: 현재 상태의 원인을 시간 거슬러 추적. 시간/인과 흐름으로 "왜 지금 이렇게 됐나" 답.

**시청자 경험**: 현재 상태가 어떤 과거 흐름의 결과인지 발견 — 시간 진행 따라가며 인과 통찰.

**핵심 어법**:

- 시간 진행 + 메커니즘 단락 삽입
- 정밀 연도 표기 (사건 서사 변주)
- 현대 도달 어법으로 시청자 현재 결합

**변주 (시간 진행 단위 10종 — 본 채널 사례 기반)**:

| 변주 | 본 채널 사례 |
|---|---|
| 발명사 | 05 락스 |
| 작물 보급사 | (분석 사례) |
| 사상 변천사 | (분석 사례) |
| 역사 사건 서사 | 06·37·38·39·12·21 |
| 외교·자원 정치사 | (분석 사례) |
| 산업·기술 발전사 | (분석 사례) |
| 영토 변천사 | (분석 사례) |
| 문화·기술 발달사 | (분석 사례) |

| 단일 대상 변천사 | 22 온돌·25 소주·34 아파트 |

→ **핵심**: 시간 진행으로 현재 도달을 추적하는 것이 본문 척추라면 인과 추적 엔진.

### 1.4 다층 원인 수렴 엔진 (신 엔진)

**작동 원리**: 본 대상의 현 상태·특성을 만든 **여러 원인을 다층 누적 후 한 결과로 수렴**.

**시청자 경험**: 본 대상의 복합 원인이 하나하나 드러나며 "아, 이런 다층 원인이 모여서 이렇게 됐구나" 통찰 도달.

**핵심 어법**:

- 원인 1·2·3 다층 나열
- 각 원인이 본 대상의 한 측면 설명
- 최종 결산에서 다층 원인이 본 대상으로 수렴

**본 채널 사례**: 09·26·27 (한국인 냄새 — 암내·피지·음식·빨래 4중 원인 수렴)

→ **핵심**: 본 대상이 한 가지 단순 원인 아닌 다층 원인 결합으로 형성됨을 보여줄 때.

### 1.5 사례 비교 엔진 (신 엔진)

**작동 원리**: 다수 사례를 평행 구조로 비교하여 공통 원리 발견 또는 차이 발견.

**시청자 경험**: 사례 A·B·C가 나열되며 시청자가 자발적으로 비교·대조·통찰 도달.

**핵심 어법**:

- 사례 진입 어법 ("X도", "Y의 경우는")
- 사례 평행 구조 (각 사례 동일 항목 비교)
- 결산에서 공통 원리 또는 차이 통찰

**변주 (사례 수에 따른 변주)**:

| 변주 | 본 채널 사례 |
|---|---|
| 2사례 평행 비교 (공통 원리 발견) | 10 |
| 3사례 비교 (차이 발견 + 핵심 사례 강조) | 32 한중일 숟가락 |
| 다국가 사례 비교 (5+) | 40 중화요리·41 소금 |

→ **핵심**: 사례 비교가 본문 척추라면 사례 비교 엔진. 단일 대상 분석이 아닌 다대상 비교 통찰.

### 1.6 사례 나열 엔진 (신 엔진)

**작동 원리**: 다수 사례를 동일 패턴으로 망라하여 본 대상의 다양성·범위 보여주기.

**시청자 경험**: 사례가 끊임없이 나오며 "이렇게나 다양하구나"·"범위가 이렇게 넓구나" 체감.

**핵심 어법**:

- 사례 나열 어법 ("X도 있고", "Y도 있다")
- 각 사례 짧은 묘사 + 다음 사례
- 결산에서 본 대상 다양성·범위 격상

**본 채널 사례**: 20

→ **핵심**: 사례 비교(통찰 도출)와 다름. 단순 다양성·범위 체감이 본문 척추.

### 1.7 다층 분기 수렴 엔진 (신 엔진)

**작동 원리**: 본 대상이 여러 분기로 갈라진 후 한 점으로 수렴. 다층 원인 수렴과 다름 — 분기 → 수렴 구조.

**시청자 경험**: 본 대상이 여러 갈래로 갈라진 모습 인지 → 그 갈래들이 한 점으로 수렴되는 통찰.

**본 채널 사례**: 31

→ **핵심**: 분기 → 수렴이 본문 척추일 때.

## 2. 엔진 단일/복합 조합 — 집필 시 선택 기준

엔진은 **단일** 또는 **복합 조합**으로 사용 가능. 주제·서사 의도에 따라 선택.

### 2.1 엔진 단일 사용 — 본질·시기

| 엔진 단일 | 선택 시기 | 시청자 경험 |
|---|---|---|
| **역설 엔진** | 본 대상에 양극 공존이 명확하고 모순·역설이 통찰 핵심일 때 | "이런 양면이 동시에 있구나" |
| **영역 확장 엔진** | 본 대상이 시청자가 인식 못한 다층 영역까지 영향이 통찰 핵심일 때 | "이렇게 광범위하구나" |
| **인과 추적 엔진** | 본 대상의 현재 상태가 어떤 과거 흐름의 결과인지가 통찰 핵심일 때 | "이런 과정으로 이렇게 됐구나" |
| **다층 원인 수렴 엔진** | 본 대상이 다층 원인 결합으로 형성됨이 통찰 핵심일 때 | "이런 다층 원인이 모여서 이렇게 됐구나" |
| **사례 비교 엔진** | 다수 사례 비교로 공통 원리 또는 차이 발견이 통찰 핵심일 때 | "이 사례들의 공통점/차이가 이거구나" |
| **사례 나열 엔진** | 본 대상의 다양성·범위 체감이 통찰 핵심일 때 | "이렇게나 다양하구나" |
| **다층 분기 수렴 엔진** | 본 대상의 다분기가 한 점으로 수렴이 통찰 핵심일 때 | "갈래들이 결국 여기로 모이는구나" |

### 2.2 엔진 복합 조합 — 본질·시기

엔진 복합은 **척추 + 보조** 관계. 척추 엔진이 본문의 주력 동력, 보조 엔진이 척추를 뒷받침.

| 복합 조합 | 본질 | 본 채널 사례 |
|---|---|---|
| **역설 엔진(척추) + 인과 추적(보조)** | 양극 공존 + 시간 진행 결합. 양극이 어떻게 형성됐는지 시간 추적 | 01 감자·23·29 |
| **인과 추적(척추) + 사례 비교(보조)** | 시대별 변천 + 지역 비교 결합. 단일 대상이 시대·지역마다 어떻게 변했는지 | 40 중화요리·41 소금 |
| **사례 비교(척추) + 인과 추적(보조)** | 다국가 비교가 척추이나 각 국가 안 시간 진행 보조 | 40 (비교 비중이 더 강함) |
| **인과 추적(척추) + 영역 확장(보조)** | 시간 진행 척추 + 사용 영역 확장 결합 | 35 콜로세움 |
| **영역 확장(척추) + 인과 추적(보조)** | 영역 확장 척추 + 시간 진행 부수적 | 02 소금·24 전기차 |
| **다층 원인 수렴(척추) + 영역 확장(보조)** | 다층 원인 척추 + 본 대상의 다층 영역 확장 보조 | 27 한국인 냄새 |
| **인과 추적(척추) + 역설(보조 작은)** | 시간 진행 척추 + 작은 역설 삽입 | 38 강뉴 부대 |

→ **핵심**: 복합 조합은 단일 엔진으로 표현 못 하는 복합 통찰을 다층으로 끌어가는 방식.

### 2.3 엔진 선택 기준 — 집필 의도별

집필 시작 단계에서 **본 대상의 통찰 핵심**을 먼저 정의:

| 통찰 의도 | 권장 엔진 |
|---|---|
| 양면 공존·역설 통찰 | 역설 엔진 |
| 광범위한 영향 통찰 | 영역 확장 엔진 |
| 현재 도달 과정 통찰 | 인과 추적 엔진 |
| 다층 원인 결합 통찰 | 다층 원인 수렴 엔진 |
| 비교 통찰 (공통 원리·차이) | 사례 비교 엔진 |
| 다양성·범위 체감 | 사례 나열 엔진 |
| 분기 → 수렴 통찰 | 다층 분기 수렴 엔진 |

**복합 조합 선택 기준**:

- 통찰이 **한 차원만으로 부족**할 때 → 척추 엔진 + 보조 엔진 결합
- 예: "본 대상의 양극 공존 + 그 양극이 형성된 시간 진행" → 역설 엔진(척추) + 인과 추적(보조)
- 보조 엔진은 척추 엔진의 통찰을 뒷받침하거나 풍부하게 만드는 역할

→ **핵심 원칙**: 엔진은 통찰의 형식이지 통찰 자체가 아님. 통찰을 정의한 후 엔진을 선택, 엔진에 통찰을 끼워맞추지 않음.

## 3. 엔진별 작동의 본질 — 정성 기준

각 엔진의 본질을 **정성 기준**으로 정리. 정량 수치 회피 (가이드 원칙 5).

### 3.1 역설 엔진 — 본질

**작동 본질**:

- A극(시청자 친숙 측면)이 시청자에게 **확신될 때까지** 충분히 누적
- 메가 SL(피벗)에서 **A극 확신 → 시청자 예상 흡수 → 본격 전환** 3축 작동
- B극(반대 측면)이 A극과 대등한 무게로 양극 공존이 인지되도록 누적
- Aha 발견: A극 스펙이 B극 약점 원인으로 역전될 때 발생 — **시청자가 자발적 연결**할 수 있도록 양극 키워드 일관 사용

**❌ 잘못된 지침 예시**: "A극 누적 4~9단락"

**✓ 본질 지침**: "A극이 시청자에게 확신될 만큼 충분히 누적된 후 피벗 진입. 갯수는 주제·강도에 따라 자유"

### 3.2 영역 확장 엔진 — 본질

**작동 본질**:

- 본 대상이 시청자가 **인식 못한 영역**까지 영향 보여주기
- 영역 확장이 **시청자 일상에 도달**할 때 정점
- 영역 확장 정점 = 도입부 약속(ML) 회수

**❌ 잘못된 지침 예시**: "영역 3개 이상 확장"

**✓ 본질 지침**: "본 대상의 영향이 시청자가 인식 못한 곳까지 도달했음을 체감할 때까지 영역 확장"

### 3.3 인과 추적 엔진 — 본질

**작동 본질**:

- 현재 상태의 **원인을 시간 거슬러** 추적
- 시간 진행 + 메커니즘 단락 (시간만 진행하면 단순 역사 나열)
- 결산에서 현재 상태로 다시 도달 — 시청자 현재 결합

**❌ 잘못된 지침 예시**: "시간 진행 단계 5개 이상"

**✓ 본질 지침**: "현재 상태의 원인이 시청자에게 충분히 이해될 때까지 시간 진행. 각 진행 단계마다 메커니즘 함께 설명"

### 3.4 다층 원인 수렴 엔진 — 본질

**작동 본질**:

- 본 대상의 현 상태를 만든 **다층 원인을 하나하나 누적**
- 각 원인이 본 대상의 한 측면을 설명
- 최종 결산에서 다층 원인이 본 대상으로 수렴

**❌ 잘못된 지침 예시**: "원인 3개 이상 누적"

**✓ 본질 지침**: "본 대상의 현 상태를 설명하기에 다층 원인이 충분히 누적된 후 수렴 결산"

### 3.5 사례 비교 엔진 — 본질

**작동 본질**:

- 다수 사례를 **평행 구조**로 비교 (각 사례 동일 항목)
- 비교 결과에서 **공통 원리** 또는 **핵심 차이** 발견
- 시청자가 자발적으로 비교·통찰 도달

**❌ 잘못된 지침 예시**: "사례 3개 비교"

**✓ 본질 지침**: "비교 통찰(공통 원리·차이)이 시청자에게 명확히 도달할 때까지 사례 누적. 사례 수는 통찰 강도에 따라 자유"

### 3.6 사례 나열 엔진 — 본질

**작동 본질**:

- 본 대상의 **다양성·범위 체감**이 핵심
- 사례 비교(통찰)와 다름 — 단순 망라
- 사례 하나하나 짧은 묘사 + 누적 효과

**❌ 잘못된 지침 예시**: "사례 5개 이상 나열"

**✓ 본질 지침**: "본 대상의 다양성·범위가 시청자에게 체감될 때까지 사례 나열"

### 3.7 다층 분기 수렴 엔진 — 본질

**작동 본질**:

- 본 대상이 **여러 분기**로 갈라진 모습 인지
- 그 분기들이 **한 점으로 수렴**되는 통찰

**❌ 잘못된 지침 예시**: "분기 4개 이상"

**✓ 본질 지침**: "본 대상의 분기 다양성과 수렴 한 점이 시청자에게 명확히 도달할 때까지"

## 4. 엔진 식별 — 분석 시 점검 질문

새 대본 집필 또는 분석 시 다음 질문으로 엔진 선판단:

1. 본 대상의 **양극 공존·역설·모순**이 본문 척추인가? → 역설 엔진
2. 본 대상의 **다층 영역 영향**이 본문 척추인가? → 영역 확장 엔진
3. **시간 흐름·인과 추적**이 본문 척추인가? → 인과 추적 엔진
4. **다층 원인이 한 결과로 수렴**이 본문 척추인가? → 다층 원인 수렴 엔진
5. **다수 사례 비교**가 본문 척추인가? → 사례 비교 엔진
6. **다수 사례 망라**가 본문 척추인가? → 사례 나열 엔진
7. **분기 → 수렴**이 본문 척추인가? → 다층 분기 수렴 엔진

복수 엔진 결합 가능. 어느 엔진이 척추(주력)이고 어느 엔진이 보조인지 명시.

---

## 5. 엔진별 보조 작동 영역 — 본문 전 영역 점검

엔진 결합형에서 보조 엔진은 본문 일부만이 아닌 **전 영역**에서 작동 가능:

- A극 (도입~정점)
- 피벗 (메가 SL)
- B극 (Aha~증명 케이스~누적)
- 현대 영향
- 미래 확장
- 결론부

→ 분석 시 보조 엔진이 어느 영역에서 작동하는지 **전수 점검** 후 표기 (가이드 원칙 4 참조).

---

## 6. 결론 — 집필 시 엔진 사용 절차

집필 시 다음 절차로 엔진 선택:

### 1단계: 통찰 정의

본 대상에 대해 **무엇을 시청자에게 통찰시키고 싶은가**? (양극 공존·다층 영향·인과 추적·다층 원인 등)

### 2단계: 엔진 단일/복합 결정

통찰이 한 차원으로 충분하면 단일 엔진, 복합 차원 필요하면 복합 조합.

### 3단계: 척추·보조 결정

복합 시 어느 엔진이 본문 주력(척추)이고 어느 엔진이 뒷받침(보조)인지 결정.

### 4단계: 본질·작동 원리 기반 구조 설계

엔진별 본질 (양극 공존·영역 확장·인과 추적 등)을 기준으로 본문 구조 설계. **정량 수치 회피, 본질 기준 적용**.

### 5단계: 본문 작성

각 단락이 엔진 본질에 부합하는지 점검하며 작성. 단락 갯수·수치는 결과일 뿐 기준 아님.
