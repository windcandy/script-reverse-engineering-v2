### 1-1. 결론 브릿지 어법 5종 정의 (구 18번 §1-1 흡수)

본문 마지막에서 결론부 진입 단락으로 넘어가는 한 문장의 연결 어법 분류. 본문 내부 BR(Type A SL 15종 + Type B 논리적 BR 3종)과는 별개 영역 — 용어정리.md §4 BR ↔ §5 결론 브릿지 어법 경계 주석 참조.

| 번호 | 명칭 | 정의 | 표지 어구 예시 |
|---|---|---|---|
| ① | 결과 호명 | 본문 내용을 결과로 정리하며 결론부 진입 | "결국 ~", "그래서 ~", "이렇게 ~된다", "정리하자면 ~" |
| ② | 이완 호명 | 본문 긴장을 풀고 시청자에게 직접 호명 | "자, 이제 ~", "그러니까 ~", "여담으로 ~", "뭐 아무튼 ~" |
| ③ | 시간 전환 | 시제 변경(과거→현재·미래)으로 결론부 진입 | "오늘날 ~", "지금 ~", "앞으로 ~", "먼훗날 ~" |
| ④ | 회수 | 도입부 또는 본문 소재·인물을 다시 호명 | "아까 ~했던 그것이...", "처음에 말한 ~", 도입부 인물·소재 직접 재등장 |
| ⑤ | 메타 | 영상 자체에 대한 언급 (시청자·채널·영상에 대한 직접 언급) | "이 영상 ~", "오늘 다룬 ~", "이렇게 보면 ~", "선원 여러분 ~" |

**다중 결합 허용:** 한 결론부가 5종 중 2~3종을 동시에 사용하는 경우가 다수 (실측 — 모순형 7편 중 5편이 다중 결합).

### 1-5. 4기준 검증 (구 §0-3 원칙 2 이동 + 원칙 3 통합)

분석 패턴 신뢰성 검증 4기준 — 억지 패턴·조건 회피 + 결과 처리 절차.

**X·Y 변수 정의:**

- **X 조건**: 분석 패턴의 입력 변수 (도입부 전략·본문 결산 양식·본문 정점 위상 등 P3·P4에서 결정되는 선행 변수)
- **Y 결과**: 분석 패턴의 출력 변수 (결론 브릿지 어법·매끄러움·결론부 직전 단락 안전장치 등 결과로 관측되는 변수)
- **자연 연동 패턴**: X → Y 인과 관계 (≥50% 발현 시 자연 연동 영역)

| 기준 | 검증 |
|---|---|
| 사례 수 | 묶음 내 ≥2건 + 다른 묶음 동일 발생 (총 ≥3건) → 자연 패턴 후보 / 미달 시 단순 유형으로만 보존 |
| 사후 합리화 회피 | X 조건이 P3·P4에서 결정되는 선행 변수인가? Y 결과로 분류된 사례의 X 공통점을 사후 추출했는가? |
| X·Y 정의 중복 회피 | X 조건이 Y 결과 정의의 일부인가? (동어반복 폐기) |
| 반례 검증 | 같은 X 조건에서 다른 Y 결과 사례 발견? → 신뢰도 약화 |

**결과 처리 4단계:**

```text
4기준 모두 통과 → 자연 연동 패턴 등재 (강제·권장 후보)
사례 수 미달 (단일 사례) → 단순 유형 분류만 보존 — 조건 강제 없음
사후 합리화 / X·Y 정의 중복 → 폐기 — 등재 부적합
반례 발견 → 신뢰도 약화 → 분포 정보로만 보존
```

### 1-6. 매끄러움 평가 (구 §1-2 분석 항목 흡수)

## 7. P4 §D-1 정비안 (3유형 통합 — 18번 §10 + 19번 §7 통합)

### 7-1. 공통 원칙 (3유형 적용 — 구 18번 §10-1 흡수)

1. **결론부의 ML 답 직접 정리 금지** — 본문 위임 절대 원칙 (3유형 0/25 실증)
2. **본문 비유 회수 금지** — 회수 대상 단일화 (도입부 OR 본문 소재 OR 자기언급 1종만 — 용어정리.md §3 #4 참조)
3. **본문 결산 양식에서 ML Q1·Q2 + 양극 균형 또는 통찰 정리 마무리** — 결론부 진입 직전
4. **결론부 진입은 결론 브릿지 어법 5종 1~3종 결합** (§1-1 참조)
5. **시제·톤 이완 필수** — 본문보다 가벼운 톤으로 이동
6. **표준 시그니처 종결** — 변주는 영상 자체의 요지가 명확할 때 허용

### 7-2. 본문 결산 양식 → 결론부 진입 어법 가이드 (구 18번 §10-2-1~4 흡수)

## 8. 후속 과제

### 8-1. 보강 사이클 — 매끄러움 ❌ 반례 사례 수집

**목적**: 4기준 (4) 미해결 영역 해소 (인과 vs 우연 구분)

**검증 영역**:

| 검증 항목 | 의미 |
|---|---|
| 매끄러움 ❌ + 안전장치 부재 비율 | 안전장치 부재가 매끄러움 ❌의 원인인지 검증 |
| 매끄러움 ✅ + 안전장치 부재 사례 | 안전장치 없이도 매끄러움 가능한지 검증 |
| 안전장치 존재 + 매끄러움 ❌ 사례 | 안전장치 무력화 사례 — 발견 시 가설 폐기 |

**판정 기준**: 추가 사례 50편에서 매끄러움 ❌ 사례 ≥5건 발견되고 80% 이상이 안전장치 부재면 인과 가설 강화. 안전장치 존재+매끄러움 ❌ 사례 발견 시 가설 폐기.

**시행 시점**: P4 §D-1-2 정비 시행 후 또는 별도 사이클로 진행.
