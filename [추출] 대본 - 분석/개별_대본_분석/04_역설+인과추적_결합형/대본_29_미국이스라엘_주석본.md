#### 메커니즘 단락 1 (01:12~03:27) — 기독교 vs 유대교 관계 | 단위: 메커니즘 단락 (게임 비유 체계)

```text
도입부: 차별 역사 점층 + 현재 역전 + 모순 명제 + 단언형 ML (본질 답 선언)
    ↓
메커니즘 단락 1: 기독교 vs 유대교 (게임 비유 체계 진입)
    ↓
사상 변천 단락 1: 성취 신학 (보편 관점)
    ↓
시간 점프 단락: 70년 → 7세기 → 중세
    ↓
한 단계 깊은 질문: 후속작 떡밥
    ↓
사상 변천 단락 2: 천년 왕국 해석 4단계 변천
    ↓
신 사상 등장 단락: 세대주의 (핵심 Aha)
    ↓
메커니즘 단락 2: 세대주의 종말 시나리오
    ↓
현재 정치 단락: 이스라엘 + 기독교 시온주의 시너지
    ↓
결론부: 정치 영향 결산 + 화자 자기 언급 + 입장 표명 + 평화 희망
```

### 기존 3유형 + 신 엔진 후보 대조

| 엔진 | 판단 | 근거 |
|---|---|---|
| 역설 엔진 (모순형) | △ 보조 | 도입부 모순 명제(미국 기독교 + 유대인 지원) + 본문 후반 답 도출 |
| 영역 확장 엔진 (이미지강화형) | ✗ | 영향 범위 확장 없음 |
| 인과 추적 엔진 (과정추적형) | ◎ 척추 | 시간 + 사상 변천 추적이 본문 척추. 신학 사상이 미국 정책 형성 인과 |
| 다층 원인 수렴 엔진 | ✗ | 단일 인과 추적이 척추. 다층 원인 누적 아님 |
| 사례 비교 엔진 | ✗ | 사례 평행 없음 |
| 사례 나열 엔진 | ✗ | 다수 사례 나열 없음 |

### 판정: **인과 추적 엔진(척추) + 역설 엔진(보조) 결합형 — 23·29 누적 2건 확정 가능**

**23·29 결합 변주 공통:**
- 도입부 모순 명제 + 본문 인과 추적 + 후반 모순 답 도출
- 시간/사상 진행이 척추, 모순이 도입부·결론부 보조
- 결론부 화자 자기 언급 + 입장 표명 + 미래 희망

**23·29 차이:**
- 23: 시간 추적 = 작물 보급사 (사물 추적)
- 29: 시간 추적 = 신학 사상 변천 (사상 추적)

→ 같은 결합 변주, 추적 대상 폭 다름. **인과 추적 엔진의 추적 대상 확장 변주 정의 가능:**

- 변주 A — 사물·정책 추적: 23 (아보카도)
- 변주 B — 사상·신학 변천 추적: 29 (기독교 시온주의)

### 인과 추적 + 역설 결합 변주 확정 판정

**누적 2건 도달 (23·29)** — 임계값 도달.

기획안 임계값:
1. 기존 3엔진 어디에 부합? 인과 추적 + 역설 두 엔진 결합 ✓
2. 최소 2~3건 이상 동일 작동 ✓ (23·29)
3. 구조적 작동 원리가 기존 단일 인과 추적과 다름 ✓

→ **인과 추적 + 역설 결합형 = 과정추적형의 변주로 확정 가능.** 기존 과정추적형 8편 재분석 시 추가 발견 검증.
