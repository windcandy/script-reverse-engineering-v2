## 1. 사전 진단 — 현재 P1 지침의 결정적 누락

| 항목 | 현재 P1 지침 상태 | 25편 실측 사실 |
|---|---|---|
| 썸네일 문구 = 단일 개념 | `§5.A`가 "썸네일 문구는 단일 문장 11~30자" — **단일 문구 가정** | **25편 100%가 메인+서브 이중 구조** (메인 문구 + 말풍선·박스 서브 문구) |
| 서브 문구 작성 기준 | **부재** | 평균 9.1자·3~16자·단일행·반말, 발화 주체 6분류 |
| 메인 문구 변주 도구 | `§5` 6단계가 정체/주어 제거만 다룸 | 4종 예외 패턴 — 25편 16% 사용 |
| 용어 표준 (00_용어정리.md) | "썸네일 문구" 단일 — **메인/서브 미등록** | 두 용어 표준화 필수 |
| §9 답 노출 금지 vs ⑤ 답 노출 | "답 유추 가능하면 탈락" 단일 — 메인/서브 미구분 | 서브 ⑤ 답·결론 노출 = 과정추적 38% 정상 패턴 |
| §8.A 자유 후보 트랙 적용 범위 | 무시 가능 제약 8건 명시 — 서브 영역 없음 | 자유 후보는 메인 단독 작성 가능 명시 필요 |

**결론:** P1 지침은 사실상 **썸네일 메인 문구만 다루는 상태**. 서브 문구·예외 패턴·메인/서브 구분은 18번 파일에만 존재 → P1 본문 통합 필요.
