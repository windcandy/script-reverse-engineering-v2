## 2. 7편 3축 사실 기술

### 2-1. 대본 01 — 감자 (조각 제시형)

**도입부 (00:00~01:34):** "전 세계 3억 8천만 톤·4위 주식 작물·오지게 많이 먹습니다·일부 지역에서는 빵보다 많이 먹어서 사실상 최중 주식" — A극(긍정·생산성) 사실 호명. 약속 영역(B극) 직접 답 부재

**본문 (01:34~):** 안데스 작물화 → 면적당 칼로리 → 콜럼버스 도입 → 강제 보급·인구 폭증 → **피벗 06:39 "이쯤 되면 인류의 구원이나 다름없는 작물이죠... 결론부터 말하면 이랬다가 진짜 큰일 날 수도 있습니다"** → B극 본격 풀이 (전염병·아일랜드 대기근·재배 난이도·보관 한계)

**결론부 (13:33~):** "감자는 앞으로 인류의 식량 안보를 책임질 핵심 작물"·"화성식민지가 개척됐을 때도 영화처럼 일단은 감자부터 재배"·"햄버거의 감자 튀김 먹으러"
- **유형: 미래 통찰 + 일상 마무리형**

**관찰:** 도입부 호명 표현 본문 동일 재호명 없음

### 2-2. 대본 03 — 한반도 (조각 제시형)

**도입부 (00:00~01:31):** "한반도는 이 모든게 정확히 반대로 돼 있는 중간만 가도 선방했다고 봐야 되는 그런 땅" — A극 호명 + "근데 또 막상 살아남았다는 거·인간의 적응력은 한계가 없고 우리 조상들도 결국엔 이 땅을 이겨냈다" — B극 예고

**본문 (01:31~):** 한반도 지세·산지 70% → 마차 안 다님 → 행정력 디버프·조선 농본주의 → 쌀 농사·강수량 몰빵 → **피벗 09:50 "장점은 없냐? 있긴 있죠"** → B극 풀이 (방어 유리·산성 우주방어·물맛)

**결론부 (13:20~):** "이런 초고난도 입지에서 지금껏 독립적인 문화와 역사를 유지할 수 있었던 것도 어찌됐든 이 투박하고 단단한 바위산들 덕분"·"이런 걸 애증이라고 하는 건가"
- **유형: 애증 통찰 회수 + 일상 마무리형**

**관찰:** 도입부 호명 표현 본문 동일 재호명 없음

### 2-3. 대본 04 — 석면 (모순 공존형)

**도입부 (00:00~01:31):** A극 풍부 노출 "불에 타지도 부식되지도 않는 물질·머리카락보다 얇은 섬유·기적의 물질" + B극 직접 호명 "거의 모든 분야에서 퇴출·끔찍한 디버프·1군 발암 물질·침묵의 살인자"

**본문 (01:31~):** 석면 정의 → 17세기 광산·산업 혁명 → 20세기 광범위 사용 → "이쯤 되면 뇌절 아닌가...이건 양반이었다" (피벗) → B극 본격 풀이 (1924 성패증·1930 발암·1970 공식 발암) → "여기까지는 ... 진짜 무서운 건 지금부터입니다" (피벗 2) → 발암 메커니즘

**결론부 (15:42~):** "우리나라는 산노화 역사 짧고 ~ 이 정도 수준에서 수습된게 기적"·"성면 슬레이트 삼겹살"·"삼겹살은 불판에다가 구워 드시길"
- **유형: 수습 통찰 + 일상 마무리형**

**관찰:** 도입부 호명 표현 본문 동일 재호명 없음

### 2-4. 대본 14 — 까트 (모순 공존형)

**도입부 (00:00~00:47):** A극 호명 "커피보다 오랜 역사·커피보다 각성 효과 더 뛰어남·생으로 씹기만 하면 되는 간편한 식물" + B극 예고 "천년 넘게 원산지에서만·절대 전 세계로 보급될 수 없는 이유·절대 그래서는 안 되는 이유"

**본문 (00:47~):** 마약류 분류 → 카트 사회 침투 → 카트 효과 (도파민·암페타민 유사) → 사회 폐해 → **"100번 양보해서 ~ 좋은 거 아니냐 ~ 문제는 농민들이 다른 농사를 안 짓는다" (피벗)** → B극 본격 풀이 (농지·물·식량 문제)

**결론부 (11:28~):** "등잔이 어둡다는게 이런 걸 말하는 거지"·"한국도 다문화 사회·미리미리 알아두고 더 철저히 단속"·"문화 상대주의라고는 하지만 마약에까지 적용할 수는 없는 일"
- **유형: 단속 통찰 + 일상 마무리형**

**관찰:** 도입부 호명 표현 본문 동일 재호명 없음

### 2-5. 대본 19 — 대통령제 (조각 제시형)

**도입부 (00:00~01:03):** A극 호명 "대통령이 곧 국가 권력의 정점이자 나라의 대표자" + B극 호명 "서구권 선진국들에서는 대통령이 아예 없거나 상징적인 자리" + 부분 답 "둘 사이에 우열 가릴 수 없다"

**본문 (01:03~):** 영국 의원내각제 기원 → 미국 대통령제·삼권 분립 → 장점 → 신생국 인기 → **"하지만 우리는 다 알고 있죠 ~ 결코 장점만 있는 제도는 아닙니다" (피벗)** → 단점 본격 풀이 (행정-입법 적대·여대야소 독재·발목잡기·탄핵 대혼란)

**결론부 (11:46~):** "지금의 시스템도 충분히 합리적·또는 결함 있는 버전"·"민주주의 자체가 완벽할 수가 없는데"·"국민들이 적극적으로 관심을 가지고 이성적이고 합리적인 판단을 해야만 잘 돌아가는게"

## 3. 3축 영역 종합 (7편 누적 사실)

### 3-1. 도입부 영역 (7편 공통)

- 7편 모두 도입부에서 약속을 100% 해소하지 않음
- **도입부 정보량 — 모순형 특유 영역:** A극 호명 또는 A극 + B극 양면 호명
  - **A극 사실 호명 + B극 본문 (1편):** 01 감자
  - **A극 + B극 양면 호명 (6편):** 03·04·14·19·23·25
  - 모순 공존형 2편 (04·14)은 양면 호명 강도 가장 풍부
- 본격 답·구체 메커니즘·정점은 모두 본문 진입

### 3-2. 본문 영역 (7편 공통)

- 7편 모두 도입부 호명 표현·수치 본문 동일 재호명 발견되지 않음
- 본문 처리 영역에 **피벗 전환** 다수 발견 (01·03·04·14·19·23 — 6편)
  - "이쯤 되면 인류의 구원이나 다름없는 작물이죠... 결론부터 말하면 이랬다가 진짜 큰일 날 수도" (01)
  - "장점은 없냐? 있긴 있죠" (03)
  - "이쯤 되면 뇌절 아닌가...이건 양반이었다" (04)
  - "100번 양보해서 ~ 좋은 거 아니냐" (14)
  - "하지만 우리는 다 알고 있죠 ~ 결코 장점만 있는 제도는 아닙니다" (19)
  - "아이러니하게도 멸종 면한 것도 인간" (23)

### 3-3. 결론부 영역 (7편)

| 편 | 결론부 유형 |
| - | - |
| 01 감자 | 미래 통찰 + 일상 마무리형 |
| 03 한반도 | 애증 통찰 회수 + 일상 마무리형 |
| 04 석면 | 수습 통찰 + 일상 마무리형 |
| 14 까트 | 단속 통찰 + 일상 마무리형 |
| 19 대통령제 | 민주주의 통찰 + 시민 호소형 |
| 23 아보카도 | 개인 영역 + 미래 희망 마무리형 |
| 25 소주 | 광고 마무리형 |

- 결론부 통찰형: 6편 / 광고 마무리: 1편 (25)
- 결론부 처리 영역은 단일 표준 아닌 분기 영역

## 4. 단정 가능 영역 vs 단정 불가 영역

### 4-1. 단정 가능 영역

- 도입부 신뢰 상환 위치: 7편 모두 본문 이연 (01 §116 명시 영역 일치)
- 도입부 약속 100% 해소 사례: 0/7
- 도입부 호명 표현 본문 동일 재호명 사례: 0/7
- 도입부 A극 호명 또는 A극+B극 양면 호명 사례: 7/7 (모순형 특유)

### 4-2. 단정 불가 영역

- "100% 해소 회피"가 의도된 채널 규칙인지, 자연 작문 결과인지 단정 불가
- "본문 동일 재호명 0건"이 회피 규칙 존재 증거인지, 자연 결과인지 단정 불가
- 결론부 회수 6/7 분기 — 작가 자율 영역

### 4-3. 새 패턴 발견 영역

- 본 분석에서 새 패턴 발견: 없음
- 기존 지침 재확인: 1건 (01 §95~116 B극 노출 금지·A극 허용·본문 이연 영역)

## 5. 과정추적형(8편) + 이미지강화형(10편) + 모순형(7편) = 25편 종합

### 25편 누적 사실 (단정 가능 영역)

- 25편 모두 도입부 100% 해소 회피 (25/25)
- 25편 모두 본문 동일 재호명 없음 (25/25)
- 결론부 회수: 약 20편 / 회수 약함·없음: 약 5편

| 결론 직접 진술 | 결과 제시형 4편 | 결과 제시형 3편 | 0편 |
| 부분 답·결과 사실·사전 정보 | 조각 제시형 4편 | 조각 제시형 7편 | 1편 (01 A극 사실) |
| A극 + B극 양면 호명 | 0편 | 0편 | 모순형 6편 |

→ **모순형은 다른 유형과 다른 도입부 정보량 영역** — A극·B극 양면 호명이 모순형 특유

### 25편 누적 사실 (단정 불가 영역)

- 25편 일관성이 의도된 채널 규칙인지 자연 작문 결과인지 단정 불가
- 세 유형 동일 영역 강한 일관성 — 채널 공통 패턴 가능성 강화

## 6. 정비 후보 영역 (분석 결과 도출)

### 정비 후보 — 부재

**점검 결과:**

- **01 §95~116:** 이미 B극 노출 금지·A극 허용·본문 이연 영역 명시
- **02 §93 (이미 정비):** 도입부 정보량 깊이 명시 영역
- **03 §106 (이미 정비):** 도입부 정보량 깊이 명시 영역
- **P3 §83~87:** 3유형 영역 표현이 다름 (모순형 본문 이연 vs 과정추적·이미지강화 도입부 수행). 통합 정비 부담 큼

## 7. 결론

본 분석은 패턴 강제 없이 사실 기술만 수행했다.

- **새 패턴 발견:** 없음
- **기존 지침 재확인:** 1건 (01 §95~116 B극 노출 금지·A극 허용·본문 이연 영역)
- **7편 공통 사실 (단정 가능):** 도입부 100% 해소 회피 / 본문 동일 재호명 회피 / A극(또는 A극+B극) 양면 호명
- **결론부 분기 (단정 불가):** 통찰형 6편 / 광고 마무리 1편 — 작가 자율 영역
- **모순형 특유 영역:** A극 + B극 양면 호명 + 본문 이연 — 다른 두 유형과 다른 영역
- **25편 통합:** 3유형 모두 도입부 100% 해소 회피·본문 동일 재호명 회피 일관

추가 정비 영역은 부재. 02 §93·03 §106 정비 영역으로 일관성 확보 완료. 01 §95~116은 기존 명시 영역으로 적정.
