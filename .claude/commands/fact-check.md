# /fact-check — 팩트 검증 스킬

대본의 수치·고유명사·날짜를 팩트리포트와 기계적으로 대조한다.

## 실행 방법

팩트검증 에이전트(`.claude/agents/팩트검증.md`)를 서브 에이전트로 호출한다.

## 실행 절차

1. 현재 작업 중인 대본 파일 경로를 확인한다
2. 같은 폴더의 `[팩트리포트] 주제명.md` 경로를 확인한다
3. 팩트검증 에이전트를 Agent tool로 호출한다:

```
Agent({
  description: "팩트 검증",
  prompt: "팩트 검증 에이전트로서 아래 두 파일을 대조하라.\n- 대본: {대본 경로}\n- 팩트리포트: {팩트리포트 경로}\n\n.claude/agents/팩트검증.md 지침을 따라 대본의 모든 수치·고유명사·날짜를 팩트리포트와 1:1 대조하고, [열화]/[출처 미확인]/[서술 위반] 결과를 반환하라."
})
```

4. 결과를 유저에게 보고한다
5. [열화] 항목은 즉시 수정을 제안한다
6. [출처 미확인] 항목은 출처 확인 여부를 유저에게 질문한다
