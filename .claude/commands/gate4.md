# /gate4 — GATE 4 자동 검증 스킬

대본 완성 후 GATE 4 체크리스트를 자동으로 검증한다.

## 실행 절차

### 1단계: 자동 검증 (기계적 측정)

현재 작업 중인 대본 파일을 읽고 아래 항목을 자동으로 측정한다:

| # | 항목 | 측정 방법 | 기준 |
|---|---|---|---|
| G1 | 글자 수 | 공백 포함/제외 카운트 | 3,500~6,000자 |
| G2 | 속어·구어 감탄 횟수 | "미쳤", "대박", "레전드", "걸작", 감탄사+판단 패턴 카운트 | 6~10회 |
| G3 | "우리/한국/여러분" 위치 | 해당 단어의 대본 내 위치(전반/중반/후반) 확인 | 후반부 |
| G4 | SL 상태 태그 | `<!-- [상태] SL` 패턴 카운트 | 모든 SL에 존재 |
| G5 | 박스 이미지 | 상태 태그의 "아하모먼트" 항목 카운트 | 최소 3개 |
| G6 | 시청자 대입 | "여러분", "생각해 보세요", "해 보신 적" 등 패턴 검색 | 최소 1회 |

### 2단계: 수동 검증 안내 (구조적 판단)

자동 측정이 불가능한 항목을 질문 형태로 출력한다:

```
[수동 확인 필요]
- 도입부 2분 내 ML이 열려 있습니까? (대본 첫 300~550자 내 질문형 ML 존재 여부)
- SL 열림 유형이 교차 배치되었습니까? (같은 유형 연속 2회 이하)
- 2분+ SL 없는 구간이 없습니까?
- 반전 질문이 레이어 전환 구간 또는 정보 밀도 피크 직후에 배치되었습니까?
- 2분 이상 이완 장치 없는 구간이 없습니까?
```

### 3단계: 팩트 검증 연계

자동 검증 완료 후 `/fact-check` 실행 여부를 유저에게 질문한다.
