# Phase 6: 퇴고 + 최종 검증

> **로드 시점:** GATE 5 통과 후
> **입력:** Phase 5 완성 대본
> **완료 조건:** 아래 A~E 전 항목 통과

---

## A. 구조 검증 — 공통 (모든 서사 유형)

| # | 점검 항목 |
|---|---|
| A1 | 핵심 역설/가치/서사가 한 문장으로 명확한가? |
| A2 | 썸네일-제목이 영상의 핵심 메시지를 각각 담고 있는가? |
| A3 | 도입부가 시청자-주제 관계에 맞는 전략을 사용하는가? |
| A4 | ML이 영상 끝까지 유효한가? (중간에 답이 나와버리지 않는가) |
| A4-1 | ML이 약속한 질문에 대한 답이 대본에 있는가? 답이 ML의 전제와 일치하는가? |
| A5 | 모든 SL이 해소되었는가? 열어놓고 안 닫은 루프 없는가? |
| A6 | SL 순서가 전제 건설 체인을 따르는가? |
| A9 | 엔딩이 클리셰("이상 알아보았습니다") 아닌가? |
| A11 | 엔딩 정보가 ML 해소/일상연결/회상트리거 중 하나에 기여하는가? |
| A12 | SL 간 전제 건설 체인에 논리 빈틈이 없는가? |
| A13 | 박스 이미지(강)와 뒷받침(약) 사이에 인과 비약이 없는가? |

## 할루시네이션 방지 — 퇴고 전용

- 퇴고 시 "통과"를 위해 실제로 고치지 않고 체크만 하는 행위 금지
- 문제를 발견하면 반드시 수정 후 해당 항목 재검증
- "이 정도면 괜찮다"는 판단 금지 — 기준에 미달하면 수정
