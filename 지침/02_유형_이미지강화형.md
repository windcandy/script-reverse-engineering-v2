## P1. 썸네일

### P1.1 이미지강화형 10편 썸네일 분류 결과

이미지강화형 썸네일 전략은 **조각 제시형이 주력**이고, **결과 제시형**이 부분 사용된다.

| 대본 | 썸네일 문구 | 판정 | 근거 |
|---|---|---|---|
| 02 입식vs좌식 | "한중일은 왜 바닥에서 식사할까?" | 조각 제시형 | `왜 ~?` 질문 |
| 05 락스 | "인류는 락스로 수명을 늘렸다" | 결과 제시형 | `~늘렸다` 단정 |
| 07 옥수수 | "당신은 매일 옥수수를 먹고 있다" | 조각 제시형 | 조각 던지기 (의외성) |
| 08 서울 | "서울은 뚫리지 않는다" | 결과 제시형 | `~않는다` 단정 |
| 09 여권 | "한국 여권이 왜 전세계 프리패스가 됐나?" | 조각 제시형 | `왜 ~?` 질문 |
| 10 종교와 고기 | "이슬람이 돼지고기를 금지한 진짜 이유" | 조각 제시형 | `이유` 부품 |
| 11 에어컨 | "인류사는 에어컨 이전과 이후로 나뉜다" | 결과 제시형 | `~나뉜다` 단정 |
| 17 서안해양성 | "왜 서유럽만 과학혁명을 주도했나?" | 조각 제시형 | `왜 ~?` 질문 |
| 20 해수면 | "어떤 나라가 제일 먼저 잠길까?" | 조각 제시형 | `어떤 ~?` 질문 |
| 24 전기차 | "한국 자동차의 진짜 미래는?" | 조각 제시형 | `~는?` 질문 |

#### 이미지강화형 썸네일 특징

- **조각 제시형 주력:** "왜/어떤/이유" 질문형
- **결과 제시형은 "이미 증명된 지위"가 압도적일 때만:** 05 락스·08 서울·11 에어컨처럼 결과 선언이 자연스러운 대상
- **모순 공존형은 사용 안 함** — 이미지강화형은 역설이 아닌 스케일이 본질

판정 기준 세부는 `00_공통_원칙.md` A.1 참조.

#### 이미지강화형 제목·썸네일 이문호공 요인 우위 (10편 실측)

| 위치 | 1순위 | 2순위 | 비고 |
|---|---|---|---|
| **제목** | 상식파괴(5)·이익제시(1) | 호기심(3) | "왜/어떻게" 질문형 + 스케일 강조 |
| **썸네일** | **이익제시(1)** | 비교(8) | 통찰적 우월감 직접 제공 |

**고유 패턴:**

- **이익제시(1)** — 이미지강화형 고유 우세 요인. "이걸 알면 이득" 신호로 시청자에게 통찰적 우월감 즉시 제공 (예: 05 락스 "인류를 지키고 있는 물질", 08 서울 "절대 뚫리지 않는다")
- **요인 분담 권장:** 제목(호기심·상식파괴)·썸네일(이익제시·비교) 분담으로 "왜?" 진입 + "이만큼 대단해" 선언 이중 후킹

## P3. 구조 설계

### P3 §1. 도입부 필수 2요소 + 전략 6종

이미지강화형 도입부는 **2가지 필수 요소**를 반드시 포함한다. 순서·흐름은 전략(A·B·C·E·G·H)이 결정한다. 구조를 별도 명세하지 않고 전략 선택만으로 도입부를 구성한다.

**필수 2요소:**

- ① **썸네일 약속 재확인** — 썸네일 문구를 동의어 치환으로 재확인
- ② **ML (질문형)** — 질문 1~2 + 시그니처 선언 (필수, §2 참조)

**필수 2요소 위치 원칙:**

- **ML:** 도입부 마지막 고정 (본문 진입 다리)
- **썸네일 약속 재확인:** 전략 앞·전략 안·전략 뒤 모두 허용 (위치 자유)

**사용 전략 (6종):**

| 전략 | 핵심 |
| --- | --- |
| **A 규모폭로** | 압도적 수치·스케일로 거대함을 즉시 증명 |
| **B 공감적이질감** | 당연히 여기는 것에 이질감 → 균열 생성 |
| **C 자조적체감** | 대상의 결함·단점을 자조적으로 먼저 제시 |
| **E 밈진입** | 밈·패러디·농담으로 가볍게 진입 |
| **G 현재불안소환** | 현재 진행형 위기·불안을 소환 |
| **H 감정직접소환** | 감정·분노·감사를 직접 소환 |

**주력 전략:** B 1순위 + A 2순위. 나머지 4종은 주제 성격에 따라 선택.

**사용 금지 전략 (이미지강화형 한정):**

- F 맥락재설정 — 안전 미확보

**신뢰 상환 위치:** **도입부 수행** (원칙). 스케일을 먼저 보여줘야 메커니즘이 궁금해짐. 증명 지연 시 "과장 아니냐" 의심 발생.

**도입부 정보량 깊이 (도입부 100% 해소 회피):**

도입부에서 썸네일 약속을 완전히 해소하지 않는다. 약속의 일부분만 도입부에서 노출하고, 본격 답·구체 과정·메커니즘은 본문에서 풀이한다. 도입부 노출은 다음 4분류 중 1종을 선택한다.

- **결론 노출형:** 도입부에 약속의 결과·결말 사실을 단정 선언으로 노출. 본격 메커니즘은 본문에서 풀이.
  - 도입부 노출 예: "사실 락스는 거시적으로 보았을 때 오히려 사람을 살리는 물질"
  - 도입부 노출 예: "휴전 이후 서울은 뼛속부터 전쟁을 위해 만들어진 요새 도시로 재건된 상태"
  - 도입부 노출 예: "에어컨은 개개인의 삶을 넘어 인류사의 향방을 바꾼 엄청난 발명품"

- **결과 사실 노출형:** 도입부에 약속과 관련된 현재 상태·통계 사실만 노출. 그렇게 된 메커니즘은 본문에서 풀이.
  - 도입부 노출 예: "대한민국 여권 ... 전 세계 226개국 중 7위 ... 193개국 무비자"

- **부분 답 노출형:** 도입부에 약속의 부분 답·단서만 노출. 본격 답은 본문에서 풀이.
  - 도입부 노출 예: "옥수수 이거 일단 사람 먹으려고 재배하는 건 아니라는 걸 미루어 짐작"
  - 도입부 노출 예: "어디까지가 더럽고 어디까지가 깨끗해야 하는지에 대한 관념 자체가 다르다"
  - 도입부 노출 예: "이 동네가 실제로 에어컨을 안 틀어도 살만하기 때문 ... 서한 해양성 기후 덕분"

- **일상 사실 노출형:** 도입부에 약속과 관련된 일상 영역·사전 정보의 사실만 노출. 약속 단어 직접 호명 없이 본문 진입.
  - 도입부 노출 예: "대한민국은 ... 자동차는 진짜로 세계 최고 수준으로 잘 만든다 ... 자동차의 미래는 누가 뭐래도 전기차"
  - 도입부 노출 예: "최근 100년간 지구의 기온이 유 없이 상승 중인만큼 해수면 따라서 높아지고 있는 상황"

**원칙:** 어떤 분류를 선택하든, 도입부 노출 정보만으로 약속의 본격 답이 풀이되어선 안 된다. 시청자가 "왜 그런가?" "어떻게 그렇게 됐나?"라는 호기심으로 본문 진입하도록 유도한다.

**예외: 본문 이연 편 (02 입식vs좌식·20 해수면):**

- **02 입식vs좌식** (B 공감적이질감): 수평 확장형 서사로 도입부에선 공감 진입(이질감)만 제시, 본문 How 레이어 진입 즉시 본문 이연 신뢰 상환 시작
- **20 해수면** (G 현재불안소환): 원인 설명 후 본문에서 지역별 Scale Jump로 증명
- 세부 상환 위치는 주제·전략에 맞는 자연스러운 위치 선택

**도입부 글자수 (공백 포함):** 300~750자. 상한 초과는 드물게 허용.

**예시 참조:** 전략별 예시는 `00_지식파일/인트로_예시_전략별.md` 해당 절 참조 — A 절(07·09)·B 절(02·24)·C 절(17)·E 절(05)·G 절(20)·H 절(11).

**도입부 공통 골격(인트로·ML 2블록 + 신뢰 상환 기능)은 `00_공통_원칙.md` B.4 참조.**

### P3 §2. 이미지강화형 ML 구조 + 4단계 구현

```
ML = 질문 1~2 + 시그니처 선언 (필수)
```

**질문 수 선택:** 주제 내용에 따라 1~2개 자연 선택.

- **질문 2개**: 단일 대상의 두 측면을 탐구할 때 (예: 07 옥수수 "어디다 쓰려고? 왜 하필 옥수수?")
- **질문 1개**: 영상 전체를 한 궁금증으로 관통할 때 (예: 11 에어컨 "파급력 어느 정도?")

**한 문장 결합 우선** (한 문장에 자연 결합 가능하면 결합 채택, 강제 분리 금지).

**이문호공 배치:**

- **필수 2요인:** 호기심(3) + 상식파괴(5)
- **추가 권장:** 1요인 이상 (총 3요인 이상 권장)
- 자연스러운 궁금증 유발이 요인 카운팅보다 우선 (공통 §B.5)

**시그니처 선언:** `00_공통_원칙.md` B.5 참조.

### P3 §3. ML 선언부 위치

이미지강화형은 ML 선언부에서 단일 사물/개체 주제일 때 **지위/인격 부여** 역할로 은유를 활용한다.

**선언부 은유 작성 원칙:**

- 대상에 **별명·지위 부여** (예: "인류의 구원자 에어컨")
- 강화문구 없이 은유 단독 (공식: `핵심 대상 + 은유`)
- ML 질문부 뒤, "~알아봅니다" 직전에 배치

**예시:**

- 11 에어컨: "인류의 구원자 에어컨"
- 09 여권: "지구 프리패스 이용권"
- 07 옥수수: "인류를 먹여 살리는 작물"
- 08 서울: "가려진 이면"

**사용 금지 조건:** 현상·시스템·비교 구조(02 입식vs좌식·05 락스 원리)에서는 은유 삽입 불가. `00_공통_원칙.md` B.3 단일 대상 조건 참조.

### P3 §4. 본문 구조 (What → How → Scale Jump → Why)

```
본문 = [What 기초레이어] → [How 메커니즘] → [Scale Jump 단계적 확대] → [Why 통찰]
```

| 레이어 | 역할 | 분량 목표 |
|---|---|---|
| What | 대상의 정의·기초 정보 (2갈래 분기 — 아래 박스 참조) | 본문의 15~20% |
| How | 숨겨진 메커니즘·작동 원리 | 본문의 20~30% |
| Scale Jump | 파급력 단계적 확대 (1~5회) | 본문의 40~50% |
| Why | 통찰·의미·문명적 위상 | 본문의 10~15% |

#### What 2갈래 분기 (대상 성격별)

What은 대상 성격에 따라 진입 방식이 두 갈래로 분기한다.

| 갈래 | 적용 대상 | 진입 방식 | 실측 편 |
|---|---|---|---|
| **시간축형 What** | 사물·기술·문화관습 | 기원·역사 진입 ("언제 시작? 어떻게 보급?") | 5편 (05 락스·07 옥수수·10 종교와 고기·11 에어컨·24 전기차) |
| **정의형 What** | 시스템·현상·구조 | 정체·시스템 분석 진입 ("이 현상의 정체는? 어떤 구조?") | 2편 (02 입식vs좌식·17 서안해양성) |

부분형(08 서울·09 여권·20 해수면)은 두 진입을 혼합 사용.

**선택 지표 (2단계 진단):**

**1단계 — 대상 성격 진단:**

1. 사물·기술·문화관습(먹을 수 있는 것·만들 수 있는 것·사람이 익혀 온 것) → **시간축형 What**
2. 시스템·현상·구조(자연·사회 메커니즘·다국 비교 구조) → **정의형 What**

**2단계 — 본문 진입 측정 (1단계 모호 시 또는 검증 시):**

본문 첫 1~2분(또는 What 단계 첫 1~2 문단)이 어떻게 시작하는지로 최종 판정한다.

| 시작 방식 | 진단 |
|---|---|
| 시대 표기(연도·시대명·"오랜 옛날"·"선사시대 때부터" 류 시간 부사)로 시작 | **시간축형 What** |
| 분류·구조 명사 또는 메커니즘 분석으로 시작(시대 표기 없음) | **정의형 What** |

**경계 사례 가이드 (정의 진입 + 시간축 진입 혼합 시):**

본문 첫 1~2분에서 두 진입이 섞이면 **핵심 비중 50%+ 우세 진입**으로 분류한다.

- 05 락스 → **시간축형 우세** (정의 + 시간축 혼합)
- 10 종교와 고기 → **시간축형 우세** (시간축 + 정의 혼합)

부분형(08·09·20)은 두 진입 비중이 50:50에 가까워 단일 분류 어려움 — "부분형" 유지하되 우세 비중 명시.

**`P3_구조설계.md` §2 본문 첫 레이어와의 관계:** 시간축형 What은 `P3_구조설계.md` §2 "기원·배경 도입"과 자연 일치. 정의형 What은 (다) "기원·배경 부재"에 해당 (02·17). 부분형은 (가) 또는 (나)에 분포 — 09는 (가), 08·20은 (나).

**핵심 동력:** How의 메커니즘이 Scale Jump마다 새 분야에 적용되는 **"범용 열쇠"** 구조 (P3 §6 참조).

### P3 §5. Scale Jump 변주 7유형 + 횟수·선택 기준

#### Scale Jump 변주 7유형

| 변주 | 명칭 | 정의 | 해당 편 |
|---|---|---|---|
| 1 | 지침 충실형 | What→How→Scale Jump(1회)→Why 단선 구조 | 09 여권 |
| 2 | 단계적 스케일업 | Scale Jump가 2~5회 단계적으로 발생 | 05 락스(2회), 11 에어컨(3회), 17 서안해양성(5회) |
| 3 | 파동형 | 장점→문제 제기→해결→더 큰 장점의 파동 구조 | 07 옥수수 |
| 4 | 누적 스케일업 | 동일 방향을 겹겹이 쌓는 구조. "끝이냐? 아니다" 반복 | 08 서울(4회) |
| 5 | 수평 확장형 | 같은 원리가 다른 문화권/지역에서도 작동함을 수평으로 보여줌 | 02 입식vs좌식, 20 해수면 상승 |
| 6 | 복수 대상 병렬형 | 단일 대상이 아닌 복수 대상을 병렬로 다루되 공통 메커니즘으로 묶음 | 10 종교와 고기 |
| 7 | 시간 역행형 | "사실 더 먼저 발명됐다"로 시간을 거슬러 올라간 뒤 현재로 복귀 | 24 전기차 |

#### Scale Jump 횟수 선정 지표

| 지표 | 횟수 선정 |
|---|---|
| **대상의 파급 분야 수** | 분야 1개 → 1회 / 2~3개 → 2~3회 / 4개 이상 → 4~5회 |
| **영상 길이** | 10분 내외 → 2~3회 / 12~15분 → 3~4회 / 15분+ → 4~5회 |
| **서사 리듬** | 빠른 전개 → 적게 (1~2회) / 차분한 전개 → 많게 (4~5회) |

**복수 지표 조합:** 3개 지표를 곱해 평균값 주변에서 결정. 예: 분야 4개 + 15분 + 차분한 전개 = 4~5회.

#### Scale Jump 변주 선택 가이드

주제의 파급 방식을 아래 질문으로 진단해 변주를 선택한다.

| 진단 질문 | Yes → 선택 변주 |
|---|---|
| 파급이 하나의 핵심 메커니즘으로 1회 증명되는가? | **1 지침 충실형** |
| 대상의 파급이 시간순으로 점진적 확대되는가? | **2 단계적 스케일업** |
| 대상이 장점·문제·해결 파동을 보이는가? | **3 파동형** |
| 대상의 파급이 동일 방향으로 겹겹이 쌓이는가? | **4 누적 스케일업** |
| 같은 원리가 다른 문화/지역에서 반복되는가? | **5 수평 확장형** |
| 복수 대상이 공통 메커니즘으로 묶이는가? | **6 복수 대상 병렬형** |
| 시간 역전("사실 더 먼저") 반전이 자연스러운가? | **7 시간 역행형** |

### P3 §6. How 메커니즘 = 범용 열쇠

How에서 공개한 메커니즘이 Scale Jump마다 새 분야에 반복 적용되어, 시청자가 "이것도 같은 원리 아니야?"를 능동적으로 유추하게 만든다.

**작동 원리:**

- What에서 대상 정의
- How에서 **핵심 메커니즘 공개** (단일 원리)
- Scale Jump마다 그 원리가 새 분야에 적용됨을 증명
- 시청자는 "범용 열쇠"를 쥐고 매 Jump를 예측·확인

**예시:**

- 05 락스: "차아염소산의 산화 작용"(How) → 표백(적용1) → 냄새 제거(적용2) → 세균 살균(적용3)
- 17 서안해양성: "1년 내내 일정한 강수"(How) → 농업(적용1) → 하천 무역(적용2) → 수력 방적기(적용3)

**핵심 체크:** How가 Scale Jump와 연결 고리를 제공하는가? 연결 고리가 약하면 스케일이 "우연한 나열"로 인식됨.

### P3 §7. Aha-moment = 스케일 발견

이미지강화형의 Aha-moment는 **"이것까지 영향이 미쳤다고?"**의 스케일 발견이다.

**위치 원칙:** % 강제 아님. 아래 **완결 조건이 충족되는 지점**에서 아하 자연 발생.

#### 이미지강화형 아하 완결 조건 체크리스트

아래 4조건 모두 충족 시 아하 배치:

1. **Scale Jump 누적:** Scale Jump가 최소 2회 이상 누적되었는가?
2. **저지선 돌파:** 시청자 기존 이미지의 마지막 저지선(일상·상식·경험)이 뚫렸는가?
3. **구체 연결 제공:** "이것까지?" 반응을 유발할 **구체 연결**(일상 사례·지위 명명·본질 대비)이 제공되었는가?
4. **변주 완결 장치:** 아래 6변주 중 하나가 완결 작동하는가?

#### 이미지강화형 6변주 — 작동 원리·사용 조건

| 변주 | 대표편 | 작동 원리 (한 줄) | 사용 조건 (한 줄) |
|---|---|---|---|
| ① 본질 통합 재정의 | 02·10·17 | 다원 사례·표면 차이를 단일 본질로 통합 압축 | 여러 사례가 단일 본질로 관통되는 주제일 때 |
| ② 지위 재정의 | 05·09 | 만만한 대상을 "양대 보루"·"극한 가성비" 등으로 지위 격상 | 시청자가 만만하게 여기는 대상에 숨은 필수성이 있을 때 |
| ③ 일상 침투 발견 | 07·11 | 원거리 스케일을 개인 일상으로 수렴 | 원거리 스케일이 시청자의 일상 행위와 직결될 때 |
| ④ 정점 비유 압축 | 08 | 누적 Scale Jump 후 최종 비유로 총합 각인 | 여러 Scale Jump를 한 비유로 압축 가능한 단일 대상 주제 |
| ⑤ Scale Jump 수평 누적 | 20 | 개별 피크 없이 반복 누적 자체가 총합 충격 | 여러 지역·사례에 동일 현상이 반복 적용될 때 |
| ⑥ 시간 역전 폭로 | 24 | 시청자 상식을 역사적 사실로 전복 | 시청자의 "새 기술·현상" 상식을 뒤집을 과거 팩트가 있을 때 |

> **유연성:** 본문 Scale Jump 수·층위·시간축에 따라 아하 지점은 주제마다 다르다. **24 전기차(초반 시간 역전)**, **20 해수면(단일 피크 부재·수평 누적)** 같이 후반 단일 피크가 아닌 경우도 유효.

**예시:**

- 05 락스: "질소 비료와 함께 현대 인류 문명을 지탱하는 양대 보루" (지위 재정의)
- 07 옥수수: "온 세상이 옥수수" — 매일 간접적으로 먹고 있다 (일상 침투 발견)
- 11 에어컨: "유튜브도 에어컨 없으면 존재 불가" (일상 침투 발견)
- 17 서안해양성: "자연 극복 vs 이용을 고민" — 문명 방향 결정 (본질 통합 재정의)
- 24 전기차: "1881년 이미 발명" — 시청자 상식 역전 (시간 역전 폭로, 초반 배치)

**설계 체크:** Aha가 단순 "스케일 수치"가 아니라 **"보이지 않던 연결"을 가시화**하는가?

### P3 §8. 결론부 문제 제기 + 부가 정보 허용 조건

이미지강화형의 결론부 고유 패턴 3종.

#### 고유 패턴

| 패턴 | 정의 | 대표 편 |
|---|---|---|
| **문제 제기** | 본문 장점 극대화 후 결론부 직전 한계/문제 제시 → 미래희망으로 착지 | 11 에어컨 (온난화 악순환), 17 서안해양성 (기후 위기) |
| **부가 정보 삽입** | ML 해소 무관 "관련 상식"을 결론부에 삽입 (보너스 정보) | 09 여권 (비자·역사), 07 옥수수 (GMO), 20 해수면 (오셔닉스) |
| **미래희망 엔딩** | "앞으로도 계속된다"로 착지 (주력) | 08·09·10·11·20·24 |

#### 허용 조건 + 비중 상한

| 패턴 | 허용 조건 | 비중 상한 |
|---|---|---|
| 문제 제기 | 본문 장점이 극단 긍정이고 5분+ 쌓였을 때 + 문제가 대상의 **미래 관련 이슈**일 때 | 결론부 전체의 **10% 이내** (짧게) |
| 부가 정보 삽입 | ML이 **본문에서 완전히 해소된 후**(신규 정보 아닌 "관련 상식") + "보너스" 성격이 명확할 때 | 결론부 전체의 **20% 이내** |
| 미래희망 엔딩 | 모든 이미지강화형 주제에서 자연 허용 | 제한 없음 |

**체크:** 문제 제기 + 부가 정보 합계가 결론부의 30%를 넘으면 **"왜 안 끝나지?"** 감이 발생. 재조정 검토.

### P3 §9. 미래희망형 엔딩 궁합 + A극/How 편차

#### 본문 결산 양식 4종

본문 마지막 단락(= 결론부 직전 단락)의 마무리 양식.

| 결산 양식 | 정의 |
|---|---|
| **A. 통찰 압축형** | 한 문장 통찰로 본문 결산 (예: "양대 보루"·"안전수칙=신앙"·"100년 200년") |

| **C. 단점→희망/시의성 결합형** | 본문 장점 극대화 후 한계·문제 제시 + 미래 희망 또는 시의성 |
| **D. 시의성형** | 현재 시의성 사건과 결합 결산 |

#### 본문 결산 양식 → 결론 브릿지 어법 자연 연동

| 결산 양식 | 결론 브릿지 어법 |
|---|---|
| A. 통찰 압축형 | **① 결과 호명** 또는 **④ 회수**를 사용한다 (양쪽 균형). |

| C. 단점→희망/시의성 결합형 | ① + ② + ④ + ⑤ 다중 결합 사용한다. |
| D. 시의성형 | ③ 시간 전환 + ④ 회수를 결합 사용한다. |

이미지강화형 결론부에서 본문 통찰을 다음 차원 중 하나로 격상시킨다.

- **미래 차원**: 미래 전망·예측·전환 영역
- **시스템 차원**: 사회·기술·자연 시스템 영역

### P4 §1. 스케일 수치 즉시 체감 + 유형별 B 전략 기능

#### 스케일 수치 즉시 체감

도입부 첫 10초 이내에 대상의 스케일을 **수치로 증명**하여 시청자가 즉각 "대단하다"를 체감하게 한다.

**예시:**

- 07 옥수수: "3억8천만 톤" (첫 10초)
- 09 여권: "193개국 무비자"
- 11 에어컨: "현재 세계 전력의 20%"
- 17 서안해양성: "1년 내내 일정한 강수"

**수치 선택 기준:**

- **압도적** (일반 상식의 10배 이상)
- **체감 가능** (비교 대상이 시청자에게 익숙)
- **1개로 충분** (2~3개 나열 금지, 오히려 체감 저하)

#### 유형별 B 전략 기능 — 이미지강화형

B 전략(공감적이질감)은 3유형 공통이지만 유형별 기능이 다르다. 이미지강화형에서 B 전략의 기능은:

| 기능 | 이미지강화형 |
|---|---|
| **공감 진입** | 시청자 일상 경험에서 출발 (예: 17 서안해양성 "안 틀어도 살만") — **출발 단위·동료 호명 톤은 `00_공통_원칙.md` B 전략 공감 진입 출발 단위 참조 (3유형 공통).** |
| **대단함 세팅** | 시청자가 당연히 여기는 것이 실은 대단함을 암시 (예: 02 입식vs좌식 "신발 신고 침대") |
| **방향성** | 이질감 → 스케일 탐구 (모순형의 "이질감 → 반전"과 다름) |

**체크:** B 전략 사용 시 "이질감 → 반전"이 아닌 "이질감 → 스케일 암시"로 연결되어야 이미지강화형.

### P4 §2. 수치로 대단함 증명 (간접 확인 기법)

썸네일 선언의 "대단함"을 수치로 입증하여 "왜만 남기는" 기법.

- **작동 방식:** 스케일 수치 → 시청자가 "왜?"만 궁금해하는 상태 → 본문에서 메커니즘·적용 탐구
- **예시:** 09 여권 "193개국 무비자" → "왜 그렇게 됐을까" 남김

`00_공통_원칙.md` C.4 간접 확인 공통 원리 참조.

### P4 §3. 역방향 진입 (Scale Jump 예고)

썸네일과 **반대 방향에서 도입부를 시작**하여 시청자의 예상을 비튼다. 본 기법은 공통원칙 `00_공통_원칙.md` §C.4 (다) "반대 방향 진입"을 이미지강화형 Scale Jump 예고 변주로 적용한 것이다. 본 기법은 주력 아님.

**작동 방식:**

- 썸네일이 결과 선언("뚫리지 않는다") → 도입부는 위험·약점에서 시작
- 시청자: "어? 위험하네?" → 본문: "사실 안전한 이유" → Scale Jump로 이어짐

**예시:**

- 08 서울: 썸네일 "뚫리지 않는다" → 도입부 "북한 40km 위험" 먼저 → 본문에서 방어 스케일업
- 05 락스: 썸네일 "수명을 늘렸다" → 도입부 "독극물 이미지" → 본문에서 긍정 효과 Scale Jump

### P4 §4. 대단함 세팅 방향 (혼합 유형별부)

이미지강화형의 간접 확인에서 **"대단함"을 세팅하는 방향성**을 유지한다.

**원칙:**

- **동의어 치환** (예: 05 락스 "지키고 있는" → "살리는") — `00_공통_원칙.md` §C.4 (가) 정합
- **속성 대체** (예: 08 서울 "뚫리지 않는다" → "지하철 83m 깊이로 이미 증명") — `00_공통_원칙.md` §C.4 (나) 정합
- **수치 직접 제시** (예: 09 여권 "193개국") — `00_공통_원칙.md` §C.4 약(단어) 허용 케이스. 핵심 결론어가 우회되면 보조 수치·단어의 직접 제시는 허용

**모순형과의 차이:**

- 모순형: "이질감 → 반전" (썸네일 A극을 깨고 B극으로)
- **이미지강화형: "이질감 → 대단함 세팅"** (썸네일 지위를 다른 각도로 재확인)

### P4 §5. 긍정 피로 방지 4전략 + 삽입 시점 정량 기준

이미지강화형은 **장점만 쌓일 때 "긍정 피로"**로 텐션이 하락한다. 다음 4전략 중 1개 이상을 반드시 삽입한다.

#### 4전략

| 전략 | 작동 방식 | 예시 |
|---|---|---|
| **1. Scale Jump 직전 "한계/문제" 삽입** | 장점 흐름에 "근데 이게 문제가 있다" 삽입 → 문제 해결 → Scale Jump | 05 락스 "엥? 근데 이걸 왜 소독에 쓰냐?" (미아즈마 설명 → 다음 Scale) / 07 옥수수 "이상한 병을 앓기 시작" (펠라그라 → 닉스타말화 해결) |
| **2. "끝이냐? 아니다" 반복** | 종결 부정형을 반복하여 기대를 계속 확장 | 08 서울 "끝이냐? 아니죠. 본 게임" → "어림도 없지" → "아닙니다. 사실 이제 절반" (4회 반복) |
| **3. 시청자 반박 선제 흡수** | "과장 아니냐?" 류 반박을 먼저 제시 | 07 옥수수 "공정이 더 복잡하면 비효율적인 거 아니냐" / 17 서안해양성 "위도만 따지면 몽골보다 더 북쪽인데?" / 11 에어컨 "이 정도면 고대인들도 좀 쳤네 싶으실 수도 있는데 함정이 있습니다" |
| **4. 새 루프 열기 (예상치 못한 또 다른)** | Scale Jump 진입 시 "예상치 못한 효과"로 새 공백 생성 | 05 락스 "예상치 못한 또 다른 효과가 있네. 냄새가 없어지더라" / 11 에어컨 "아 맞다 중요한 걸 하나 까먹었는데" |

#### 삽입 시점 정량 기준

| 조건 | 삽입 빈도 |
|---|---|
| **Scale Jump 간격이 3분 이상** | Jump 간 1회 필수 (전략 1·4 주력) |
| **장점 수치 3개 이상 연속** | 수치 3개째에 1회 삽입 (전략 3 주력) |
| **누적 스케일업 구조** | 매 Jump마다 전략 2 반복 |
| **Scale Jump 1회만 있는 단선 구조** | 삽입 불필요 (지침 충실형 09 여권) |

**체크:** 4전략 중 최소 1개 이상 사용 확인. 장점만 연속 5분 이상 쌓이면 텐션 하락 경고.

### P4 §6. Scale Jump 전환 브릿지 5유형

Scale Jump 간 전환에 사용하는 브릿지 5유형. 세부는 `00_공통_원칙.md` §C.9 참조.

| 유형 | 예시 | 사용 편 |
|---|---|---|
| "그런데 예상치 못한 효과가" | "그런데 이게 예상치 못한 또 다른 효과가 하나 더 있네"(05) | 05, 11 |
| "끝이냐? 아니다" | "그럼 끝날까요? 아니죠. 이제부터가 본 게임"(08) | 08 |
| 시청자 반박→반전 | "엥? 논리가 왜 이렇게 연결되냐"(05) → 미아즈마 설명 → "엥? 근데 진짜로" | 05, 07 |
| 인과 연쇄 | "이처럼 독특한 진화 전략을 선택한 덕분에" | 10, 17 |
| 문제 제기→해결 | "이상한 병을 앓기 시작"(07) → 원인 규명 → 해결 → 다음 장점 | 07, 11 |

**핵심 원리:** 이미지강화형의 브릿지는 **Scale Jump 간 전환(복수 회)**이며, **"아직 더 있다"의 기대감**으로 관성을 만든다.

### P4 §8. 본문 주력 브릿지 (이미지강화형)

본문 레이어 전환 시 이미지강화형 주력 브릿지 조합. (`00_공통_원칙.md` C.9 브릿지 Type A/B 세부 정의 참조)

#### 주력 서브 오픈 루프 (Type A-1)

- **인과관계 추적형** (최우선): `대체 이거 어디다 쓰려고 이렇게 많이 재배하는 걸까요? 왜 하필 옥수수일까요?` (07) — **이미지강화형 메커니즘 탐구 주력**
- **스케일업**: `사실상 도하 작전이 아닌 상륙 작전` (08), `바닷속에 노다지이자 살아 있는 광맥` — 대상 판·규모를 키워 무게감 부여

#### 주력 SL 선언·예고형 (Type A-2 / 구 심화 브릿지)

- **"예상치 못한 또 다른" 형**: `예상치 못한 또 다른 효과가 하나 더 있네` (05) — **Scale Jump 진입 기본**
- **"마침내" 형**: `마침내 1890년대 클로르 알칼리 공정이 상용화됩니다` (05), `마침내 내연 기관의 독주도 끝나가고 있습니다` (24) — 스케일/변화의 도달
- **"아 맞다" 형**: `아 맞다 중요한 걸 하나 까먹었는데` (11) — 추가 Scale 열기 (비공식 톤)

#### 피벗/정점 유형

- **Scale Jump 진입 = "예상치 못한 또 다른"** 조합 (Type A-2 기본)
