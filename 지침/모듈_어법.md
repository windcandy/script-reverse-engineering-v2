# 모듈_어법

> **본 모듈 역할:** 본 채널 "이거봐봐" 어법 카탈로그 단일 저장소.
> 12 언어장치 + 도입부 예시 + 4단계 말맛 변주 + 4역할 × 어법 결합 매트릭스 + 8엔진별 어법 권장 + 3유형별 어법 결합 권장.
>
> **자체 완결.** 외부 파일 직접 참조 없음.
>
> **호출 시점:** P1(도입부 어법 일부) · P4(전체) · P5(검수 영역).

### 0.2 단위 정의

- **어법** — 화자 발화 방식. 종결 어미·진입 어휘·치환·의인화·호흡·시제·말맛 표현을 모두 포괄.
- **언어장치** — 어법 중 SL 본질·서사 골격에 결합된 도구. §1의 12종이 본체.
- **말맛** — 권위 정보를 비권위 옷으로 옮기는 변환 메커니즘. §3 본체.

### 0.4 차용 원칙 (예문 처리 — 본 모듈 전체 관통 규칙)

본 모듈의 모든 예문은 **참고용 패턴 예시**이다. 그대로 차용하지 말고 **변환 메커니즘**만 흡수하여 매 주제·맥락에 맞춰 새 표현을 창의적으로 생성한다.

- ✅ 패턴 골격(예: `[추상 X] + [구체 Y]` 형식) 흡수
- ✅ 핵심 어휘 1~2개 참고
- ❌ 완성 문장 그대로 복사
- ❌ 동일 표현 다회 반복 사용

> 본 §0.4 차용 원칙은 §2·§3 전체에 일괄 적용. 하위 §에서 재선언 없음. 매듭은 §6.4.

## §1. 12 언어장치 본체

> 본 §1은 본 채널 고유 12 언어장치 카탈로그 본체. 채널 공통 3대 시그니처 + 비-모순형 공통 1종 + 가정 도구 2종 + 유형별 분기 도구 6종 구성.
>
> **3계층 등급:** 🔴 강제(100% 적용) / 🟠 권장(50% 이상 활용) / ⚪ 선택(맥락 부합 시).
>
> **시점 분류:** 🔴 강제 → P4 강제 / 🟠 권장 → P4 권장 또는 P5 전담 / ⚪ 선택 → P5 전담.

### 1.0 채널 화자 페르소나

- **정체성:** 세상 돌아가는 이야기를 가장 재미있게 풀어주는 똑똑한 친구. 관찰자·강사 톤 금지, 적극 논평자.
- **톤 — 냉소+온정:** 사건·세력·시스템에는 냉소, 사람·약자·피해자·시청자에는 온정. 후자 대상 비하 + 무거운 구간(추모·재난) 위트 회피.
- **화법:** 합쇼체 기본(~합니다·~입니다)으로 신뢰감 깔되, 친근·강조·자조·위트 위치에서 해요체(~거든요·~잖아요·~죠)와 반말 펀치(~거든·~지·~인데) 변주 혼합.

**[재미 원칙 — 작성 시점 즉시 발동]**

- **즉시 변환 원칙:** 추상 수치·속성·메커니즘 등장 시 그 자리에서 현대인이 체감할 수 있는 단위·사물·캐릭터로 변환. (예: 720만 달러 → "강남 아파트", 정교함 → "아이폰 설계도")
- **위트 꼬집기 원칙:** 사건·세력·시스템의 어리석음·무지·오판은 위트로 꼬집어 친근하게 풀이.

### 1.1 채널 공통 3대 시그니처 (3유형 강제 — 🔴)

3유형 모두 강제. SL 본질·서사 유형 무관 일관 적용.

#### ① 1인칭 화자 🔴 (P4 강제)

- **정의:** "저는·제가" 우세. 권위 인격 제거.
- **사용 조건:** 모든 화자 발화에 일관 적용. 3인칭·익명 톤 발생 시 1인칭 교체.
- **정량 기준:** 모든 화자 발화에 1인칭 자연 등장 → 자동 충족. 횟수 채우기 강제 삽입 금지.
- **사용 예시 (참고용):** "저는 ~", "제가 ~", "저도 처음엔 ~"
- **금지:** 3인칭 객관화 톤 / "본 영상에서는" 식 화자 비인격화 / "X가 등장합니다·다음 단계는 ~" 같은 사물 주어 SL 진입 톤.

#### ② 속어/밈 🔴 (P4 강제)

- **정의:** 채널 톤 시그니처 표현 (속어·인터넷 슬랭·인터넷 밈 포함, 친근 톤 강화).
- **사용 조건:** 정보 피크 직후·반전 임팩트 직후·자조 위치. 사물/상황만 대상 (인물/집단 비하 금지). 도입부 첫 30초 사용 금지 — 단 도입부 밈이 주제 진입의 핵심 후킹 역할인 경우 예외 허용 (전략 E 밈진입 본질).
- **정량 기준:** 편당 6~10회. 한 단락 안 동일 표현 반복 금지.
- **사용 예시 (참고용):** "개사기", "미친", "찐"
- **금지:** 욕설 수준 표현 금지 / 무거운 구간(추모·재난) 사용 금지.

#### ③ 어려운 용어 풀이 🔴 (P4 강제)

- **정의:** 대중적으로 잘 안 쓰이거나 시청자가 모를법한 용어 등장 시 쉬운 말 대체·간단 설명 동반·장면 묘사형 대체.
- **효과:** 시청자 부담 완화 + 신뢰감 형성.
- **금지:** 어려운 용어 단독 사용 / 어렵지 않은 용어 풀이 동반 (시청자 우롱).

**자가 진단:** "시청자가 정확한 의미·작동 방식까지 즉시 떠올리는가?"

- Yes → 풀이 금지 (일상어·친숙 고유명사)
- No → 풀이 필요 (약자·영문 직역체·추상 용어·학술/전문/정책 권위어)

**결합 방식 3종:**

| 방식 | 사용 기준 | 예시 (참고용) |
| --- | --- | --- |
| ① 쉬운 말 대체 | 일상어로 흔적 없이 교체 가능 | "차관" → "빌린 돈" |
| ② 간단한 설명 동반 | 용어 유지 + 풀이 결합 | "전향력 즉 코리올리 효과" |
| ③ 장면 묘사형 | 개념의 느낌·현실감 전달이 충분할 때 | "감자의 수확 후 가공 용이성" → "흙만 털어내면 바로 조리됩니다" |

**② 결합 어구 (참고용 5종):** 그러니까 / 다시 말해 / 쉽게 말해 / 정확히는 / 말 그대로 — 일반 한국어 부연 어구라면 다른 어구도 허용. 동일 어구 한 단락 2회 또는 편당 3회 이상 시 변주.

### 1.2 비-모순형 공통 도구 (1종)

#### ④ 자문자답 압축형 🟠 (P4 권장 — 이미지강화·과정추적 한정)

- **정의:** 가상 시청자 의문을 호명한 후 물음표 직후 즉답으로 압축. 의문과 답이 한 호흡(짧은 인접 두 문장) 안에서 압축.
- **사용 조건:** 정보 피크 직전·반전 임팩트 직전·전환 브릿지 위치. 전환점 하나당 최대 1회.
- **정량 기준:** 이미지강화·과정추적 편당 1회 이상.
- **사용 예시 (참고용):**
  - "그러면 일본은? 이쪽도 심각성이 만만치 않습니다"
  - "그럼 끝났을까요? 아니죠. 이제부터가 본 게임입니다"
- **효과:** 시청자 의문 즉시 해소 + 호흡 압축 + 임팩트 강화.
- **금지:** **모순형 차용 금지(0회).** 1회 이상 사용 시 SL 골격 어긋남.

### 1.3 가정 도구 (2종)

#### ⑤ 반사실 가정 ⚪ (모순형 사용 금지)

- **정의:** "~이 없었다면" — 추상 가치·인물·발명 영향 강조.
- **사용 조건:** 추상 가치 강조 위치.
- **금지:** **모순형 사용 금지** (현재 폭로 vs 과거 가정 충돌).

#### ⑥ 가정법 시뮬레이션 ⚪ (모순형 추상 제도 한정)

- **정의:** "만약 ~한다면" — 추상 제도·정책·미래 시나리오 위치.
- **사용 조건:** 미래·시뮬레이션 주제에서 미래 단정 시제와 결합 가능 — "~할 예정"·"~될 거"·"~할 가시권".
- **사용 예시 (참고용):** "도심 항공 모빌리티 더 이상 꿈이 아니게" / "에어컨 보급률 90% 넘어가면 ~할 예정"
- **금지:** 모순형의 자연·생물·문화 주제 사용 시 SL 어긋남.

**미사용 시 근거 기술 의무:** 두 도구 모두 미사용 시 P3 조감도에 그 근거를 명시한다 (의식적 결정 보장).

### 1.4 유형별 분기 도구

각 유형별 SL 본질에 맞는 분기 도구. 채널 공통 3대 시그니처 위에 추가 적용.

#### 1.4.1 모순형 분기 도구

- **채널 공통 속어/밈 우세 사용:** 모순형은 속어/밈 빈출 (모순 발견 자조 톤과 결합).
- **시리즈 연결형 예외:** 시리즈 후속편 한정 도입부 2단계 축소 패턴 사용 가능 (자세한 패턴은 §2.5.2).

##### ⑦ 양면 인격 의인화 🔴 (모순형 강제 / P4 강제)

- **정의:** 동일 대상에 상반된 양면 인격 부여 — 모순형 SL "양면 발견" 골격.
- **사용 조건:** 도입부·본문 양면 발견 단계. 양면 모두 동등 강도 부여 (한쪽 약화 금지).
- **사용 예시 (참고용):** "효자 상품 ↔ 악마의 풀" / "착한 과일 ↔ 살인 작물" / "값싼 천한 술 ↔ 고귀한 명문 술"
- **효과:** 모순 진입 셋업 골격.

##### ⑧ 양보 후 반박 🟠 (모순형 권장 / P4 권장)

- **정의:** 시청자 반론을 미리 양보한 뒤 즉시 핵심 반박. 시청자 대입 변형이나 양보 단계 명시가 핵심.
- **사용 조건:** 모순 진입 직전·시청자 반론 예상 위치·결과 공유 셋업 직후. 편당 1~2회.
- **사용 예시 (참고용):**
  - "100번 양보해서 ~ 좋은 거 아니냐라고 하실 수도 있는데요 → 문제는 ~"
  - "~한다고 하실 수도 있는데 → 이런 식으로 가다가 ~"
- **효과:** 시청자 반박 선제 흡수 + 본 모순 강조.
- **금지:** **비-모순형 차용 금지** (이미지·과정 0%).

##### 모순형 ⚪ 선택 도구 (4종 — P5 전담 또는 맥락 적용)

| 도구 | 정의 | 사용 예시 (참고용) |
| --- | --- | --- |
| 결과 공유 셋업 | "결론 알고 보는 우리들" 식 화자·시청자 결과 공유 | "결국엔 이렇게 됐죠" / "사실상 ~ 급이라고 볼 수 있지요" |
| 단속음 반전 | → §3.5.3 본체 위임 (정보 → 반전 단속 리듬) | "커피는 전 세계인이 사랑하는 기호식품이죠 → 사실 ~ 더 강력한 각성제가" |
| 모순 발견 갸우뚱 | 시청자 감정 동조 (갸우뚱 같은 편 호명) | "저도 이거 처음 알았을 때 갸우뚱했어요" / "이상하지 않으세요?" — 편당 2~3회 |
| 출처 인간적 인용 | 권위적 출처를 인간적 톤으로 풀어쓰기 | "어느 학자가 이런 말을 했는데" — 출처 정확성 손상 금지 |

#### 1.4.2 이미지강화형 분기 도구

- **진입:** 사고 실험 초청 + 시청자 일상 동기화.
- **비유 영역:** 단일 인격 강화.

##### ⑨ 단일 인격 강화 의인화 🟠 (이미지강화형 권장 / P4 권장)

- **정의:** 거시 대상에 단일 격상 인격 부여 — 이미지강화 SL "거시 풍경 가이드" 골격. **편 전체 변주 반복 적용** (도입 → 본문 → 결론 회수).
- **사용 조건:** 거시 풍경 가이드 1단계 + 편 전체 변주 반복 위치.
- **정량 기준:** 편당 평균 6~7회 변주 반복 권장.
- **사용 예시 (참고용):** "락스 = 인류 구원자·문명 보루" / "에어컨 = 인류 구원자" / "옥수수 = 인류 부양자"
- **주의:** 비교 분석 주제(예: 입식 vs 좌식·종교 비교)는 단일 주인공 부재로 권장 강도 — 양측 각각 캐릭터화 또는 메타 인격 부여로 변주.

##### 이미지강화형 ⚪ 선택 도구 (3종)

| 도구 | 정의 | 사용 예시 (참고용) |

### 2.0 도입부 구조 라벨 용어 정리 (본 §은 유형별 도입부 구조 본체 정의 영역)

- **모순형 4요소:** ① 상식적 맥락 제시 / ② 썸네일 약속 재확인 / ③ 반전 암시 / ④ ML(메인 루프 질문형)
- **모순형 4가지 구조:** `기본형 [① → ② → ③ → ④]` / `대안형 [② → ① → ③ → ④]` / `압축형 [① → (②+③) → ④]` / `예외형 [① → (스케일+③) → ④]`
- **이미지 강화형 3가지 구조:** `표준 3단` / `권위 인용형` / `미래 제시형`
- **과정 추적형 구조:** `표준 3단 [① → ② → ③]` / `맥락 선행형 4단` / `상식 전복 확장 4단`

### 2.1 전략 B (공감적이질감) — 주력 1순위

> **본질:** 시청자가 당연히 여기는 상식에 이질감을 심는다. 본 채널 도입부 최다 발동.

#### 2.1.1 모순형 — 대안형 패턴

**썸네일 3분류:** 조각 제시형 · **구조:** 대안형 [② → ① → ③ → ④] · **신뢰 상환:** 본문 이연

**패턴 골격:**

> **[② 썸네일 약속 재확인]** [현상·풍경 단정 한 줄] — 시청자 직관 진입
>
> **[① 상식적 맥락 제시]** [이게 ~문제만이 아닙니다 + 시스템 맥락 제시]
>
> **[③ 반전 암시]** [역사적·구조적 결과 한 줄]
>
> **[④ ML(질문형)]** [상식 라벨] 정말 [부정 라벨]일까요? 아니면 [반전 라벨]일까요?

**참고 예시 (1편 압축):** 한반도 — "지평선 축제·평지 귀한 땅" 단정 → "땅 문제만이 아닌 대륙-해양 접점" → "전장 운명·자연재해 집중" → "저주받은 땅일까 축복일까".

#### 2.1.2 모순형 — 기본형 패턴

**구조:** 기본형 [① → ② → ③ → ④] · **신뢰 상환:** 본문 이연

**선택 기준:** 상식 라벨(대비 축)을 먼저 세워야 ②썸네일 재확인이 임팩트 발생 시 → 기본형.

**패턴 골격:**

> **[①]** [상식 정의 한 줄] + [추가 부연]
>
> **[②]** 그런데 [상식 부정 한 줄].
>
> **[③]** [구체 사례 누적 + 일반 결론]
>
> **[④]** 그렇다면 [궁금증 1]? [궁금증 2]?

**참고 예시 (1편 압축):** 대통령제 — "대통령 = 국가 권력 정점" 상식 → "안 그런 나라가 더 많다" 재확인 → "서구권 의원 내각제 기본값" → "대통령제 장점·우리나라 채택 이유" 질문.

#### 2.1.3 모순형 — 압축형 패턴

**구조:** 압축형 [① → (②+③) → ④] — 한 문장 자연 결합 가능 시.

**참고 예시 (1편 압축):** 아보카도 — "착한 과일" 상식 → "그런데 막상 ~ 사람이 죽고 환경까지 박살 나고" 압축 결합 → "언제부터 먹기 시작? 왜 사람을 죽이고 환경을 파괴?".

#### 2.1.4 과정 추적형 — 표준 3단 패턴

**구조:** [현재 결과 제시 → 과거 시간 역행 암시 → ML "어쩌다 이렇게 됐을까?"]

**참고 예시 (1편 압축):** 온돌 — "한국 귀신이 미국 귀신보다 덜 무서운 이유" → "한국 바닥 따뜻 / 외국 차가움" → "언제부터 우리는 바닥을 데워서 살았고 이게 어떻게 한국 문화의 일부가 됐을까요?".

#### 2.1.5 이미지 강화형 — 미래 제시형 패턴

**구조:** [① 공감 진입 → 미래 제시 → ③ ML]

**참고 예시 (1편 압축):** 전기차 — "한국 제조업·자동차 세계 최고" → "그런데 미래는 전기차·EV 시대" → "전기차 2.0 시대에도 한국 산업 건재할까요?".

#### 2.1.6 과정 추적형 — 상식 전복 확장 4단 패턴

**구조:** [상식 전복 → ① 현재 영향력 제시 → ② 시간 역행 암시 → ③ ML]

**참고 예시 (1편 압축):** 가야 — "삼국 시대 → 4국 시대로 불러야 한다 (상식 전복)" → "가야금·김해김씨 (영향력)" → "기록 부족·고분군 (역행 암시)" → "가야는 어떤 나라? 고분군 비밀은?".

### 2.2 전략 A (규모폭로) — 주력 2순위

> **본질:** 스케일 수치로 즉시 증명 가능. 도입부 첫 문장에서 수치 폭로.

#### 2.2.1 모순형 — 예외형 (A 전용) 패턴

**구조:** [① 상식 → (스케일+③ 통합) → ④]

**선택 기준:** 썸네일이 수치로 즉시 증명되므로 ②썸네일 재확인을 스케일 폭로로 대체.

**참고 예시 (1편 압축):** 감자 — "밀·쌀 문명" 상식 → "감자 3억 8천만 톤·주식 4위" 스케일+반전 → "언제부터 먹었나? 밀·쌀과 장단점?".

#### 2.2.2 이미지 강화형 — 표준 3단 패턴 (스케일 충격 진입)

**구조:** [① 스케일 충격 → ② 숨겨진 메커니즘 암시 → ③ ML]

**패턴 골격:**

> **[①]** [수치 1위/연간 N톤·M개국 등 스케일 단정] + [매일 먹고 있다·이미 누리고 있다 같은 시청자 직격]
>
> **[②]** [원리·메커니즘 한 줄 암시]
>
> **[③]** [궁금증 ML]

**참고 예시 (2편 압축):**

- 옥수수 — "3억 8천만 톤·1위" → "C4 식물·테오신테 품종 개량" → "왜 인류 문명 기반? 매일 먹는다는 게 무슨 소리?"

#### 2.2.3 과정 추적형 — 표준 3단 패턴 (스케일 진입)

**참고 예시 (1편 압축):** 고래잡이 — "19세기 연간 8만 마리·대왕고래 1%" → "기름·뼈 산업 재료" → "어쩌다 끝판왕급 잡았나?".

### 2.3 전략 F (맥락재설정) — 중빈도

## §3. 4단계 말맛 분석 결과

> 본 §3은 본 채널 말맛 변주 카탈로그. 권위 정보 → 비권위 옷 입힘 변환 메커니즘 본체.

### 3.1 친밀톤·화자 스탠스 변주

> 합쇼체 기본 흐름 + 결정적 순간 해요체·반말 변주로 호흡 단절·시청자 즉시 동조.

#### 3.1.1 합쇼체 기본형 (~합니다·~입니다)

- **적용 영역:** 도입부 진입·본문 정보 단정·결론부 거시 결산
- **변환 메커니즘:** 권위 정보 진입 시 안정감·신뢰도 확보
- **예문 (참고용):** "~이었습니다" / "~인 거지요" / "~한 셈입니다" / "~할 정도였습니다"

#### 3.1.2 해요체 친근형 (~거든요·~잖아요·~죠)

- **적용 영역:** 풀이형 부연 · 통념 정정 진입 · 자문자답 답
- **변환 메커니즘:** 친밀 어미로 친근감 확보 + 시청자 동조 유도
- **예문 (참고용):**
  - "~ 거든요" (이유 풀이 표준)
  - "~ 잖아요" (시청자 이미 알고 있다는 전제 + 동조)
  - "~ 죠" (단정 + 친근)
  - "~ 라는 거예요" (밈 종결)

#### 3.1.3 반말 펀치형 (~거든·~지·~인데·~다는 거)

- **적용 영역:** [전] 정점 임팩트 · 자문자답 자답 · 결산 단속
- **변환 메커니즘:** 합쇼체 흐름 중 반말 펀치 1회로 호흡 단절 + 강도 증폭
- **예문 (참고용):** "어림도 없지" / "응, ~ 야" / "그렇다는 거" / "~ 한다는 거지요" / "~ 라는 거" (명사형 종결 미끼·폭로 어법)

#### 3.1.4 결합 변주 패턴 (위치별)

| 위치 | 결합 패턴 |
| --- | --- |
| 도입부 후킹 | 단속 명사형 종결 → 자문자답 → 합쇼체 풀이 |
| 본문 빌드업 | 합쇼체 정보 + 해요체 풀이 + 반말 펀치 임팩트 |
| [전] 정점 | 자문자답 반말 자답 + 합쇼체 위상 부여 결산 |
| 결론부 | 합쇼체 거시 결산 + 해요체 시청자 동조 + 위트 반말 종결 |

### 3.2 치환·의인화 변주

#### 3.2.1 단일 인격 강화 의인화 (이미지강화형 주)

> 사물·시스템·국가를 하나의 친근 캐릭터로 호명하여 위상 격상.

- **변환 메커니즘:** 추상 시스템 → 단일 캐릭터·친근 호명
- **적용 영역:** 영역확장형 진입부 · 친근 화제 진입
- **예문 (참고용):** "캐리어 아저씨" / "캐리어 형님"·"서유럽 형님들" / "이쪽 동네"·"옆나라"·"바로 옆동네" / "좌식러·입식러" (인격화 접미사 "~러") / "남산이 버티고 있어서"·"강남이 버티고 있기 때문" / "이놈의 미국 땅은 적당함을 모르지요"

#### 3.2.2 양면 인격 의인화 (모순형 주)

> 같은 대상의 두 얼굴을 양극 인격으로 분리하여 모순 시각화.

- **변환 메커니즘:** 단일 대상 → 두 얼굴 양극 인격 분리
- **적용 영역:** 모순형 피벗 진입 · 결합형 [전] 그림자
- **예문 (참고용):** "산업 무적 ↔ 인체 저주" / "한국(여름 한정 숭배) vs 열대(1년 내내 유일신)" / "미끼 → 가해자 → 귀감 3단 둔갑" / "기름진 과일 → 걸어다니는 자연재해"

#### 3.2.3 시간 여행 가이드 시점 (과정추적형 주)

> 화자가 시청자 손 잡고 시간순으로 안내.

- **변환 메커니즘:** 시간 흐름 → 화자 동행 안내
- **적용 영역:** 인과추적형 [승]·[전] 블록 진입 · 시기 전환점
- **예문 (참고용):** "그러던 [시기]" / "마침내 [연도]" / "그리고 마침내" / "그러다가 ~ 더니 ~까지" / "X세기까지 ~"·"때마침 ~년" / "시즌1 종영 → 시즌2 주인공" (드라마·OTT 비유)

#### 3.2.4 권위 인물 친근 호칭

- **변환 메커니즘:** 권위 인물 → 친근 별명·호칭
- **적용 영역:** 역사 인물·강대국·통치자 친근 격하·격상
- **예문 (참고용):** "캐리어 아저씨" / "백종원 아저씨 나오면 뽑는다" / "동네 형 호출" / "어린 돼지 김일성" / "찐따 출신 국가 / 일진 출신 국가"

#### 3.2.5 국가·시스템 의인화

- **변환 메커니즘:** 국가·법·자연·시스템 → 능동 동작 의인화
- **적용 영역:** 외교·제도·기후·시장 같은 추상 시스템 친근화
- **예문 (참고용):** "수틀리면 침공" / "미쳐 날뛰는 기름값" / "천연 장벽이 고립시키고" / "얘네가 수틀려 가지고" / "유교적 알고리즘" / "느슨해진 미국" / "위기는 ~ 찾아왔다"

#### 3.3.5 자문자답 호흡

- **변환 메커니즘:** 시청자 추리 자극 + 답 공개로 호흡 동기화
- **적용 영역:** 블록 전환 · 메커니즘 풀이 진입 · 정점 직전

#### 3.5.5 §3.5 사용 규칙

1. **위치 정합 우선** — 각 어법은 위치별 효과가 결정. 위치 어긋남 시 톤 어긋남 위험.
2. **변환 메커니즘만 흡수** — 예문 그대로 차용 금지. 매 주제 창의 생성.
3. **유형 분기 도구와 결합** — §3.5 위치별 어법 + §1.4 유형 분기 도구 결합으로 발동.
