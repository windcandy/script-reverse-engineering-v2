# 채널 운영: 에버그린 주제 풀

> **이 파일의 역할:** 개별 영상 제작과 독립적으로, 채널 전체 관점에서 주제 후보를 관리하는 운영 전략이다. P1~P6의 제작 지침과 별개로 운영한다.

---

## 1. 주제 풀 운영 원칙

에버그린 주제 후보를 **20~30개** 수준으로 상시 유지한다. 각 후보는 P1의 GATE 1 기준 중 핵심 질문 정의와 역설 테스트를 완료한 상태여야 한다.

```
[평소]
  에버그린 주제 후보 확보 → 역설 테스트 완료 상태로 풀에 적재

[시의성 이벤트 발생 시]
  풀에서 연결 가능한 주제 우선 제작 → 시의성 트리거 영상으로 전환

[이벤트 없을 시]
  가중치 공식(P1 §6) 점수가 가장 높은 주제부터 순차 제작
```

---

## 2. 주제 풀 등록 기준

풀에 등록하려면 아래 세 항목이 정의된 상태여야 한다.

| 항목 | 내용 |
| ---- | ---- |
| 핵심 질문 | "에? 왜?" 반응 테스트 통과한 한 문장 |
| 역설/가치/서사 유형 | A(역설) / B(숨겨진 가치) / C(과정 추적) 중 하나 |
| 가중치 점수 | P1 §6 공식으로 계산한 수치 |

완전한 P1 작업(Q1~Q8, 프레임, 결론, 경쟁 환경 점검)은 실제 제작 시작 시 수행한다.

---

## 3. 시의성 트리거 연결 전략

뉴스 이벤트가 발생했을 때 풀에서 연결 가능한 주제를 찾는 절차다.

```
[이벤트 발생]
  ↓
풀에서 연관 키워드 검색
  ↓
연결 가능한 주제 있음 → 해당 주제 우선 제작 (시의성 도입부 전략 선택)
연결 가능한 주제 없음 → 이벤트 자체를 신규 주제로 등록 + 역설 테스트 수행
```

**연결 판단 기준:** 이벤트가 해당 주제의 "A극" 또는 "Aha-moment"를 강화하면 연결 가능하다.

---

## 4. 주제 풀 관리 방법

### 등록 양식

```
주제명:
핵심 질문:
유형: A / B / C
가중치 점수:
등록일:
메모 (차별화 각도, 시의성 연결 가능 이벤트 등):
```

### 관리 규칙

- 풀이 20개 미만이면 신규 주제 발굴을 우선 수행한다
- 등록 후 6개월이 지난 항목은 시의성/경쟁 환경을 재점검한다
- 제작이 완료된 항목은 풀에서 제거한다
- Pivot + Cross-Over(P1 §2)를 활용해 기존 제작 영상에서 파생 주제를 정기적으로 발굴한다

---

## 5. 신규 주제 발굴 루틴

정기적으로 아래 루틴을 수행하여 풀을 채운다.

| 루틴 | 방법 | 빈도 |
| ---- | ---- | ---- |
| 트렌드 탐색 | Google Trends 키워드 비교, YouTube 인기 급상승 확인 | 주 1회 |
| 경쟁 채널 분석 | 유사 지식 채널의 최근 영상 주제 확인 | 격주 |
| 파생 주제 발굴 | 기존 제작 영상에 Pivot + Cross-Over 적용 | 월 1회 |
| 시청자 반응 수집 | 댓글/커뮤니티에서 "이것도 다뤄줘" 요청 파악 | 수시 |
