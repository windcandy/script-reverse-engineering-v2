## 1. 주제 유형 판별 (Q1~Q8)

아래 8개 질문에 순서대로 답변하여 주제의 성격을 확정한다.

| # | 질문 | 판별 기준 |
|---|------|----------|
| Q1 | "물질/현상"인가, "사건/인간"인가? | 물질/현상 → **인지적 채무** (몰랐던 사실을 알게 됨) / 사건/인간 → **감정적 채무** (감정이 움직임) |
| Q2 | 역설은 어떤 유형인가? | 장점-단점 역전 / 상식 파괴 / 인과 단절 / 미스터리 |
| Q3 | B극은 하나의 반전인가, 여러 층위인가? | 단일 반전 vs 다층 반전 구조 파악 |
| Q4 | A극과 B극은 분리 가능한가? | 분리 가능 여부에 따라 구조 전략이 달라짐 |
| Q5 | 가치는 "있어서 좋은 것"인가, "없으면 고통인 것"인가? | 긍정적 가치 강조 vs 결핍/고통 프레임 |
| Q6 | 시의성이 있는가? | 뉴스(즉시) / 기념일(주기) / 에버그린(상시) |
| Q7 | 시간축이 긴 주제인가? | 단기 사건 vs 장기 역사·추이 |
| Q8 | 여러 사례를 다뤄야 하는가? | 단일 사례 집중 vs 복수 사례 비교 |

## 2. 핵심 질문 발굴 (3가지 유형)

### 유형 A: 역설 (모순형)

**공식:** [시청자가 아는 것 A] + [모순되는 사실 B]

| 하위 유형 | 설명 |
|-----------|------|
| 장점-단점 역전형 | 좋다고 알려진 것이 실은 해롭다 (또는 그 반대) |
| 상식 파괴형 | 누구나 믿는 상식이 사실이 아니다 |
| 인과 단절형 | 원인과 결과가 직관과 반대로 작동한다 |
| 미스터리형 | 설명 불가능한 현상이 존재한다 |

### 유형 B: 숨겨진 가치 (이미지 강화형)

| 하위 유형 | 설명 |
|-----------|------|
| 이면 폭로형 | 겉으로 보이지 않는 이면을 드러낸다 |
| 가치 증명형 | 과소평가된 대상의 진정한 가치를 입증한다 |
| 전략 해부형 | 성공/실패 뒤에 숨겨진 전략을 분석한다 |

### 유형 C: 과정 추적 (서사형)

| 하위 유형 | 설명 |
|-----------|------|
| 사건 서사형 | 하나의 사건을 시간 순서로 추적한다 |
| 역사 서사형 | 긴 역사적 흐름을 압축하여 전달한다 |
| 비교 서사형 | 둘 이상의 대상을 비교하며 서사를 전개한다 |

### 핵심 질문 테스트

> 발굴한 핵심 질문에 대해 시청자가 **"에? 왜?"** 반응을 보이는가?
> 이 반응이 나오지 않으면 핵심 질문을 재설정한다.

### 주제 발굴 원칙

트렌드 키워드에서 **"사건 자체"**가 아니라 **"핵심 대상"**을 추출한다.
사건은 시의성과 함께 소멸하지만, 핵심 대상에서 출발하면 **에버그린 지식**이 된다.

## 3. 프레임 선택

| 프레임 | 본론 중심 | ML 방향 | 톤 |
|--------|----------|---------|-----|
| 기술적 | 메커니즘, 공정 | "이 기술이 어떻게 가능한가?" | 분석적 |
| 문화적 | 역사, 정체성 | "왜 이것이 마음을 잡았는가?" | 서사적 |
| 경제적 | 시장, 산업 구조 | "왜 이런 경제 현상이?" | 시사적 |
| 지정학적 | 국가 관계, 자원 | "왜 이곳이 운명을 좌우?" | 긴박한 |
| 과학적 | 생물, 화학, 물리 | "원리는 무엇인가?" | 지적 |

## 4. 최종 결론 + 지적 전리품 정의

| 항목 | 정의 |
|------|------|
| **최종 결론** | 시청자가 영상을 다 본 뒤 소유하게 되는 결론 |
| **지적 전리품** | 시청자가 세상을 다르게 보게 되는 방식 |
| **논리 엔진** | 최종 결론에 도달하기 위한 핵심 추론 경로 |

세 항목 모두 Phase 1 단계에서 명확히 정의해야 이후 구조 설계(Phase 2)로 진행할 수 있다.

## 5. 주제 선정 가중치 공식

```
기대 조회수 = (한국 관련도 × 3) + (일상 연결성 × 2) + (역설 강도 × 2) + (보편 궁금증 × 1)
```

각 항목을 1~5점으로 채점한 뒤 가중 합산하여 주제 간 우선순위를 판단한다.

## 7. 할루시네이션 방지 규칙 (Phase 1 전용)

- 존재하지 않는 트렌드나 통계를 만들어내지 않는다.
- 주제의 역설 강도를 과장하지 않는다 -- 실제 사실로 뒷받침 가능한 역설만 채택한다.
- "모든 주제에 역설이 있어야 한다"는 강박을 갖지 않는다 -- 과정 추적형(유형 C)도 유효한 핵심 질문 유형이다.
