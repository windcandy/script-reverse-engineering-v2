## §1. ML 구조 공통 원리

ML(메인 루프)은 **도입부 말미에서 본문이 답할 핵심 질문을 공식화하는 단위**이다. 서사 유형에 따라 변주하며, 유형별 구체 구조와 자체 형태 변주는 모듈_엔진 ML 영역 참조.

### 1.1 ML 3대 실용 기능

1. **본문 로드맵 제시** — ML 질문이 본문 파트를 예고 (예: "언제부터? 장단점?" → 역사 + 비교분석 2파트)
2. **본문 진입 선언 (선택)** — ML 질문 직후 본문으로 자연스럽게 넘어가면 생략 가능. 삽입 시 "~알아봅니다" / "한번 보시죠" / "지금부터 보시죠" 등 진입 표현만 허용. 채널명·화자 소개 삽입 금지
3. **서사 방향 설정** — 유형별 고유 기능

**ML 봉인 형태 (질문 개수):** 양면 봉인(2개 질문 — "왜? 어떻게?")이 기본. 단면 봉인(1개 질문)도 본문 단일 방향 추적 영상에서 활용 가능.

**이문호공 요인 운영:** 본문 ML·해소 = 9 기본 요인 중심 / 썸네일·제목 후킹 = 10요인 + 부정·감정 변주. 단일 본체 → `지침/모듈/모듈_이문호공.md`.

### 1.2 본문 운영 원칙

본문은 메인 루프(ML) 1개를 축으로, 여러 블록이 ML을 분산 해소하며 **ML로 수렴**한다. SL(서브 오픈루프)은 ML로 답할 수 없는 별도 의문이 떠오를 때 조건 발동하는 보조 장치다 (정의·발동 = 모듈_집필기법 §8).

### 1.3 본 채널 3유형별 ML 유지·강화·닫기 구현

| 단계 | 모순형 | 이미지강화형 | 과정추적형 |
| --- | --- | --- | --- |
| **유지** | Q2를 보류하여 공백 유지 (Q1은 A극에서 분산 해소) | 영역 A 빌드업 미니 회수 후 영역 전환점이 새 공백 생성 | 시간순 맥락을 축적하며 핵심 사건·인과를 보류 |
| **강화** | 피벗 직전 A극 확신 극대화 | 영역 전환점 진입 시 영역 B 새 루프 열기 | 인과 폭로로 절정 직전 텐션 극대화 |
| **닫기** | B극에서 3~4회 분할 제공. Q1·Q2 해소 위치 5패턴 허용 | 영역 B 확장에서 스케일 각인 — 영역 전환 1회 | 절정의 사건 자체가 해소 + 결론부에서 현재 의미 부여 — 전환점 2~6회 |

## §2. 엔딩 8유형

결론부 착지(엔딩)는 아래 8유형 중 1~2개를 조합한다.

| # | 유형 | 착지 방식 |
| --- | --- | --- |
| ① | 일상행위연결 | "오늘 ~해 보세요" 식 실천 제안 |
| ② | 감정적여운 | "~한번쯤 생각해 보는 것도" 식 감정 착지 |
| ③ | 미래희망 | "~평화가 계속되길" 식 미래 기원 |
| ④ | 시의성마무리 | 현재 시점 이슈로 연결 |
| ⑤ | 위트반전 | 위트 있는 반전·유머로 착지 |
| ⑥ | 통찰형 | 본문 전체를 관통하는 한 문장 통찰 제시 |
| ⑦ | 경고형 | 대상의 한계·문제·위험 경고로 착지 |
| ⑧ | 광고·협찬형 | 협찬·제품 홍보와 결합한 착지 |

**⑦ 경고형 보편성 원칙 (필수 조건):** 경고 대상자가 시청자 대다수와 일치할 때만 채택한다. 일부 소수에게만 적용되는 경고는 ⑦ 채택 금지.

### 2.1 본 채널 3유형별 결론부 궁합 요약

- **모순형:** 미래희망(③) + 위트(⑤) 혼합 주류. 심각 주제는 경고형(⑦) 단독 허용
- **이미지강화형:** 미래희망(③) 주력. 감정적여운(②) 비사용
- **과정추적형:** 감정적여운(②) + 시의성(④) 주류. 위트(⑤) 비사용
