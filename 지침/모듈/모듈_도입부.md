# 모듈_도입부 — 썸네일·도입부 카탈로그

> **역할:** 본 채널 도입부 영역 카탈로그. 썸네일 판정 3분류 · 약속·도입부·본문 역할 분담 원리 · 자발적 발견 단일 정의 · 도입부 전략 A-H + 본 채널 3유형별 주력.

---

## §1. 썸네일 판정 기준 — 3분류

썸네일 문구의 형태에 따라 3가지 전략으로 구분한다.

| 전략 | 판정 기준 | 문형 신호 | 참고 매핑 |
| --- | --- | --- | --- |
| **결과 제시형** | 결론·지위·단정을 직접 선언 | `~이다`, `~나뉜다`, `~하는 X`, `~할 수 없다` | 과정추적형, 이미지강화형 일부 |
| **조각 제시형** | 질문 형태 또는 특정 부품 던지기 | `~이유`, `~일까?`, `~라고?`, `얼마나 ~?`, `어떤 ~?` | 모순형, 이미지강화형 |
| **모순 공존형** | 썸네일 문구 내에 A극 속성과 B극 속성이 동시 제시 | `[긍정 수식] + [부정 명사]`, `[신적·절대적 표현] + [위험·문제 표현]` 조합 | 모순형 우세 |

**참고 매핑 운영:** 위 매핑은 빈도 경향이며 강제 아님. 본 채널 3유형 모두 3종 분류 자유 활용 가능.

### 모순 공존형 예시

- "악마의 풀" — 마성의 이미지 + 식물 속성
- "축복의 저주" 식 대조 — A극·B극 속성 충돌

## §2. 썸네일 약속·도입부·본문 역할 분담 원리

용어 구분은 모듈_레고이론 §0.2 참조.

- **썸네일·제목 = 썸네일 약속** — 최종 통찰 암시
- **도입부 = 썸네일 약속이 왜 의외인지의 전제 조건 세팅** + (도입부 신뢰 상환 전략일 경우) 신뢰 상환 초기 증거 제공
- **본문 = 단서들이 최종 통찰을 향해 수렴** + 아하 모먼트 발생 지점

썸네일 문구를 반복하면 "이미 알고 있는 걸 또 말하네"가 되어 이탈한다. 전제 조건만 깔면 시청자가 스스로 궁금증을 생성하여 자발적 몰입이 일어난다.

### 자발적 발견 (썸네일 약속 보호) — 직접 인용 금지 단일 정의

**썸네일 약속 메인 문구 자체**를 그대로 인용·반복하지 않는다. 동의어 치환·간접 확인으로 처리하여 시청자가 본문에서 아하 도약을 통해 최종 통찰을 자발 발견하도록 전제 조건만 세팅한다 (노출 강도 기준 모듈_집필기법 §4 간접 확인 원리 — 약·중 강도 허용 / 강 강도 회피).

- **적용 대상:** 썸네일 **메인 문구** 자체 + 제목 (예: "인류를 지키고 있는 물질")
- **예외 — 썸네일 서브 문구:** 서브 문구는 본문/결론부에서 **직접 인용 허용**. 서브는 메인 미끼의 결과·보충 선언이라 본문 노출 시 후킹 약화 영향이 미미하고, 본문 통찰 강화 효과가 더 크다
- **목적:** 시청자 능동 추리 유도 (모듈_레고이론 §0.1 능동 추리자 원리·§0.6 지적 유능감 자극 정합). "이미 아는 걸 또 듣는다" 이탈 방지
- **단일 정의:** 본 §2에서만 정의한다. 우회 기법(동의어 치환·속성 대체·반대 방향 진입)은 모듈_집필기법 §4 간접 확인 원리에서만 정의한다

### 도입부 공통 골격 — 인트로 + ML + 신뢰 상환

도입부는 **인트로(블록) + ML(블록) + 신뢰 상환(기능)**으로 구성된다.

| 구성 | 분류 | 공통 원칙 |
| --- | --- | --- |
| **인트로** | 블록 | ML 이전 진입 영역 — 유형별 차등 (모순형 3요소 / 이미지강화·과정추적 2요소) |
| **ML (메인 루프)** | 블록 | 제목·썸네일의 약속을 질문·선언으로 문장화 (도입부 말미) |
| **신뢰 상환** | 기능 | 시청자에게 "영상이 약속을 지킬 것"이라는 신호 제공 |

**신뢰 상환 전략별 도입부 기능:**

| 전략 | 도입부 기능 |
| --- | --- |
| **도입부 신뢰 상환** | 전제 조건 세팅 + 신뢰 상환(초기 증거 제공) 겸함 |
| **본문 이연 신뢰 상환** | 전제 조건 세팅만 (증명은 본문에서) |

각 구성의 구체 구현(진입 단계 수·내용·신뢰 상환 위치)은 모듈_엔진 해당 엔진 절 참조.

## §3. 도입부 전략 A~H 정의 + 본 채널 3유형별 주력

도입부 첫 블록에서 사용하는 진입 전략. 서사 유형 + 주제 성격에 따라 1개 주력 전략을 선택한다.

### A~H 각 전략 정의

| 코드 | 명칭 | 정의 |
| --- | --- | --- |
| A | 규모폭로 | 압도적 수치·스케일로 대상의 거대함을 즉시 증명 |
| B | 공감적이질감 | 시청자가 당연히 여기는 것에 이질감을 심어 균열 생성 |
| C | 자조적체감 | 대상의 결함·단점을 자조적으로 먼저 제시 |
| E | 밈진입 | 밈·패러디·농담으로 가볍게 진입 |
| F | 맥락재설정 | 시청자의 상식적 맥락을 뒤집는 새 맥락 제시 |
| G | 현재불안소환 | 현재 진행형 위기·불안을 소환 |
| H | 감정직접소환 | 감정·분노·감사를 직접 소환 |

> D 결번 — 과거 통계 진입형 D 코드가 A 규모폭로와 기능 중복으로 폐기, 번호는 안정성 유지.

### 본 채널 3유형별 주력 전략

- **모순형**: B·F 우선 (A·C·E·G·H 회피 권장 — 부수 작동 허용)
- **이미지강화형**: B 우선 고려, 주제 적합 시 A·C·E·G·H 선택 (F 회피 권장)
- **과정추적형**: B 우선 고려, 주제 적합 시 A·F·G·H 선택 (C·E 회피 권장)

**8엔진 ↔ 3유형 매핑:** 역설·다층원인수렴 → 모순형 / 영역확장·사례비교 → 이미지강화형 / 인과추적 → 과정추적형 / 결합형(역설+인과추적) → 모순형 + 과정추적형 동시 적용 (F는 모순형 영역 한정 허용).

**범용 전략 안내:** B(공감적이질감)는 3유형 모두에서 우선 고려. 상식 전복 구조로 서사 유형 불문 범용성이 높다.

**B 전략 공감 진입 출발 단위 (3유형 공통):** 시청자 일상 경험에서 출발. 출발점은 이 주제에만 성립하는 구체적 **공통 인지·상식·경험·일상 장면·일상 행동·감각**이어야 한다. 주제 이름만 바꾸면 다른 대본에도 쓸 수 있는 문장은 금지. — **동료 호명 톤 사용** (예: "우리 다 아는 ~"·"~ 한 번쯤 들어봤죠"·"다들 ~ 경험 있죠").

## §4. 도입부 예시 카탈로그 (전략 A~H별 압축 예시)

> 본 §4는 전략 A~H별 도입부 압축 예시 영역. 전략 정의·진단표는 §3 참조.

### 4.1 전략 B (공감적이질감) — 주력 1순위

> **본질:** 시청자가 당연히 여기는 상식에 이질감을 심는다. 본 채널 도입부 최다 발동.

- **모순형 예시 (한반도):** "지평선 축제·평지 귀한 땅" 단정 → "땅 문제만이 아닌 대륙-해양 접점" → "전장 운명·자연재해 집중" → "저주받은 땅일까 축복일까".
- **모순형 예시 (대통령제):** "대통령 = 국가 권력 정점" 상식 → "안 그런 나라가 더 많다" → "서구권 의원 내각제 기본값" → "대통령제 장점·우리나라 채택 이유" 질문.
- **과정 추적형 예시 (온돌):** "한국 귀신이 미국 귀신보다 덜 무서운 이유" → "한국 바닥 따뜻 / 외국 차가움" → "언제부터 우리는 바닥을 데워서 살았고 이게 어떻게 한국 문화의 일부가 됐을까요?".
- **이미지 강화형 예시 (전기차):** "한국 제조업·자동차 세계 최고" → "그런데 미래는 전기차·EV 시대" → "전기차 2.0 시대에도 한국 산업 건재할까요?".

### 4.2 전략 A (규모폭로) — 주력 2순위

> **본질:** 스케일 수치로 즉시 증명 가능. 도입부 첫 문장에서 수치 폭로.

- **모순형 예시 (감자):** "밀·쌀 문명" 상식 → "감자 3억 8천만 톤·주식 4위" 스케일 폭로 → "언제부터 먹었나? 밀·쌀과 장단점?".
- **이미지 강화형 예시 (옥수수):** "3억 8천만 톤·1위" → "C4 식물·테오신테 품종 개량" → "왜 인류 문명 기반? 매일 먹는다는 게 무슨 소리?"
- **과정 추적형 예시 (고래잡이):** "19세기 연간 8만 마리·대왕고래 1%" → "기름·뼈 산업 재료" → "어쩌다 끝판왕급 잡았나?".

### 4.3 전략 F (맥락재설정) — 중빈도

> **본질:** 시청자의 상식 맥락을 뒤집을 명확한 새 맥락이 있다. 큰 프레임을 먼저 세워 임팩트 발생.

- **모순형 예시 (까트):** "커피 = 전 세계 각성제 기호식품" → "그런데 커피 원산지 에티오피아에 더 강한 각성제 까트 존재" → "커피는 600년간 전 세계 퍼졌으나 까트는 원산지에만" → "왜? 절대 전 세계 보급 안 되는 이유는?".
- **과정 추적형 예시 (루프탑 코리안):** "미국 수정헌법 2조 (큰 프레임)" → "1992년 LA 폭동 옥상 위 한국인" → "한인 자경대·예비역·월남전 경험" → "모범적 소수가 어쩌다 옥상 위에서 총을 들었나?".

### 4.4 전략 H (감정직접소환) — 중빈도

> **본질:** 감정(감사·경외·공감)을 직접 소환한다.

- **이미지 강화형 예시 (에어컨):** "인류 최고 발명품? 저는 에어컨" → "리콴유 아저씨 '에어컨은 인류 역사상 가장 중요한 발명품'" → "이전과 이후 어떻게 달라졌나?".
- **과정 추적형 예시 (유엔군):** "작고 가난한 나라를 위해 22개국이 피를 흘렸습니다" → "1950년 유엔 첫 집단 안보 발동" → "왜 그 나라들이 와줬을까? 어떤 활약?".

### 4.5 전략 G (현재불안소환) — 중빈도

> **본질:** 현재 진행형 위기·시의성. G 전략은 2단계 축소 경향(② 메커니즘 암시 거의 생략).

- **과정 추적형 예시 (호르무즈):** "2026년 이란 봉쇄 선언·유가 30%↑·유조선 200척 갇힘·한국 석유 200일분" → "세계 원유 20% 멈춤" → "어쩌다 세계 경제 급소? 우회 방법?".

> **시리즈 연결형 예외 안내:** 이미지 강화형 시리즈 후속편 한정 2단계 축소 가능 ("지난 영상 이어서 ~" → "대책 마련" → "ML"). 일반 선택지 사용 금지. 시리즈물 한정.

### 4.6 전략 C (자조적체감) — 저빈도

> **본질:** 대상의 결함·단점을 자조적으로 먼저 드러낸다.

- **이미지 강화형 예시 (서안해양성 기후):** "영국 날씨 = 비·안개·우중충·음식 맛없음" → "근데 이 애매한 날씨가 농사 자동사냥급 — 편서풍·난류·일정 강수량" → "서유럽 과학혁명 주도도 이 기후 때문?".

### 4.7 전략 E (밈진입) — 저빈도

> **본질:** 밈·유머 소재가 주제와 자연스럽게 연결된다. 도입부 밈 사용 허용 (속어/밈 첫 30초 금지 원칙의 본 전략 한정 예외 — 모듈_어법 §1.1 ② 참조).

- **이미지 강화형 예시 (락스):** "바텐더, 락스 한잔 (밈)" → "현대 인류 인구 8배·전염병 사라짐·차아염소산나트륨" → "독극물 취급이 어떻게 인류 살렸나?".

### 4.8 도입부 예시 카탈로그 사용 규칙

1. **압축 예시만 흡수** — 완성 문장 그대로 차용 금지. 매 주제 창의적 새 표현 생성.
2. **신뢰 상환 위치 명시** — 본문 이연 / 도입부 수행 중 P3 조감도에 기술.
3. **글자수 권장:** 도입부 200-330자 권장 (참고용 — 강제 아님).
