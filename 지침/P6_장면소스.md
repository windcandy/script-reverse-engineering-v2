# Phase 6: 장면 소스 제작

> **로드 시점:** P5 퇴고 완료 후
> **입력:** 최종 대본 + 연출 가이드
> **완료 조건:** 전체 장면에 소스 배정 완료

---

## 방식 선택

대본 완성 후 아래 두 방식 중 하나를 선택한다.

| | 방식 A — 애니메이션 | 방식 B — 실사 혼합 |
|---|---|---|
| 스타일 | 일관된 애니메이션 캐릭터 | 실사 + 부분 생성 이미지 |
| 소스 | Auto Whisk 프롬프트 전체 | 실사 링크 + Freepik + Auto Whisk |
| 크레딧 소모 | 많음 | 적음 |
| 신뢰도 | 보통 | 높음 (실제 사진) |
| 적합한 주제 | 개념/경제/과학 등 추상적 주제 | 사건/인물/장소가 명확한 주제 |

---

## 방식 A — 애니메이션 전체 구성

### 진행 순서

```
[유저] 참고 이미지 1장 제공 (스타일 기준)
  ↓
[나] 대본을 7~10초 단위로 장면 분할
[나] 각 장면별 Auto Whisk 프롬프트 생성 (영어)
[나] 프롬프트 목록 텍스트 파일로 저장
  ↓
[유저] Auto Whisk에 참고 이미지 + 프롬프트 목록 입력 → 일괄 생성
[유저] 01_영상, 썸네일/ 폴더에 저장
```

### 프롬프트 작성 규칙

- 영어로 작성
- 장면 사이 **한 줄 공백** 필수 (Auto Whisk 구분자)
- 참고 이미지 스타일과 일관성 유지하는 묘사 포함
- 텍스트/숫자가 필요한 장면은 별도 표기

### 출력 형식 예시

```
[장면 01] "한국에만 존재하는 독특한 주거 제도"

이미지 프롬프트: A small cartoon figure standing in front of a giant apartment building, black and white sketch style, dark dramatic atmosphere

연출: 건물 전체 부감 → 인물 클로즈업으로 줌인
AI 영상화: Camera slowly zooms in on a tiny figure standing before a massive apartment complex, dark and oppressive atmosphere, cinematic


[장면 02] ★ "보증금 3억이 하루아침에 사라졌습니다"

이미지 프롬프트: A cartoon figure staring at an empty open vault with shocked expression, money scattered on the floor, high contrast lighting

연출: 금고 문이 열리며 텅 빈 내부 드러남 — 충격 정점
AI 영상화: Vault door slowly swings open to reveal emptiness, shocked cartoon character, dramatic lighting shift from warm to cold
```

## 방식 B — 실사 혼합 구성

### 소스 우선순위

| 순위 | 소스 | 장면 유형 |
|---|---|---|
| 1순위 | 보도자료 / 정부 / 국제기구 실사진·영상 링크 | 실제 사건, 인물, 장소, 통계 |
| 2순위 | Freepik 웹 링크 (API 미사용) | 감성 이미지, 배경, 오브젝트 |
| 3순위 | Auto Whisk 프롬프트 | 인포그래픽, 데이터 시각화, 실사 없는 개념 장면 |

### 진행 순서

```
[나] 대본을 7~10초 단위로 장면 분할
[나] 각 장면별 소스 배정:
  - 1순위: 실사 검색 → 공개 출처 링크 + 이미지 위치 설명
  - 2순위: Freepik 검색 → 웹 링크
  - 3순위: Auto Whisk 프롬프트 생성
[나] 모든 장면에 연출 설명 + AI 영상화 프롬프트 추가
[나] 장면 소스 목록 파일로 저장
  ↓
[유저] 실사 다운로드 + Auto Whisk 실행 + Freepik 저장
[유저] 연출 설명 참고하여 편집 / AI 영상화 필요 시 G-Labs Veo 사용
[유저] 01_영상, 썸네일/ 폴더에 저장
```

### 실사 출처 기준

| 출처 유형 | 예시 |
|---|---|
| 국내 공공기관 | 통계청, 국토교통부, 한국은행, LH공사 |
| 국제기구 | UN, IMF, World Bank, WHO |
| 보도사진 (CC라이선스) | AP, Reuters, Wikimedia Commons |
| 정부 공식 자료 | 대한민국 정책브리핑, 각 부처 보도자료 |

---

## 공통 규칙

- 장면 분할 기준: **7~10초 단위** (13분 영상 기준 약 80~100장면)
- 박스 이미지(강) 장면은 반드시 별도 표기하여 강조 처리
- 저작권 있는 이미지 링크 제공 금지
- 장면 번호는 연출 가이드 번호와 일치시킨다
