## 1. 리서치 목적

팩트가 모든 의사결정에 선행한다. 이 Phase의 목적은 이후 모든 단계(구조설계, 집필)의 재료가 되는 팩트 리포트를 완성하는 것이다.

확정된 제목/썸네일이 리서치 방향타다. 팩트 리포트의 핵심 과제는 제목 약속을 증명할 **신뢰 상환 전략**을 결정하는 것이다.

**신뢰 상환 전략 후보**:

- **도입부 신뢰 상환:** 수치·권위 발언 등으로 도입부에서 즉시 증명
- **본문 이연 신뢰 상환:** A극 자체 상환 / 피벗 직후 투하 / B극 분할 / 원인→Scale Jump / 역사·메커니즘 후 핵심 등 구조 변주별 전략

팩트 리포트에 신뢰 상환 전략 후보 1~2개를 명시한다. 세부 선택은 P3에서 확정.

> **특수 예외 사례 (각주):** 22 온돌은 제목("유독 조선땅에 귀신이 없는 이유")이 해석적 주장이라 도입부·본문 이연 2종 체계로 증거 제시 불가능한 단일 사례. 서사 전체가 누적 인증으로 작동. 주 2종 체계 밖 특수 예외이며, 유사 특성의 제목을 확정할 경우에만 개별 대응. **표준 명칭: "분산 신뢰 상환"** (`00_용어정리.md` §5 정합).

## 2. 리서치 방향 설정

리서치 시작 전, 확정된 제목/썸네일을 기반으로 아래 네 가지를 명시한다.

| 항목 | 확정된 제목/썸네일에서 도출 | 리서치에서 찾아야 할 것 |
|---|---|---|
| 핵심 질문 | 제목이 암시하는 "왜/어떻게 X가 Y인가?" | X와 Y를 연결하는 실제 메커니즘 |
| 최종 통찰(L3) | 제목이 약속하는 지적 전리품 (공통 §0.1.5) | 최종 통찰을 뒷받침하는 팩트와 반례 |
| 추론 경로 (P2 자체 운영) | 제목 → 최종 통찰(L3)에 이르는 단계별 추론 | 각 단계를 채울 팩트 |

**서사 유형 방향별 강조점** (P1에서 잡은 방향에 따라 위 4항목 수집 시 가중치 적용):

- **모순형 방향:** 핵심 질문을 A극(기존 인식) vs B극(실제)의 대립 구조로 정식화. A극/B극 낙차를 벌리는 수치·사례 우선 수집. 추론 경로는 "상식 → 반전 근거 → 재해석".
- **이미지강화형 방향:** 핵심 질문을 "대상이 얼마나 대단한지(Scale)" + "왜 그런지(How/Why)"로 이원화. Scale Jump용 수치와 숨겨진 메커니즘 팩트를 동등 비중으로 수집. 추론 경로는 "표면 인식 → 스케일 증폭 → 메커니즘 해부".
- **과정추적형 방향:** 핵심 질문을 "어쩌다/어떻게/언제부터 현재에 이르렀나"로 정식화. 전환점 간 인과 + 시간축 팩트 우선 수집. 추론 경로는 "현재 상태 → 기원 복귀 → 전환점 연쇄 → 현재 도달".

> **주의:** 위 방향은 P1에서 메모한 가능성에 따른 가중치일 뿐, 유형 확정은 P3에서 한다. 여러 방향 가능성이 있으면 공통 수집(§3) + 모든 가능성 유형의 강조점을 병행 수집한다.

## 3. 서사 요소별 팩트 발굴

### 공통 수집 항목

아래 3개는 모든 편 공통 필수. 선택 3개는 유형·주제 성격에 따라 존재 여부가 갈림.

**필수:**

| 요소 | 수집 내용 |
|---|---|
| **기원** | 대상이 시작된 시점, 맥락, 최초 사건 |
| **규모/수치** | 정량적 데이터 (최신 연도 기준, 출처 명시) |
| **현재 상태** | 지금 이 순간의 실제 상황 |

**선택 (유형·주제에 따라):**

| 요소 | 수집 내용 | 필요 조건 |
|---|---|---|
| **전환점** | 흐름이 바뀐 결정적 사건이나 시점 | 과정추적형 필수, 모순형 권장 |
| **Aha-moment 후보** | 시청자가 "그래서 그랬구나" 반응을 보일 팩트 | 전 유형 권장, 주제에 강한 단일 팩트 존재 시 |
| **반례/이면** | 주장을 강화하는 예외 사례 또는 숨겨진 측면 | 모순형 필수, 기타 유형 권장 |

### 서사 유형별 추가 수집 항목

**모순형으로 가능성이 있는 경우:**
- A극(기존 인식)을 강화하는 팩트
- B극(실제/반전)을 뒷받침하는 팩트
- 두 극의 낙차를 극대화하는 수치나 사례

**이미지강화형으로 가능성이 있는 경우:**
- 시청자가 현재 갖고 있는 이미지(표면적 인식)
- 그 이미지를 뛰어넘는 Scale Jump 팩트
- 숨겨진 메커니즘(How)을 설명하는 팩트

**과정추적형으로 가능성이 있는 경우:**
- 사건 서사형: 발단→전개→위기→결말 각 단계 팩트
- 역사 서사형: 시대별 전환점과 분기 팩트
- 비교 서사형: 비교 대상 간 공통점/차이점 팩트

> **주의:** 위 과정추적형 3분류(사건/역사/비교)는 **본문 구조 층위**다. P3 §1에서 확정하는 **도입부 전략 층위**(전략 5종 선택)와 독립이다. 두 층위에 필요한 팩트를 별개로 수집한다.

### 도입부 전략·ML 연동 팩트 (P3 확정 대비)

P3 §1에서 도입부 전략을 확정할 때 필요한 팩트를 선제 수집한다. 확정은 P3에서 하되, 팩트 부족으로 재리서치하는 상황을 예방한다.

| 유형 | 도입부 전략별 필요 팩트 |
|---|---|
| **모순형** | 공통 수집(§3 필수 3 + A극/B극 팩트)으로 충분 — 추가 요건 없음 |
| **이미지강화형** | 공통 수집(§3 필수 3 + Scale Jump 팩트)으로 충분 — 추가 요건 없음 |
| **과정추적형** | 공통 수집(§3 필수 3 + 전환점 팩트)으로 충분 — 추가 요건 없음 |

**공통 추가 수집 항목:**

- **해소 은유 재료:** 추상 개념을 일상으로 치환할 비유 대상 1~2개 (예: 복잡한 지정학 → 교실 권력 다툼)
- **ML 1차 소재:** 도입부에 투하할 임팩트 재료(수치·권위 발언·감정 선언·상식 전복 팩트 등) 중 주제에서 가용한 자원 1~2개 사전 식별

> **주의:** 이 단계에서 서사 유형과 도입부 전략을 확정하지 않는다. P1에서 메모한 가능성을 기반으로 해당 유형의 팩트를 집중 수집하되, 팩트가 충분하지 않으면 다른 유형으로 전환한다. 확정은 P3에서 한다.

## 5. 팩트 품질 기준

### 팩트 분류 마킹

팩트 리포트 작성 시 각 항목 앞에 아래 마킹을 붙인다.

| 마킹 | 기준 | 집필 시 서술 방식 |
|---|---|---|
| [사실] | 복수 출처로 확인된 객관적 데이터 | 단정형 사용 가능 |
| [추정] | 단일 출처 또는 예측/전망 데이터 | "~로 알려져 있다", "~로 추정된다" |
| [해석] | 전문가/기관의 분석·해석 | "~라는 분석이 있다", "~로 보는 시각이 있다" |
| [논쟁] | 상반된 주장이 존재하는 사안 | "~라는 견해와 ~라는 견해가 있다" |

> **팩트검증 에이전트 정합:** [추정]·[해석]·[논쟁] 마킹 정보를 P4 집필 시 단정형으로 서술하면 팩트검증 에이전트가 **[서술 위반]**으로 분류하여 GATE 4 불통과. P2 마킹 ↔ P4 집필 ↔ 팩트검증 에이전트 4분류([일치]·[열화]·[미수록]·[서술 위반]) 정합 필수.

### 출처 신뢰도 등급

| 등급 | 출처 유형 |
|---|---|
| 1순위 | 정부기관, 국제기구 공식 발표 |
| 2순위 | 학술 논문, 국책·공공 연구기관 보고서 |
| 3순위 | 권위 있는 언론사 1차 보도 |
| 4순위 | 전문가 인터뷰, 공신력 있는 칼럼 |

**분야별 주요 출처**

| 분야 | 1~2순위 출처 | 3순위 출처 |
|---|---|---|
| 경제/금융 | 통계청, 한국은행, 기획재정부, KDI, IMF, World Bank, OECD | 한국경제, 매일경제, Bloomberg, FT |
| 부동산/주거 | 국토교통부, 한국부동산원, 국토연구원, 금융감독원 | 연합뉴스, 조선/중앙/동아 |
| 과학/기술 | NASA, WHO, Nature/Science 논문, MIT Technology Review | BBC, 과학동아 |
| 의학/건강 | WHO, 질병관리청, PubMed 게재 논문 | 의학신문, Reuters Health |
| 역사/문화 | 국사편찬위원회, 각국 국립기록원, 브리태니커 | 역사 전문 학술지 |
| 환경/기후 | IPCC, 환경부, UNEP | 한겨레, Guardian |
| 국제정치 | UN, 외교부, Council on Foreign Relations | NYT, BBC, Reuters |
| 법/제도 | 국회 입법조사처, 법원 판결문, 감사원 | 법률신문 |
| 사회/인구 | 통계청, 한국보건사회연구원, OECD | 연합뉴스 |

**적용 규칙:**
- 1순위 출처가 있으면 하위 순위로 대체하지 않는다 — 하위 출처로 교체하면 퇴고(C2, C6)에서 출처 검증 실패 위험이 커진다
- 3순위 이하 출처는 [추정] 또는 [해석]으로 마킹한다
- 4순위 단독 출처는 Aha-moment 후보로 채택하지 않는다 — Aha-moment는 영상의 핵심 보상이므로, 단일 전문가 의견에 의존하면 팩트 오류 시 영상 전체 신뢰가 무너진다

### 수집 금지 출처

- 개인 블로그
- 출처가 불분명한 커뮤니티 글 (네이버 카페, 디시인사이드 등)
- 개인의 주관적 의견 (뇌피셜)
- 작성 연도가 불명확한 자료
- 운영 주체가 불분명한 웹사이트

### 기타 품질 기준

- **수치:** 최신 연도 기준, 주체/범위/조건이 명시된 것만 채택
- **출처 기록:** 기관명, 보고서명, 연도를 함께 기록. 출처 실존 여부 확인
- **시의성 주제 유효기간:** 수치 데이터는 집필 시점(P4 진입 시) 재확인. 리서치 후 7일 이상 경과한 수치는 갱신 여부 점검

---

## 6. 할루시네이션 방지 규칙

- 존재하지 않는 통계나 연구를 만들어내지 않는다
- 상관관계를 인과관계로 단정하지 않는다
- 확인되지 않은 팩트를 Aha-moment 후보로 채택하지 않는다
- 팩트가 부족한 경우 추가 검색을 수행한다. "팩트가 없는 구조"를 만들지 않는다
