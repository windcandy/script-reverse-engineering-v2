## 1. 사용 메타 원칙

지식 해적단 채널 25편 정밀 분석으로 도출된 **언어 장치 종합 도구 체계**.
**4영역(의인화·수치 체감 치환·전문 용어 치환·기타 치환) + 문장 구조·서술 시제·메타 운영 도구**로 분류된 전체 도구를 P4·유형별 P4 집필 시 표준 자료로 사용한다.

본 사전은 단순 친밀 톤 도구가 아닌 **SL 본질·서사 전체 골격에 작용하는 언어 장치 종합 사전**이다.

3계층 + 2부속 구조:

- **계층 1 — 채널 공통 강제 시그니처 (5대):** 모든 대본에 강제 적용 (1인칭 화자·한국 정체성 환산·인터넷 슬랭·**권위어 친근 풀이**·**직접 비교 정량 나열**)
- **계층 2 — 채널 공통 권장 (3종):** 모든 대본에 권장 적용 (자기 한계 인정·자문자답 호명·**외국어/사료 어원 풀이**)
- **§4 비-모순형 공통 도구 (2종):** 자문자답 압축형·단계 진입 평행 구조 — 이미지강화·과정추적 강제 / 모순형 부재
- **§5 이미지+과정 공유 도구 (2종):** 반사실 가정·가정법 시뮬레이션 — 이미지강화·과정추적 공유 / 모순형 부재
- **계층 3 — 유형별 분기 도구 (모순 13개 / 이미지강화 14개 / 과정추적 16개):** 모순/이미지강화/과정추적별 분기 적용

### 1.1 시점 분류 — P4 강제 / P4 권장 / P5 전담

본 사전의 모든 도구는 **3계층 등급 분류(🔴 강제·🟠 권장·⚪ 선택)** 와 별개로 **적용 시점**에 따라 P4 강제·P4 권장·P5 전담 3가지로 재분류된다. 모든 도구를 P4 집필과 P5 퇴고 두 시점에 분배하여 인지 부담을 시점별로 분산한다.

**시점 분류 정의:**

| 시점 분류 | 정의 | 적용 방식 |
| --- | --- | --- |
| **P4 강제** | 채널 톤 골격·SL 본질에 직결. 사후 추가 시 SL 흐름 손상 | 집필 중 매 SL 사전 선언에 추적 |
| **P4 권장** | SL 위치·서사 흐름이 명확한 보조 도구. 사후 추가 시 효과 약화 | 집필 중 의식적 사용 (정량 강제 없음) |
| **P5 전담** | 어휘 빈도·정량 카운트형. 사후 검증 가능 | P5_퇴고.md §B-2로 일괄 검증 |

**판정 3축:**

1. **구조 결합도** — SL 골격 결합 강도 (강·중·약)
2. **위치 종속성** — 특정 SL 위치(Aha 정점·반전 직전 등) 발현 여부 (높음·낮음)
3. **사후 검증 가능성** — 정량 카운트로 재검 가능 여부 (정성·정량)

**매핑 규칙:**

- 결합도 강 + 위치 종속 높 → **P4 강제**
- 결합도 중 + 위치 종속 높 + 정성 → **P4 권장**
- 결합도 중 + 위치 종속 높 + 정량 → **P4 권장 + P5 정량 재검증** (이중)
- 결합도 약 + 위치 종속 낮 + 정량 → **P5 전담**

**등급 분류와 시점 분류의 관계:**

- 🔴 강제 등급 → 모두 **P4 강제** (구조 결합도 강 + 위치 종속 높)
- 🟠 권장 등급 → **P4 권장** 또는 **P5 전담** (위치 종속성에 따라 분기)
- ⚪ 선택 등급 → 모두 **P5 전담**

**시청 지속 시간 우선 원칙:**

위치 종속성이 높고 사후 추가 시 SL 흐름이 손상되는 도구는 정량 빈도가 있어도 P4 권장으로 분류한다. 이 원칙으로 다음 5개 도구가 P4 권장으로 격상되었다.

- 권위어 밀집 나열 시그니처 (이미지) — 첫 30초 이탈 방지
- 시간 여행 가이드 시점 (과정) — 시기 전환 이탈 방지
- 시간 감정 여운 (과정) — 결말 다음 영상 클릭률
- 국제 정치 학교 싸움 의인화 (과정, 외교·국제 주제 한정) — 외교 주제 진입 장벽 완화
- 미래 시뮬레이션 시제 (이미지, 미래·시뮬레이션 주제 한정) — 결말 임팩트

**개별 도구의 시점 분류:** §2~§4의 각 도구 표 "분류" 열 참조. P5 검증 시 `P5_퇴고.md` §B-2 참조.

---

> **갱신 원칙:** 본 사전은 단일 지침 문서이다. P4·유형별 P4 4개 파일이 본 사전을 참조하므로, 사전 갱신 시 4개 P4 계열 파일 모두 동기화 검증을 거친다.

## 2. 계층 1 — 채널 공통 5대 시그니처 (3유형 강제)

3유형 모두 강제 사용. SL 본질·서사 유형과 무관하게 일관 적용한다.

### 1인칭 화자

- **정의:** "저는·제가" 우세. 권위 인격 제거
- **시점 분류:** **P4 강제** (계층 1 — 채널 톤 골격)
- **사용 조건:** 모든 화자 발화에 일관 적용. 3인칭·익명 톤 발생 시 1인칭으로 교체
- **정량 기준:** 모든 화자 발화에 1인칭이 자연스럽게 등장하면 자동 충족. 횟수 채우기 목적의 강제 삽입 금지.
- **사용 예시:** "저는 ~", "제가 ~", "저도 처음엔 ~"
- **금지:** 3인칭 객관화 톤·"본 영상에서는" 식 화자 비인격화 + "X가 등장합니다·~번째 ~은요·다음 단계는 ~" 같은 사물 주어 SL 진입 톤. SL 진입 시 정보 자체로 진입 (메타 호명 불필요).

### 한국 정체성 환산

- **정의:** 외국 정보를 한국 일상 단위·맥락으로 환산
- **시점 분류:** **P4 강제** (계층 1 — 채널 톤 골격)
- **사용 조건:** 외국 인명·지명·역사·통화·면적·인구 등장 시 즉시 환산. 환산 직후 원본 정보 보완 가능
- **정량 기준:** 외국 정보 등장 시 100% 환산 (외국 정보당 1회)
- **적용 영역:** 외국 인명·지명·역사·통화·면적·인구
- **사용 예시:** "서울에서 부산까지 거리", "한국으로 치면 ~", "우리나라로 치면 ~"

### 인터넷 슬랭

- **정의:** 채널 톤 시그니처 표현 (친근 톤 강화)
- **시점 분류:** **P4 강제** (계층 1 — 채널 톤 골격)
- **사용 조건:** 정보 피크 직후·반전 임팩트 직후·자조 위치. 도입부 첫 30초 사용 금지 (첫인상 가벼워져 이탈 상승)
- **정량 기준:** 편당 6~10회. 한 단락 안 동일 표현 반복 금지
- **사용 예시:** "개사기", "미친", "찐"
- **금지:** 욕설 수준 표현 금지 (CLAUDE.md 핵심 금지 원칙 정합). 무거운 구간(추모·재난) 사용 금지

### 권위어 친근 풀이

- **정의:** 권위 학술어·전문 용어 (학술·전문·정책·역사 + **영문 직역체·세무·경제·금융 추상 용어** 포함) + 친근 비유·일상 풀이 결합. 결합 방식은 아래 4종 중 조건에 맞게 선택 — ① 수식어 삽입(풀이가 짧아 수식어로 자연스럽게 녹을 때: "액화천연가스 운반선인 LNG선"), ② 보충 어구(풀이가 길어 수식어로 넣으면 문장이 무거워질 때: "LNG선, 그러니까 액화천연가스 운반선이죠"), ③ 앞 문장 서술(용어보다 개념이 중요하고 용어는 이름표처럼 나중에 붙여도 될 때), ④ 완전 대체(뉴스·보고서에는 쓰이나 일상 대화에서는 쓰지 않는 용어 → 더 쉬운 일상어로 흔적 없이 교체: "차관" → "빌린 돈")
- **시점 분류:** **P4 강제** (계층 1 — 채널 톤 골격)
- **사용 조건:** 학술·전문·정책·역사 권위어 + **영문 직역체(예: worldwide income → "전 세계 소득") · 세무·경제·금융 추상 용어(예: 자본이득세·시가총액·순자산·공시지가·비거주자 세제)** 등장 시 풀이 결합 의무. 권위어 단독 사용 금지
- **면제 조건:** "권위어 단독 사용"은 약자(영문/한자) 단독 사용 한정. 한국어로 풀어쓴 단일어(예: "한국개발연구원" — 원문 "KDI") 사용 시 그 자체로 풀이 결합 인정. 단, 일반 시청자 인지도 낮은 풀어쓴 단일어는 추가 친근 풀이 결합 의무.
- **정량 기준:** 편당 5~15회
- **사용 예시:**
  - "악성 중피종... 흉막이라는 조직이 워낙에 섬세하고 회복이 어려운 부위" (04 석면)
  - "차아 염소산이 핵심 성분... 자꾸만 분리되려고 하기 때문" (05 락스)
  - "전향력 즉 코리올리 효과... 디스코 팡팡" (17 서안해양성)
  - "유엔군 평화 유지군 말하는 건가 싶으실 수도 있는데" (21 유엔군)
- **효과:** 권위 + 친밀 양립 (시청자 부담 완화 + 신뢰감 형성)
- **금지:** 권위어 단독 사용 (시청자 부담 증가 + 이탈 상승)

### 직접 비교 정량 나열

- **정의:** 같은 단위 정량 직접 병치 — 환산이 아닌 직관 비교 ("A 단위 N / B 단위 N")
- **시점 분류:** **P4 강제** (계층 1 — 채널 톤 골격)
- **사용 조건:** 비교 대상 2개 이상 등장 시 같은 단위로 직접 병치
- **나열 종결:** 마지막 항목은 동사형으로 자연스럽게 마무리한다. 명사·수치 단독으로 마침표 찍어 종결하지 않는다 (P4 §A 15번 정합)
- **정량 기준:** 편당 4~10회
- **사용 예시:**
  - "쌀이 3만 kcal, 밀이 1.8만 kcal, 감자가 5.4만 kcal나 됩니다" (01 감자)
  - "북미가 약 110만 대, 유럽이 64만 대, 인도가 61만 대 팔렸어요" (24 전기차)
  - "유엔군 154,832명이 약 14,825km 떨어진 곳에서 모였죠" (21 유엔군)
  - "남한 면적의 절반이고, 본토보다 무려 50배나 됩니다" (16 그린란드)
- **효과:** 시청자 직관 비교 유도 (환산 없이 같은 단위 병치)
- **금지:** 단일 수치 단독 (직관 비교 효과 부재) / 명사·수치 단독 마침표 종결 (§A 15번 정합)

## 3. 계층 2 — 채널 핵심 보조 3종 (3유형 권장)

3유형 모두 권장 사용. 언어 장치 강화 보조 도구.

### 자기 한계 인정

- **정의:** 화자가 자기 한계·모름을 친근하게 인정
- **시점 분류:** **P4 권장** (계층 2 — 보조 도구)
- **사용 조건:** 어려운 정보·논쟁 영역·추정 영역 등장 시. 출처 인용 직전·직후
- **정량 기준:** 편당 1~2회
- **사용 예시:** "저도 사실 잘 몰랐는데", "여기까지가 제가 아는 한계입니다", "정확한 건 저도 모르겠어요"
- **효과:** 권위 평탄화 + 시청자 부담 완화 + 신뢰감 형성
- **주의:** 3회 이상이면 신뢰 약화. 사실 영역(단정 가능)에서 사용 금지

### 자문자답 호명

- **정의:** 화자가 시청자 의문을 한 단락에 걸쳐 선제 호명한 후 자답 — 의문 호명과 자답이 별개 문장 또는 단락 단위로 분리. **(§4 자문자답 압축형과 구분: 압축형은 한 문장 안에 질문→즉답 압축)**
- **시점 분류:** **P4 권장** (계층 2 — 보조 도구)
- **사용 조건:** 모순 진입 직전·정보 피크 직전·전환 브릿지 위치. 시청자 의문 발생 예상 지점. 단락 단위 호흡 보조 (한 문장 압축이 아닌 의문 호명·자답 분리 적용)
- **정량 기준:** 시청자 의문 발생 예상 지점이 부재하면 횟수 채우기 금지.
- **사용 예시:** "왜 그럴까요? 답은 ~", "이상하지 않나요? 사실은 ~"
- **효과:** 갸우뚱 진입 셋업 + 시청자 의문 선제 흡수
- **주의:** 의문 호명 후 즉시 자답 (호명만 하고 답 지연 금지)

### 외국어/사료 어원 풀이

- **정의:** 외국어·외국 명칭·한자어·사료 어휘의 어원 풀이 (다국어 어원 + 한자 직역 + 사료 풀이 광역)
- **시점 분류:** **P4 권장** (계층 2 — 보조 도구)
- **사용 조건:** 외국 명칭·한자어·고대 사료 어휘 등장 시 즉시 어원 풀이
- **정량 기준:** 외국 명칭·사료 등장 시 100% 풀이 (편당 평균 2~5회)
- **사용 예시:**
  - "어스베스토스 → 아스베스토스 → 불멸" (04 석면)
  - "아오아카트 → 아고와카테 → 아보카도" (23 아보카도)

- **금지:** 외국 명칭/사료 부재 시 사용 강제 안 됨

## 4. 비-모순형 공통 도구 (2종 — 이미지강화·과정추적 양쪽 강제)

이미지강화·과정추적 두 유형에 모두 강제 적용. 모순형은 부재 또는 약형이므로 적용 안 함.

### 자문자답 압축형

- **정의:** 가상 시청자 의문을 한 문장에 압축한 후 즉시 단답으로 해소 — 의문과 답이 한 문장 안에서 압축 ("엥? ~? 아니죠" / "X가 나타난다? Y가 나타난다"). **(계층 2 자문자답 호명과 구분: 호명은 단락 단위 분리·압축형은 한 문장 압축)**
- **시점 분류:** **P4 강제** (§4 비-모순형 공통 — 이미지강화·과정추적 한정 / **모순형 편당 0~1회**)
- **사용 조건:** 정보 피크 직전·반전 임팩트 직전·전환 브릿지 위치 — 한 문장 압축 호흡 (이미지·과정에서 단계 진입·반전 임팩트 즉시 해소)
- **정량 기준:** 이미지강화 편당 3~6회 / 과정추적 편당 13회 이상 / **모순형 편당 0~1회**
- **사용 예시:**
  - "엥? 그럼 남미 원주민들은 뭐임? 얘네 옥수수 원 아님" (07 옥수수)
  - "그럼 끝났을까요? 아니죠. 이제부터가 본 게임" (08 서울)
  - "그러면 어떻게 된다? 결국 ~" (12 가야)
  - "이제 끝났냐? 어 아닙니다" (08 서울)
- **효과:** 시청자 의문 즉시 해소 + 호흡 압축 + 임팩트 강화
- **금지:** 모순형 편당 0~1회 한도. **2회 이상 사용 시 SL 골격 어긋남 위험** (모순 진입 마이크로 빌드업과 충돌).

### 단계 진입 평행 구조

- **정의:** "그럼 끝났을까요? 아니죠. 이제부터가 본 게임" 같은 평행 구조로 단계 진입 — 반전 질문 + 부정 + 다음 단계 슬로건 3박자
- **시점 분류:** **P4 강제** (§4 비-모순형 공통 — 이미지강화·과정추적 한정 / 모순형 차용 금지)
- **사용 조건:** 다음 단계 진입 위치·하이라이트 누적 사이클 (Scale Jump·연표 누적·전환점)
- **정량 기준:** 이미지강화 편당 1~5회 / 과정추적 편당 5~8회 / 모순형 사용 강제 안 됨
- **사용 예시:**
  - "그러면 일본은? 이쪽도 심각성이 만만치 않습니다" (20 해수면)
  - "그러면 인도랑 파키스탄은 안전하냐?" (20 해수면)
  - "그러면 방장 사기 맵으로 유명한 북아메리카는 안전하냐?" (20 해수면)
- **효과:** 단계 진입 임팩트 + 시청자 호흡 압축 + 평행 구조 리듬
- **금지:** 모순형 대본에 차용 금지 (SL 골격 어긋남)

## 5. 이미지+과정 공유 도구 (2종 — 모순형 부재)

이미지강화·과정추적 두 유형이 공유. 모순형은 사실상 부재 (0~14%).

### 반사실 가정

- **정의:** "~이 없었다면" / "만약 ~ 안 했다면" 변형 구문 — 추상적 가치를 구체적 부재로 체감시키는 도구
- **시점 분류:** **P4 권장** (§5 이미지+과정 공유 — 이미지강화·과정추적 한정 / 모순형 사용 금지)
- **사용 조건:** 추상 가치·인물·발명 영향 강조 위치 / 거시 시스템 영향 결정타
- **정량 기준:** 이미지강화 편당 1~3회 / 과정추적 편당 0~2회 / **모순형 사용 금지**
- **사용 예시:**
  - "낙스를 비롯한 염소 살균제가 없었다면 불가능했다고 보시면" (05 락스)
  - "윌리스 캐리어 아저씨가 없었다면 이 푹 짓는 날씨를 어떻게 버텼겠습니까" (17 서안해양성)
  - "여권 파워가 받쳐주지 않았다면 이것도 불가능" (09 여권)
  - "이거 안 뜯어봤으면 교과서 원삼국 시대 부분은 사진 하나 없을 수도" (12 가야)
- **효과:** 추상 가치 체감 + 거시 시스템 무게 강조
- **금지:** 모순형 대본에 차용 시 SL 골격 어긋남 (현재 폭로 vs 과거 가정 충돌)

### 가정법 시뮬레이션

- **정의:** "만약 ~한다면" 조건 분기로 현재·미래 시나리오 시뮬레이션
- **시점 분류:** **P4 권장** (§5 이미지+과정 공유 — 이미지강화·과정추적 한정 / 모순형 추상 제도 주제 한정 0~2회)
- **사용 조건:** 추상 제도/정책·미래 시뮬레이션·시스템 영향 영역
- **정량 기준:** 이미지강화 편당 1~6회 (추상 주제·미래 시뮬레이션 시 강) / 과정추적 편당 0~2회 / **모순형 추상 제도 주제 한정 0~2회**
- **사용 예시:**
  - "그냥 옥수수만 심는다 → 그 결과는 우리 위 동네 북한" (07 옥수수)
  - "도심 항공 모빌리티 더 이상 꿈이 아니게" (24 자동차)
  - "호르무즈 해협이 막혀 버리면 그냥 석유 수입의 절반이 스톱" (13 호르무즈)
  - "만약에 이 타협이 제대로 안 이루어진다 → 내각 공중분해" (19 대통령제)
- **효과:** 시나리오 직관 + 거시 시스템 영향 시뮬레이션
- **주의:** 모순형 대본에서 사용 시 19편 같은 추상 제도/정책 주제 한정. 자연·생물·문화 주제 사용 시 SL 어긋남
