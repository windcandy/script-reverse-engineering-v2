## 1. 사용 메타 원칙

지식 해적단 채널 25편 정밀 분석으로 도출된 **언어 장치 종합 도구 체계**.
**4영역(의인화·수치 체감 치환·전문 용어 치환·기타 치환)**으로 분류된 약 44개 도구를 P4·유형별 P4 집필 시 표준 자료로 사용한다.

본 사전은 단순 친밀 톤 도구가 아닌 **SL 본질·서사 전체 골격에 작용하는 언어 장치 종합 사전**이다. 25편 100% 분석 검증 (모순 7 + 이미지강화 10 + 과정추적 8).

3계층 구조:

- **계층 1 — 채널 공통 강제 시그니처 (6대):** 모든 대본에 강제 적용 (위계 평탄화·1인칭 화자·한국 정체성 환산·인터넷 슬랭·**권위어 친근 풀이**·**직접 비교 정량 나열**)
- **계층 2 — 채널 공통 권장 (4종):** 모든 대본에 권장 적용 (자기 한계 인정·자문자답 호명·갸우뚱 선제 인정·**외국어/사료 어원 풀이**)
- **계층 3 — 유형별 분기 도구 (3유형 × 11~12개):** 모순/이미지강화/과정추적별 분기 적용

> **갱신 원칙:** 본 사전은 단일 지침 문서이다. P4·유형별 P4 4개 파일이 본 사전을 참조하므로, 사전 갱신 시 4개 P4 계열 파일 모두 동기화 검증을 거친다.

## 2. 계층 1 — 채널 공통 6대 시그니처 (3유형 강제)

3유형 모두 강제 사용. SL 본질·서사 유형과 무관하게 일관 적용한다.

> **25편 누적 검증 결과 (2026-05-01 사이클 11):** 기존 4대 시그니처 + 신규 격상 2개 (권위어 친근 풀이 25/25 100% / 직접 비교 정량 나열 23/25 92%) = **6대 시그니처 절대 확정**.

### 위계 평탄화

- **빈도:** 160회 / 26편 (100%)
- **정의:** 화자 = 시청자 동등 위치 톤. "같이 봐요" 자세
- **사용 조건:** 모든 문장에 일관 적용. 권위 톤 발생 시 즉시 평탄화 톤으로 교체. 도입부·전환·정점·해소·엔딩 모두 적용
- **정량 기준:** 편당 6회 이상
- **사용 예시:** "같이 보시죠", "어떻게 생각하세요?", "우리도 한번 생각해 봅시다"
- **금지:** 권위적 강의자 자세·"제가 알려드리겠습니다" 톤

### 1인칭 화자

- **빈도:** 156회 / 26편 (100%)
- **정의:** "저는·제가" 우세. 권위 인격 제거
- **사용 조건:** 모든 화자 발화에 일관 적용. 3인칭·익명 톤 발생 시 1인칭으로 교체
- **정량 기준:** 편당 6회 이상
- **사용 예시:** "저는 ~", "제가 ~", "저도 처음엔 ~"
- **금지:** 3인칭 객관화 톤·"본 영상에서는" 식 화자 비인격화

### 한국 정체성 환산

- **빈도:** 154회 / 23편 (92%)
- **정의:** 외국 정보를 한국 일상 단위·맥락으로 환산
- **사용 조건:** 외국 인명·지명·역사·통화·면적·인구 등장 시 즉시 환산. 환산 직후 원본 정보 보완 가능
- **정량 기준:** 외국 정보 등장 시 100% 환산 (외국 정보당 1회)
- **적용 영역:** 외국 인명·지명·역사·통화·면적·인구
- **사용 예시:** "서울에서 부산까지 거리", "한국으로 치면 ~", "우리나라로 치면 ~"
- **유형별 균등:** 모순 50회 / 이미지강화 51회 / 과정추적 50회 (3유형 균등 빈출)

### 인터넷 슬랭

- **빈도:** 42회 / 19편 (76%)
- **정의:** 채널 톤 시그니처 표현 (위계 평탄화 강화)
- **사용 조건:** 정보 피크 직후·반전 임팩트 직후·자조 위치. 도입부 첫 30초 사용 금지 (첫인상 가벼워져 이탈 상승)
- **정량 기준:** 편당 6~10회. 한 단락 안 동일 표현 반복 금지
- **사용 예시:** "개사기", "미친", "찐"
- **유형별 차이:** 모순형 우세 (모순 발견 자조 톤과 결합)
- **금지:** 욕설 수준 표현 금지 (CLAUDE.md 핵심 금지 원칙 정합). 무거운 구간(추모·재난) 사용 금지

### 권위어 친근 풀이

- **빈도:** 25/25편 (100%) — **채널 100% 절대 시그니처** (사이클 11 격상)
- **정의:** 권위 학술어·전문 용어 + 직후 친근 비유·일상 풀이 결합
- **사용 조건:** 학술·전문·정책·역사 권위어 등장 시 즉시 친근 풀이 결합. 권위어 단독 사용 금지
- **정량 기준:** 편당 5~15회 (모순 18.4·이미지강화 14.5·과정추적 12.75 — 모순 우세)
- **사용 예시:**
  - "악성 중피종... 흉막이라는 조직이 워낙에 섬세하고 회복이 어려운 부위" (04 석면)
  - "차아 염소산이 핵심 성분... 자꾸만 분리되려고 하기 때문" (05 락스)
  - "전향력 즉 코리올리 효과... 디스코 팡팡" (17 서안해양성)
  - "유엔군 평화 유지군 말하는 건가 싶으실 수도 있는데" (21 유엔군)
- **효과:** 권위 + 친밀 양립 (시청자 부담 완화 + 신뢰감 형성)
- **금지:** 권위어 단독 사용 (시청자 부담 증가 + 이탈 상승)

### 직접 비교 정량 나열

- **빈도:** 23/25편 (92%) — **채널 절대 시그니처** (사이클 11 격상)
- **정의:** 같은 단위 정량 직접 병치 — 환산이 아닌 직관 비교 ("A 단위 N / B 단위 N")
- **사용 조건:** 비교 대상 2개 이상 등장 시 같은 단위로 직접 병치
- **정량 기준:** 편당 4~10회 (모순 7.7·이미지강화 13.2·과정추적 13.6 — 과정추적·이미지강화 우세)
- **사용 예시:**
  - "쌀 3만 kcal / 밀 1.8만 kcal / 감자 5.4만 kcal" (01 감자)
  - "북미 약 110만 / 유럽 64만 / 인도 61만 대" (24 전기차)
  - "유엔군 154,832명 / 약 14,825km 떨어진" (21 유엔군)
  - "남한 면적의 절반·본토보다 무려 50배" (16 그린란드)
- **효과:** 시청자 직관 비교 유도 (환산 없이 같은 단위 병치)
- **금지:** 단일 수치 단독 (직관 비교 효과 부재)

## 3. 계층 2 — 채널 핵심 보조 4종 (3유형 권장)

3유형 모두 권장 사용. 언어 장치 강화 보조 도구.

> **25편 누적 검증 결과 (2026-05-01 사이클 11):** 기존 3종 + 신규 1개 (외국어/사료 어원 풀이 17~18/25 한정 빈출) = **4종 절대 확정**.

### 자기 한계 인정

- **정의:** 화자가 자기 한계·모름을 친근하게 인정
- **사용 조건:** 어려운 정보·논쟁 영역·추정 영역 등장 시. 출처 인용 직전·직후
- **정량 기준:** 편당 1~2회
- **사용 예시:** "저도 사실 잘 몰랐는데", "여기까지가 제가 아는 한계입니다", "정확한 건 저도 모르겠어요"
- **효과:** 권위 평탄화 + 시청자 부담 완화 + 신뢰감 형성
- **주의:** 3회 이상이면 신뢰 약화. 사실 영역(단정 가능)에서 사용 금지

### 자문자답 호명

- **정의:** 화자가 시청자 의문을 선제 호명한 후 자답
- **사용 조건:** 모순 진입 직전·정보 피크 직전·전환 브릿지 위치. 시청자 의문 발생 예상 지점
- **정량 기준:** 편당 3~5회
- **사용 예시:** "왜 그럴까요? 답은 ~", "이상하지 않나요? 사실은 ~"
- **효과:** 갸우뚱 진입 셋업 + 시청자 의문 선제 흡수
- **주의:** 의문 호명 후 즉시 자답 (호명만 하고 답 지연 금지)

### 갸우뚱 선제 인정

- **정의:** 시청자 갸우뚱·반박을 선제 인정
- **사용 조건:** 반박 가능 영역 등장 시·결과 공유 직후·자기 셋업 부정 위치
- **정량 기준:** 편당 1~3회
- **사용 예시:** "여기서 갸우뚱하셨죠?", "이상하다고 느끼시는 게 맞아요"
- **효과:** 시청자 의문 흡수 + 반박 선제 차단
- **주의:** 인정 후 즉시 해소 동작 (인정만 하고 해소 지연 금지)

### 외국어/사료 어원 풀이

- **빈도:** 17~18/25편 (68~72%) — 외국 명칭/사료 등장 시 한정 빈출 (사이클 11 신설)
- **정의:** 외국어·외국 명칭·한자어·사료 어휘의 어원 풀이 (다국어 어원 + 한자 직역 + 사료 풀이 광역)
- **사용 조건:** 외국 명칭·한자어·고대 사료 어휘 등장 시 즉시 어원 풀이
- **정량 기준:** 외국 명칭·사료 등장 시 100% 풀이 (편당 평균 2~5회)
- **사용 예시:**
  - "어스베스토스 → 아스베스토스 → 불멸" (04 석면)
  - "아오아카트 → 아고와카테 → 아보카도" (23 아보카도)
  - "강남곡 → 강남이 → 강냉이" (07 옥수수)
  - "온돌·구들·쪽구들·고래·아궁이" (22 온돌)
  - "소주·아락·아라크" 3중 어원 (25 소주)
- **효과:** 권위 + 친근 양립 (외국어/사료 본래 의미 즉시 전달)
- **금지:** 외국 명칭/사료 부재 시 사용 강제 안 됨

## 4. 계층 3 — 유형별 분기 도구

각 유형별 SL 본질·호흡 특성에 맞는 분기 도구. 채널 공통 6대 + 보조 4종 위에 추가 적용.

> **표 5컬럼 의미:** 도구 = 도구 명칭 / 정의 = 한 줄 핵심 정의 / 사용 조건 = 적용 시점·트리거 / 사용 예시 = 14번 분석 파일에서 인용한 실제 표현 또는 재구성 / 효과·주의 = 친밀 톤 효과 + 사용 시 주의 사항. 14번 분석 파일과의 코드 매핑은 본 사전 부록 매핑 표 참조.
>
> **강제·권장·선택 분류 기호:** 🔴 강제 (SL 본질 — 100% 적용) / 🟠 권장 (SL 본질 보조 — 50% 이상 활용) / ⚪ 선택 (SL 본질에 맞을 때만)

### 4-1. 모순형 분기 도구 ("같이 발견하는 친구")

- **화자 스탠스:** 시청자와 같은 출발선에서 모순을 같이 발견
- **호흡 단위:** 마이크로 반전 사이클
- **친밀 진입:** 자문자답 호명 (계층 2 도구 활용)
- **비유 영역:** 게임 용어 매핑
- **채널 공통 인터넷 슬랭 우세 사용:** 모순형은 계층 1 인터넷 슬랭 빈출 (모순 발견 자조 톤과 결합)

## 9. 친밀 톤 검증 체크리스트 (P5 퇴고 시 활용)

집필 후 P5 퇴고 단계에서 본 체크리스트로 친밀 톤 적용 검증한다. 모든 항목 통과해야 친밀 톤 적합 판정.

### 9-1. 채널 공통 4대 시그니처 누락 검증 (강제)

- [ ] **위계 평탄화** 적용 — "같이 보시죠"·"어떻게 생각하세요" 등 편당 6회 이상
- [ ] **1인칭 화자** 우세 — "저는·제가" 빈도 높음 (편당 6회 이상)
- [ ] **한국 정체성 환산** 적용 — 외국 정보 등장 시 100% 환산
- [ ] **인터넷 슬랭** 적용 — 편당 6~10회, 정보 피크 직후 위치

### 9-2. 채널 핵심 보조 3종 적용 검증 (권장)

- [ ] **자기 한계 인정** 편당 1~2회 (어려운·논쟁·추정 영역에서)
- [ ] **자문자답 호명** 편당 3~5회 (모순 진입·정보 피크 직전)
- [ ] **갸우뚱 선제 인정** 편당 1~3회 (반박 가능 영역)

### 9-3. 유형별 분기 강제 도구 적용 검증

- [ ] **(모순형)** 모순 진입 마이크로 빌드업 적용 (단계적 의문 누적 후 반전)
- [ ] **(모순형)** 결과 공유 셋업 적용 (도입부 또는 모순 전환 직전)
- [ ] **(이미지강화형)** Scale Jump 다단 누적 사이클 4~6단 적용
- [ ] **(과정추적형)** 결말 40% 역전 적용 (본문 약 40% 지점)
- [ ] **(과정추적형)** 자기 셋업 부정 적용 (결말 40% 역전과 연동)

### 9-4. 무리 사용·금지 톤 검증

- [ ] 인터넷 슬랭 편당 10회 이내 (무리 사용 금지)
- [ ] 한 단락 안 동일 도구 반복 없음
- [ ] 권위적 강의자 자세 없음
- [ ] 3인칭 객관화 톤 없음
- [ ] 욕설 수준 표현 없음 (CLAUDE.md 정합)
- [ ] 인터넷 슬랭 도입부 첫 30초 사용 없음
- [ ] 무거운 구간(추모·재난) 슬랭·위트 사용 없음

### 9-5. 타 유형 도구 차용 검증 (충돌 회피)

- [ ] **(모순형 대본)** 발명가·역사 인물 친근 호칭 차용 없음 (이미지강화형 분기)
- [ ] **(모순형 대본)** Scale Jump 다단 누적 사이클 차용 없음 (이미지강화형 분기)
- [ ] **(모순형 대본)** 단일 인격 강화 의인화 차용 없음 (이미지강화형 골격)
- [ ] **(이미지강화형 대본)** 시간 척도 환산·사료 친근 인용 차용 없음 (과정추적형 분기)
- [ ] **(이미지강화형 대본)** 결말 40% 역전·연표 누적 시그니처 차용 없음 (과정추적형 분기)
- [ ] **(이미지강화형 대본)** 양면 인격 의인화 차용 없음 (모순형 골격)
- [ ] **(과정추적형 대본)** 게임 용어 매핑 차용 없음 (모순형 분기)
- [ ] **(과정추적형 대본)** 거시 시스템 놀이기구 비유 차용 없음 (이미지강화형 분기)
- [ ] **(과정추적형 대본)** 양면 인격 의인화·단일 인격 강화 의인화 차용 없음 (다른 유형 골격)

### 9-6. 신규 도구 검증 (사이클 11 추가 — 25편 검증 결과)

- [ ] **권위어 친근 풀이** 적용 — 권위 학술어 등장 시 즉시 친근 풀이 결합 (편당 5~15회) — **25/25편 100% 채널 시그니처**
- [ ] **직접 비교 정량 나열** 적용 — 비교 대상 2개 이상 같은 단위 직접 병치 (편당 4~10회) — **23/25편 92% 채널 시그니처**
- [ ] **외국어/사료 어원 풀이** 적용 — 외국 명칭/사료 등장 시 100% 어원 풀이
- [ ] **(모순형)** 양면 인격 의인화 적용 — 도입부·본문 양면 발견 단계 100% (모순 7/7편 골격)
- [ ] **(이미지강화형)** 5대 강제 도구 적용 — 단일 인격 강화 의인화 (편당 6.7회) + 이미지 결정타 슬로건 (정점 1~2회) + 스케일 누적 환산 + 학술 풀이 누적 사이클 + 누적 거시 슬로건 결정타
- [ ] **(과정추적형)** 결말 40% 역전 (본문 40% 지점 ±15초)·자기 셋업 부정 (편당 4회)·연표 누적 시그니처 (편당 13단) 3대 강제 적용
- [ ] **(과정추적형)** 외교·국제 주제 시 국제 정치 학교 싸움 의인화 100% 적용 (선빵·동네 형 부르기·찐따·뚝배기 깨다 등)
- [ ] **(과정추적형)** 시스템 한계·종말 위험 영역 시 자조적 종말론 결정타 적용
