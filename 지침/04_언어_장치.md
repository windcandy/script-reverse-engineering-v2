## 1. 사용 메타 원칙

지식 해적단 채널 25편 정밀 분석으로 도출된 **언어 장치 종합 도구 체계**.
**4영역(의인화·수치 체감 치환·전문 용어 치환·기타 치환) + 문장 구조·서술 시제·메타 운영 도구**로 분류된 **63개 도구 + 4개 변주 = 67개 항목**을 P4·유형별 P4 집필 시 표준 자료로 사용한다.

본 사전은 단순 친밀 톤 도구가 아닌 **SL 본질·서사 전체 골격에 작용하는 언어 장치 종합 사전**이다. 25편 100% 분석 검증 (모순 7 + 이미지강화 10 + 과정추적 8).

3계층 + 2부속 구조:

- **계층 1 — 채널 공통 강제 시그니처 (6대):** 모든 대본에 강제 적용 (위계 평탄화·1인칭 화자·한국 정체성 환산·인터넷 슬랭·**권위어 친근 풀이**·**직접 비교 정량 나열**)
- **계층 2 — 채널 공통 권장 (4종):** 모든 대본에 권장 적용 (자기 한계 인정·자문자답 호명·갸우뚱 선제 인정·**외국어/사료 어원 풀이**)
- **§3-A 비-모순형 공통 도구 (2종):** 자문자답 압축형·단계 진입 평행 구조 — 이미지강화·과정추적 강제 / 모순형 부재
- **§3-B 이미지+과정 공유 도구 (2종):** 반사실 가정·가정법 시뮬레이션 — 이미지강화·과정추적 공유 / 모순형 부재
- **계층 3 — 유형별 분기 도구 (모순 13개 / 이미지강화 14개 / 과정추적 16개 + 변주 4개):** 모순/이미지강화/과정추적별 분기 적용
- **§7 부속 메타 운영 (채널 공통 강제 2종):** G-8 정보 밀도 사이클·G-9 정보 분류 단정형

### 1.1 시점 분류 — P4 강제 / P4 권장 / P5 전담 (2026-05-03 신설)

본 사전의 모든 도구는 **3계층 등급 분류(🔴 강제·🟠 권장·⚪ 선택)** 와 별개로 **적용 시점**에 따라 P4 강제·P4 권장·P5 전담 3가지로 재분류된다. 67개 도구를 P4 집필과 P5 퇴고 두 시점에 분배하여 인지 부담을 시점별로 분산한다.

**시점 분류 정의:**

| 시점 분류 | 정의 | 적용 방식 |
| --- | --- | --- |
| **P4 강제** | 채널 톤 골격·SL 본질에 직결. 사후 추가 시 SL 흐름 손상 | 집필 중 매 SL 사전 선언에 추적 |
| **P4 권장** | SL 위치·서사 흐름이 명확한 보조 도구. 사후 추가 시 효과 약화 | 집필 중 의식적 사용 (정량 강제 없음) |
| **P5 전담** | 어휘 빈도·정량 카운트형. 사후 검증 가능 | P5 §9 체크리스트로 일괄 검증 |

**판정 3축:**

1. **구조 결합도** — SL 골격 결합 강도 (강·중·약)
2. **위치 종속성** — 특정 SL 위치(Aha 정점·반전 직전 등) 발현 여부 (높음·낮음)
3. **사후 검증 가능성** — 정량 카운트로 재검 가능 여부 (정성·정량)

**매핑 규칙:**

- 결합도 강 + 위치 종속 높 → **P4 강제**
- 결합도 중 + 위치 종속 높 + 정성 → **P4 권장**
- 결합도 중 + 위치 종속 높 + 정량 → **P4 권장 + P5 정량 재검증** (이중)
- 결합도 약 + 위치 종속 낮 + 정량 → **P5 전담**

**등급 분류와 시점 분류의 관계:**

- 🔴 강제 등급 → 모두 **P4 강제** (구조 결합도 강 + 위치 종속 높)
- 🟠 권장 등급 → **P4 권장** 또는 **P5 전담** (위치 종속성에 따라 분기)
- ⚪ 선택 등급 → 모두 **P5 전담**

**시청 지속 시간 우선 원칙:**

위치 종속성이 높고 사후 추가 시 SL 흐름이 손상되는 도구는 정량 빈도가 있어도 P4 권장으로 분류한다. 이 원칙으로 다음 5개 도구가 P4 권장으로 격상되었다.

- 권위어 밀집 나열 시그니처 (이미지) — 첫 30초 이탈 방지
- 시간 여행 가이드 시점 (과정) — 시기 전환 이탈 방지
- 시간 감정 여운 (과정) — 결말 다음 영상 클릭률
- 국제 정치 학교 싸움 의인화 (과정, 외교·국제 주제 한정) — 외교 주제 진입 장벽 완화
- 미래 시뮬레이션 시제 (이미지, 미래·시뮬레이션 주제 한정) — 결말 임팩트

**개별 도구의 시점 분류:** §2~§4의 각 도구 표 "분류" 열 참조. P5 검증 시 §9 체크리스트 참조.

## 2. 계층 1 — 채널 공통 6대 시그니처 (3유형 강제)

3유형 모두 강제 사용. SL 본질·서사 유형과 무관하게 일관 적용한다.

> **25편 누적 검증 결과 (2026-05-01 사이클 11):** 기존 4대 시그니처 + 신규 격상 2개 (권위어 친근 풀이 25/25 100% / 직접 비교 정량 나열 23/25 92%) = **6대 시그니처 절대 확정**.

### 위계 평탄화

- **빈도:** 160회 / 26편 (100%)
- **정의:** 화자 = 시청자 동등 위치 톤. 어미는 ~요계와 ~다계 자연 혼합 사용. 한 톤 일색 금지.
- **시점 분류:** **P4 강제** (계층 1 — 채널 톤 골격)
- **사용 조건:** 모든 문장에 일관 적용. 권위 톤 발생 시 즉시 평탄화 톤으로 교체. 도입부·전환·정점·해소·엔딩 모두 적용
- **정량 기준:** 편당 6회 이상
- **사용 예시:** "같이 보시죠", "어떻게 생각하세요?", "우리도 한번 생각해 봅시다"
- **금지:** 권위적 강의자 자세·"제가 알려드리겠습니다" 톤

### 1인칭 화자

- **빈도:** 156회 / 26편 (100%)
- **정의:** "저는·제가" 우세. 권위 인격 제거
- **시점 분류:** **P4 강제** (계층 1 — 채널 톤 골격)
- **사용 조건:** 모든 화자 발화에 일관 적용. 3인칭·익명 톤 발생 시 1인칭으로 교체
- **정량 기준:** 25편 평균 편당 6회 이상 등장 (통계 참고). 사용 조건이 정량보다 상위 — 모든 화자 발화에 1인칭이 자연스럽게 등장하면 자동 충족. 횟수 채우기 목적의 강제 삽입 금지.
- **사용 예시:** "저는 ~", "제가 ~", "저도 처음엔 ~"
- **금지:** 3인칭 객관화 톤·"본 영상에서는" 식 화자 비인격화 + "X가 등장합니다·~번째 ~은요·다음 단계는 ~" 같은 사물 주어 SL 진입 톤. SL 진입 시 정보 자체로 진입 (메타 호명 불필요).

### 한국 정체성 환산

- **빈도:** 154회 / 23편 (92%)
- **정의:** 외국 정보를 한국 일상 단위·맥락으로 환산
- **시점 분류:** **P4 강제** (계층 1 — 채널 톤 골격)
- **사용 조건:** 외국 인명·지명·역사·통화·면적·인구 등장 시 즉시 환산. 환산 직후 원본 정보 보완 가능
- **정량 기준:** 외국 정보 등장 시 100% 환산 (외국 정보당 1회)
- **적용 영역:** 외국 인명·지명·역사·통화·면적·인구
- **사용 예시:** "서울에서 부산까지 거리", "한국으로 치면 ~", "우리나라로 치면 ~"
- **유형별 균등:** 모순 50회 / 이미지강화 51회 / 과정추적 50회 (3유형 균등 빈출)

### 인터넷 슬랭

- **빈도:** 42회 / 19편 (76%)

- **사용 조건:** 정보 피크 직후·반전 임팩트 직후·자조 위치. 도입부 첫 30초 사용 금지 (첫인상 가벼워져 이탈 상승)
- **정량 기준:** 편당 6~10회. 한 단락 안 동일 표현 반복 금지
- **사용 예시:** "개사기", "미친", "찐"
- **유형별 차이:** 모순형 우세 (모순 발견 자조 톤과 결합)
- **금지:** 욕설 수준 표현 금지 (CLAUDE.md 핵심 금지 원칙 정합). 무거운 구간(추모·재난) 사용 금지

### 권위어 친근 풀이

- **빈도:** 25/25편 (100%) — **채널 100% 절대 시그니처** (사이클 11 격상)
- **정의:** 권위 학술어·전문 용어 (학술·전문·정책·역사 + **영문 직역체·세무·경제·금융 추상 용어** 포함) + 직후 친근 비유·일상 풀이 결합

- **금지:** 권위어 단독 사용 (시청자 부담 증가 + 이탈 상승)

## 3. 계층 2 — 채널 핵심 보조 4종 (3유형 권장)

3유형 모두 권장 사용. 언어 장치 강화 보조 도구.

> **25편 누적 검증 결과 (2026-05-01 사이클 11):** 기존 3종 + 신규 1개 (외국어/사료 어원 풀이 17~18/25 한정 빈출) = **4종 절대 확정**.

### 자기 한계 인정

- **정의:** 화자가 자기 한계·모름을 친근하게 인정
- **시점 분류:** **P4 권장** (계층 2 — 보조 도구)
- **사용 조건:** 어려운 정보·논쟁 영역·추정 영역 등장 시. 출처 인용 직전·직후
- **정량 기준:** 편당 1~2회
- **사용 예시:** "저도 사실 잘 몰랐는데", "여기까지가 제가 아는 한계입니다", "정확한 건 저도 모르겠어요"
- **효과:** 권위 평탄화 + 시청자 부담 완화 + 신뢰감 형성
- **주의:** 3회 이상이면 신뢰 약화. 사실 영역(단정 가능)에서 사용 금지

### 자문자답 호명

- **정의:** 화자가 시청자 의문을 한 단락에 걸쳐 선제 호명한 후 자답 — 의문 호명과 자답이 별개 문장 또는 단락 단위로 분리. **(§3-A 자문자답 압축형과 구분: 압축형은 한 문장 안에 질문→즉답 압축)**
- **시점 분류:** **P4 권장** (계층 2 — 보조 도구)
- **사용 조건:** 모순 진입 직전·정보 피크 직전·전환 브릿지 위치. 시청자 의문 발생 예상 지점. 단락 단위 호흡 보조 (한 문장 압축이 아닌 의문 호명·자답 분리 적용)
- **정량 기준:** 25편 평균 편당 3~5회 등장 (통계 참고). 사용 조건이 정량보다 상위 — 시청자 의문 발생 예상 지점이 부재하면 횟수 채우기 금지.
- **사용 예시:** "왜 그럴까요? 답은 ~", "이상하지 않나요? 사실은 ~"
- **효과:** 갸우뚱 진입 셋업 + 시청자 의문 선제 흡수
- **주의:** 의문 호명 후 즉시 자답 (호명만 하고 답 지연 금지)

### 갸우뚱 선제 인정

- **정의:** 시청자 갸우뚱·반박을 선제 인정
- **시점 분류:** **P4 권장** (계층 2 — 보조 도구)
- **사용 조건:** 반박 가능 영역 등장 시·결과 공유 직후·자기 셋업 부정 위치
- **정량 기준:** 편당 1~3회
- **사용 예시:** "여기서 갸우뚱하셨죠?", "이상하다고 느끼시는 게 맞아요"
- **효과:** 시청자 의문 흡수 + 반박 선제 차단
- **주의:** 인정 후 즉시 해소 동작 (인정만 하고 해소 지연 금지)

### 외국어/사료 어원 풀이

- **빈도:** 17~18/25편 (68~72%) — 외국 명칭/사료 등장 시 한정 빈출 (사이클 11 신설)
- **정의:** 외국어·외국 명칭·한자어·사료 어휘의 어원 풀이 (다국어 어원 + 한자 직역 + 사료 풀이 광역)
- **시점 분류:** **P4 권장** (계층 2 — 보조 도구)
- **사용 조건:** 외국 명칭·한자어·고대 사료 어휘 등장 시 즉시 어원 풀이
- **정량 기준:** 외국 명칭·사료 등장 시 100% 풀이 (편당 평균 2~5회)
- **사용 예시:**

- **효과:** 권위 + 친근 양립 (외국어/사료 본래 의미 즉시 전달)

## 3-A. 비-모순형 공통 도구 (2종 — 이미지강화·과정추적 양쪽 강제)

이미지강화·과정추적 두 유형에 모두 강제 적용. 모순형은 부재 또는 약형이므로 적용 안 함.

> **25편 누적 검증 결과 (16번 분석 사이클 1~10):** 모순형 14% 미만 사실상 부재 / 이미지강화 80~90% / 과정추적 100% — 비-모순형 18/18 = 100% 절대 확정. **비-모순형 공통 도구로 신설**.

### 자문자답 압축형

- **빈도:** 모순형 14% (1/7) / 이미지강화 90% (9/10) / 과정추적 100% (8/8) — 비-모순형 18/18 = 100%
- **정의:** 가상 시청자 의문을 한 문장에 압축한 후 즉시 단답으로 해소 — 의문과 답이 한 문장 안에서 압축 ("엥? ~? 아니죠" / "X가 나타난다? Y가 나타난다"). **(계층 2 자문자답 호명과 구분: 호명은 단락 단위 분리·압축형은 한 문장 압축)**
- **시점 분류:** **P4 강제** (§3-A 비-모순형 공통 — 이미지강화·과정추적 한정 / **모순형 편당 0~1회**)
- **사용 조건:** 정보 피크 직전·반전 임팩트 직전·전환 브릿지 위치 — 한 문장 압축 호흡 (이미지·과정에서 단계 진입·반전 임팩트 즉시 해소)
- **정량 기준:** 이미지강화 편당 3~6회 / 과정추적 편당 13회 이상 / **모순형 편당 0~1회**
- **사용 예시:**
  - "엥? 그럼 남미 원주민들은 뭐임? 얘네 옥수수 원 아님" (07 옥수수)
  - "그럼 끝났을까요? 아니죠. 이제부터가 본 게임" (08 서울)
  - "그러면 어떻게 된다? 결국 ~" (12 가야)
  - "이제 끝났냐? 어 아닙니다" (08 서울)
- **효과:** 시청자 의문 즉시 해소 + 호흡 압축 + 임팩트 강화
- **금지:** 모순형 편당 0~1회 한도. **2회 이상 사용 시 SL 골격 어긋남 위험** (모순 진입 마이크로 빌드업과 충돌).

### 단계 진입 평행 구조

- **빈도:** 모순형 14% (1/7) / 이미지강화 80% (8/10) / 과정추적 100% (8/8)
- **정의:** "그럼 끝났을까요? 아니죠. 이제부터가 본 게임" 같은 평행 구조로 단계 진입 — 반전 질문 + 부정 + 다음 단계 슬로건 3박자
- **시점 분류:** **P4 강제** (§3-A 비-모순형 공통 — 이미지강화·과정추적 한정 / 모순형 차용 금지)
- **사용 조건:** 다음 단계 진입 위치·하이라이트 누적 사이클 (Scale Jump·연표 누적·전환점)
- **정량 기준:** 이미지강화 편당 1~5회 / 과정추적 편당 5~8회 / 모순형 사용 강제 안 됨
- **사용 예시:**
  - "그럼 끝날까요? 아니죠 이제부터가 본 게임입니다" (08 서울)
  - "그러면 일본은? 이쪽도 심각성이 만만치 않습니다" (20 해수면)
  - "그러면 인도랑 파키스탄은 안전하냐?" (20 해수면)

## 3-B. 이미지+과정 공유 도구 (2종 — 모순형 부재)

이미지강화·과정추적 두 유형이 공유. 모순형은 사실상 부재 (0~14%).

> **25편 누적 검증 결과 (16번 분석 사이클 1~10):** 모순형은 "현재 상태의 모순"에 집중하므로 가정 표현을 거의 사용하지 않음. 이미지강화·과정추적은 거시 시뮬레이션·역사 분기점 가정으로 활용. **모순형 부재 / 비-모순형 공유 도구로 신설**.

### 반사실 가정

- **빈도:** 모순형 0% (0/7) / 이미지강화 80% (8/10) / 과정추적 50% (4/8)
- **정의:** "~이 없었다면" / "만약 ~ 안 했다면" 변형 구문 — 추상적 가치를 구체적 부재로 체감시키는 도구
- **시점 분류:** **P4 권장** (§3-B 이미지+과정 공유 — 이미지강화·과정추적 한정 / 모순형 사용 금지)
- **사용 조건:** 추상 가치·인물·발명 영향 강조 위치 / 거시 시스템 영향 결정타
- **정량 기준:** 이미지강화 편당 1~3회 / 과정추적 편당 0~2회 / **모순형 사용 금지**
- **사용 예시:**
  - "낙스를 비롯한 염소 살균제가 없었다면 불가능했다고 보시면" (05 락스)
  - "윌리스 캐리어 아저씨가 없었다면 이 푹 짓는 날씨를 어떻게 버텼겠습니까" (17 서안해양성)
  - "여권 파워가 받쳐주지 않았다면 이것도 불가능" (09 여권)
  - "이거 안 뜯어봤으면 교과서 원삼국 시대 부분은 사진 하나 없을 수도" (12 가야)
- **효과:** 추상 가치 체감 + 거시 시스템 무게 강조
- **금지:** 모순형 대본에 차용 시 SL 골격 어긋남 (현재 폭로 vs 과거 가정 충돌)

### 가정법 시뮬레이션

- **빈도:** 모순형 14% (1/7, 19편 단독 6회 집중) / 이미지강화 70% (7/10) / 과정추적 62.5% (5/8)
- **정의:** "만약 ~한다면" 조건 분기로 현재·미래 시나리오 시뮬레이션
- **시점 분류:** **P4 권장** (§3-B 이미지+과정 공유 — 이미지강화·과정추적 한정 / 모순형 추상 제도 주제 한정 0~2회)
- **사용 조건:** 추상 제도/정책·미래 시뮬레이션·시스템 영향 영역
- **정량 기준:** 이미지강화 편당 1~6회 (추상 주제·미래 시뮬레이션 시 강) / 과정추적 편당 0~2회 / **모순형 추상 제도 주제 한정 0~2회**
- **사용 예시:**
  - "그냥 옥수수만 심는다 → 그 결과는 우리 위 동네 북한" (07 옥수수)
  - "도심 항공 모빌리티 더 이상 꿈이 아니게" (24 자동차)

## 4. 계층 3 — 유형별 분기 도구

각 유형별 SL 본질·호흡 특성에 맞는 분기 도구. 채널 공통 6대 + 보조 4종 위에 추가 적용.

> **표 5컬럼 의미:** 도구 = 도구 명칭 / 정의 = 한 줄 핵심 정의 / 사용 조건 = 적용 시점·트리거 / 사용 예시 = 14번 분석 파일에서 인용한 실제 표현 또는 재구성 / 효과·주의 = 친밀 톤 효과 + 사용 시 주의 사항. 14번 분석 파일과의 코드 매핑은 본 사전 부록 매핑 표 참조.
>
> **강제·권장·선택 분류 기호:** 🔴 강제 (SL 본질 — 100% 적용) / 🟠 권장 (SL 본질 보조 — 50% 이상 활용) / ⚪ 선택 (SL 본질에 맞을 때만)

### 4-1. 모순형 분기 도구 ("같이 발견하는 친구")

- **호흡 단위:** 마이크로 반전 사이클
- **친밀 진입:** 자문자답 호명 (계층 2 도구 활용)
- **비유 영역:** 게임 용어 매핑
- **채널 공통 인터넷 슬랭 우세 사용:** 모순형은 계층 1 인터넷 슬랭 빈출 (모순 발견 자조 톤과 결합)

#### 모순형 분기 도구 표 (13개)
