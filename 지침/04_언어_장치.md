## 1. 사용 메타 원칙

**언어 장치 종합 도구 체계**.
**4영역(의인화·수치 체감 치환·전문 용어 치환·기타 치환) + 문장 구조·서술 시제·메타 운영 도구**로 분류된 전체 도구를 P4·유형별 P4 집필 시 표준 자료로 사용한다.

본 사전은 단순 친밀 톤 도구가 아닌 **SL 본질·서사 전체 골격에 작용하는 언어 장치 종합 사전**이다.

2계층 + 2부속 구조:

- **계층 1 — 채널 공통 강제 시그니처 (3대):** 모든 대본에 강제 적용 (1인칭 화자·속어/밈·**어려운 용어 풀이**)
- **§4 비-모순형 공통 도구 (1종):** 자문자답 압축형 — 이미지강화·과정추적 권장 / 모순형 차용 금지
- **§5 가정 도구 (2종):** 반사실 가정·가정법 시뮬레이션 — 도구별 사용 범위 본체
- **계층 2 — 유형별 분기 도구 (모순 6개 / 이미지강화 4개 / 과정추적 5개):** 모순/이미지강화/과정추적별 분기 적용

### 1.1 시점 분류 — P4 강제 / P4 권장 / P5 전담

본 사전의 모든 도구는 **3계층 등급 분류(🔴 강제·🟠 권장·⚪ 선택)** 와 별개로 **적용 시점**에 따라 P4 강제·P4 권장·P5 전담 3가지로 재분류된다. 모든 도구를 P4 집필과 P5 퇴고 두 시점에 분배하여 인지 부담을 시점별로 분산한다.

**시점 분류 정의:**

| 시점 분류 | 정의 | 적용 방식 |
| --- | --- | --- |
| **P4 강제** | 채널 톤 골격·SL 본질에 직결. 사후 추가 시 SL 흐름 손상 | 집필 후 §C-2 사후 점검으로 추적 |
| **P4 권장** | SL 위치·서사 흐름이 명확한 보조 도구. 사후 추가 시 효과 약화 | 집필 중 의식적 사용 (정량 강제 없음) |
| **P5 전담** | 어휘 빈도·정량 카운트형. 사후 검증 가능 | P5_퇴고.md §2.1·§2.2로 일괄 검증 |

**판정 3축:**

1. **구조 결합도** — SL 골격 결합 강도 (강·중·약)
2. **위치 종속성** — 특정 SL 위치(Aha 정점·반전 직전 등) 발현 여부 (높음·낮음)
3. **사후 검증 가능성** — 정량 카운트로 재검 가능 여부 (정성·정량)

**매핑 규칙:**

- 결합도 강 + 위치 종속 높 → **P4 강제**
- 결합도 중 + 위치 종속 높 + 정성 → **P4 권장**
- 결합도 중 + 위치 종속 높 + 정량 → **P4 권장 + P5 정량 재검증** (이중)
- 결합도 약 + 위치 종속 낮 + 정량 → **P5 전담**

**등급 분류와 시점 분류의 관계:**

- 🔴 강제 등급 → 모두 **P4 강제** (구조 결합도 강 + 위치 종속 높)
- 🟠 권장 등급 → **P4 권장** 또는 **P5 전담** (위치 종속성에 따라 분기)
- ⚪ 선택 등급 → 모두 **P5 전담**

**개별 도구의 시점 분류:** §2~§4의 각 도구 표 "분류" 열 참조. P5 검증 시 `P5_퇴고.md` §2.1·§2.2 참조.

---

> **갱신 원칙:** 본 사전은 단일 지침 문서이다. P4·유형별 P4 4개 파일이 본 사전을 참조하므로, 사전 갱신 시 4개 P4 계열 파일 모두 동기화 검증을 거친다.

## 2. 계층 1 — 채널 공통 3대 시그니처 (3유형 강제)

3유형 모두 강제 사용. SL 본질·서사 유형과 무관하게 일관 적용한다.

### 1인칭 화자

- **정의:** "저는·제가" 우세. 권위 인격 제거
- **시점 분류:** **P4 강제** (계층 1 — 채널 톤 골격)
- **사용 조건:** 모든 화자 발화에 일관 적용. 3인칭·익명 톤 발생 시 1인칭으로 교체
- **정량 기준:** 모든 화자 발화에 1인칭이 자연스럽게 등장하면 자동 충족. 횟수 채우기 목적의 강제 삽입 금지.
- **사용 예시:** "저는 ~", "제가 ~", "저도 처음엔 ~"
- **금지:** 3인칭 객관화 톤·"본 영상에서는" 식 화자 비인격화 + "X가 등장합니다·~번째 ~은요·다음 단계는 ~" 같은 사물 주어 SL 진입 톤. SL 진입 시 정보 자체로 진입 (메타 호명 불필요).

### 속어/밈

- **정의:** 채널 톤 시그니처 표현 (속어·인터넷 슬랭·인터넷 밈 포함, 친근 톤 강화)
- **시점 분류:** **P4 강제** (계층 1 — 채널 톤 골격)
- **사용 조건:** 정보 피크 직후·반전 임팩트 직후·자조 위치. 사물/상황만 대상 (인물/집단 비하 금지). 도입부 첫 30초 사용 금지 (첫인상 가벼워져 이탈 상승). 해당 팩트에만 쓸 수 있는 맥락 특이적 표현 사용
- **정량 기준:** 편당 6~10회. 한 단락 안 동일 표현 반복 금지
- **사용 예시:** "개사기", "미친", "찐"
- **금지:** 욕설 수준 표현 금지 (CLAUDE.md 핵심 금지 원칙 정합). 무거운 구간(추모·재난) 사용 금지
- **비유 도구 분리:** 인터넷 밈을 비유 도구로 사용 시(호명·동사·형용 등치 또는 정도·수준 환산) → 공통 §C.8 "인터넷 밈 은유·환산" 참조. 본 도구 빈도 카운트와 분리.

### 어려운 용어 풀이

- **정의:** 대중적으로 잘 사용하지 않거나 시청자가 모를법한 용어 등장 시 쉬운 말 대체, 간단한 설명 동반, 또는 장면 묘사형 대체
- **시점 분류:** **P4 강제** (계층 1 — 채널 톤 골격)
- **효과:** 시청자 부담 완화 + 신뢰감 형성
- **금지:** 어려운 용어 단독 사용 (시청자 부담 증가) / 어렵지 않은 용어 풀이 동반 (시청자 우롱)

#### 자가 진단

"시청자가 정확한 의미·작동 방식까지 즉시 떠올리는가?"

- **Yes → 풀이 금지** (어렵지 않은 용어 — 일상어·친숙 고유명사 등)
- **No → 풀이 필요** (약자·영문 직역체·추상 용어·학술/전문/정책 권위어 등)

#### 결합 방식 3종

| 방식 | 사용 기준 | 예시 |
| --- | --- | --- |
| **① 쉬운 말 대체** | 일상어로 흔적 없이 교체 가능 | "차관" → "빌린 돈" / "worldwide income" → "전 세계 소득" |
| **② 간단한 설명 동반** | 용어 유지 + 풀이 결합 (수식어/보충 어구/앞 문장 중 흐름에 맞는 형태) | "차아 염소산이 핵심 성분..." / "전향력 즉 코리올리 효과" / "유엔군 평화 유지군 말하는 건가 싶으실 수도 있는데" |
| **③ 장면 묘사형** | 개념의 정확한 정의보다 '어떻게 느껴지고 보이는가'가 중요할 때. 추상적 사회현상·생활상·감정·행동 원리 등을 시청자가 경험할 수 있는 장면으로 대체. **②와 구분: 용어의 정확한 정의를 전달해야 하면 ②, 개념의 느낌·현실감을 전달하면 충분하면 ③** | "감자의 수확 후 가공 용이성" → "쌀은 껍질 벗겨야 하고 밀은 가루까지 내야 하는 반면, 감자는 흙만 털어내면 바로 조리됩니다" |

**② 결합 어구 (대표 5종):** 그러니까 / 다시 말해 / 쉽게 말해 / 정확히는 / 말 그대로 — 일반 한국어 부연 어구라면 다른 어구도 허용. 동일 어구 한 단락 2회 또는 편당 3회 이상 시 변주

#### 정량 기준

사용 조건 따름 (편당 횟수 별도 기준 없음).

- **금지:** 단일 수치 단독 (직관 비교 효과 부재)

## 4. 비-모순형 공통 도구 (1종)

### 자문자답 압축형

- **정의:** 가상 시청자 의문을 호명한 후 물음표 직후 즉답으로 압축 — 의문과 답이 한 호흡(짧은 인접 두 문장) 안에서 압축 ("X가 나타난다? Y가 나타난다").
- **시점 분류:** **P4 권장** (§4 비-모순형 공통 — 이미지강화·과정추적 한정)
- **사용 조건:** 정보 피크 직전·반전 임팩트 직전·전환 브릿지 위치. 전환점 하나당 최대 1회.
- **정량 기준:** 이미지강화·과정추적 편당 1회 이상
- **사용 예시:**
  - "그러면 일본은? 이쪽도 심각성이 만만치 않습니다"
  - "그럼 끝났을까요? 아니죠. 이제부터가 본 게임입니다"
  - "그러면 어떻게 된다? 결국 ~"
  - "이제 끝났냐? 아닙니다"
- **효과:** 시청자 의문 즉시 해소 + 호흡 압축 + 임팩트 강화
- **금지:** **모순형 차용 금지(0회).** 1회 이상 사용 시 SL 골격 어긋남 위험.

## 5. 가정 도구 (2종)

- **반사실 가정** ("~이 없었다면"): 추상 가치·인물·발명 영향 강조 위치 / **모순형 사용 금지** (SL 골격 어긋남 — 현재 폭로 vs 과거 가정 충돌)
- **가정법 시뮬레이션** ("만약 ~한다면"): 추상 제도/정책·미래 시나리오 위치 / **모순형 추상 제도 주제 한정** (자연·생물·문화 주제 사용 시 SL 어긋남) / 미래·시뮬레이션 주제에서 미래 단정 시제와 결합 가능 — "~할 예정"·"~될 거"·"~할 가시권" (예: "도심 항공 모빌리티 더 이상 꿈이 아니게" / "에어컨 보급률 90% 넘어가면 ~할 예정")

**미사용 시 근거 기술 의무:** 두 도구 모두 미사용 시 P3 §12 조감도에 그 근거를 명시한다 (의식적 결정임을 보장 — 누락 방지).

## 6. 계층 2 — 유형별 분기 도구

각 유형별 SL 본질·특성에 맞는 분기 도구. 채널 공통 3대 시그니처 위에 추가 적용.

> **표 5컬럼 의미:** 도구 = 도구 명칭 / 정의 = 한 줄 핵심 정의 / 사용 조건 = 적용 시점·트리거 / 사용 예시 = 실제 표현 또는 재구성 / 효과·주의 = 친밀 톤 효과 + 사용 시 주의 사항.
>
> **강제·권장·선택 분류 기호:** 🔴 강제 (SL 본질 — 100% 적용) / 🟠 권장 (SL 본질 보조 — 50% 이상 활용) / ⚪ 선택 (SL 본질에 맞을 때만)

### 6-1. 모순형 분기 도구

- **채널 공통 속어/밈 우세 사용:** 모순형은 계층 1 속어/밈 빈출 (모순 발견 자조 톤과 결합)

#### 모순형 분기 도구 표 (6개)

| 도구 | 분류 | 정의 | 사용 조건 | 사용 예시 | 효과·주의 |
| --- | --- | --- | --- | --- | --- |
| **양면 인격 의인화** | 🔴 강제 | 동일 대상에 상반된 양면 인격 부여 — 모순형 SL "양면 발견" 골격 | 도입부·본문 양면 발견 단계 | "효자 상품 ↔ 악마의 풀" / "착한 과일 ↔ 살인 작물" / "값싼 천한 술 ↔ 고귀한 명문 술" | 모순 진입 셋업 골격 / **주의:** 양면 모두 동등 강도 부여 — 한쪽 약화 금지 |
| **결과 공유 셋업** | ⚪ 선택 | "결론 알고 보는 우리들" 식 화자·시청자 결과 공유 자세 | 내용 흐름에 맞게 적용 — 셋업 톤·SL 회고 마커·결과 정리 위치 등 | "결국엔 이렇게 됐죠" / "결국 ~ 한 거예요" / "이거 알고 보면 ~" / "사실상 ~ 급이라고 볼 수 있지요" / "이렇게 고착화 됐습니다" / "지금까지 ~로 남은 겁니다" / "이미 ~ 뒤였습니다" | 같은 편 셋업 / 시청자 부담 완화 / 1인칭 동조 유지 / **주의:** 단정형 사용 시 권위 톤 발생 위험 |

## 7. 사용 규칙 (P4 집필 시)

1. **계층 1 (3대 시그니처) 강제 적용** — 모든 대본에 SL 본질·서사 유형과 무관하게 일관 적용. 1인칭 화자·속어/밈·**어려운 용어 풀이** 3개 모두 빠뜨리지 않는다.
2. **계층 2 (분기 도구) 유형별 선택 적용** — 해당 유형의 분기 도구만 선택. 타 유형 도구 차용 시 톤 어긋남 위험 (예: 모순형 대본에 발명가·역사 인물 친근 호칭 차용 금지).
3. **무리 사용 금지** — 속어/밈 등 자연스러운 사용. 빈도 강제·수치 채우기 금지. 한 편당 무리 빈출 시 부담감·부자연스러움 발생.
4. **금지 톤** — **권위적 강의자 자세·가르치는 톤·"제가 알려드리겠습니다" 톤**·3인칭 객관화 톤·딱딱한 학술어. 욕설 수준 표현(CLAUDE.md 금지 원칙 정합).
