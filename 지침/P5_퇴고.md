# Phase 5: 퇴고 + 최종 검증
> **로드 시점:** GATE 4 통과 후
> **입력:** Phase 4 완성 대본
> **완료 조건:** 아래 A~E 전 항목 통과

---

## A. 구조 검증

| # | 점검 항목 |
|---|---|
| A1 | 역설이 한 문장으로 명확한가? |
| A2 | 썸네일-제목이 역설의 양극을 각각 담고 있는가? |
| A3 | 도입부가 시청자-주제 관계에 맞는 전략을 사용하는가? |
| A4 | ML이 영상 끝까지 유효한가? (중간에 답이 나와버리지 않는가) |
| A5 | 모든 SL이 해소되었는가? 열어놓고 안 닫은 루프 없는가? |
| A6 | SL 순서가 전제 건설 체인을 따르는가? |
| A7 | A극 정점에 피벗 BR이 있는가? |
| A8 | B극이 What→How→Why→현재의미 순서인가? |
| A9 | 엔딩이 클리셰("이상 알아보았습니다") 아닌가? |
| A10 | A극에 B극 핵심 정보가 미리 노출되지 않았는가? |
| A11 | 엔딩 정보가 ML 해소/일상연결/회상트리거 중 하나에 기여하는가? |

## C. 무결성 검증

| # | 점검 항목 |
|---|---|
| C1 | 지침 내부 용어(역설엔진, A극/B극, SL, BR, ML, 이문호공, 박스 이미지, 피벗, 쿨다운)가 대본에 노출 안 되었는가? |
| C2 | 인용 정보가 사실과 일치하는가? (웹 검색 재확인) |
| C3 | 확인 안 된 사실을 추측으로 서술하지 않았는가? |
| C4 | 비속어 대상이 사물/상황으로만 한정? 인물/집단 비하 없는가? |
| C5 | 제목/썸네일 기대감을 대본 내용이 충족하는가? (낚시 아닌가) |
| C6 | 인용한 출처(기관, 논문, 연구)가 실제로 존재하는가? (출처명 실존 확인) |
| C7 | 상관관계를 인과관계로 단정 서술하지 않았는가? |
| C8 | 사건의 시간 순서(선후 관계)가 정확한가? |

| # | 점검 항목 |
|---|---|
| D1 | 뒷받침 문장이 실제 데이터/출처/구체적 사실인가? 또 다른 수사 아닌가? |

| D8 | 수치가 최신이고 원래 맥락(주체/범위/조건)이 보존되었는가? |

## 결론 최종 검증 (05 결론 설계 기반)

1. **차별성:** 지난 영상과 결론 구조가 80% 이상 다른가?
2. **트리거:** 시청자가 내일 특정 사물을 볼 때 이 영상이 떠오르는가?
3. **성취감:** "나는 방금 성장했다"고 느낄 수 있는가?

## ⛔ 할루시네이션 방지 — 퇴고 전용

- 퇴고 시 "통과"를 위해 실제로 고치지 않고 체크만 하는 행위 금지
- 문제를 발견하면 반드시 수정 후 해당 항목 재검증
- "이 정도면 괜찮다"는 판단 금지 — 기준에 미달하면 수정
