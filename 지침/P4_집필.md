## PART A: 공식 문장 + 이문호공

### 공식 문장 공식

```
공식 문장 = (핵심 대상 + 비유/은유) + 강화문구(이문호공)
```

### 비유 vs 은유

- **비유(직유):** "~처럼" → 이해를 돕는다
- **은유:** "A는 B다" → 뇌에 새 개념을 박는다 (더 강력)

### 공식 문장 2가지 역할

- **여는 박스:** 비유 먼저 → 궁금증 (SL 열림)
- **닫는 박스:** 팩트 후 비유 → 지적 보상 (SL 해소)

### 배치 규칙: 한 편당 최소 3개

| 위치 | 필수/권장 |
|---|---|
| A극 정점 (피벗 직전) | 필수 |
| B극 핵심 해소 | 필수 |
| SL 해소 | 권장 |
| 엔딩 직전 | 권장 |

### 강약 구조

```
팩트 투입 → 연결 문장 → 공식 문장(강) → 논리적 뒷받침(약) → 브릿지/다음 전개
```

- 뒷받침 없으면 "뇌피셜" 위험
- 연결 문장 없으면 "비약" 위험
- **예외:** 팩트 임팩트형 (숫자 자체가 충격이면 비유 불필요)

### 이문호공 9요인

| # | 요인 | 설명 |
|---|---|---|
| 1 | 이익제시 | 시청자가 얻을 실질적 이익 |
| 2 | 문제제기 | 현재 상태의 문제를 드러냄 |
| 3 | 호기심 | 알고 싶게 만드는 정보 제시 |
| 4 | 공감 | 시청자 경험/감정에 연결 |
| 5 | 상식파괴 | 기존 믿음을 뒤집음 |
| 6 | 반전 | 예상과 반대되는 결론 |
| 7 | 경고 | 위험/손실 가능성 환기 |
| 8 | 비교 | 대상 간 차이를 부각 |
| 9 | 최신정보 | 시의성 있는 새 데이터 |

### 이문호공 배치

- **복합적 배치(수평):** 문장당 1요인 → 서사 연속 구간
- **전략적 중첩(수직):** 문장 내 2요인+ → 핵심 전환점
- **강도 조절:** 수직적 중첩을 모든 마디에 쓰면 피로감

## PART B: 서브루프 집필

### SL 열림 5유형

| # | 유형 | 형태 | 비중 |
|---|---|---|---|

### 시청자 예상 반박 선제 흡수

"~라고 하실 수도 있는데"

## PART C: 13개 언어 장치

### 1. 스포츠 메타포
한 편당 5~8회. 같은 단락 내 종목 1개 유지. 무거운 구간은 절제.

### 2. 의인화
국가/물질/체내 반응을 캐릭터처럼. 직후 객관적 사실 보완.

### 3. 수치 체감 치환
즉시 체감 불가한 수치만 치환. 충격 수치/연도는 원본 유지. 권위 구축 목적이면 원본 밀집 나열.

### 4. 반전 질문 → 즉시 해소
한 편당 5회+. 되묻기/확인사격/시청자대입/침묵여백/뒤집기후재강화/유머전환 교차.

### 5. 구어체 리듬
짧은 확인 사격, 한 줄 요약, 감탄사 리셋. 30초+ 긴 문장 연속 금지.

### 6. 시청자 대입
초반 2분 내 1회 필수(2인칭). 중반: 생활권 재해석. 후반: 화자 1인칭(선택).

### 7. 비속어/밈
6~10회. 정보 피크 직후. 사물/상황만 대상. 같은 표현 반복 금지. 무거운 구간 자제.

### 8. 현재형 서술
배경=과거형 → 핵심=현재형 → 가상="만약~하면" 현재형 가정.

### 9. 대조 쌍
"A가 아니라 B다"가 "A다"보다 항상 강하다. 교차 대조(Chiasmus) A-B-B-A.

### 10. 엔딩 감정 전환
마지막 20~30초. 희망직접제시/기원감정토로/비교상대화 중 택1.

### 11. 반사실 가정
"~이 없었다면". 1~3회. 구문 매번 변형.

### 12. 화자 감정 삽입
1~2회. 1인칭 감정. 남용 시 주관적 채널 인식.

### 13. "이미 이렇게 돼 있다" 현실 폭로
일상 속 대상의 숨겨진 사실. 모든 주제에 적용 불가.

### 이완 장치 강제 규칙

- 팩트 3개+ 연속 후 반드시 이완 1회
- 2분 이상 이완 없으면 수정 대상

### 정보 밀도 파동 사이클

- 60~90초 단위 반복
- 15분 기준 10~12회

## PART D: 도입부 + 엔딩 집필 상세

### 도입부 2단계

1. **신뢰 상환:** 썸네일 약속 확인 ("네, 이게 맞습니다")

### 결론 설계 3모델

| 모델 | 구성 |
|---|---|
| Problem-Solver | Context + Action |
| Story-Teller | Reflection + Insight |

### 결론 4자산

| 자산 | 역할 |
|---|---|
| Insight | 수미상관 |
| Context | 연상트리거 |
| Reflection | 가치연결 |
| Action | 효능감확장 |

### 결론부 쿨다운 규칙 ★

결론부는 본문에서 올린 긴장감/무거움을 **해소**하는 쿨다운 역할이다.  
교훈/경고/조언 톤으로 끝내면 피로도가 누적되고 시청자에게 숙제를 남긴다.

**3단계 구조:**

1. 짧은 정리 1~2문장
2. 가벼운 전환 (일상 행동 제안 또는 유머)
3. 시그니처 아웃트로

**규칙:**

- 결론부 분량은 본문 대비 5% 이내로 짧게
- "교훈"이나 "경고"로 끝내지 말 것
- 일상 행동 제안으로 착지 (예: "오늘 집 가면서 계란이나 사세요")
- 도입부 소재를 회수해 수미상관으로 착지하는 것도 유효
- 희망적/낙관적 또는 유머러스한 전환 사용

## 할루시네이션 방지 규칙 (집필 Phase 전용)

- **지침 예시 문구 복제 금지:** "칼로리 치트키", "침묵의 살인자", "아파트 지옥" 등은 지식해적단의 것. 새로운 비유 생성.
- **화자가 자기 정보에 놀라는 척 금지:** 문장 어미가 전제하는 대화 상대가 실제로 존재하는지 확인.
- **비유의 메커니즘 일치 확인:** 표면적 유사성이 아니라 작동 원리가 일치하는지 검증.
- **수정 후 정합성 확인:** 문장 수정 시 같은 단락 전체를 다시 읽는다.
- **리서치 → 대본 변환 시:** 전문 용어는 "들으면 이해되는" 구어체로 변환.
- **수치 최신성 확인:** 현재 시점 기준 최신인지 확인. 주체/범위/조건이 보존되었는지 확인.
