## P1. 썸네일

### P1.1 모순형 6편 썸네일 분류 결과

모순형 썸네일 전략은 **조각 제시형이 주력**이고, **모순 공존형**(14)이 부분 사용된다. 결과 제시형은 거의 없음.

| 대본 | 썸네일 문구 | 판정 | 근거 |
| --- | --- | --- | --- |
| 01 감자 | "전세계를 먹여 살리는 이 작물의 숨겨진 진실?" | 조각 제시형 | `~진실?` 질문 |
| 03 한반도 | "한반도는 얼마나 저주받은 땅일까?" | 조각 제시형 | `얼마나 ~?` 질문 |
| 14 까트 | "**악마의 풀** 까트의 정체" | **모순 공존형** | 악마(부정) + 풀(식물) 공존 |
| 19 대통령제 | "대통령 없는 나라가 더 많다고?" | 조각 제시형 | `~고?` 질문 |
| 23 아보카도 | "이 과일이 사람을 죽인다고?" | 조각 제시형 | `~고?` 질문 |
| 25 소주 | "소주의 진짜 정체는?" | 조각 제시형 | `~는?` 질문 |

#### 모순 공존형 작동 원리 (모순형 전용)

A극 속성(긍정·신적·절대적)과 B극 속성(부정·위험·금지)을 **썸네일 문구 한 구절에 동시 배치**하여 즉각적 충돌을 일으킨다. 시청자는 "왜 이 대상에 상반된 속성이 공존하는가?"를 본능적으로 추리하게 된다.

**작성 원칙:**

- A극 수식어 + B극 명사 조합 (예: "악마의 풀")
- A극·B극 간 의미 거리가 멀수록 충돌 강도↑
- 조각 제시형과 달리 질문 문형이 없어도 됨 — 충돌 자체가 공백을 생성

판정 기준 세부는 `00_공통_원칙.md` A.1 참조.

#### 모순형 제목·썸네일 이문호공 요인 우위 (6편 실측)

| 위치 | 1순위 | 2순위 | 비고 |
| --- | --- | --- | --- |
| **제목** | 상식파괴(5) | 반전(6) | A극↔B극 대비를 이성적 후킹으로 표현 |
| **썸네일** | 상식파괴(5) | **도덕공분(10)** | 모순 공존형(14)에서 도덕공분 우세 |

**고유 패턴:**

- **도덕공분(10)** — 모순 공존형 썸네일에서만 나타나는 모순형 고유 요인. A극(긍정)·B극(부정) 충돌이 분노를 유발하는 구조
- **요인 분담 권장:** 제목(상식파괴)·썸네일(도덕공분 또는 시간후회) 분담으로 이성·감정 이중 후킹

## P3. 구조 설계

### P3 §1. 도입부 필수 2요소 + 전략 B·F

모순형 도입부는 **2가지 필수 요소**를 반드시 포함한다. 순서·흐름은 전략(B·F)이 결정한다. 구조를 별도 명세하지 않고 전략 선택만으로 도입부를 구성한다.

**필수 2요소:**

- ① **썸네일 약속 재확인** — 썸네일 문구를 동의어 치환으로 재확인
- ② **ML (질문형)** — 질문 2개 + 시그니처 선언 (§2 참조)

**필수 2요소 위치 원칙:**

- **ML:** 도입부 마지막 고정 (본문 진입 다리)
- **썸네일 약속 재확인:** 전략 앞·전략 안·전략 뒤 모두 허용 (위치 자유)

**ML 질문 수:**

- 표준 = 질문 2개 + 시그니처 선언 (P3 §2 참조)
- 변주 = ML 압축형(질문 1개) 허용 (P4 §4 참조)
- 질문 2개 구성 시 **한 문장 결합 우선** (한 문장에 자연 결합 가능하면 결합 채택, 강제 분리 금지)

**사용 전략 (B·F 한정):**

| 전략 | 핵심 | 반전 신호 |
| --- | --- | --- |
| **B 공감적이질감** | 당연히 여기는 것에 이질감 → 균열 생성 | 균열 자체가 반전 신호 내장 |
| **F 맥락재설정** | 상식 맥락을 뒤집는 새 맥락 제시 | 뒤집음 자체가 반전 내장 |

**사용 금지 전략 (모순형 한정):**

- A·E·G·H — 반전 신호 내장 없음 → 모순형 서사 정체성 약화
- C — 반전 내장은 있으나 안전 미확보로 금지
- D — 일반 선택지 금지

> **B극 팩트 도입부 노출 금지:**
>
> B·F 전략이 반전 신호를 내장하더라도, B극 신뢰 상환 핵심 팩트는 본문 이연을 위해 도입부 노출 금지.
>
> **❌ 도입부 노출 금지:**
>
> - B극 답의 **구체 행동·반응·표현** (예: "환호했다"·"자기 집에 모셨다"·"벼슬 내렸다"·"후원했다")
> - B극 신뢰 상환의 **고유명사·인물명·기관명·구체 사건명** (예: 흥선대원군·운현궁·선달·박유전)
> - B극의 **인과 메커니즘 풀이** (예: "안전 밸브였다"·"길들여진 욕")
>
> **✅ 도입부 허용:**
>
> - B극 **존재의 추상적 암시** ("오히려 정반대였다"·"전혀 다른 일이 벌어졌다"·"화내지도 처벌하지도 않았다")
> - B극 **결과의 추상 묘사** (예: 23 아보카도 "사람이 죽고 환경 파괴" — 인물·기관 미명시)
> - B극 **사회적 정황 사실** (예: 19 대통령제 "의원 내각제가 세계 기본값")
>
> **자가 점검:**
>
> 1. 도입부만 보고 "B극 답의 구체 행동"을 추측 가능한가? → 예이면 너무 강한 노출
> 2. 본문 SL4 폭로 팩트(인물·기관·구체 증거)가 도입부에 누설됐는가? → 예이면 본문 이연 신뢰 상환 충격 약화

**신뢰 상환 위치:** **본문 이연** (원칙). 모순형은 역설 암시만으로 도입부 성립. 신뢰 상환(썸네일 약속 증명) 지점은 **주제 성격에 따라 다르다**. 세부 위치를 강제하지 않는다 — 주제·전략에 맞는 자연스러운 위치 선택.

**도입부 글자수 (공백 포함):** 300~750자. 상한 초과는 드물게 허용.

**예시 참조:** B 전략 예시는 `00_지식파일/인트로_예시_전략별.md` B.1·B.2·B.3 / F 전략 예시는 동 파일 F.1 (모순형 한정).

**도입부 공통 골격(인트로·ML 2블록 + 신뢰 상환 기능)은 `00_공통_원칙.md` B.4 참조.**

### P3 §2. 모순형 ML 특유 구조 [Q2+S1] + 4단계 구현

```
ML 표준 = 질문 2개 + 시그니처 선언 (필수)
```

**질문 2개의 역할:** 본문 **A극 파트(Q1) + B극 파트(Q2)의 로드맵**. 각 질문은 해당 파트에서 해소된다. **결합 형태 우선:** 두 질문이 의미상 한 문장에 자연스럽게 결합 가능하면 결합 형태 채택 (강제 두 문장 분리 금지). 예: "이 작은 나라가 어떻게 211년을 버텼는지, 진짜 운이었는지 한번 보시죠."

**변주 허용:** ML 압축형(질문 1개) 등 변주 — 사용 조건·금지 조건은 P4 §4 참조. **Q1·Q2 의미 분화 자가점검:** "Q1을 답하면 Q2도 자동으로 답되는가?" → YES면 (a) 두 질문이 한 문장에 자연 결합 가능한가? → YES면 **결합형**, NO면 **압축형**(질문 1개).

**예시:**

- 03 한반도: "어떤 시련을 안겨 줬나? 어떻게 생존해 왔나?" → A극 시련 + B극 생존 (표준)
- 23 아보카도: "언제부터 먹기 시작? 왜 사람을 죽이고 환경을 파괴?" → A극 기원 + B극 해악 (표준)
- 25 소주: "원래 모습은? 어쩌다 갑산 술?" → A극 과거 + B극 변천사 (표준)
- 역전형 변주 (가상 예시 — 향후 25편 실측 사례 발굴 예정): "왜 갑자기 사라졌나? 어떻게 그렇게 오래 사랑받았나?" → Q1=B극 결과 + Q2=A극 역사 (P4 §4 ③ 역전형)
- 14 까트(ML 압축형 변주): "왜일까요?" 단일 질문 + 시그니처에 두 로드맵 압축 (P4 §4 질문 수 변주 영역)

**이문호공 배치:** 질문에 **호기심(3)** + **상식파괴(5)** 내장.

**시그니처 선언:** `00_공통_원칙.md` B.5 참조.

### P3 §3. 기능 C (A극/B극 방향 설정)

ML의 3대 기능 중 **서사 방향 설정**은 모순형에서 **A극/B극 역설 방향 설정**으로 작동한다.

**표준 작동 방식:**

- Q1은 A극(상식·긍정) 방향으로 질문 배치
- Q2는 B극(반전·부정) 방향으로 질문 배치
- 두 질문의 방향 대비 자체가 역설을 예고

**변주 허용:** Q1·Q2 방향이 표준과 다른 변주(역전형·암시적 루프 해소 등)는 **P3 §2 변주 표 + P4 §4 패턴 정의 참조** (단일 지침 문서 원칙).

**모순형 판정 기준:** 본문 A극·B극 양극 구조 + 피벗 존재로 판정한다 (P3 §4 본문 구조 정합).

### P3 §4. 본문 구조 (A극→피벗→B극) + A극 3단 vs 2단 구분

```
본문 = [A극 누적] → [피벗] → [B극 분할]
```

#### A극 누적 방식 — 3단 상승 vs 2단 구분 지표

A극을 쌓는 방식은 2가지로 구분된다. 주제 성격에 따라 선택한다.

| 구분 | 구조 | 선택 지표 | 예시 |
| --- | --- | --- | --- |
| **3단 상승** | 기초SL → 심화SL → 정점SL | A극 속성 수 4개 이상 / A극 섹션 길이 본문의 40% 이상 / 인지부담 높은 주제 | — (추후 발굴) |
| **2단 구분** | 기초 + 정점 | A극 속성 수 3개 이하 / A극 섹션 길이 본문의 30% 이하 / 직관적 주제 | 01 감자, 23 아보카도 |

#### B극 분할 원칙

B극은 **문제·원인·메커니즘 3요소**를 연속 제공한다. 3요소가 얽혀 역설을 증명한다. 상세 분할 방식은 P4 §4 참조.

**압축 금지 (단락 분리 의무):** 3요소를 한 단락 안에 동시 등장시키면 시청자 능동 추리가 무너지고 보상의 비례(공통 §B.1 ①)가 깨진다. 각 요소는 **별도 단락 또는 SL**로 분리하여 단계별로 노출한다.

| 위반 (압축형) | 정상 (분할형) |
| --- | --- |
| 한 단락에 문제·원인·메커니즘 동시 등장 | 1단계 문제 단락 → 2단계 원인 단락 → 3단계 메커니즘 단락 |
| 시청자가 정답을 한 번에 받음 | 시청자가 단계별로 스스로 추리 |
| 아하 모먼트 임팩트 분산 | 3단계 누적 후 정점 발생 |

**P5 검증:** P5 §A-1 M2가 본 분할 정합을 검증한다.

### P3 §5. A극→피벗 공백 설계 + 비율 불균형 허용 + A극 성격별 편차

#### A극 공백 설계 원리

A극 누적 단계에서 **"이렇게 좋은데 왜?"** 형태의 공백을 점진적으로 확대한다. 공백 확대의 정점에서 피벗이 발생한다.

**공백 설계 체크:**

- A극 마지막 SL에서 "이 정도면 완벽해 보이는데" 수준의 긍정 정점 형성
- 피벗 직전 공백 강도가 도입부 ML 공백의 2배 이상

#### 비율 불균형 허용 조건

원칙적으로 A극:B극 = 50:50. 단 아래 조건 충족 시 불균형 허용.

| 허용 유형 | 비율 | 조건 |
| --- | --- | --- |
| A극 우세 | A 60% · B 40% | A극이 인지부담 높아 충분한 설명 필요 + B극이 단일 강한 반전으로 압축 가능 |
| B극 우세 | A 40% · B 60% | A극이 짧은 상식 확인으로 충분 + B극이 복수 문제·원인·메커니즘을 다뤄야 함 |
| 극단 불균형 금지 | A 70%+ 또는 B 70%+ | 금지 — 역설 효과 상실 |

**"충분히 흥미로운" 정량 기준:** 피벗 후 B극에서 이문호공 2요인 중첩이 최소 3회 발생해야 B극 우세 허용.

#### A극 성격별 브릿지 편차 허용 (`00_공통_원칙.md` B.9 참조)

A극 성격에 따라 기본 레이어 전환 비율에서 편차를 허용한다.

| A극 성격 | 허용 편차 |
| --- | --- |
| **속성·현상 나열** | SL 상한 60% → **최대 86%** 허용 |
| **시간순 역사 서사** | 선언·예고형 상한 35% → **최대 57%** 허용 |
| **제도/메커니즘 설명** | 논리적 BR 상한 25% → **최대 40%** 허용 |

### P3 §6. A극 속성 B극 역전 Aha 설계

모순형 Aha-moment는 **"A극의 바로 그 속성이 B극에서 역전됨"**을 발견하는 순간이다.

**위치 원칙:** % 강제 아님. 아래 **완결 조건이 충족되는 지점**에서 아하 자연 발생.

**설계 공식:**

```
A극 속성 X (긍정 프레임) ← 누적
         ↓
피벗
         ↓
B극에서 X가 문제·위험·한계의 원인으로 재해석
         ↓
Aha! "그 X가 이런 결과를 낳았구나!"
```

#### 모순형 아하 완결 조건 체크리스트

아래 4조건 모두 충족 시 아하 배치:

1. **A극 속성 누적:** A극 속성 X가 최소 2회 이상 누적되었는가?
2. **피벗 후 재해석:** 피벗 이후 B극에서 X가 반대 결과의 원인으로 재해석되었는가?
3. **연결 단서 제공:** 시청자가 "A극 X = B극 재앙의 원인"을 스스로 연결할 단서가 제공되었는가?
4. **변주 완결 장치:** 아래 7변주 중 하나가 완결 작동하는가?

#### 모순형 7변주 — 작동 원리·사용 조건

| 변주 | 대표편 | 작동 원리 (한 줄) | 사용 조건 (한 줄) |
| --- | --- | --- | --- |
| ① 수치 대비형 | 01 감자 | A극 수혜 수치 ↔ B극 피해 수치 직접 대비 | A극·B극 양쪽에 강한 수치가 모두 존재할 때 |
| ② 관점 전환형 | 03 한반도 | 프레임 이동 (내부↔외부, 주민↔외세) | 동일 속성이 두 관점에서 정반대 의미를 가질 때 |
| ③ 메커니즘 해설형 | — (추후 발굴) | 과학적 기전으로 장점=단점의 원천 증명 | A극 속성의 물리·생물·화학 원리가 B극 결과의 원인일 때 |
| ④ 연쇄 인과 폭로형 | 14 까트 | 외부 축복이 내부 저주임 (양면성 중첩) | 대상의 특성이 외부와 내부에 정반대 결과를 낳을 때. **본문 첫 레이어는 정의·각성제 진입(기원·배경 부재) — 모순형 유일 예외**. 다른 편은 P3 §2 권장대로 기원·배경 도입 |
| ⑤ 메타 선언형 | 19 대통령제 | "뒤집어 생각하면 단점" 메타 선언 + 항목별 재해석 | A극 장점들을 항목별로 단점화할 수 있을 때 |
| ⑥ 2단 아이러니 순환형 | 23 아보카도 | A극 중반 1차 아하 + B극 2차 재역전 | 주체 역할이 "파괴자→구원자→파괴자"처럼 순환할 때 |
| ⑦ 역사 순환형 | 25 소주 | 외부 압력 연쇄 파괴 + 경제 성장 복원 | 대상이 외부 연쇄로 변형되고 후대에 복원되는 역사일 때 |

> **변주 ⑥ 2단 아이러니 순환형 — 정보 중복 회피**
>
> 1차 아하 단락 + 2차 재역전 단락 구조에서 **같은 사실을 두 단락에 걸쳐 두 번 인용하지 않는다.**
>
> - ❌ 위반: 1차 단락 "X 사실 제시" → 2차 단락 "양반이 X를 했다" (단어만 다르게 재인용)
> - ✅ 정합: 1차 단락 "X 사실 제시" → 2차 단락은 X의 **결과·재해석**으로 연결
>
> **자가 점검:** 1차·2차 단락을 나란히 놓고 같은 명사·동작·사실이 재등장하는지 검사. 재등장 시 통합 또는 결과 연결로 변경.

> **유연성:** 본문 구조 변주(A극/B극 비율, 피벗 위치, 통찰 말미 배치 등)에 따라 아하 지점은 주제마다 다르다. 위치는 % 범위가 아닌 조건 충족 여부로 판단.

**예시:**

- 23 아보카도: A극 "슈퍼푸드 인기" → B극 "바로 그 인기가 환경·사회 문제의 원인" (2단 아이러니 순환)

### P3 §7. 모순형 엔딩 궁합

**궁합 원리:** 모순형은 "A극·B극 양면성 발견"으로 착지하는 것이 자연스러우므로 **미래희망(③) + 위트(⑤) 혼합**이 주력. 단 심각한 주제(14 까트)는 경고형 단독도 허용.

엔딩 8유형 정의는 `00_공통_원칙.md` B.6 참조.

### P3 §8. 본문 결산 양식 + 결론부 진입 어법 (모순형)

본 절은 모순형 고유 결산 양식 4종 정의 + 자연 연동 매트릭스. 운영 절차·강제 규칙·5종 어법 정의는 P4 §D-1-2-B·C 위임.

#### 본문 결산 양식 4종

본문 마지막 단락(= 결론부 직전 단락)의 마무리 양식.

| 결산 양식 | 정의 |
|---|---|
| **A. 정리·복귀형** | 본문 ML 답을 결과로 정리하거나 복귀 서사로 마무리 |
| **B. 통찰 압축형** | 한 문장 통찰로 본문 결산 (예: "애증"·"이성적 판단") |
| **C. 부가정보+통찰 결합형** | 부가 정보 추가 후 통찰·호소로 마무리 |
| **D. 자기언급+갈등형** | 시청자 일상 갈등 + 자기언급 결산 |

#### 본문 결산 양식 → 결론 브릿지 어법 자연 연동

선택한 결산 양식에 따라 결론 브릿지 어법을 다음과 같이 사용한다.

| 결산 양식 | 결론 브릿지 어법 |
|---|---|
| A. 정리·복귀형 | **① 결과 호명**을 사용한다. |
| **B. 통찰 압축형** | **② 이완 호명을 필수로 사용한다.** 직접 ① 결과 호명 점프는 금지 ("성급한 마무리" 차단 — P4 §D-1-2-B 강제 규칙 2). |
| C. 부가정보+통찰 결합형 | ④ 회수 또는 ⑤ 메타를 분산 사용한다. |
| D. 자기언급+갈등형 | ② 이완 + ④ 회수를 결합 사용한다. |

#### 통찰 격상 차원

모순형 결론부에서 본문 통찰을 다음 차원 중 하나로 격상시킨다.

- **사회 차원**: 사회 구조·제도·집단 영역
- **인생 차원**: 개인·삶·인생관 영역

> **위임:** 5종 어법 정의 → P4 §D-1-2-C / 강제 규칙 4건 → P4 §D-1-2-B

### P3 §9. 암시적 루프 특수 패턴 (모순형 고유)

모순형에는 **ML에서 직접 던지지 않은 질문을 시청자가 자동 예측하여 루프로 기능**하는 특수 패턴이 있다.

#### 원리

모순형은 장점/단점, 성공/실패 같은 **대칭 구조**이므로, 한쪽만 보여줘도 시청자는 반대쪽을 자동으로 예측한다. ML이 명시하지 않아도 B극 진입이 자연스럽다.

#### 대표 사례 — 19 대통령제

- ML: "어떤 장점? 왜 채택?"
- A극 해소: Q1(장점 5가지) + Q2(신생국에 유리, 냉전 영향)
- B극 진입: "결코 장점만 있는 제도는 아닙니다" → **단점** 해소 시작
- 시청자 뇌: "장점을 이만큼 보여줬으면 당연히 단점이 나올 차례"라는 **암묵적 기대**가 형성되어 있음

#### 사용 조건

- 서사가 대칭 구조를 전제할 때 (모순형 피벗 구조)
- Q1·Q2가 A극에서 모두 해소되어 명시적 루프가 닫힌 상태

#### 주의

이미지강화형·과정추적형에서는 대칭 구조가 약하므로 암시적 루프 작동이 어렵다. **모순형 고유 패턴**이다.

| --- | --- | --- |
| Q2가 시청자 직관에 더 가까움 | Q2 답이 시청자가 쉽게 예상 가능 | — (추후 발굴) |
| Q1이 Q2 해소 후 더 흥미로워짐 | Q1 답이 Q2 답의 의미를 확장 | 14 까트: Q2(사회 해악) → Q1(48시간 약효) |
| 본문 후반 피크가 Q1 해소가 됨 | Aha-moment가 Q1에서 발생 | 메커니즘이 보상 클라이맥스 |

**설계 시점 판정:** 본문 구조 설계 시 "Aha가 Q1에서 발생하는가"를 먼저 확인. 그렇다면 역전형 적용.

#### 패턴 선택 가이드

```
1. Q1·Q2 모두 A극에서 해소되는가?
   → YES: ⑤ 암시적 루프 해소
   → NO: 다음 질문

2. B극을 문제/원인/메커니즘 순차 분할로 구성하는가?
   → YES: ④ 분할 제공 (가장 일반적)
   → NO: 다음 질문

3. Aha-moment가 Q1에서 발생하는가?
   → YES: ③ 역전형
   → NO: 다음 질문

4. Q1·Q2가 연결되는가?
   → YES: ② 교차형
   → NO: ① 순차형
```

#### ML 질문 수 변주 — ML 압축형 (질문 1개)

표준은 질문 2개 + 시그니처 선언 (필수, P3 §2)이지만, 일부 사례에서 명시적 질문이 1개로 압축되고 시그니처 선언에 두 로드맵이 통합되거나 본문에서 분할 해소되는 형태가 발견된다.

| 사례 | ML 질문 (단일) | 두 로드맵 처리 방식 |
| --- | --- | --- |
| 14 까트 | "왜일까요?" | 시그니처에 통합 — "절대 보급될 수 없는 이유 + 절대 그래서는 안 되는 이유" |
| 26 미국 원유 | "이거 뭐 대체 어떻게 된 일일까요?" | 본문 분할 해소 — 경질류 vs 중질류 메커니즘 + 정유 인프라·법적 제약 |

**사용 조건:**

- A극·B극의 두 방향이 한 단어/한 질문에 자연 결합 가능할 때 ("왜일까요?" 하나로 양극 모두 함의)
- 두 로드맵의 인과 연결이 강해 별도 질문 분리가 오히려 어색할 때
- 도입부 ③ 반전 암시가 충분히 강하여 시청자가 두 로드맵을 자동 예측할 때

**금지 (게으름·축약 차단):**

- 명확한 사용 조건 없이 단순 분량 축약 목적으로 1개로 압축하면 안 됨
- 압축 시 시그니처 선언에 두 로드맵의 단서를 반드시 명시 또는 본문 진입부에서 즉시 두 갈래를 명시

#### ML 질문 역할 변주 — 배경+통합형

표준은 Q1=A극, Q2=B극 (P3 §3)이지만, 일부 사례에서 Q1이 공통 배경(역사·기원·정의)을 묻고 Q2가 A극·B극을 통합 비교하는 형태가 발견된다.

| 사례 | Q1 (공통 배경) | Q2 (A+B 통합) |
| --- | --- | --- |
| 01 감자 | "언제부터 먹어 왔을까?" | "어떤 장단점?" |

**사용 조건:**

- A극·B극이 한 대상의 두 측면이라 통합 비교가 자연스러울 때 (감자 = 장점·단점이 같은 대상의 양면)
- 공통 배경(역사·기원·정의)이 본문 진입에 필요한 사전 정보일 때
- 본문이 "배경 → A극 → 피벗 → B극" 4단 구조로 풀릴 때

**금지:**

- 단순 배경 정보가 본문 진입에 불필요한데 Q1을 배경 질문으로 채우면 안 됨 (이탈 위험)

### P4 §5. 모순형 편별 해소 사례

모순형 **해소 은유 공식**(`00_공통_원칙.md` C.1) 적용 시 **보조 요인은 경고(7)가 최다**. 6편 편별 사례:

| 대본 | 해소 문장 | 이문호공 2요인 |
| --- | --- | --- |
| 01 감자 | "**주식의 왕좌를 위협한 그 작물**이 사실은 인류를 먹여 살렸다" | 상식파괴(5) + 경고(7) |
| 03 한반도 | "저주받은 땅이 아니라 **살아남은 땅**" | 상식파괴(5) + 공감(4) |
| 14 까트 | "**커피의 탈을 쓴 지옥의 문**" | 상식파괴(5) + 경고(7) |
| 19 대통령제 | "**대통령 없는 게 더 흔한 정상**" | 상식파괴(5) + 비교(8) |
| 23 아보카도 | "**슈퍼푸드의 이름 아래 치러지는 대가**" | 상식파괴(5) + 경고(7) |
| 25 소주 | "**순수해 보이지만 사실은 블렌딩된 칵테일**" | 상식파괴(5) + 반전(6) |

**적용 원칙:**

- 상식파괴(5)는 모순형 전수 적용 (필수 요인)
- 경고(7)는 보조 요인 (`00_공통_원칙.md` §C.1)
- 배치: 중첩(1문장에 2요인)

### P4 §6. 모순형 비유 특화 예시

모순형 비유는 **"역설 압축"** 역할 (`00_공통_원칙.md` C.7 4역할 중 하나)을 수행한다.

모순형 도입부 ML 단계에서 비유는 A극↔B극 역설을 한 표현에 압축할 때 사용한다. 14 까트 "커피의 탈을 쓴 지옥의 문"은 본문 해소 은유로 도입부 ML 단계 비유에는 포함하지 않는다.

#### 모순형 비유 작성 원칙

- A극·B극 속성을 **한 비유에 동시 담기**
- "탈을 쓴", "얼굴을 한", "가면 뒤의" 등 이중성 표현 활용
- 종교·신화·극단어 활용 ("저주", "지옥", "악마", "신의" 등)

#### 편별 비유 예시

| 대본 | 비유 | 충돌 구조 |
| --- | --- | --- |
| 14 까트 | 커피의 탈을 쓴 지옥의 문 | 익숙함(A) + 위험(B) |
| 23 아보카도 | 뭔놈의 과일이 사람을 죽이고 | 과일(A) + 살인(B) |

### P4 §7. A극 강화 변주 4종 (피벗 직전 확신 극대화)

A극 누적이 길어질 때 피벗 직전 확신을 최고조로 끌어올려 피벗 낙차를 극대화하는 기법.

#### 4종 변주

| # | 변주 | 정의 | 사용 조건 | 실제 대본 예시 |
| --- | --- | --- | --- | --- |
| ① | **확신 극대화** | 피벗 직전 A극 확신을 최고조로 끌어올려 피벗 낙차 극대화 | A극이 속성·역사 나열이고 피벗에서 강한 낙차가 필요할 때 | `인류의 구원이나 다름없는 작물이죠` → 직후 `이랬다가 진짜 큰일 날 수도` |
| ② | **시청자 대입** | 시청자 본인 상황으로 맥락 전환하여 추상적 정보를 체감으로 변환 | A극이 추상적 제도·정치·사회 주제일 때 | `어 듣다 보니까 그냥 우리나라 얘기죠` |
| ③ | **A극 내 미니 역설** | A극 안에 작은 역설을 삽입하여 관심 재점화 (모순형적 요소를 A극에 심음) | A극이 시간순 성공 스토리일 때 단조로움 방지 | `아이러니하게도 아보카도가 멸종을 면한 것 역시 인간 때문` |
| ④ | **질문→확인 2단** | 시청자에게 질문을 유도하고 직후 단호하게 확인사격 | A극에서 시청자의 의심을 선제 처리해야 할 때 | `이쯤 되면 의문이 하나 생기죠. 마약이 맞냐?` (질문) → 효과 설명 → `누가 봐도 마약 맞죠` (확인) |

#### 선택 가이드 (A극 성격별)

| A극 성격 | 권장 강화 변주 |
| --- | --- |
| 속성·현상 나열 | ① 확신 극대화 + ④ 질문→확인 |
| 시간순 성공 스토리 | ③ A극 내 미니 역설 |
| 추상적 제도·정치 | ② 시청자 대입 |

### P4 §8. 본문 주력 브릿지 (모순형)

#### 주력 서브 오픈 루프 (Type A-1)

- **완벽함 뒤의 함정** (최우선): `결론부터 말하면 이랬다가 진짜 큰일 날 수도`
- **종결 부정형** (A극 누적 중): `~면 끝이냐? 아니다` 반복

#### 주력 SL 선언·예고형 (Type A-2 / 구 심화 브릿지)

- **"진짜는 지금부터" 형**: `진짜 핵심은 지금부터입니다` (가상 예시 — 추후 25편 사례 발굴 예정, 피벗 직전 위치)
- **"결정적 사건" 형**: `결정적인 사건은 역시 18세기 말에 일어난 산업 혁명이었습니다`

#### 피벗/정점 유형

- **피벗 = 완벽함 뒤 함정 + 사회적 통념 반전** 조합 (Type A-1 2종 교차)
