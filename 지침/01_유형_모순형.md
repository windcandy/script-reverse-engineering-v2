## P1. 썸네일

### P1.1 모순형 7편 썸네일 분류 결과

모순형 7편의 썸네일 전략은 **조각 제시형이 71%(5/7)**로 주력이고, **모순 공존형**(04·14)이 29%(2/7)로 사용된다. 결과 제시형은 거의 없음.

| 대본 | 썸네일 문구 | 판정 | 근거 |
|---|---|---|---|
| 01 감자 | "전세계를 먹여 살리는 이 작물의 숨겨진 진실?" | 조각 제시형 | `~진실?` 질문 |
| 03 한반도 | "한반도는 얼마나 저주받은 땅일까?" | 조각 제시형 | `얼마나 ~?` 질문 |
| 04 석면 | "**신의 물질** — 왜 금지됐나?" | **모순 공존형** | 신적(긍정) + 물질(금지·위험) 공존 |
| 14 까트 | "**악마의 풀** 까트의 정체" | **모순 공존형** | 악마(부정) + 풀(식물) 공존 |
| 19 대통령제 | "대통령 없는 나라가 더 많다고?" | 조각 제시형 | `~고?` 질문 |
| 23 아보카도 | "이 과일이 사람을 죽인다고?" | 조각 제시형 | `~고?` 질문 |
| 25 소주 | "소주의 진짜 정체는?" | 조각 제시형 | `~는?` 질문 |

#### 모순 공존형 작동 원리 (모순형 전용)

A극 속성(긍정·신적·절대적)과 B극 속성(부정·위험·금지)을 **썸네일 문구 한 구절에 동시 배치**하여 즉각적 충돌을 일으킨다. 시청자는 "왜 이 대상에 상반된 속성이 공존하는가?"를 본능적으로 추리하게 된다.

**작성 원칙:**

- A극 수식어 + B극 명사 조합 (예: "신의 물질", "악마의 풀")
- A극·B극 간 의미 거리가 멀수록 충돌 강도↑
- 조각 제시형과 달리 질문 문형이 없어도 됨 — 충돌 자체가 공백을 생성

판정 기준 세부는 `00_공통_원칙.md` A.1 참조.

## P3. 구조 설계

### P3 §1. 도입부 4요소 + 4가지 구조 (7편 실측 기반)

모순형 도입부는 **4가지 필수 요소**를 반드시 포함하되, **순서와 압축**은 주제·전략에 따라 선택한다. 7편 전수 분석 결과 단일 고정 구조가 아니라 4가지 구조가 병존한다.

**필수 4요소:**

- ① **상식적 맥락 제시 (A극 또는 시청자 인식)** — 시청자 당연시 상식·인식·대비 축
- ② **썸네일 약속 재확인** — 썸네일 문구를 동의어 치환으로 재확인
- ③ **반전 암시 (B극 예고)** — "그런데…" "사실은…"으로 B극 존재 암시
- ④ **ML (질문형)** — 질문 2개 + 시그니처 선언 (§2 참조)

**순서 원칙:**

- ③ → ④는 **고정**
- ① ↔ ② 순서는 주제 조건에 따라 **교체 가능**

**4가지 구조 (7편 실측 분포):**

| 구조 | 순서 | 실측 편수 | 선택 조건 | 대표 사례 |
|---|---|---|---|---|
| **기본형** | ① → ② → ③ → ④ | **3편 (43%, 최다)** | 대비 축을 먼저 세워야 하는 주제 | 14 까트·19 대통령제·25 소주 |
| **대안형** | ② → ① → ③ → ④ | 1편 (14%) | A극 상식과 B극이 시·인과적으로 명확 분리 | 03 한반도 |
| **압축형** (3단) | ① → (②+③) → ④ | 1편 (14%) | 썸네일·반전이 한 문장에 자연 결합 | 23 아보카도 |
| **예외형** (3단, A 전용) | ① → (스케일 폭로+③) → ④ | 1편 (14%) | A 규모폭로 전략 + 썸네일이 수치로 즉시 증명 | 01 감자 |

**집필 판단 플로우 (4단계 질문):**

1. A 규모폭로 전략? → **예외형**
2. 대비 축(커피 vs 까트형)을 먼저 세워야 하나? → **기본형(역순)**
3. 썸네일·반전이 한 문장에 자연 결합 가능? → **압축형**
4. A극·B극 시·인과적 분리 명확? → **대안형(원순)**

**특수 예외 (기본 체계 외):**

- **D 속성폭격형** (04 석면, 1편): [속성 5개 이상 나열 → 썸네일 지위 선언] → [상식] → [반전+신뢰상환] → [ML] 4단. D 전략은 25편 중 1편만 사용된 저빈도·특수 조건으로 **집필 시 일반 선택지로 사용하지 않는다.** 단 본문 차원의 변주 ③ 메커니즘 해설형(04 석면 본문 패턴)은 일반 선택지로 사용 가능 — 도입부 차원의 D 전략과 본문 차원의 변주 분류는 별개 층위이기 때문이다 (P3 §6 변주 표 참조). 분석 파일(`[추출] 대본 - 분석/01_도입부_분석_모순형.md` §2 특수 예외) 참조.

**단계별 분량 목표 (4단 구조 기준):**

| 요소 | 분량 비중 |
|---|---|
| ① 상식적 맥락 제시 | 도입부의 40~50% |
| ② 썸네일 약속 재확인 | 도입부의 10~15% |
| ③ 반전 암시 | 도입부의 15~25% |
| ④ ML | 도입부의 15~20% |

3단 구조(압축형·예외형)는 ①+③ 또는 스케일+③이 결합되므로 결합 블록 분량은 위 표의 ①+③ 합계를 따른다.

**신뢰 상환 위치:** **본문 이연** (원칙). 모순형은 역설 암시만으로 도입부 성립. 신뢰 상환(썸네일 약속 증명) 지점은 **구조 변주·주제 성격에 따라 다르다** — 6편 실측 4가지 패턴:

| 변주 패턴 | 상환 위치 | 해당 편 |
|---|---|---|
| **A극 자체가 상환** | 본문 10~25% | 03 한반도·23 아보카도 |
| **A극+B극 분할** | 본문 30~45% | 25 소주 |
| **피벗 직후 B극** | 본문 44% | 14 까트 |
| **B극 분할 투하** | 본문 50~68% | 01 감자·19 대통령제 |

**세부 위치를 강제하지 않는다** — 주제·변주에 맞는 자연스러운 위치 선택. 실측 가이드는 `[추출] 대본 - 분석/09_신뢰_상환_위치_분석.md` §7 참조. 단 D 속성폭격형은 도입부 수행(특수 예외).

**도입부 글자수 (공백 포함):** 300~750자 (평균 632자). 상한 초과는 드물게 허용 (예: 03 한반도 793자).

**예시 참조:** 4가지 구조의 대표 예시는 `00_지식파일/인트로_예시_전략별.md` B.1 (대안형) · B.2 (기본형) · B.3 (압축형) · A.1 (예외형) + 특수 예외 D.1 (속성폭격).

**도입부 공통 골격(인트로·ML 2블록 + 신뢰 상환 기능)은 `00_공통_원칙.md` B.4 참조.**

### P3 §2. 모순형 ML 특유 구조 [Q2+S1] + 4단계 구현

```
ML = 질문 2개 + 시그니처 선언 1개 (고정)
```

**질문 2개의 역할:** 본문 **A극 파트(Q1) + B극 파트(Q2)의 로드맵**. 각 질문은 해당 파트에서 해소된다.

**예시:**

- 01 감자: "언제부터? 장단점?" → A극 역사 + B극 비교분석
- 04 석면: "어떻게 그렇게 쓰였나? 왜 금지됐나?" → A극 쓰임 + B극 금지 이유
- 14 까트: "정체가 뭔가? 왜 못 끊나?" → A극 정체 + B극 중독성

**이문호공 배치:** 질문에 **호기심(3)** + **상식파괴(5)** 내장. 모순형은 상식파괴 사용률이 3유형 중 최고.

**시그니처 선언:** "~알아봅니다" / "~알아보았습니다" 종결. `00_공통_원칙.md` B.5 참조.

#### 모순형 ML 4단계 구현

ML은 도입부에서 열어 본문 후반부에서 닫는 생애주기 4단계를 따른다 (`00_공통_원칙.md` B.7 참조). 모순형의 단계별 구현:

| 단계 | 모순형 구현 |
|---|---|
| **열기** | 도입부 ML에서 Q1·Q2 선언 |
| **유지** | **Q2를 보류**하여 공백 유지 (Q1은 A극에서 분산 해소) |
| **강화** | **피벗 직전 A극 확신 극대화** (P4 §7 강화 변주 4종 참조) |
| **닫기** | B극에서 **3~4회 분할 제공** (실측 평균 3.3회). ML 순서 역전 가능 (P4 §4 참조) |

### P3 §3. 기능 C (A극/B극 방향 설정)

ML의 3대 기능 중 **서사 방향 설정**은 모순형에서 **A극/B극 역설 방향 설정**으로 작동한다.

**작동 방식:**

- Q1은 A극(상식·긍정) 방향으로 질문 배치
- Q2는 B극(반전·부정) 방향으로 질문 배치
- 두 질문의 방향 대비 자체가 역설을 예고

**체크:** Q1과 Q2가 같은 방향이면 모순형이 아니다. 서사 유형 재검토 필요.

### P3 §4. 본문 구조 (A극→피벗→B극) + A극 3단 vs 2단 구분

```
본문 = [A극 누적] → [피벗] → [B극 분할]
```

#### A극 누적 방식 — 3단 상승 vs 2단 구분 지표 (모호성 04-01 해결)

A극을 쌓는 방식은 2가지로 구분된다. 주제 성격에 따라 선택한다.

| 구분 | 구조 | 선택 지표 | 예시 |
|---|---|---|---|
| **3단 상승** | 기초SL → 심화SL → 정점SL | A극 속성 수 4개 이상 / A극 섹션 길이 본문의 40% 이상 / 인지부담 높은 주제 | 04 석면 (속성 5개 + 산업 사례) |
| **2단 구분** | 기초 + 정점 | A극 속성 수 3개 이하 / A극 섹션 길이 본문의 30% 이하 / 직관적 주제 | 01 감자, 23 아보카도 |

#### B극 분할 원칙

B극은 **문제·원인·메커니즘 3요소**를 연속 제공한다. 3요소가 얽혀 역설을 증명한다. 상세 분할 방식은 P4 §4 참조.

### P3 §5. A극→피벗 공백 설계 + 비율 불균형 허용 + A극 성격별 편차

#### A극 공백 설계 원리

A극 누적 단계에서 **"이렇게 좋은데 왜?"** 형태의 공백을 점진적으로 확대한다. 공백 확대의 정점에서 피벗이 발생한다.

**공백 설계 체크:**

- A극 마지막 SL에서 "이 정도면 완벽해 보이는데" 수준의 긍정 정점 형성
- 피벗 직전 공백 강도가 도입부 ML 공백의 2배 이상

#### 비율 불균형 허용 조건 (모호성 04-02 해결)

원칙적으로 A극:B극 = 50:50. 단 아래 조건 충족 시 불균형 허용.

| 허용 유형 | 비율 | 조건 |
|---|---|---|
| A극 우세 | A 60% · B 40% | A극이 인지부담 높아 충분한 설명 필요 + B극이 단일 강한 반전으로 압축 가능 |
| B극 우세 | A 40% · B 60% | A극이 짧은 상식 확인으로 충분 + B극이 복수 문제·원인·메커니즘을 다뤄야 함 |
| 극단 불균형 금지 | A 70%+ 또는 B 70%+ | 금지 — 역설 효과 상실 |

**"충분히 흥미로운" 정량 기준:** 피벗 후 B극에서 이문호공 2요인 중첩이 최소 3회 발생해야 B극 우세 허용.

#### A극 성격별 브릿지 편차 허용 (`00_공통_원칙.md` B.9 참조)

A극 성격에 따라 기본 레이어 전환 비율에서 편차를 허용한다.

| A극 성격 | 허용 편차 | 실전 근거 |
|---|---|---|
| **속성·현상 나열** | SL 상한 60% → **최대 86%** 허용 | 03 한반도 (종결 부정형 3연속) |
| **시간순 역사 서사** | 심화 BR 상한 35% → **최대 57%** 허용 | 25 소주, 04 석면 ("결정타", "진짜는 지금부터") |
| **제도/메커니즘 설명** | 논리적 BR 상한 25% → **최대 40%** 허용 | 19 대통령제 (인과 연쇄 필수) |

### P3 §6. A극 속성 B극 역전 Aha 설계

모순형 Aha-moment는 **"A극의 바로 그 속성이 B극에서 역전됨"**을 발견하는 순간이다.

**위치 원칙:** % 강제 아님. 아래 **완결 조건이 충족되는 지점**에서 아하 자연 발생. 실측 분포 55~93%(편차 큼) — 참고만.

**설계 공식:**

```
A극 속성 X (긍정 프레임) ← 누적
         ↓
피벗
         ↓
B극에서 X가 문제·위험·한계의 원인으로 재해석
         ↓
Aha! "그 X가 이런 결과를 낳았구나!"
```

#### 모순형 아하 완결 조건 체크리스트

아래 4조건 모두 충족 시 아하 배치:

1. **A극 속성 누적:** A극 속성 X가 최소 2회 이상 누적되었는가?
2. **피벗 후 재해석:** 피벗 이후 B극에서 X가 반대 결과의 원인으로 재해석되었는가?
3. **연결 단서 제공:** 시청자가 "A극 X = B극 재앙의 원인"을 스스로 연결할 단서가 제공되었는가?
4. **변주 완결 장치:** 아래 7변주 중 하나가 완결 작동하는가?

#### 모순형 7변주 — 작동 원리·사용 조건

| 변주 | 대표편 | 작동 원리 (한 줄) | 사용 조건 (한 줄) |
|---|---|---|---|
| ① 수치 대비형 | 01 감자 | A극 수혜 수치 ↔ B극 피해 수치 직접 대비 | A극·B극 양쪽에 강한 수치가 모두 존재할 때 |
| ② 관점 전환형 | 03 한반도 | 프레임 이동 (내부↔외부, 주민↔외세) | 동일 속성이 두 관점에서 정반대 의미를 가질 때 |

| ④ 연쇄 인과 폭로형 | 14 까트 | 외부 축복이 내부 저주임 (양면성 중첩) | 대상의 특성이 외부와 내부에 정반대 결과를 낳을 때. **본문 첫 레이어는 정의·각성제 진입(기원·배경 부재) — 7편 중 유일한 예외**(다른 6편은 P3 §2 권장대로 기원·배경 도입) |
| ⑤ 메타 선언형 | 19 대통령제 | "뒤집어 생각하면 단점" 메타 선언 + 항목별 재해석 | A극 장점들을 항목별로 단점화할 수 있을 때 |
| ⑥ 2단 아이러니 순환형 | 23 아보카도 | A극 중반 1차 아하 + B극 2차 재역전 | 주체 역할이 "파괴자→구원자→파괴자"처럼 순환할 때 |
| ⑦ 역사 순환형 | 25 소주 | 외부 압력 연쇄 파괴 + 경제 성장 복원 | 대상이 외부 연쇄로 변형되고 후대에 복원되는 역사일 때 |

편별 실측 상세(아하 문장·트리거·선행 맥락)는 `[추출] 대본 - 분석/11_아하모먼트_재정의.md` §1 참조.

### P3 §8. 썸네일 판정 경계 사례 처리 (모호성 01-03 해결)

복합 문형 썸네일의 판정 원칙을 추가한다.

#### 복합 문형 처리 규칙

| 문형 예시 | 판정 | 근거 |
|---|---|---|
| "A의 B가 된 이유" | 조각 제시형 | `이유` = 부품 던지기 |
| "A는 사실 B이다" | 결과 제시형 | `~이다` 단정 |
| "신의 A — 왜 금지?" | 모순 공존형 + 조각 | A극(신의)+B극(금지) 공존 + 질문 |
| "A의 B한 C" | 모순 공존형 후보 | B가 A와 의미 충돌할 때만. 단순 수식은 결과 제시형 |

**판정 우선순위:** 모순 공존형 > 조각 제시형 > 결과 제시형.

**경계 판정 기준:**

1. A극·B극 속성 충돌이 명시적이면 **모순 공존형 우선**
2. 질문 문형(? · 이유 · 정체 등)이 있으면 **조각 제시형**
3. 단정 선언이 있으면 **결과 제시형**

### P3 §9. 암시적 루프 특수 패턴 (모순형 고유)

모순형에는 **ML에서 직접 던지지 않은 질문을 시청자가 자동 예측하여 루프로 기능**하는 특수 패턴이 있다.

#### 원리

모순형은 장점/단점, 성공/실패 같은 **대칭 구조**이므로, 한쪽만 보여줘도 시청자는 반대쪽을 자동으로 예측한다. ML이 명시하지 않아도 B극 진입이 자연스럽다.

#### 대표 사례 — 19 대통령제

- ML: "어떤 장점? 왜 채택?"
- A극 해소: Q1(장점 5가지) + Q2(신생국에 유리, 냉전 영향)
- B극 진입: "결코 장점만 있는 제도는 아닙니다" → **단점** 해소 시작
- 시청자 뇌: "장점을 이만큼 보여줬으면 당연히 단점이 나올 차례"라는 **암묵적 기대**가 형성되어 있음

#### 사용 조건

- 서사가 대칭 구조를 전제할 때 (모순형 피벗 구조)
- Q1·Q2가 A극에서 모두 해소되어 명시적 루프가 닫힌 상태

#### 주의

이미지강화형·과정추적형에서는 대칭 구조가 약하므로 암시적 루프 작동이 어렵다. **모순형 고유 패턴**이다.

- B극 존재를 명시적 언급(직접 인용 아님)

기본: 반전 암시는 도입부에서 완료. 단 아래 조건 충족 시 본문 10% 지점까지 이연 허용.

| 조건 | 기준 |
|---|---|
| A극이 너무 복잡하여 도입부에서 충분한 상식 확인이 어려움 | A극 속성 4개 이상 + 각 속성이 상호 의존적 |

**예시:** 01 감자 — 썸네일 "숨겨진 진실" + A극 역사가 복잡하여 도입부에서는 "장단점이 있다"로 암시, 본문 초반에서 B극 반전 명시.

### P4 §3. 피벗 브릿지 7유형 + 선택 가이드 (모호성 04-03 해결)

A극→B극 전환에 사용하는 피벗 브릿지 7유형.

#### 7유형 정의

| # | 유형 | 예문 | 효과 |
|---|---|---|---|
| 1 | 직접 선언형 | "그런데 사실은..." | 간결·단호 |
| 2 | 반어형 | "근데 이 좋은 게 왜 사라졌을까요" | 호기심 유발 |
| 3 | 단호형 | "하지만 이게 전부가 아닙니다" | 긴장 고조 |
| 4 | 대비 선언형 | "빛이 있으면 그림자도 있는 법" | 프레임 전환 |
| 5 | 유보형 | "다만 모든 게 완벽하진 않았어요" | 부드러운 전환 |
| 6 | 질문 투하형 | "근데 왜 지금은 사라졌을까요?" | 직접 공백 생성 |
| 7 | 증거 역전형 | "그런데 이런 수치가 나왔습니다" | 팩트로 반전 |

#### A극 성격 → 피벗 유형 매핑 가이드

| A극 성격 | 권장 피벗 유형 | 이유 |
|---|---|---|
| A극이 감정적·공감적 | 5 유보형 / 2 반어형 | 감정 이탈 방지 |
| A극이 수치·팩트 중심 | 1 직접 선언형 / 7 증거 역전형 | 팩트로 대응 |
| A극이 오래 누적 (3단 상승) | 3 단호형 / 6 질문 투하형 | 긴 A극 이후 강한 전환 필요 |
| A극이 짧음 (2단 구분) | 1 직접 선언형 / 4 대비 선언형 | 간결 전환 |
| A극이 극단 긍정 | 4 대비 선언형 / 2 반어형 | 극단 대비 효과 |

**복수 조합 가능:** 주력 1 + 보조 1로 2단 피벗도 허용.

### P4 §4. ML Q1·Q2 해소 5패턴 + 순서 역전 판정 (모호성 04-04 해결)

모순형 ML Q1·Q2 해소 방식은 총 5패턴으로 정리된다 (기존 3패턴 + 닫기 변주 2패턴).

#### 5패턴

| 패턴 | Q1 해소 위치 | Q2 해소 위치 | 특징 | 사용 조건 |
|---|---|---|---|---|
| **① 순차형** | A극 완료 시점 | B극 완료 시점 | 표준 | 기본. Q1·Q2가 독립적일 때 |
| **② 교차형** | A극~피벗 사이 | B극 중반 | Q1·Q2 연결 | Q1·Q2가 연결될 때 |
| **③ 역전형** | B극 초반 | A극 후반 | Q1·Q2 순서 뒤집기 | 특수 조건 (아래 참조) |
| **④ 분할 제공** | B극에서 문제/원인/메커니즘을 **3~4회 순차 제공** (실측 평균 3.3회) | — | **B극 분할 방식** | Q2가 명시적으로 B극에 해소되는 일반적 경우 (예: 01 감자 전염병→대기근→재배난이도→보관한계) |
| **⑤ 암시적 루프 해소** | Q1·Q2 모두 A극에서 해소 | — | B극이 "예측 가능한 역전"을 수행 | ML Q1·Q2가 모두 A극에서 해소되고 B극이 대칭 역전일 때 (P3 §9 참조) |

#### 순서 역전 사전 판정 기준 (패턴 ③ 역전형)

Q1·Q2 순서가 역전되는 조건:

| 조건 | 기준 | 예시 |
|---|---|---|
| Q2가 시청자 직관에 더 가까움 | Q2 답이 시청자가 쉽게 예상 가능 | 04 석면: Q1(메커니즘)·Q2(발견 과정) → Q2가 먼저 |
| Q1이 Q2 해소 후 더 흥미로워짐 | Q1 답이 Q2 답의 의미를 확장 | 14 까트: Q2(사회 해악) → Q1(48시간 약효) |
| 본문 후반 피크가 Q1 해소가 됨 | Aha-moment가 Q1에서 발생 | 메커니즘이 보상 클라이맥스 |

**설계 시점 판정:** 본문 구조 설계 시 "Aha가 Q1에서 발생하는가"를 먼저 확인. 그렇다면 역전형 적용.

#### 패턴 선택 가이드

```
1. Q1·Q2 모두 A극에서 해소되는가?
   → YES: ⑤ 암시적 루프 해소
   → NO: 다음 질문

2. B극을 문제/원인/메커니즘 순차 분할로 구성하는가?
   → YES: ④ 분할 제공 (가장 일반적)
   → NO: 다음 질문

3. Aha-moment가 Q1에서 발생하는가?
   → YES: ③ 역전형
   → NO: 다음 질문

4. Q1·Q2가 연결되는가?
   → YES: ② 교차형
   → NO: ① 순차형
```

|---|---|---|
| 01 감자 | "**주식의 왕좌를 위협한 그 작물**이 사실은 인류를 먹여 살렸다" | 상식파괴(5) + 경고(7) |
| 03 한반도 | "저주받은 땅이 아니라 **살아남은 땅**" | 상식파괴(5) + 공감(4) |
| 04 석면 | "**무적이지만 몸에 들어오면 저주**" | 상식파괴(5) + 경고(7) |
| 14 까트 | "**커피의 탈을 쓴 지옥의 문**" | 상식파괴(5) + 경고(7) |
| 19 대통령제 | "**대통령 없는 게 더 흔한 정상**" | 상식파괴(5) + 비교(8) |
| 23 아보카도 | "**슈퍼푸드의 이름 아래 치러지는 대가**" | 상식파괴(5) + 경고(7) |
| 25 소주 | "**순수해 보이지만 사실은 블렌딩된 칵테일**" | 상식파괴(5) + 반전(6) |

**통계:**

- 상식파괴(5) 사용률: 7/7 = 100%
- 경고(7) 사용률: 4/7 ≈ 57% (3유형 중 최고)
- 배치: 전부 **중첩**(1문장에 2요인)

### P4 §6. 모순형 비유 특화 예시

모순형 비유는 **"역설 압축"** 역할 (`00_공통_원칙.md` C.7 4역할 중 하나)을 수행한다.

**도입부 ML 단계 비유 사용률 2/7 = 29% (3유형 중 최저).** 분모 = 모순형 전체 7편 중 도입부 ML 단계 비유 사용 편수 2편 (04 석면 "금단의 물질" · 23 아보카도 "뭔놈의 과일이 사람을 죽이고"). 14 까트 "커피의 탈을 쓴 지옥의 문"은 본문 닫기 해소 비유로, 도입부 ML 단계 비유 사용 편수에는 포함하지 않는다.

#### 모순형 비유 작성 원칙

- A극·B극 속성을 **한 비유에 동시 담기**
- "탈을 쓴", "얼굴을 한", "가면 뒤의" 등 이중성 표현 활용
- 종교·신화·극단어 활용 ("저주", "지옥", "악마", "신의" 등)

#### 편별 비유 예시

| 대본 | 비유 | 충돌 구조 |
|---|---|---|
| 04 석면 | 금단의 물질 / 저주 | 쓰임(A) + 위험(B) |
| 14 까트 | 커피의 탈을 쓴 지옥의 문 | 익숙함(A) + 위험(B) |
| 23 아보카도 | 뭔놈의 과일이 사람을 죽이고 | 과일(A) + 살인(B) |

### P4 §7. A극 강화 변주 4종 (피벗 직전 확신 극대화)

A극 누적이 길어질 때 피벗 직전 확신을 최고조로 끌어올려 피벗 낙차를 극대화하는 기법.

#### 4종 변주

| # | 변주 | 정의 | 사용 조건 | 실제 대본 예시 |
|---|---|---|---|---|
| ① | **확신 극대화** | 피벗 직전 A극 확신을 최고조로 끌어올려 피벗 낙차 극대화 | A극이 속성·역사 나열이고 피벗에서 강한 낙차가 필요할 때 | `인류의 구원이나 다름없는 작물이죠` → 직후 `이랬다가 진짜 큰일 날 수도` (01 감자) / `이건 양반이었다` → `담배 필터까지` → `다들 아시죠? 발암물질` (04 석면) |
| ② | **시청자 대입** | 시청자 본인 상황으로 맥락 전환하여 추상적 정보를 체감으로 변환 | A극이 추상적 제도·정치·사회 주제일 때 | `어 듣다 보니까 그냥 우리나라 얘기죠` (19 대통령제) |
| ③ | **A극 내 미니 역설** | A극 안에 작은 역설을 삽입하여 관심 재점화 (모순형적 요소를 A극에 심음) | A극이 시간순 성공 스토리일 때 단조로움 방지 | `아이러니하게도 아보카도가 멸종을 면한 것 역시 인간 때문` (23 아보카도) |
| ④ | **질문→확인 2단** | 시청자에게 질문을 유도하고 직후 단호하게 확인사격 | A극에서 시청자의 의심을 선제 처리해야 할 때 | `이쯤 되면 의문이 하나 생기죠. 마약이 맞냐?` (질문) → 효과 설명 → `누가 봐도 마약 맞죠` (확인) (14 까트) |

#### 선택 가이드 (A극 성격별)

| A극 성격 | 권장 강화 변주 |
|---|---|
| 속성·현상 나열 | ① 확신 극대화 + ④ 질문→확인 |
| 시간순 성공 스토리 | ③ A극 내 미니 역설 |
| 추상적 제도·정치 | ② 시청자 대입 |

### P4 §8. 본문 주력 브릿지 (모순형)

본문 레이어 전환 시 모순형 주력 브릿지 조합. (`00_공통_원칙.md` C.9 브릿지 Type A/B 세부 정의 참조)

#### 주력 서브 오픈 루프 (Type A-1)

- **완벽함 뒤의 함정** (최우선): `결론부터 말하면 이랬다가 진짜 큰일 날 수도` (01 감자)
- **종결 부정형** (A극 누적 중): `~면 끝이냐? 아니다` 반복 (03 한반도, 모순형 속성 나열형)

#### 주력 심화 브릿지 (Type A-2)

- **"진짜는 지금부터" 형**: `진짜 무서운 건 지금부터입니다` (04 석면 피벗 직전)
- **"결정적 사건" 형**: `결정적인 사건은 18세기 말 산업 혁명` (04 석면 전환점)
- **"양반이었다" 형**: `이건 양반이었다는 거` (04 석면 A극 계단식 확장)

#### 피벗/정점 유형

- **피벗 = 완벽함 뒤 함정 + 사회적 통념 반전** 조합 (Type A-1 2종 교차)

#### A극 성격별 브릿지 조정 (`00_공통_원칙.md` C.10 참조)

| A극 성격 | 브릿지 조정 |
|---|---|
| 속성·현상 나열 | 종결 부정형 증가. "~면 되냐? 아니다" 3연속 허용 |
| 시간순 역사 서사 | 심화 BR 증가. "결정적 사건", "마침내", "진짜는 지금부터" |
