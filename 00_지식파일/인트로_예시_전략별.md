> **모순형 도입부 체계 (필수 4요소 + 순서·압축 원칙, 7편 실측 기반):**
>
> 모순형 도입부는 4가지 필수 요소를 반드시 포함한다:
>
> - ① **상식적 맥락 제시 (A극 또는 시청자 인식)** — 시청자 당연시 상식·인식·대비 축
> - ② **썸네일 약속 재확인** — 썸네일의 미끼를 도입부에서 동의어 치환으로 확인
> - ③ **반전 암시 (B극 예고)** — "그런데…" "사실은…"으로 B극 존재 암시
> - ④ **ML (질문형)** — 질문 2개 + 시그니처 선언 (표준). 변주 4종(ML 압축형·역전형·암시적 루프 해소·배경+통합형) 허용 — `지침/01_유형_모순형.md` P3 §2 + P4 §4 참조.
>
> **순서 원칙:** ③→④는 고정. ①↔② 순서는 주제 조건에 따라 교체.
>
> **4가지 구조:**
>
> - **기본형** (실측 최다): `① → ② → ③ → ④` — 대비 축을 먼저 세워야 하는 주제
> - **대안형**: `② → ① → ③ → ④` — A극·B극이 시·인과적으로 명확 분리되는 주제
> - **압축형** (②+③ 결합 3단): `① → (②+③) → ④` — 썸네일·반전이 한 문장에 자연 결합되는 주제
> - **예외형** (A 규모폭로 전용, 3단): `① → (스케일 폭로+③) → ④` — 썸네일이 수치로 즉시 증명되는 주제
>
> **집필 판단 플로우:**
>
> 1. A 규모폭로 전략? → 예외형
> 2. 대비 축(커피 vs 까트형)을 먼저 세워야? → 기본형
> 3. 썸네일·반전이 한 문장 압축 가능? → 압축형
> 4. A극·B극 시·인과적 분리? → 대안형
>
>
>---
> **이미지 강화형 도입부 체계 (필수 3요소 + 3가지 구조, 10편 실측 기반):**
>
> 이미지 강화형 도입부는 3가지 필수 요소를 반드시 포함하되, 2단계 구현 방식은 주제 조건에 따라 선택한다.
>
> - ① **1단계: 스케일 충격 or 공감 진입**
> - ② **2단계: 숨겨진 메커니즘 암시 / 권위 인용 / 미래 제시**
> - ③ **ML (질문형)**
>
> **3가지 구조:**
>
> - **표준 3단** (실측 최다, 60%): ① → [숨겨진 메커니즘 암시] → ③ — 구체적 메커니즘·원리 존재 주제
> - **권위 인용형** (10%): ① → [권위 인용] → ③ — H 감정직접소환 + 권위자 명시 인용 가능
> - **미래 제시형** (10%): ① → [미래 제시] → ③ — 현재→미래 시간축 + 흐름이 서사 중심
>
> **집필 판단 플로우:**
>
> 1. H 감정직접소환 + 권위자 인용 가능? → 권위 인용형
> 2. 주제가 현재→미래 시간축 전개 + 흐름이 핵심? → 미래 제시형
> 3. 구체적 메커니즘·원리 존재? → 표준 3단 (기본)
>
> **특수 예외 (기본 체계 외):**
>
> - **ML 외부화 (10 종교와 고기)**: ML이 도입부 밖 제목으로 이전. 집필 시 일반 선택지 아님
> - **2단계 축소 (20 해수면)**: 시리즈 연결 + 현재 상태 제시만으로 ML 직진. 시리즈물 특수 조건
>
>---
> **과정 추적형 도입부 체계 (필수 3요소 + 3가지 구조, 7편 실측 기반):**
>
> 과정 추적형 도입부는 3가지 필수 요소를 반드시 포함한다. 7편 전수 검증 결과 3가지 구조가 병존하며, 13 호르무즈(G 전략 특성상 2단계 축소 경향)는 분류 제외하고 표준 3단의 하위 변형으로 간주.
>
> - ① **1단계: 현재 상태/결과 제시** — 결과를 먼저 보여주고 "어쩌다 이렇게?" 궁금증 유발
> - ② **2단계: 과거로 시간 역행 암시** — 시대적 배경·촉발 사건 암시로 과거 탐구 예고
> - ③ **ML ("어쩌다 이렇게 됐을까?" 질문형)**
>
> **3가지 구조:**
>
> - **표준 3단** (실측 최다, 71.4%): `① → ② → ③` — 썸네일이 현재 상태·결과 직접 드러냄
> - **맥락 선행형** (4단): `[상식적 맥락] → ① → ② → ③` — F 맥락재설정 + 대비 축을 먼저 세워야
> - **상식 전복 확장** (4단): `[상식 전복] → ① → ② → ③` — B 공감적이질감 + 용어·개념·분류 전복
>
> **집필 판단 플로우:**
>
> 1. F 맥락재설정 전략 + 대비 축 선행 필요? → 맥락 선행형
> 2. B 전략 + 용어·개념·분류 전복 필요? → 상식 전복 확장
> 3. 썸네일이 현재 상태·결과 직접 드러냄? → 표준 3단 (기본)
>
> **1단계 구현 다양성 (표준 3단):** A 스케일 충격 / B 공감·질문 진입 / H 기념일·감정 진입 / G 현재 위기 직입(2단계 축소 경향)
>
> **글자수 기준:** 모든 예시의 `[총 글자수]`는 **공백 포함** 기준 (CLAUDE.md 표준: 공백 포함 분당 약 300자).

### B.3 23 아보카도 (모순형 — 압축형 3단)
