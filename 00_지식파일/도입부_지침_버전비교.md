## 버전 A: 수정 전 (간결 버전)

```
### 도입부 3단계

1. **후킹 (첫 15초):** P3에서 선택한 도입부 전략(A~H)을 실행한다. 이문호공 **2요인 이상**을 중첩 또는 복합으로 적용한다.
   - **중첩:** 한 문장/장면에 2개 이상의 요인을 동시에 작동시킨다. (예: 상식파괴(5) + 경고(7))
   - **복합:** 연속된 2문장에 서로 다른 요인을 배분하여 감정을 누적시킨다. (예: 문제제기(2) → FOMO(12))
   - 주제와 서사 유형에 맞는 요인 조합을 전략적으로 선택한다.
2. **신뢰 상환:** 썸네일이 약속한 정보의 실체를 구체적 팩트·수치·사건으로 보여준다. 썸네일 문구를 대본에 그대로 인용하지 않는다.
3. **ML 생성:** 영상 전체를 관통하는 핵심 궁금증으로 연결. **2문장 이내, 이문호공 3요인 이상** 적용한다. 한 문장에 3요인을 모두 담거나, 2중첩 + 1복합으로 구성한다. 주제와 내용 흐름에 맞는 요인 조합을 전략적으로 선택한다.
```

**특징:**
- 신뢰 상환에 분량 기준 없음
- 신뢰 상환 ≠ ML 분리 원칙 없음
- 서술 기법(소화 문장, 일상 언어 풀이 등) 없음
- ML 필수 요인(호기심/상식파괴) 지정 없음

## 버전 B: 현재 (호흡 확장 버전)

```
### 도입부 3단계

1. **후킹 (첫 15초):** P3에서 선택한 도입부 전략(A~H)을 실행한다. 이문호공 **2요인 이상**을 중첩 또는 복합으로 적용한다.
   - **중첩:** 한 문장/장면에 2개 이상의 요인을 동시에 작동시킨다. (예: 상식파괴(5) + 경고(7))
   - **복합:** 연속된 2문장에 서로 다른 요인을 배분하여 감정을 누적시킨다. (예: 문제제기(2) → FOMO(12))
   - 주제와 서사 유형에 맞는 요인 조합을 전략적으로 선택한다.
2. **신뢰 상환 (30~60초):** 썸네일이 약속한 내용을 실제 팩트/현실로 증명한다. 팩트 나열이 아니라 팩트의 체감 시간을 늘린다. 썸네일 문구를 대본에 그대로 인용하지 않는다. — 시청자가 "이 채널 믿을 만하다"는 느낌을 받아야 ML이 자연스럽게 열린다
   - **신뢰 상환 ≠ ML:** 신뢰 상환은 "이미 일어난 일/현재 상태"를 팩트로 보여주는 것이고, ML은 "왜/어떻게"를 묻는 질문이다. 같은 문장에 섞지 않는다.
   - **필수:** 팩트 후 소화 문장(한 박자 쉬며 의미를 짚는 문장)을 삽입한다. 단, 팩트 자체가 충분히 직관적이면 삽입하지 않는다. 앞 문장의 단어를 반복하는 것은 소화 문장이 아니다
   - **필수:** 숫자/전문 개념은 일상 언어로 풀어서 서술한다
   - **선택:** 시청자 인식과 팩트의 괴리를 명시한다 (모순형, 이질감/맥락재설정 전략일 때)
   - **선택:** 시청자 본인의 상황으로 대입한다 (생활 밀착형, 불안소환/자조적체감 전략일 때)
3. **ML 생성:** 영상 전체를 관통하는 핵심 궁금증으로 연결. **2문장 이내, 이문호공 3요인 이상** 적용한다. 한 문장에 3요인을 모두 담거나, 2중첩 + 1복합으로 구성한다. **호기심(3) 또는 상식파괴(5) 중 하나를 반드시 포함**한다. 나머지 요인은 주제와 팩트 흐름에 맞게 선택한다.
```

**특징:**
- 신뢰 상환 30~60초 분량 기준
- 신뢰 상환 ≠ ML 분리 원칙
- 서술 기법 필수 2개 + 선택 2개
- 소화 문장 금지 조건 (직관적이면 생략, 단어 반복 금지)
- ML 필수 요인: 호기심(3) 또는 상식파괴(5) 중 하나 반드시 포함
