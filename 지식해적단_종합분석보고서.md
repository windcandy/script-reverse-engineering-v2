
15분 영상 기준 약 10~12회 반복.

### 5-1. 제목 구조

```
💀 [감정적 훅 + 감탄 이모티콘] / 💀 [주제명 + 형식 표기]
```

| 요소 | 규칙 | 예시 |
|---|---|---|
| 앞부분 | 역설의 한 극을 감정적으로 압축 | "미군도 점령 못한다는 '서울'의 위엄 ㄷㄷ" |
| 뒷부분 | SEO 키워드 + 부제 | "서울의 방어력은 어느정도일까?" |
| 감탄 이모티콘 | ㄷㄷ, ;;, ??? | 감정 방향에 따라 선택 |

### 5-2. 썸네일 전략

| 전략 | 사용 비율 | 효과 |
|---|---|---|
| 모순형 (제목과 반대 방향) | ~40% | "왜 이 모순이?" → 클릭 |
| 이미지 강화형 (제목과 같은 방향) | ~60% | "진짜야? 근거는?" → 클릭 |

- 최대 15~20자
- 선언적 문구 (질문형 아님)

## 6부: 지침 대비 차이점 및 반영 포인트

### 기존 지침에 이미 잘 반영된 것
- ✅ 역설 엔진 (A극→피벗→B극)
- ✅ 공식 문장 = 핵심 대상 + 비유 + 이문호공
- ✅ SL 열림 5유형, BR 5유형
- ✅ 정보 밀도 파동 사이클
- ✅ 13개 언어 장치

### 추가 발견 — 지침 업데이트 권장 사항

#### 1. 주제 선정 가중치 공식
```
기대 조회수 = (한국 관련도 × 3) + (일상 연결성 × 2) + (역설 강도 × 2) + (보편 궁금증 × 1)
```
- 한국인의 자부심/위기감을 건드리는 주제가 조회수 2~5배 차이
- "특정 국가 한정" 주제는 아무리 구조가 좋아도 30만 이하

#### 2. 최적 영상 길이 범위
- **스윗 스팟: 13~17분** (상위 10편 평균 15.2분)
- 11분 이하: 정보 밀도 부족 체감
- 18분 이상: 완주율 하락 (단, 시리즈물 예외)

#### 3. 도입부 규모 폭로의 "숫자 3개 법칙"
- 상위 편들의 도입부 15초 내에 반드시 구체적 수치 3개 이상 투입
- 예: 08편 "486억원, 605km², 372km, 964만명" → 시청자가 "이 정도인 줄 몰랐지?" 반응

#### 4. 피벗 강도와 조회수 상관
- **정점 확신 → 배신** 낙차가 클수록 조회수 높음
- 04편 석면: "기적의 물질" → "발암물질" (205만뷰)
- 01편 감자: "개사기 작물" → "이랬다가 큰일" (76만뷰, 피벗 낙차 중간)

#### 5. "우리" 배치의 중요성
- 상위 편들은 반드시 후반부에 "우리나라", "한국", "여러분" 등으로 시청자 생활권 재연결
- 08편: "여러분이 자주 다니시는 올림픽대로" → 군사 시설 재해석
- 11편: "에어컨 없었으면 한국도 지금만큼 발전 못했을 거예요"

#### 6. 연쇄형 SL의 텐션 유지 조건
- 연대기형 주제에서 SL 간 연쇄가 작동하려면:
  - 각 SL이 "이전보다 더 놀라운" 정보여야 함 (스케일 상승)
  - SL 사이에 BR이 반드시 있어야 함 (빈틈 없는 연결)
  - 18편(미국영토)이 대표 사례: 13개 식민지→루이지에나→텍사스→알래스카 (스케일 상승)

#### 7. 쇼츠 vs 본편 전략 분리
- 쇼츠: 평균 15~30만뷰, 구독 유입 채널 역할
- 본편: 평균 60~90만뷰, 시청시간/광고수익 핵심
- **교차 활용:** 본편 주제의 단일 팩트를 쇼츠로 선행 공개 → 본편 유입 유도

## 7부: 대본 집필 시 적용 체크리스트 (지침 보완용)

### 구조설계 단계
- [ ] 주제가 "한국 관련 + 일상 연결성" 2개 이상 충족하는가?
- [ ] 핵심 질문을 한 문장으로 말했을 때 "에? 왜?" 반응이 나오는가?
- [ ] 최종 결론, 지적 전리품, 논리 엔진이 정의되었는가?
- [ ] 영상 길이가 13~17분 범위인가?

### 도입부 단계
- [ ] 첫 15초 내에 구체적 수치가 3개 이상 있는가?
- [ ] 2분 이내에 메인 루프 질문이 열리는가?
- [ ] 신뢰 상환이 이루어졌는가?

### 본론 단계
- [ ] A극 SL들이 "확신 스케일 확대" 방향으로 배열되었는가?
- [ ] 피벗 낙차가 충분한가? (정점 확신 → 완전 배신)
- [ ] 2분 이상 SL 없는 구간이 없는가?
- [ ] 공식 문장이 A극 정점, B극 해소에 최소 1개씩 있는가?
- [ ] "우리/한국/여러분" 배치가 후반부에 있는가?

### 언어 장치 단계
- [ ] 정보 밀도 파동이 60~90초 단위로 작동하는가?
- [ ] 공식 문장(강) 직후에 뒷받침 문장(약)이 있는가?
- [ ] 반전 질문이 5회 이상 교차 배치되었는가?
- [ ] 동일 비속어/밈이 반복 사용되지 않았는가?
