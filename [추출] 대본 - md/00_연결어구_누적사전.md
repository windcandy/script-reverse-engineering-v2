# 연결 어구 누적 사전 (모순형부터)

> **목적:** 폐지된 18종(사후 분류·부정확) 대신, 실측 연결 어구를 유형별로 누적한다.
> **추출 기준 (가):** "실제 연결 어법" = **다음 블록 진입 전환 어구** (다음 블록 첫머리 전환사 — 하지만·또한·실제로·등등 ~만 등). 자기 블록 끝 어구가 아님. 블록 경계 = 대본 ### 헤더 기준.
> **운영 (동적·비고정):** 대본 분석 시 어구 추출 → 기존 유형에 추가(출처 명시) / 안 맞으면 새 유형 신설 / 비슷하면 통합. **유형 개수·이름은 데이터가 결정 — 표 고정 금지.** 대본 추가할 때마다 갱신.
> **현 상태:** 모순형 5편(감자·한반도·석면·까트·대통령제) 입력. 이후 이미지강화·과정추적형 등 전 대본 추가 예정.

## 핵심 발견 (5편 누적)

1. **★ "정보 직접 진입(브릿지 없음)" 다수 (한반도·까트·대통령제)** — (가) 기준으로 보면 상당수 블록 전환이 전환사 없이 정보로 바로 시작. **모든 블록 연결에 브릿지 어법이 있는 게 아니다** → 18종이 "모든 연결에 코드 부여"를 전제했던 게 실측과 배치.
2. **자문자답이 모순형 최빈 연결** — 5편 전편에 다수. 시청자 의문 지점마다 자문→즉답.
3. **역접·첨가·부연(A 담화 표지)이 정보 흐름 따라 반복** — 하지만·또한·실제로·다만 등. 정보 배치가 정해지면 거의 자동 발생하는 구어.
4. **피벗 전환은 별도 어구군** — "그런데 다들 아시죠"·"100번 양보"·"자 하지만 우리는 다 알고 있죠" = A극→B극 전환 진입 신호.
