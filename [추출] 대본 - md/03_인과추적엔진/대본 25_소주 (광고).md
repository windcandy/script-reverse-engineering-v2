1. 시기 단절 진입 — "그런데 잠깐, 요 녀석 알고 보면" / "그러면 ~ 됐을까. 안타깝게도 ~" 자문자답 반전 진입
2. 격상 비유 (세계 동급) — "위스키·사케·럼 같은 ~ 진짜 [정체성]이라고 소개해도 손색이 없는" 글로벌 동급 3종 병렬
3. 한자·외래어 친근 풀이 — "[글자]는 [의미]로 우리가 [친숙 단어]할 때 쓰는 그 [글자]자" 친숙 매개 어원
4. 위계 점층 친숙 비교 — "막걸리(탁주) → 청주 → 소주" 3단 친숙 사례 점층 학술 위계
5. 도구 의인화 + 동선 시각화 — "얘를 [위치]에 올리고 ~ 찬물 갈아주면 ~ 흘러내려 ~" 작동 동선 따라가기
6. 수율 진폭 + 위상 격상 — "[X] 써서 겨우 [Y] 남짓 ~ 과연 [위상]이었다" 통계 → 위상 부여
7. 게임·전투 슬랭 정점 — "결정타를 날린 건" / "제끼고 시장 장악" 인과 정점 슬랭 격상
8. 다층 인과 압축 통찰 — "[현재 인식]은 이렇게 고착화 됐습니다" / "[친숙 사례]가 [속성]으로 유명했던 것도 이런 이유라는 거" 한 줄 회수
9. 군중·민족 의인화 — "얘네가 ~ 살던 민족이라" / "너도 나도 뛰어들어" 친근 호칭·집단 동작
10. 발견 순간 의태어 + 결과 풀이 — "송글송글 맺히네 ~ 봤더니 ~ 이더라 이거였습니다" 의태어 발견 + B 풀이

**광고 어법 (광고 편 한정 — 광고 편에서만 활용):**

1. 광고 제품 정체성 결산 진입 — "그것이 바로 [제품명]. [3요소]로 만든 [위상] [범주]이죠" 호명 + 3요소 + 위상
2. 광고 제조 공정 정형 설명 — "어떻게 구현했다는 거냐. 첫째 ~ 둘째로 ~ 마지막으로 ~" 자문 + 정형 나열
3. 광고 다요소 압축 정체성 — "재료부터가 토종이라 할 수 있어요" 다요소 → 정체성 D 압축
4. 광고 시음·체험 후기 + 작명 회수 — "저희도 미리 [체험]해봤는데요 ~ 이름 잘 지었다는 생각" 1인칭 후기 + A 격찬
5. 광고 개발 과정 위상 격상 — "과연 [기업]의 야심작이라 할 수 있을 것 같습니다" 다층 개발 → A 야심작
6. 광고 CTA 권유 — "여러분도 [시기] ~ [일상 동사] 보시는 거 어떨까요?" 호명 + 일상 동사 + 친근 권유
7. 광고 결산 + 도전 격상 + 엔딩 — "[구매처]에서 만나볼 수 있으니 모두 도전해 보시기 바라며 이만 영상 마치겠습니다"

**광고·정보성 공용 가능:**

1. 시대 정서 공감 + 위상 부여 — "각자의 [가치]가 존중받는 시대. 이처럼 [대상]은 [위상]"
2. 제품명·이름 어원 친근 풀이 — "[이름]은 순 우리말로써 [의미]를 의미하죠"
