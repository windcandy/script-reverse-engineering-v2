## 지침 정정 후보 (기존 지침에서 고칠 항목)

### 정정 후보 ① — "SL 6~10개" vs 모순형 실제 (본문 직접 검증으로 전면 수정)

- **이전 분석 오류 정정:** "SL=레이어 7개"로 봤으나, 본문 직접 정독 결과 — 레이어/블록 7개는 **정보·단점 나열 묶음**이지 "지적 공백을 만드는 독립 SL"이 아님.
- **엄격 SL(답 미뤄 공백 만드는 것) 실측 (5편 전수):** 감자 **1** / 한반도지형 **0** / 석면 **0** / 까트 **0~1** / 대통령제 **0~1**
- **모순형 실제 구조:** ML 양면(Q1=한 극/Q2=반대 극) + 독립 SL **0~1** + **피벗**(A극→B극 양극 전환). [승]·[전]은 ML 양면이 분할 해소하고, 피벗은 SL이 아닌 양극 전환.
- → 지침 **"SL 6~10개"가 모순형 실제와 극단 불일치** (실제 독립 SL 0~1개).
- **정정 방향:** 모듈_본문구조 §1.2 "SL 6~10개"를 "모순형 = ML 양면 + 피벗 골격, 독립 SL 0~2"로 재검토. 이미지강화·과정추적형은 SL 수가 다를 수 있음(별도 확인).
- **확정:** 모순형 5편 본문 직접 검증 완료 — **모순형 한정 확정.** 단 이미지강화·과정추적형 SL 수 교차 확인 후 §1.2 본체 정정 착수 (엔진별 SL 수 차이 가능성).
- **연동:** 모듈_본문구조 §1.2 + P4 §C-0(449줄). SL 정의가 모듈_엔진·P3 조감도까지 연쇄

### 정정 후보 ② — §8.6 모순형 "Type B 회피" 표현 (영역 1·3 통합) [2편에서 서로 다른 위치 충돌]

- **현 지침:** §8.6 "모순형은 [B] 논리적 브릿지 사용 회피(인과는 A2에 녹임)"
- **충돌이 두 양상으로 분화:**
  - **한반도지형 — [결] 통찰 결산:** B3 2회 + 장점 연결 B2 1회 (Type B 약 20%)
  - **석면·대통령제 — [승] A극 빌드업:** A극이 시간순 역사 계보일 때 B1·B2 명백 사용. 석면(고대 4문명 병렬 B1 + 산업화 B2) / 대통령제(영국 명예혁명→미국 독립→워싱턴→신생국→한국 시간순 B2)
- **까트 — [승] 비역사형:** A극·B극이 해악·경제 설명(시간순 계보 없음) → Type B 적고 A2 자문·B2 인과 위주. → **A극이 시간순 역사형일 때만 B1·B2 증가**가 5편으로 확증
- **핵심 쟁점:** "회피"가 절대 금지로 오독되나, 실측은 **A극이 시간순 역사 서사일 때 Type B(B1 계보·B2 인과) 자연 발생**. §8.3 [A-선언]도 "A극·How가 시간순 역사 서사면 S1~S9 증가" 안내 → "인과는 A2에 녹임 / 시간순 계보는 B1·S 선언형 병용"이 실측에 부합
- **정정 방향:** "회피" → "A극이 시간순 역사 서사일 때 B1·S 선언형 병용 허용, 피벗 인과는 A2에 녹임" (기존 한 줄 정밀화, 신설 아님)
- **✅ 확정:** 5편 검증 완료 — A극 시간순 역사형(석면·대통령제·한반도 [결])에서 Type B 발생, 비역사형(까트)에서 미발생. §8.6 본체 정정 착수 가능.

### 정정 후보 ③ — 결론부 분석 관련 지침 혼선·부재 (2026-06-11 발견)

- **엔딩 유형 두 체계 혼선:** 모듈_어법 §3.5.4 "엔딩 보상 어법 8종"(거시위상부여·위트반전 등 — 표현 도구)과 모듈_본문구조 §2 "결론부 8유형"(①일상행위연결 ②감정적여운 ③미래희망 ④시의성 ⑤위트 ⑥통찰형 ⑦경고형 ⑧광고 — 엔딩 유형 분류)이 별개. **정식 엔딩 유형 = 결론부 8유형**(CLAUDE.md 헤더 정합). §56 "**모순형 = 미래희망(③)+위트(⑤) 주류**".

## 이미지강화형 골격 (입식좌식 1편 — 잠정, 추가 편 분석 후 확정)

**ML 단일 질문 → 영역 확장(지역·사례 나열, 각 영역이 ML을 분할 해소) → 종합 결론 → 시청자 적용 → 엔딩.**

- ML이 "왜 X인가?" 단일 질문이고, 본문이 **여러 영역(지역·기후·사례)으로 확장**하며 각 영역이 ML 답을 한 조각씩 제공(이미지 강화).

## 과정추적형 골격 (락스 1편 — 잠정, 추가 편 분석 후 확정)

**모순형·이미지강화형과 명확히 다른 제3 골격.** 결과물(사물·제도·사건)의 **발명·형성 과정을 시간순 인과 사슬로 추적**한다.

- **ML:** 결과물의 **발명 과정(어떻게 생겼나) + 작동 원리(어떤 메커니즘)** 2축 질문
- **[승]·[전]:** 시간순 인과 사슬 — 각 단계가 "문제 발생 → 누가 등판 → 해결 → 새 문제 → 다음 등판" 미니 인과로 연결. 중간에 **메커니즘 회수 블록**(ML "원리?" 분할 회수)
- **용도 확장 분기점:** [승]→[전] 전환. 부수효과·새 국면 발견("예상치 못한 또 다른 효과가 있네")으로 다음 인과 영역 진입. 이미지강화형 "영역 전환점"과 기능 유사하나 **인과 사슬 맥락**(공간/분류 영역이 아니라 용도·국면 전환)
- **[결]:** 거시 의의(문명 부양·위상) + (위험·반전 단서) + 엔딩
- **모순형과 차이:** 양극(A극/B극)·피벗 없음 — 대립이 아니라 시간순 발전
- **이미지강화형과 차이:** 공간/분류 영역 나열이 아니라 **연도별 인과 단계 추적**
- **시그니처 어법:** 시기 단절점 진입("X세기까지"·"때마침 X년"·"그리고 마침내") · 인물 등판("X가 등판") · 인과 종착 부사("그래서"·"결국"·"마침내") · 자문 도발(미니 오픈루프로 다음 단계 견인)

| 15 | 최초·최후 산업+제로솔트+엔딩(13:54) | 결산+엔딩([결]) | "소금은 이렇듯 인류 최초의 상품이자 산업입니다"(→15, 결산) |

**정합 확인:** 과정추적형 골격 **17편째(마지막)**. ML 2축 + 시간순 인과 사슬([승] 소금 산업사 / [전] 근대 국력) + 국면 전환점(블록12) + [결]. 골격 동일. 문명별 사례 비교 나열 + 시간 대칭 결산이 변주.

**핵심 발견 4가지:**
1. **2축 ML** — "왜 집착?(생리 원인)" + "어떤 역할?(역사)". 집착은 [승] 앞 생리·농경(블록3-4) 빠른 회수, 역할은 사례·국력 전체.
2. **국면 전환점 = 산업→국력 지표**(블록12 "이처럼 ~ 패권을 가늠할 수 있는 지표") — 의미 전환(소금 산업→국력 지표). 걸프전·미국영토 계열 의미/초점 전환.
3. **[승] 문명별 사례 비교 나열**(블록6-11 로마·중국·중세 유럽·식민) — 소금 역사를 문명별 사례로 나열. 콜로세움·걸프전·차우셰스쿠 나열과 유사하나 문명/지역별 사례 비교.
4. **시간 대칭 결산**(블록15 "최초의 상품이자 산업, 아마 최후의 상품이자 산업") — 소금을 "최초↔최후" 시간 대칭으로 거시 위상 부여.

**결론부 분석 ([결] 블록15):**
- 결론 브릿지: "소금은 이렇듯 인류 최초의 상품이자 산업입니다" → 결산 진입("이렇듯") + 시간 대칭("최초이자 최후")
- 엔딩 유형: **⑤ 위트**("제로 솔트 국밥도 나왔으면") + **① 일상행위연결**("적당한 염분 섭취로 건강한 생활") + 시청자 호명 · 채널 시그니처 "지식해적단의 키드"
