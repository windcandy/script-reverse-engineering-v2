### 인류와 고기 — 사치제 + 가축 비효율 (00:00)

오랜 옛날 인류가 수렵과 채집을 하던 선사시대 때부터 고기는 아주 특별한 음식이었습니다. 너무나 맛있었지만 구하기는 쉽지 않았죠. 사실 여자들이 채집해 온 열매와 씨앗들만으로도 뭐 버틸만 했습니다. 정 단백질이 부족하면 생선이나 벌레 같은 걸로 필수 영양소를 맞출 수도 있었고요. 하지만 그럼에도 불구하고 온혈 동물의 붉은 고기가 먹고 싶었기 때문에 부족 내에 건장한 남자들은 목숨을 걸고 사냥해 나가 몇날 며칠 허탕을 치는 비효율을 감수했죠. 농경이 시작된 이후에도 이 집착은 사라지지 않아서 고기는 권력과 특권을 투영하는 하나의 사치재로 자리잡았습니다. 가축을 기르더라도 젖이나 알 같은 부산물을 공급받거나 노동력을 얻기 위해서였지 이걸 잡아서 먹어버리는 건 매우 비효율적인 일이었거든요. 하지만 인간의 욕심은 끝이 없고 같은 실수를 반복하죠. 가난한 사람들이 자기네 집 가축을 잡아먹었다가 다음 해 농사를 망치고 사람들이 목초지를 만들기 위해 땅을 독식하면서 유랑민이 생겨났어요. 심지어 마땅한 가축이 없는 일부 사회에서는 사람끼리 잡아먹는 일도 드물지 않았고요.

  - 2단 1차 가공 [영역: 호흡·리듬 + 어휘 치환]: "현대인의 새 해법 등장 가능성" [화자 의견] → "~ 찾아내지 않을까요" (학술 → 시청자 호명 반문 종결)
  - 3단 2차 가공:
    - 추가 문장: "그때의 현자들처럼 지금 시대에 맞는 새로운 해법을 찾아내지 않을까요"
    - 메커니즘: **E. 시청자 인식 전환 부연** ("고대 안전수칙" 통념을 "현대 새 해법"으로 시청자 사고 확장)
    - 효과: 과거 사례 분석을 미래 해법 모색으로 인식 전환 엔딩
  - 4단 재현 패턴:
    - 변환 메커니즘: 미래 전망 결산 → "그때의 X 처럼 지금 시대에 맞는 Y 를 ~ 않을까요" 시청자 호명 반문
    - 예문 (확장 가능 — 참고용): "찾아내지 않을까요" / "마련해내지 않겠어요?" / "만들어내지 않을까요"
    - 적용 영역: 미래 전망 엔딩
    - 적용 원칙: 위 예문은 참고용. 주제·맥락에 맞춰 창의적으로 새 표현 생성 가능

- 산출 보정 (한 표현 다층 메커니즘 별도 산입): 없음 → **2차 가공 총 20건**
- **2차 가공 비율: 20/49 = 41%** — 경계 (30~50%)
- 메커니즘 분포: A 6건 / B 3건 / C 1건 / D 6건 / E 4건
- 평가: 정합 (사례비교엔진 — 사례 A·B 2영역을 깊이 빌드업하는 구조상 단순 학술 정보 누적 블록이 다수. 1차 가공 어휘 치환·평행 누적·자문자답 호흡이 자연 우세. 핵심 정점([승] 5·6, [전] 3·4·5, [결] 1·2)에 A·D·E 추가 문장 집중 분포 — 메커니즘 분배 정상)
- 사례비교엔진 잠정 가변 폭 (±10%p)을 감안하면 41%는 정합 범위. 표본 작은 엔진 (3편) 특성상 첫 분석편으로서 향후 32·40 분석 시 비교 기준점 제공

- 출처 차용: 약 6건 (게임 슬랭·IT 어휘·관용구·격언·음식점 슬랭)
- 비유: 약 8건 (가족·신체·물리 속도·운명·종교 인격)
- 인격 부여: 약 7건 (종교 출산·브라만·환경·소·시스템 의인화)

### 이거봐봐 채널 흡수 핵심 자산

- **사례 A·B 분기 평행 결산 — "X 하니까 X 하고 Y 하니까 Y 한 겁니다"**: "허용할 만 하니까 허용하고 금지해야 하니까 금지" / "유럽인은 ~ 갔고 인도인은 ~ 갔습니다" — 사례비교엔진의 두 사례 분기 결산을 평행 인과로 한 줄 압축
- **종교·시스템 의인화 — 출산·인격 동작**: "유대교가 ~ 기독교를 낳고 ~ 이슬람교를 낳습니다" / "수세에 몰린 브라만들 ~ 진화시켜 대응" — 학술 종교·제도를 능동 인격으로 환원하여 친근 동력화
- **자문자답 미니 오픈루프 — "왜 굳이 ~ 했을까요? 의외로 ~"**: "안 할 짓인데 왜 굳이 공식적으로 못 박아가면서까지 금지를 때렸을까요? 의외로 ~ 때문입니다" — 블록 내 미니 오픈루프와 즉시 봉인을 한 호흡으로 묶기
- **시점 대비 한 줄 결산 — "원래는 X 했는데 점점 더 Y 다는 거예요"**: "원래는 돼지를 잘 먹었는데 점점 더 못 먹게 됐다는 거예요" — 복잡한 시간축 변천을 두 시점 대비로 한 줄 압축
- **역전 결산 — "X 사람이 아니라 이제 Y 사람"**: "도축하고 바치는 사람이 아니라 이제 ~ 보호하는 계급" — 학술 정체성 전환의 본질을 한 줄 역전으로 압축
- **시청자 호명 + 단정 종결 — "당연히 ~ 수밖에요 / 거 아니에요"**: "당연히 들고 일어날 수밖에요" / "또 기를 거 아니에요" — 학술 인과를 시청자 추리 호명으로 즉시 동기화
- **수량 평행 누적 — "몇십 몇 백마리씩 / 작은 X 하나 작은 Y 하나씩"**: "소를 몇십 몇 백마리씩 죽이는" / "작은 소 한 마리 작은 땅 하나씩" — 대규모 통계·자원 분배를 평행 수량 누적으로 직관 환원
- **누적 부사 호흡 — "가뜩이나 / 거기다 / 콕 집어서 / 끝끝내"**: "가뜩이나 식량이 부족한데" / "거기다 ~ 전염병이 창궐한다" / "콕 집어서 아예 금지" / "끝끝내 이슬람을 몰아냈" — 누적·단정·운명 부사로 강도 압축
- **위상 격상 결산 — "최고의 X / 만능 X / 권력과 특권을 투영하는 X"**: "최고의 안전장치" / "만능 가축" / "권력과 특권을 투영하는 사치재" — 단일 사실을 사회·역사 위상으로 격상
- **출산·가족·음식점 위계 슬랭 차용**: "4대장 / 원조 / 형제 종교 / 공통의 조상 / 사기급 / 빡세다 / 디폴트 값" — 학술 분류·계보·강도 어휘를 친숙 슬랭으로 환원
- **의태어·의성어 감각 환원**: "픽픽 쓰러지는데 / 꾸역꾸역 길렀던 / 찰랑거리는 / 빛의 속도로" — 학술 동작·물성을 의태어로 직관 환원
- **운명 관용구 비유 — "막다른 길 / 벼랑 끝"**: "막다른 길에 들어설지도" — 추상 위기 전망을 관용구 비유로 압축
- **거시 결산 — "시대를 불문하고 ~ 가장 ~ 바로 X / 고대로부터 전해진 ~"**: "시대를 불문하고 ~ 가장 고품질의 영양섭취원이 바로 고기" / "고대로부터 전해진 안전수칙" — 전 영상 누적 명제를 시간 보편 부사 + 격상 결산으로 압축
