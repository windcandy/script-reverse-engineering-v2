# 5단계 지침 변환 매핑표 — 산출물 1·2·3·4 → P3·P4

> **본 파일 목적:** 3단계 산출물 4종(위치-유형 매핑·레고 작동·엔진별 분산·부가 표기 영역)이 P3 구조설계·P4 집필 지침의 어느 절로 흡수·참조되는지 1:1 매핑. 단계 1~5 수정의 기준 자산.
>
> **작성일:** 2026-06-04
> **현재 단계:** 5단계 단계 0 (매핑표 작성)
> **다음 단계:** 단계 1 (P3 §3 3유형별 구조 템플릿 — 영역 배치 보강)

## § 1. 매핑표 목적·활용 흐름

### 1.1 목적

- 산출물 4종 → P3·P4 흡수 시 매번 재탐색되는 부담 제거
- 단계 1~5 진입 시 어느 산출물 어느 절을 어느 지침 어느 절로 옮길지 1:1 결정
- 누락·오매핑 위험 사전 차단

### 1.2 활용 흐름

1. 단계 1 진입 → 본 매핑표 §2 (P3 흡수) 참조 → 변경안 보고 → 수정
2. 단계 2 진입 → 본 매핑표 §2 (P3 흡수) 추가 절 참조 → 변경안 보고 → 수정
3. 단계 3 진입 → 본 매핑표 §5·§6 참조 → 변경안 보고 → 수정
4. 단계 4 진입 → 본 매핑표 §3·§4 (P4 흡수) 참조 → 변경안 보고 → 수정
5. 단계 5 진입 → 본 매핑표 §8 체크리스트 항목 반영 → GATE 3·4 보강
6. 단계 5 완료 후 본 매핑표 보존 여부 재검토 (§7 원칙에 따라)

## § 7. 흡수 vs 참조 분리 원칙 (비대화 방지)

### 7.1 정량 정의

- **흡수** = P3·P4 본문에 1줄 이상 추가 (영역 명칭·핵심 패턴·권장 범위 표·가변성 원칙)
- **참조** = P3·P4에 "산출물 N §M 참조" 한 줄만 명시 (결정 트리·분포 표·통계 표)

### 7.2 흡수 대상

1. 본 채널 3유형 ↔ 8엔진 매칭표
2. 양식별 [승]·[전] 영역 명칭
3. 양식별 강세 정보유형 권장 범위 (표 1개씩)
4. 양식별 핵심 패턴(강제) / 변주 허용(가변) 분리 명시
5. [전] 진입 용어 8엔진 정합표
6. 다양한 주제 가변성 인정 원칙
7. 4위치 강세 정보유형 권장 범위

### 7.3 참조 대상

1. 결정 트리 7~8단계 상세
2. 9유형 × 8엔진 분포 % 표기
3. 이문호공 10요인 × 4역할 상세 카탈로그
4. BR 흐름 권장 패턴 상세
5. 양식별 효과 분산 표

### 7.4 금지 사항

- 분포 % 표기 흡수 금지 (산출물 4에만 보존)
- 통계 출처·근거 부연 흡수 금지
- 표 컬럼명 "비율 %·빈도" 흡수 금지
- 산출물 4 결정 트리 7~8단계 본문 흡수 금지 (절 번호 참조만)

4. **모순** — 산출물 4 §11.4 매칭표와 정합 확인
5. **불필요한 복잡성** — 흡수 vs 참조 §7 원칙 준수, 결정 트리 본문 흡수 금지
6. **불필요한 지침 외 문구** — 분포 %·통계 출처·근거 부연 흡수 금지
7. **모든 경우의 수** — 표본 작은 엔진(§5·§6·§7·§8·§9) 잠정 한계 명시, 1:다 매칭(이미지 강화형 = 영역확장 + 다층분기수렴) 분기 블록 흡수

### 8.3 단계별 변경안 보고 형식

```
[단계 N — 대상 절]
1. 매핑표 §M 참조
2. 흡수 대상 (한 줄씩)
3. 7점검 결과 — 수정 필요 N건 / 이상 없음 영역
4. 비대화 방지 — 추가 글자수 + 신설 건수
5. 근본 원인 해결 점검 — 흡수 vs 참조 결정 사유
6. 승인 요청
```

## § 9. 표본 작은 엔진 잠정안 한계 명시

### 9.1 표본 작은 엔진 5종

| 엔진 | 표본 | 본 채널 3유형 | 한계 |
|---|---|---|---|
| 결합형 (역설+인과) | 2편 | 모순형 | 핵심 패턴(강제) 가능, 세부 비중 가변 폭 ±10%p 확대 |
| 다층원인수렴 | 3편 | 과정 추적형 | 동일 |
| 사례비교 | 3편 | 과정 추적형 | 동일 |
| 사례나열 | 1편 | 과정 추적형 | 잠정안 — 핵심 패턴만 권장, 비중은 주제 적응 우선 |
| 다층분기수렴 | 1편 | 이미지 강화형 | 동일 |

### 9.2 P3·P4 흡수 시 한계 표기 의무

표본 작은 엔진을 P3 §5·§6·§7 분기 블록으로 흡수 시 다음 한 줄 반드시 동반:

> "본 양식은 표본 N편 기반 잠정안. 핵심 패턴은 강제 유지하되 세부 비중은 가변 폭 ±10%p 확대 — 후속 분석 누적 후 재검증 예정."

### 9.3 잠정안 격상 위험 방지

- 표본 작은 엔진을 본 채널 3유형 골격으로 격상 금지
- 본 채널 3유형은 표본 5편 이상(역설·영역확장·인과추적) 양식만 골격 유지
- 잠정안 5종은 "분기 옵션"으로만 표기

## § 10. 본 매핑표 보존 여부 재검토 (단계 5 완료 후)

- **단계 5 완료 후** 본 매핑표는 작업 가이드 역할 종료
- **유지 옵션:** 후속 산출물 추가 시 (예: 새 대본 분석 누적) 동일 매핑 기준 재사용 → 유지 권장
- **폐기 옵션:** 단계 5 완료 후 본 매핑표 내용이 P3·P4에 모두 흡수됨 → 폐기 가능
- **결정 시점:** 단계 5 완료 후 유저 확인
