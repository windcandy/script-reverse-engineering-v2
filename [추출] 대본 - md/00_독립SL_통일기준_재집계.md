# 독립 SL 재집계 (통일 기준) — 39편 정밀 분석

> **목적:** SL(서브 오픈루프) 정의를 엔진 불문 하나로 통일하여 39편 독립 SL을 재집계한다.
> **작성:** 2026-06-12. 각 대본 본문 직접 정독 기반.

---

## 통일 기준

**SL = 지적 공백(궁금증)을 열고 즉시 답하지 않은 채 1블록 이상 답을 미루는 루프.** ML(메인 오픈루프)은 이 원리의 영상 전체 최상위 버전, SL은 본문 중간의 작은 버전 — 정의는 동일.

**SL 아님 (제외 대상):**
- **자문자답** — 같은 블록에서 질문+즉답("그럼 ~ 되냐? 아니죠"). 지적 공백을 지속시키지 않음.
- 정보 제시 / ML·피벗(모순 양극 전환) / ML 해소
- 영역 전환점(이미지강화·과정추적형 국면 전환) / 개방형 엔딩(클리프행어) / 정서 정점

**판정 방식:** 본문 정독 → 블록별 판정 → SL이 작동하는 블록은 대본 헤더에 `【SL{N} ▶ "공백"】`·`【SL{N} 해소】` 표시.

**배경:** 기존 브릿지 분석이 "자문 다수"를 SL로 카운트했던 것을 정정. 자문자답은 즉답이라 지적 공백을 1블록 이상 미루지 않으므로 SL이 아니다.

## 엔진별 패턴

| 엔진 | 골격 | 독립 SL 범위 |
| --- | --- | --- |
| 모순형 | ML 양면 + 피벗이 본체 | **0-1** (감자만 1) |
| 이미지강화형 | ML 단일 + 영역 나열 | **0-2** (영역 나열형 0 / 미스터리 삽입형 1-2) |
| 과정추적형 | 시간순 인과 사슬 | **0-1** (대부분 0 / 별도 축 미스터리 삽입 2편만 1) |

## 핵심 발견

1. **통일 기준 적용 시 전 엔진 독립 SL이 0-2개로 수렴.** 모순형(0-1)·이미지강화(0-2)·과정추적(0-1)이 거의 같은 수준.
2. **지침 "SL 6-10개"(모듈_본문구조 §1.2)가 전 엔진과 극단 불일치.**
3. **"자문 다수 ≠ SL"이 데이터로 확인.** 당뇨(자문 4회→독립 SL 1)·서울요새/에어컨/서안해양성(자문 다수→0) 등. 기존 분석이 자문자답을 SL로 셌던 것이 정정됨.
4. **독립 SL이 생기는 조건:** ML(영상 전체 화두)과 **다른 축의 미스터리**를 본문 중간에 명시적으로 열고 1블록 이상 미룰 때. (호르무즈 "왜 석유 많나?", 루프탑 "왜 코리아타운?", 옥수수 "왜 펠라그라?" 등)
