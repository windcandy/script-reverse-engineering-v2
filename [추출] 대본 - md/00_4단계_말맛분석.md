## §2. 친밀톤·화자 스탠스 변주 카탈로그

> 39편 전수에서 확인된 화자 톤 변주 패턴. 합쇼체를 기본 흐름으로 하고 결정적 순간 해요체·반말로 변주하여 호흡 단절·시청자 즉시 동조 유도.

### 2.1 합쇼체 기본형 (~합니다·~입니다)

- **적용 영역:** 도입부 진입·본문 정보 단정·결론부 거시 결산
- **변환 메커니즘:** 권위 정보 진입 시 합쇼체로 안정감·신뢰도 확보
- **예문 (참고용):** "~이었습니다" / "~인 거지요" / "~한 셈입니다" / "~할 정도였습니다"
- **반례:** 도입부 후킹은 종종 단속음 단정·반말 펀치로 시작하여 합쇼체와 대비

### 2.2 해요체 친근형 (~거든요·~잖아요·~죠)

- **적용 영역:** 풀이형 부연(B 메커니즘) · 통념 정정 진입 · 자문자답 답
- **변환 메커니즘:** 권위 정보 풀이 시 친밀 어미로 친근감 확보 + 시청자 동조 유도
- **예문 (참고용):**
  - "~ 거든요" (이유 풀이 표준)
  - "~ 잖아요" (시청자 이미 알고 있다는 전제 + 동조)
  - "~ 죠" (단정 + 친근)
  - "~ 라는 거예요" (밈 종결)
- **출현 빈도:** 39편 전수 거의 모든 블록 등장

### 2.3 반말 펀치형 (~거든·~지·~인데·~다는 거)

- **적용 영역:** [전] 정점 임팩트 · 자문자답 자답 · 결산 단속
- **변환 메커니즘:** 합쇼체 흐름 중 반말 펀치 1회로 호흡 단절 + 강도 증폭
- **예문 (참고용):**
  - "어림도 없지" / "응, ~ 야" / "그렇다는 거" / "~ 한다는 거지요"
  - "할 수 없잖아" / "~ 라는 거" (명사형 종결 미끼·폭로 어법)
- **출현:** 08 서울요새도시·09 한국여권·19 대통령제·39 차우셰스쿠 등에서 결정적 순간 발동

### 2.4 결합 변주 패턴 (위치별)

| 위치 | 결합 패턴 |
| --- | --- |
| 도입부 후킹 | 단속 명사형 종결 → 자문자답 → 합쇼체 풀이 |
| 본문 빌드업 | 합쇼체 정보 + 해요체 풀이 + 반말 펀치 임팩트 |
| [전] 정점 | 자문자답 반말 자답 + 합쇼체 위상 부여 결산 |
| 결론부 | 합쇼체 거시 결산 + 해요체 시청자 동조 + 위트 반말 종결 |

## §3. 치환·의인화 변주 패턴

- **변환 메커니즘:** 추상 시스템 → 단일 캐릭터·친근 호명

  - "캐리어 형님"·"서유럽 형님들" (17 서안해양성)

  - "좌식러·입식러" (02 입식좌식 — 인격화 접미사 "~러")
  - "남산이 버티고 있어서"·"강남이 버티고 있기 때문" (08 서울요새도시)
  - "이놈의 미국 땅은 적당함을 모르지요" (26 미국석유)
- **적용 영역:** 영역확장형 영역 진입부 · 친근 화제 진입

### 3.2 양면 인격 의인화 (모순형 주)

> 같은 대상의 두 얼굴(이로운/해로운, 강한/약한)을 양극 인격으로 분리하여 모순 시각화

- **변환 메커니즘:** 단일 대상 → 두 얼굴 양극 인격 분리
- **예문 (참고용):**

  - "한국(여름 한정 숭배) vs 열대(1년 내내 유일신)" (11 에어컨 — 캐리어 양면)

  - "기름진 과일 → 걸어다니는 자연재해" (23 아보카도)
- **적용 영역:** 모순형 피벗 진입 · 결합형 [전] 그림자

### 3.3 시간 여행 가이드 시점 (과정추적형 주)

- **예문 (참고용):**

- **적용 영역:** 인과추적형 [승]·[전] 블록 진입 · 시기 전환점

### 3.4 권위 인물 친근 호칭

- **변환 메커니즘:** 권위 인물 → 친근 별명·호칭
