#### 1.1 영역 A — 제목/썸네일 결정 분석

**A.1 약속 유형:**

| 텍스트 | 약속 유형 | 시청자 미끼 |
| --- | --- | --- |
| 제목 1 ("쓰레기 땅에서도 개사기 작물") | 장점 폭로 | "왜 개사기인가" |
| 제목 2 ("감자가 치트키인 이유") | 장점 폭로 | "왜 치트키인가" |
| 썸네일 ("주식으로 먹지 못하는 결정적 이유") | **단점 폭로** | "왜 못 먹는가" |

→ **제목 = 장점 / 썸네일 = 단점** 정반대 약속

**A.2 노출 수준:**

| 위치 | 노출 |
| --- | --- |
| 제목 | 부분 노출 — "개사기·치트키"라는 라벨만, 구체 이유 미제시 |
| 썸네일 | 부분 노출 — "결정적 이유"라고만, 구체 이유 미제시 |

→ 둘 다 **부분 노출**. 시청자는 "장점도·단점도" 모두 궁금

**A.3 제목 ↔ 썸네일 역할 분담:**

- 제목 = 장점 약속
- 썸네일 = 단점 약속
- 두 약속이 **정반대** = **모순 공존 구조**
- 결합 효과: 시청자가 보면 "장점인데 왜 못 먹어?" 자동 호기심 발생

**A.4 메인 ↔ 서브(말풍선) 역할 분담:**

- 본 대본 헤더는 단일 "썸네일 문구" 표기. 메인/서브 구분 정보 없음
- **분석 한계:** 실제 영상 확인 시 메인/서브 구조 추가 확인 필요
- 후속 7단계(이미지 분석)에서 보강 예정

**A.5 본 채널 3유형 신호:**

| 신호 | 본 채널 3유형 매칭 |
| --- | --- |
| 제목(장점) + 썸네일(단점) = 모순 공존 | **모순형**(역설 엔진) 신호 명확 |

→ 시청자는 제목·썸네일만 보고도 "양면성 영상"임을 예측 가능

#### 1.2 영역 B — 약속 → 도입부 연결 분석

**B.1 첫 30초 이탈 방지 신호:**

도입부 첫 블록 (line 14-90, "의외의 4위 감자 — 화제 던지기"):

```
지구상의 문명들은 크게 밀을 재배하는 문명과 쌀을 재배하는 문명으로 나뉩니다.
[...양강 구도 제시...]
그런데 이 뒤를 바짝 쫓아오고 있는 작물이 뭐다? 감자다.
실제로 감자는 현재 전 세계에서 연간 무려 3억 8천만 톤이 생산되고 있는
인류의 대표 작물 중 하나입니다.
[...]
```

| 신호 | 효과 |
| --- | --- |
| 첫 문장 "밀·쌀 문명" → "감자" | 시청자가 보고 들어온 "감자 영상" 대상 즉시 확인 (대상 확인 신호) |
| "3억 8천만 톤·6위·4위" 수치 폭격 | 의외성 투하로 호기심 유지 |
| "어, 그렇게 많이 먹나? 오지게 많이" 자문자답 | 시청자 가상 반응 인용으로 친근감 |

→ **대상 확인 신호 강함**, **약속 자체 직접 확인 신호 약함** (의도된 호기심 유지)

**B.2 약속 처리 방식:**

| 위치 | 처리 방식 |
| --- | --- |
| 도입부 첫 블록 | **정반대 정보형** — 약속("못 먹는")과 반대인 "주식 4위 의외로 많이 먹는다" 투하 |
| 메인 오픈루프 (line 193) | **추상화 우회형** — 약속("주식 못 먹는 이유")을 "장단점?"으로 상위 추상화 |

→ **혼합형** — 정반대 정보 + 추상화 우회 결합

**B.3 약속 ↔ 메인 오픈루프 관계:**

메인 오픈루프 (line 193):

```
그렇다면 감자는 대체 언제부터 먹어 왔을까요?
밀과 쌀이랑 비교했을 때 어떤 장단점이 있을까요?
지식해적단이 간단히 알아보았습니다.
```

| 구성 | 약속과의 관계 |
| --- | --- |
| "언제부터?" | 약속 외 영역 (새 호기심 도구) |
| **"장단점이 있을까?"** | **약속의 추상화** — "단점" 영역 안에 썸네일 약속("주식 못 먹는 이유") 포함 |

→ **결론: 썸네일 약속 = 메인 오픈루프의 정보 조각 중 하나** (단점 영역의 추상화)
→ 별도 후킹 도구 아님

**B.4 도입부 회수 여부:**

- **도입부에서 약속 회수 = 안 함**
- 도입부 첫 블록 (line 14-90)에서 정반대 정보("주식 4위") 투하로 호기심 강화만 수행
- 메인 오픈루프 (line 193)에서 약속을 "장단점"으로 추상화하여 자연 연결 — 회수 아닌 약속 재진입

→ **도입부는 회수 안 함, 호기심 강화 + 약속 추상화 진입만 수행**
→ 본격 회수는 본 분석 범위 외 ([승]·[전] 영역)

**B.5 약속 ↔ 도입부 톤·어조 일치 여부:**

| 위치 | 톤·어조 |
| --- | --- |
| 썸네일 | 단정형 ("결정적인 이유") |
| 제목 1·2 | 게임 슬랭형 ("개사기·치트키 ㄷㄷ") |
| 도입부 첫 블록 | 자문자답·강조 슬랭 ("오지게·1황·뭐다? ~다") |

→ 톤 자체는 **불일치** (썸네일 단정형 ↔ 도입부 자문자답형)
→ 그러나 어휘 게임 슬랭은 제목 ↔ 도입부 일관 ("개사기/치트키" ↔ "오지게/1황")
→ 결합: 썸네일은 후킹 / 제목·도입부는 본 채널 톤 신호

#### 1.3 영역 C — 결정 + 연결 통합

**C.1 결정 → 연결 / 연결 → 결정 양방향:**

- 본 대본 가설: **결정 시점에 도입부 연결 가능성을 미리 본 흔적**이 있음
  - 모순 공존 약속(제목 장점 + 썸네일 단점) 선택 = 도입부에서 정반대 정보 투하 가능 조건 확보
  - 추상화 메인 오픈루프(장단점) 선택 = 썸네일 약속을 자연 흡수 가능
- 즉, **결정 → 연결** 정방향이지만 **연결 가능성을 결정 시점에 미리 검토**한 흔적
- 39편 확장 분석 후 본 패턴이 일반화 가능한지 확인

**C.2 결정 흐름차트 기여 (감자 사례):**

본 사례에서 도출된 추천 경로 1건 (다른 7편 분석 후 확장):

```text
[주제: 감자 — 대단한 작물인데 단점이 있음]
   ↓
[본 채널 3유형 후보: 모순형 (장점·단점 양면 존재)]
   ↓
[결정 패턴: 제목=장점 약속 / 썸네일=단점 약속 → 모순 공존 구조]
   ↓
[연결 패턴: 도입부=정반대 정보(장점) 투하 + 메인 오픈루프=추상화("장단점")]
   ↓
[효과: 시청자 호기심 이중 강화 + 약속 직접 인용 회피]
```

→ 다른 7편 분석 후 흐름차트 추천 경로 누적·통합
