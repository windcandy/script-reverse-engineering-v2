# 하이브리드 대본 색인 — 43편 구조 룩업표

> **용도:** P1 §6.1 레퍼런스 대본 선정용. 새 주제 입력 → 이 표에서 뼈대를 참고할 지식해적단 대본 1편을 잠정 매칭한다(확정은 P3 — P2·P3에서 팩트가 드러나면 적합성 재점검).
> **재료:** `00_도입부_2축분석_누적.md` (도입부 두축 분석) + 엔진 폴더(서사 유형 분류).
> **범위:** 도입부 골격까지만 색인. **본문 전개 뼈대는 색인하지 않는다** — 선정된 1편만 P3에서 정독해 뽑는다.

---

## 매칭 방법 (P1 §6.1 절차 — 우선순위 순)

1. **1차 필터 — 서사 유형:** 새 주제가 모순형(상식 뒤집기)/이미지강화형(대상 스케일 각인)/과정추적형(어쩌다 이렇게?) 중 무엇인가.
2. **2차 매칭 — 진입 프레임:** 새 주제의 썸네일 약속을 어떤 방식으로 열 것인가(아래 범례). **소재보다 후킹 방식이 뼈대 이식에 중요** — 여기가 핵심 매칭 축.
3. **3차 tiebreaker — 주제군:** 음식·지리기후·역사전쟁·국제정치·물질과학·문화사회·건강신체산업 중 가까운 군(동점 시 소재 어휘·비유 감각 참고).
4. **1편 잠정 선정 → 뼈대 참고 + 살 100% 창작.** 맞는 대본이 없으면 **지침 단독 fallback**. 확정은 P3(P2·P3에서 팩트가 드러나면 적합성 재점검).

### 진입 프레임 범례 (축②)

| 프레임 | 뜻 |
| --- | --- |
| 반대정보빌드업 | 약속과 정반대 정보를 여러 개 쌓아 낙차 생성(→ 뒤에서 반전 확인) |
| 반대방향진입 | 약속과 반대 방향으로 시작해 한 번 뒤집기 |
| 속성대체 | 약속을 다른 속성으로 우회해 암시 |
| 동의어 | 약속을 동의어로 바로 명시 |
| 위상빌드업 | 대상의 위상을 격상하며 진입 |
| 직접노출 | 약속(답)을 도입부에서 바로 노출 후 ML 재점화 |
| 사례 | 시청자 경험·사례 나열로 진입 |

### 골격 표기

`N블록 · 블록1 요인 · 약속확인 위치 · ML봉인` (봉인: 양면=질문2개+/단면=1개/묵시=보편원리/다중=3질문+)

## 매칭 예시

- **새 주제가 "음식인데 의외로 X" (모순형/음식):** 감자·까트·옥수수·참외 후보 → 진입 프레임으로 좁힘(반대정보빌드업이면 감자·참외).
- **새 주제가 "어쩌다 이 나라가 이렇게" (과정추적/역사전쟁):** 가야·미국영토·차우셰스쿠·강뉴부대 후보 → 단일 인과면 단면 봉인 계열(미국영토·차우셰스쿠).
- **새 주제가 "대상이 알고 보니 엄청남" (이미지강화):** 에어컨·옥수수·콜로세움 후보.

> **주의:** 진입 요인·골격은 참고용 뼈대다. 실제 문장·팩트·비유는 100% 새로 창작하며, 선정 대본과 연속 4단어 이상 겹치지 않게 한다(검열 회피).

## 본문 정보 (정독 시 누적)

> **용도:** 도입부만으로 매칭하면 본문 구조가 안 맞을 수 있으므로(예: 몰락 파트가 레퍼런스에 없는 경우), 본문 구조·정보유형까지 대조하기 위한 정보.
> **원칙:** 43편 전수 정독은 비용 과다 → **집필하며 원문 정독한 대본만 여기에 누적**한다. 정독할수록 색인이 똑똑해진다.
> **참고용 주의:** 여기 본문 정보유형은 참고일 뿐이다. **새 주제 고유의 더 강한 후킹 정보를 이 유형에 가두지 말 것** — 후킹 강도 우선(P2 §2 서칭 제1원칙).

### 감자 (모순형 / 역설엔진)

- **본문 구조:** [기] 도입 → [승] 위력 빌드업(A극) → [전] 피벗+함정 빌드업(B극) → [결] 종합·미래. **모순 공존**(개사기 작물인데 주식 못 됨 — 위력 쌓다가 피벗으로 함정 반전)
- **블록 흐름·정보유형:**
  - [승] A극: 기원(안데스) → **통계**(칼로리 압도) → 비교(조리·전쟁 안전) → 기원(유럽 전래) → 메커니즘(솔라닌 독성) → 사례(돼지 사료 신세) → 인과(위기·전쟁 강제보급) → 통계(인구 폭증)
  - [전] B극: 피벗(해석 판정) → 메커니즘(전염병 취약) → **사례**(아일랜드 대기근) → 비교(노동·환경·보관·운송) → 사례(잉카 추뇨 극복)
  - [결]: 인과(냉동 기술 정착) → 해석(미래 식량안보)
- **정보유형 분포:** 통계·수치 최다 · 비교·대비 다수 · 사례·일화(대기근·잉카·성경) · 메커니즘 · 기원·배경 · 인과 · 해석·결산
