# 3단계 3-패스 산출물 2 — 레고 이론 작동 패턴 카탈로그

> **목적:** 39편 679블록의 [레고작동] 셀에서 4역할(오픈루프·의도적 지연·보상 투하·충전·빌드업)·끌리는 이유(이문호공 10요인) 작동 메커니즘을 사례 카탈로그로 정렬하여, 5단계 P4 집필 가이드의 직접 재료를 제공한다.
> **정합 기준:** `00_가이드_분석_방법.md` §4.14.1·§7.3 준수.

## § 1. 작업 개요

### 1.1 분석 대상

- **대본 편수:** 39편 (광고 6편 포함)
- **블록 총량:** 679블록
- **4역할 분류:**
  - 오픈루프 — 지적 공백 생성·질문 제시 (답 차단·궁금증 유발)
  - 의도적 지연 — 예상 비틀기·추가 정보 투입 (단서 제공해 시청자 추측 자극)
  - 보상 투하 — Aha-moment·역전 팩트·통찰 회수
  - 충전·빌드업 — 다음 보상을 위한 사전 누적
- **끌리는 이유:** 이문호공 10요인 (전략 결정 지점 한정 표기, 363블록 729요인 발동)

### 1.2 카운트 방식

- 한 블록에 복수 역할(예: "보상 투하 + 오픈루프") = 각 역할 +1 (중복 카운트)
- 끌리는 이유 = 전략 결정 지점 블록에만 표기 (총 363블록), 한 블록에 2~3요인 동반 발동
- 본 카탈로그는 5단계 P4 집필 시 작동 패턴 직접 인용 가능하도록 구체 발화·헤딩 인용 우선

### 1.3 작업 절차

1. 39편 [레고작동] 셀 전수 추출
2. 4역할 × 8엔진 매트릭스 + 10요인 분포 카운트
3. 패턴별 사례 카탈로그 작성 (구체 발화 인용)
4. 통합 사이클 패턴 + P4 변환 권장안 도출

## § 3. 오픈루프 패턴 카탈로그

### 3.1 도입 후킹 오픈루프 사례 ([기] 블록)

도입부에서 메인 오픈루프 직전, 화제 던지기로 시청자 시선을 붙드는 1차 후킹.

| 대본 | 헤딩 | 후킹 방식 |
|---|---|---|
| 01 감자 | 의외의 4위 감자 — 화제 던지기 (00:00) | 통념 충돌 — "밀·쌀 양강" → "4위 감자" 의외 순위 |
| 13 호르무즈 | 중동 전쟁 + 한국이 처맞는 상황 (00:00) | 최신 정보 + 자국민 직격 — "2026 진행 중 사태" + "우리나라 = 피해자" |
| 18 미국영토 | 미국 영토 1등 + 개꿀 (00:00) | 통념 강화 후 반전 예고 |
| 17 서안해양성기후 | 유럽 에어컨 보급률 3~5% (00:37) | 통계 충격 — "선진국 = 보급률 91%" 통념 단번 부정 |
| 30 당뇨 | 오줌 당분 = 무슨 문제? (00:00) | 사소해 보이는 증상에 미니 오픈루프 |

**작동 메커니즘:** 도입 후킹 = "익숙한 화제를 통념 비틀기로 즉시 의문 발생". 메인 오픈루프 전 2~3분 안에 시청자 이탈 차단.

### 3.2 메인 오픈루프 사례 (영상 전체 화두)

영상 전체를 끌고 가는 핵심 질문. 끌리는 이유는 3 호기심형 단독 또는 3+5(상식파괴)·3+8(비교) 조합이 가장 흔하다.

| 대본 | 메인 오픈루프 발화 | 회수 위치 |
|---|---|---|
| 01 감자 | "언제부터? 장단점?" (00:43) | [전] 피벗에서 단점 회수 → [결]에서 종합 |
| 02 입식좌식 | "왜 누구는 신발 신고 누구는 벗는가?" (00:37) | [결] 미국 PPL Aha-moment |
| 07 옥수수 | "사람도 안 먹는데 왜 1위? 어디다 씀?" (00:41) | [전] "96%는 산업용" 영역 전환점 |
| 08 서울요새도시 | "서울 방어력? 한강의 기적 이면?" (01:51) | [결] 우크라이나 비교 회수 |
| 11 에어컨 | "에어컨 파급력?" (01:04) | [전] "거주 가능 구역 확장" Aha-moment |
| 14 까트 | "왜 천년 동안 보급 안 됐나? 절대 그래서는 안 되는 이유?" (00:47) | [전] "현지 경제 인질화" 사회 차원 회수 |
| 17 서안해양성기후 | "왜 유럽은 안 쓸까?" (00:37) | [전] 메커니즘 회수 |
| 19 대통령제 | "대통령제 장점? 한국은 왜 채택?" (01:03) | [전] 피벗 "장점 = 단점" 등치 반전 |
| 24 전기차 | "전기차 시대 한국 건재?" (00:00 끝) | [전] "100년 존버 끝 → 한국 축복받은 땅" |
| 30 당뇨 | "왜 걸리나? 예방?" | [결] 본인 위험 + 행동 권고 |

**작동 메커니즘:**
- 두 질문 동시 개방 ("언제부터? 장단점?", "왜? 어디다 씀?") — 정보 공백 진폭 확대
- 통념 정정 + 호기심 결합 ("선진국 = 보급률 높다 통념을 3~5%로 부정 → 왜?") — 5+3 조합
- 자국민 직격 + 위기 ("한국은 왜?·한국 건재?") — 4 공감 + 7 경고

### 3.3 중간·새 오픈루프 사례 (미니 오픈루프·시기 전환 오픈루프)

영상 중간 새로운 지적 공백을 추가 개방하는 사례.

| 대본 | 헤딩 | 작동 |
|---|---|---|
| 02 입식좌식 | 침상 = 일본군 유래 (03:35) | "왜 한국군 침상이 일본군에서?" 미니 오픈루프 |
| 03 한반도지형 | 피벗 — 장점은 없냐? (09:50) | 단점 일변도 → 장점 방향 재가동 (6 반전 + 3 호기심) |
| 07 옥수수 | 또 다른 함정 — 지력 소모 (10:12) | C4 효율의 보상이 토양 황폐화 (6 반전) |
| 08 서울요새도시 | 영역 전환점 — 아직 강남이 남았다 (09:37) | "이만하면 됐다"를 절반으로 강등 (6+7) |
| 11 에어컨 | 영역 전환점 — 최초 에어컨은 종이 때문 (07:27) | "사람 위해 발명" 통념 정정 (5+6) |
| 17 서안해양성기후 | 영역 전환점 — 단점 없는 건 아니다 (13:05) | "최고 기후 = 무적" 통념 비틀기 (6+3) |
| 24 전기차 | 영역 전환점 — 100년 존버 끝, 상용화 개막 (05:20) | 시기 전환 + 새 영역 진입 |

**작동 메커니즘:**
- **영역확장 엔진:** "영역 A 결산 + 영역 B 게이트" 이중역할 블록 = 영역 전환점에 새 오픈루프 개방
- **인과추적 엔진:** "옛 시기 결산 + 새 시기 진입" 이중역할 = 시기 전환점에 새 오픈루프 개방 (대본 06 루프탑·15 고래잡이·18 미국영토·25 소주 등 광범위)
- **역설 엔진:** 피벗 직전에 미니 오픈루프 재개방 ("장점은 없냐?"·"진짜 큰일")

## § 5. 의도적 지연 패턴 카탈로그

전체 34건 중 인과추적 엔진(17건) 집중. 영역확장 6건. 사례비교 1건.

## § 8. 시청자 추리·보상 사이클 통합 패턴

### 8.1 짧은 루프 + 즉각 보상 (사례비교·사례나열 우세)

- **루프 길이:** 30초~2분
- **사례 엔진:** 대본 10 종교 고기 금기·대본 32 숟가락·대본 40 중화요리·대본 20 해수면상승
- **작동:** 사례 1개 던지고 → 메커니즘 즉시 공개 → 다음 사례. 충전 비중 27~41%로 낮음, 보상 비중 50~59%로 높음
- **시청자 경험:** 사례마다 작은 Aha-moment 누적 — 피로 적음

### 8.2 긴 루프 + 강한 보상 (역설·인과추적 우세)

- **루프 길이:** 5~13분 (영상 전체 메인 오픈루프 회수까지)
- **사례 엔진:** 대본 01 감자·대본 14 까트·대본 06 루프탑·대본 08 서울요새도시
- **작동:** 메인 오픈루프 도입부 개방 → 5~9블록 빌드업 → 피벗/시기 전환점에서 1차 회수 → [결]에서 종합 회수
- **시청자 경험:** 회수가 늦지만 진폭 큼 — 통념 정정 + 통찰 + 자기 일상 변환 동시 발동
- **보상 불균형 방지 장치:** 회수 다단 — 같은 질문에 3~5번 점진 답 공급

### 8.3 다층 회수 패턴 (다층원인수렴·다층분기수렴 우세)

- **루프 길이:** 영상 전체 (10~15분)
- **사례 엔진:** 대본 09 한국여권·대본 26 미국석유수입·대본 27 채취(광고)·대본 31 흑각궁
- **작동:** 원인/분기 4~6개 차례 누적 → 후반 1~2블록에 수렴 정점 폭발
- **충전 비중:** 다층분기수렴 65% (최고) — 빌드업이 영상 2/3
- **시청자 경험:** 누적 부담 큰 대신 수렴 보상이 압도적

## § 9. P4 집필 지침 변환 권장안

### 9.1 양식 선택 후 P4 집필 시 작동 패턴 적용 가이드

| 서사 유형 (이거봐봐 3유형) | 대응 엔진 | 권장 작동 패턴 |
|---|---|---|
| **모순형** | 역설·결합형 | 위력 빌드업(A극) → 피벗(6+7) → 함정 빌드업(B극) → 통합 회수. 보상 다단 회수 + 5+6 조합 |
| **이미지 강화형** | 영역확장·사례비교 | 영역 A 사례 누적 → 영역 전환점(5+6 또는 6+8) → 영역 B Aha → 시청자 일상 변환 회수 |
| **과정 추적형** | 인과추적·다층원인수렴·다층분기수렴 | 시기·원인 차례 누적 → 시기 전환점마다 "옛 시기 결산 + 새 시기 진입" 이중역할 → 수렴 정점 |

### 9.2 본 채널 집필 시 작동 메커니즘 권장 (블록 단위)

**(1) 도입 후킹 ([기] 1~2블록):**
- 통념 충돌 + 시의성 또는 자국민 직격
- 이문호공 5+3 또는 6+9 조합 권장
- 발화 예: "[익숙한 화제] 그런데 [의외의 통계·반전]"

**(2) 메인 오픈루프 ([기] 마지막 블록):**
- 두 질문 동시 개방 (예: "언제부터? 장단점?", "왜? 어디다 씀?")
- 이문호공 3+8 또는 3+5 조합 권장
- 영상 전체 화두를 한 문장으로 압축

**(3) 충전·빌드업 ([승]):**
- 5~9블록 차례 누적
- 매 블록에 작은 보상 1개씩 (미니 Aha-moment)
- 충전 비중 35% 전후 유지 (다층분기수렴 외)

**(4) 피벗/영역 전환점/시기 전환점 ([전] 진입):**
- "옛 시기·영역 A 결산 + 새 시기·영역 B 진입" 이중역할 블록
- 이문호공 5+6 또는 6+7 조합 권장
- 새 미니 오픈루프 동반 개방

**(5) Aha-moment 정점 ([전] 중후반):**
- 메인 오픈루프 핵심 회수
- 시각 증거 또는 친근 변환 ("반지하 = 방공호", "압구정 = 군사 거점")
- 이문호공 5+6 + 8 또는 4 조합

**(6) 엔딩 ([결]):**
- 시청자 일상 회수 ("우리가 사는 아파트가 무기였다")
- 위트 반전 또는 시민 책임 환기
- 이문호공 4+6 또는 10+4 조합

### 9.3 보상 불균형 방지 체크리스트

- 루프 길이 vs 보상 밀도 매칭 확인 (가이드 §4.14.1 축 D)
- 메인 오픈루프 5분 이상 미회수 시 → 미니 Aha-moment 또는 1차 부분 회수 삽입
- 충전 비중 50% 초과 시 → 보상 정점 강도 격상 필요 (다층분기수렴 패턴)
- 끌리는 이유 5+6 단독 의존 회피 — 3·1·4·9 보조 요인 다양화

## § 10. 비고

### 10.1 광고 편 6편 변주 (대본 12·21·24·25·27·34)

- "보상 투하 (광고 CTA)" 역할 명시 — 9건 (전체 광고 편)
- 패턴: 본 주제 메인 오픈루프 회수 직후 → CTA 진입 → 광고 메시지 투하 + 엔딩 결합
- 이문호공 1 이익 제시 + 4 공감 조합이 광고 변주의 핵심

### 10.2 결합형 2편 이중 작동 (대본 23 아보카도·29 기독교시온주의)

- 역설(속성 대립) + 인과추적(시기 대립) 골격 동시 작동
- 빌드업 비중 34% — 역설보다 충전 강함 (이중 골격 누적 비용)
- 의도적 지연 3건 — 양면성 단서 점진 공급에 활용

### 10.3 다층분기수렴 단일 표본 (대본 31 흑각궁) 한계

- 충전 비중 65% = 1편 표본에서만 관찰
- 분기 4편 누적 후 한반도 수렴 = 매우 특수한 구조
- 일반화는 추가 분석 후 판단
