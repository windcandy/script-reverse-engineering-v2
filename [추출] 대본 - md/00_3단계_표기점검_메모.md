#### 횡단 점검 — 같은 부담 등급이 일관 기준인가

**등급별 기준 정리 (5편 횡단 추출):**

- **낮음** — 일반 상식 + 친숙 어휘 / 새 개념 0~1개 / 일상 사례 위주 (예: 대본 01 "조리 간편 + 전쟁 안전" / 대본 03 첫 블록 "지평선 축제" / 대본 14 "한국 사례 — 제주 예멘")
- **중간** — 학술 용어 1~2개 + 다중 사례·수치 동시 / 친숙도 보통 (예: 대본 01 "안데스 기원 + 척박지 생존력" / 대본 04 "석면 정체 — 돌솜" / 대본 19 "삼권분립 + 강력한 행정부")
- **높음** — 학술·전문 용어 다수 + 다층 인과 / 단위·메커니즘 복합 (예: 대본 03 "화강암·자원·침식" / 대본 04 "석면 종류·다양한 활용성" / 대본 04 "위험 원리" / 대본 14 "옹호론 vs 실제 해악")

#### 어긋난 사례

**사례 B-1. 대본 01 "위기(소빙하기·전쟁) → 강제 보급" 블록 (05:55)**

- 현재 표기: 등급 = 중간 / 근거 = "다층 인과(기후 + 전쟁 + 정책 + 심리) 동시 등장, 다만 화자가 빠르게 흘려 부담 분산"
- 점검 대비: 대본 03 "지역 이동·도로 불가 — 행정력 디버프"(05:04)도 다층 인과(지형→교통→정치)인데 등급 = 중간. 대본 19 "신생국 인기 — 식민지 출신 + 미·소 양강"(06:22)은 5팩트 거시 인과 누적인데 등급 = **높음**.
- **불일치 의심**: 동일하게 4단계 이상 인과 사슬인데 대본 19만 "높음" 분류. 다만 대본 19는 거시 사례 5개 동시 등장(중남미·2차대전·신생국·냉전·미국 의도)으로 정보 밀도가 한 단계 더 높음. **차별 근거 있음 — 어긋남 아님**.

**사례 B-2. 대본 04 "솔라닌 독성 → 악마 식물 낙인" (04:29) vs 대본 01 동명 블록**

- 대본 04에는 솔라닌 단독 블록 없음 (석면 주제이므로 N/A). 대본 01 "솔라닌 독성"(04:29): 등급 = 중간 / "솔라닌 낯선 화학명 + 증상 6종 나열"
- 점검 대비: 대본 04 "위험 원리"(10:01) = 높음 — "좌절 식작용·산화 스트레스·사이토카인" 7단계 메커니즘 + 게임 슬랭 풀이.
- 동일하게 "독성/위험 메커니즘"이지만 정보층 수에 따라 중간↔높음으로 갈림. **차별 근거 있음 — 어긋남 아님**.

**사례 B-3. 대본 14 "옹호론 vs 실제 해악" (03:17)**

- 현재 표기: 등급 = 높음 / 근거 = "카티논·암페타민·도파민·노르에피네프린·향정신성·ADHD 치료제 등 전문 용어 다층"
- 점검 대비: 대본 04 "위험 원리" 등급 = 높음 (분자병리학 7단계). 두 블록 모두 약리·분자 영역 전문 용어 다수 + 메커니즘. **일관 정합**.

**사례 B-4. 대본 03 "70% 산지 — 산악국가 기준 미달" (02:16)**

- 현재 표기: 등급 = 중간 / 근거 = "환태평양 조산대 학술 용어 + 다중 수치 비교"
- 점검 대비: 대본 01 "칼로리·생장 속도 압도"(02:19): 등급 = 중간 / "1헥타르당 kcal 단위 + 3작물 수치 동시 비교"
- 두 블록 모두 "낯선 학술 1개 + 다중 수치 비교" 패턴. **일관 정합**.

**경계 사례 (보정 검토 필요)**

**사례 B-5. 대본 19 "장점② 임기·연임 제한 → 장기 정책 + 독재 방지" (05:18)**

- 현재 표기: 등급 = 중간 / 근거 = "30년 해먹는 정량 + 독재 메커니즘 동시 처리"
- 비교 대상: 대본 19 "장점③ 정치 외부 전문가 기용"(05:18 끝) = 낮음 / "백종원 매핑으로 즉시 직관"
- 점검: 장점②는 "임기 제한 → 장기 정책 → 자기 반박(독재 우려) → 30년 사례" 4단계. 그러나 모든 정보가 친숙 키워드(임기·독재·30년). **현재 "중간"은 다소 보수적** — 본 5편의 다른 "친숙 키워드 + 짧은 자기 반박" 블록은 "낮음"으로 분류된 경우 많음(예: 대본 01 "감자의 정체"). 다만 학계 통설 정합 측면에서 큰 차이 없으므로 보정 강제성 낮음.

**결론 (축 B):** 5편 전체 어긋남 거의 없음. 경계 사례 1건 (B-5)만 등급 재검토 여지 있으나 보정 강제성 낮음. **보정 권장 사례 — 사례 B-5 1건만 선택적 검토**.

#### 점검 — ❌ 누락 의심 사례 검토

**의심 사례 검토:**

- 대본 01 블록 17 "단점② 재배 노동력 多"(09:39): 루프 = 짧음 / 보상 = 중 — ✓ 정합

- 대본 04 블록 5 "석면 종류·다양한 활용성 + 산업혁명 후 폭증"(04:23): 정보 밀도 매우 높지만 보상도 강 — ✓ 정합
- 대본 14 블록 9 "자연·1차산업 무너짐 + 휘발성 의문"(07:38): 루프 = 약 50초 / 보상 = 약 (의도적 지연 단계) — 보상 약하지만 루프도 짧으므로 ✓ 정합
- 대본 19 블록 4 "워싱턴 2연임 자진 사퇴 → 대통령제 정착"(03:10 끝): 루프 = 짧음 / 보상 = 약 — ✓ 정합

다만 **"보상 = 약"인 블록이 ✓로 일관 유지**되는 패턴이 본 5편 공통. 가이드의 ❌ 기준 "긴 루프(2분+) + 약한 보상"이 본 5편에는 해당 사례 자체가 없는 것으로 보임. 지식해적단 대본의 보상-루프 설계가 정밀하다는 방증.

**결론 (축 D — 보상균형):**

- 5편 전체 ✓ 일관. ❌ 누락 의심 사례 없음.
- 다만 "보상 = 약" 블록도 ✓ 유지되는 패턴이 보편적이라, "약" 보상이 ❌ 트리거가 되지 않는 가이드 정의의 자연스러운 결과.

## § 5. 역설엔진 5편 공통 패턴 발견

본 점검을 통해 드러난 역설엔진 엔진 특성:

### 5.1 구조 패턴

- 메인 오픈루프 = **"A극 측면 + B극 측면" 이중 질문 패턴**

근거: 5편 실측 시 A극이 "긍정"인 경우와 "부정"인 경우 모두 존재. "긍정·부정"으로 묶이지 않음. 본질 = A극·B극 양쪽 정보 공백 개방.

> **셀 숫자 정의:** 셀 = 해당 편의 메인 오픈루프 텍스트·매핑.

| 편 | 메인 오픈루프 텍스트 | A극 매핑 | B극 매핑 |
|---|---|---|---|
| 01 감자 | "언제부터? + 장단점?" | 언제부터(A극 기원) | 장단점(A·B 양쪽 동시) |
| 03 한반도 | "어떤 시련? + 어떻게 생존?" | 생존(A극 장점) | 시련(B극 단점) |
| 04 석면 | "왜 발암? + 왜 수천 년간 못 알아챘나?" | 못 알아챔(A극 마법) | 발암(B극 저주) |
| 14 까트 | "왜 보급 못? + 왜 그래서는 안?" | 보급 못(영역 이동) | 그래서는 안(B극 함정) |
| 19 대통령제 | "왜 채택? + 장단점?" | 채택(A극 장점) | 장단점(A·B 양쪽 동시) |

- **피벗 = 짧은 반전 선언 패턴**: 5편 모두 짧고 정보 밀도 낮으며 다음 블록에서 본격 단점 정보 펼쳐짐. "그런데/하지만/100번 양보해도" 류 강한 반전 신호어 사용.

- **Aha-moment = [전] 본격 블록 (5편 중 4편)**: 대부분 [전] 진입 직후 2~3번째 블록에서 메인 오픈루프 핵심 회수. 대본 01만 [전] 2번째(아일랜드 대기근)와 [전] 마지막(잉카 추뇨)에 2회 Aha 분산.

- **A↔B 대칭 반전 패턴**: [승] 위력 빌드업에서 쓰인 학술 정보가 [전]에서 정반대 의미로 회수되는 메커니즘. 가장 강한 예 = 대본 04 (석면 = 0.02μm·청>갈>백석면이 위력 근거였다가 그대로 발암 기전 근거로 뒤집힘) / 대본 03 (산투성이·화강암이 단점이었다가 방어 자원으로 재해석).

### 5.2 정보 유형 분포 패턴

- 9유형 중 **해석·결산 (19) / 사례·일화 (12) / 인과 (12) / 메커니즘 (11)** 4유형이 본 5편 주축
- 인용·증언 주 유형 0건 — 권위 인사 직접 인용보다 화자 해석으로 가공하는 경향
- 통계·수치는 메인 오픈루프 직후 첫 정점 블록·Aha-moment 직전 보강 블록에 집중 (예: 대본 01 칼로리 비교·대본 03 70% 산지·대본 19 54개국)

### 5.3 인지 부담 분포 패턴

- 낮음 35 · 중간 39 · 높음 11 — **중간이 약 절반**
- 높음 11건 중 다수가 메커니즘 또는 다층 인과 블록 — Aha-moment 직전·직후 학술 정보 폭격 블록에 집중
- 낮음 35건 중 다수가 [기] 후킹 + 일상 회귀 [결] 블록 — 진입·마무리는 부담 낮춤

### 5.4 레고 작동 패턴

- 보상 투하 40 (47%) > 충전·빌드업 26 (31%) > 오픈루프 16 (19%) > 의도적 지연 3 (4%)
- **보상 투하가 약 절반** — 역설엔진의 "정점·Aha 반복" 구조 정합
- 의도적 지연이 본 5편 전체에서 3건뿐 — 본 표기이 가이드상 보편 적용보다 특수 상황에 한정될 가능성

### 5.5 끌리는 이유 패턴

역설엔진 공통 강세 4요인:

1. **6 반전형 (14)** — 1위 / 역설엔진 핵심 (A↔B 뒤집기)
2. **3 호기심형 (13)** = **5 상식 파괴형 (13)** — 공동 2위 / 도입·피벗·Aha 정점에 집중
3. **7 경고형 (12)** — 4위 / 단점 빌드업·결산에서 위기 자극

이 4요인 외 사용도가 낮은 요인: **9 최신 정보형 (3)·10 도덕적 공분 (3)·2 문제 제기형 (1)**.

→ 역설엔진은 시간적 최신성보다는 통념 반전·호기심 자극·위기 환기 중심.

## § 6. 다음 묶음(영역확장 7편) 진입 전 주의사항

### 6.1 가이드 정의 사전 명확화 필요 항목

- **피벗 역할 표기 통일안 (사례 C-2)** — 영역확장 7편 분석 진입 전 유저 결정 필요.
  - 권장 A: "오픈루프 + 1차 반전 폭로 동반" 통일
  - 권장 B: "보상 투하 + 오픈루프 이중 역할" 통일
  - 본 메모 권장은 A안.

- **의도적 지연 표기 적용 기준** — 영역확장엔진은 "영역 A → 영역 B 이동" 구조이므로 의도적 지연 발생 패턴이 다를 가능성. 영역 전환 직전 사전 충전 블록을 의도적 지연으로 볼지 충전·빌드업으로 볼지 사전 정렬 권장.

### 6.2 영역확장엔진 표기 예상 차이점

- **부가 표기**: 영역 A / 영역 B (역설의 긍정/부정 vs 영역확장의 첫 영역/두 번째 영역) — 본질적으로 다른 대비 구조이므로 끌리는 이유에서 6 반전형 빈도 감소 예상, 8 비교형 빈도 증가 예상
- **Aha-moment 위치**: 역설엔진은 [전] 진입 직후 Aha. 영역확장엔진은 두 영역의 공통 원리·차이 통찰이 Aha일 가능성 — 위치 예측 미리 가설로 두고 검증
- **메인 오픈루프 패턴**: 역설엔진의 "장점? 단점?" 이중 질문 대신 "왜 영역 A는 X이고 영역 B는 Y인가?" 단일 통합 질문 가능성

### 6.3 작업 운영 권고

- 영역확장 7편(대본 02·07·08·11·17·24광고·30)은 본 5편보다 블록 수 많음 (18~24블록 × 7편 = 약 140블록). 묶음 단위 점검·보정 시 본 메모와 동일 수준의 횡단 비교를 위해 분석 표기 일관성 즉시 점검 권장.
- 본 5편에서 발견된 "메인 오픈루프·피벗·Aha-moment·엔딩 4개 핵심 지점 끌리는 이유 표기" 패턴은 영역확장 7편에도 동일 적용 권장.

**역설엔진 §3.1 보정 권장안(피벗 = "오픈루프(+1차 반전 폭로 동반)" 통일)과의 정합 검토:**

- 역설엔진 피벗 본질 = A극↔B극 반전 신호 + 다음 빌드업 예고 → "오픈루프" 본질이 더 강함.
- 영역확장엔진 영역 전환점 본질 = **영역 A 결산 + 영역 B 시작** 이중 기능. 영역 A의 메인 오픈루프 회수 자체가 강한 보상이며 동시에 영역 B 진입 게이트.
- **본 7편 실측 양상**: 영역 A 결산 강도가 큰 편(02·07·11·24)은 "보상 투하" / 영역 B 게이트 강도가 큰 편(08·17·30)은 "오픈루프"로 분기. **본질적으로 이중 역할 블록**.
