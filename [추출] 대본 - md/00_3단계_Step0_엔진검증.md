# 3단계 Step 0 — 엔진 분류 신뢰성 검증

> 작성일: 2026-06-02
> 근거 가이드: `00_가이드_분석_방법.md` §7.3 Step 0
> 검증 대상: 지식해적단 39편 (8엔진 기존 분류)

---

## 1. 작업 개요

### 1.1 목적

3단계 정보 배치 분석(편당 4축 동시 분석)에 진입하기 전, 기존 8엔진 분류가 본 분석의 출발점으로 적절한지 검증한다. 일치율 ≥80% → 기존 엔진 분류 유지. 미만 → 재분류 또는 정의 수정 권장.

### 1.2 방법

39편 각각에 대해 4축 빠른 판정 (편당 깊은 분석 아님, 본문 [기]·[승]·[전]·[결] 구조 + 부가 라벨 + 토픽 블록 흐름 기반):

| 축 | 판정 값 |
|---|---|
| 시간 구조 | 시간순 / 비시간순 |
| 대립 구조 | A극↔B극 있음 / 없음 |
| 인과 깊이 | 단일 인과 / 다층 인과 |
| 사례 수 | 단일 깊이 / 복수 비교 / 복수 나열 |

### 1.3 판정 기준 (운영 정의)

- **시간순**: [승]·[전]이 연대순(고대→근대 등) 또는 단일 사건 흐름을 따라 진행
- **비시간순**: 영역·주제 단위 묶음 (예: 좌식 영역 / 입식 영역) 또는 동시 다발 분기
- **A극↔B극 있음**: 부가 라벨이 두 대립 영역(긍정/부정·장점/단점·전·후 등) 구분 + 피벗 명시
- **단일 인과**: 한 사건이 다음 사건으로 사슬형 연결 (A→B→C→D)
- **다층 인과**: 여러 원인이 한 결과로 수렴 또는 한 원인이 여러 결과로 분기
- **단일 깊이**: 한 대상(주제)을 깊게 추적
- **복수 비교**: 2~3개 대상 명확 비교 (A vs B)
- **복수 나열**: 4개 이상 사례 병렬 나열

## 3. 4축 조합 → 엔진 매핑 규칙

39편 패턴 관찰 기반으로 4축 조합과 기존 8엔진의 매핑 규칙을 정리한다.

| 엔진 | 시간 구조 | 대립 구조 | 인과 깊이 | 사례 수 |
|---|---|---|---|---|
| 01_역설엔진 | 혼재 (시간순 3/비시간순 2) | A극↔B극 있음 (5/5) | 다층 (5/5) | 단일 깊이 (5/5) |
| 02_영역확장엔진 | 비시간순 우세 (4/7) | A극↔B극 있음 (7/7) | 다층 (6/7) | 단일 깊이 우세 (5/7) |
| 03_인과추적엔진 | 시간순 (17/17) | 혼재 (있음 5/없음 12) | 단일 (17/17) | 단일 깊이 (17/17) |
| 04_결합형 | 시간순 (2/2) | A극↔B극 있음 (2/2) | 다층 (2/2) | 혼재 |
| 05_다층원인수렴 | 비시간순 (3/3) | 혼재 | 다층 (3/3) | 복수 비교 우세 |
| 06_사례비교 | 비시간순 (3/3) | 혼재 | 혼재 | 복수 비교/나열 (3/3) |
| 07_사례나열 | 비시간순 (1/1) | 없음 (1/1) | 단일 (1/1) | 복수 나열 (1/1) |
| 08_다층분기수렴 | 비시간순 (1/1) | 있음 (1/1) | 다층 (1/1) | 복수 나열→단일 (1/1) |

### 3.1 4축 → 엔진 추론 규칙 (도출)

- **시간순 + A극↔B극 + 다층 + 단일** → 역설엔진 또는 결합형
- **비시간순 + A극↔B극 + 다층 + 단일/복수 비교** → 영역확장엔진
- **시간순 + 대립 없음 + 단일 인과 + 단일** → 인과추적엔진 (전형)
- **시간순 + A극↔B극 + 단일 인과 + 단일** → 인과추적엔진 (시기 대비형) — 본 분류에서 5편 발견
- **비시간순 + 다층 + 복수 비교** → 다층원인수렴 또는 사례비교
- **비시간순 + 대립 없음 + 단일 + 복수 나열** → 사례나열
- **비시간순 + A극↔B극 + 다층 + 복수 나열→단일** → 다층분기수렴

## 4. 기존 엔진 vs 4축 분류 일치율

### 4.1 일치 판정 기준

각 대본의 4축 판정값이 §3.1 매핑 규칙의 해당 엔진 전형 패턴과 부합하면 "일치", 부합하지 않으면 "불일치".

### 4.2 엔진별 일치율

| 엔진 | 편 수 | 일치 | 불일치 | 일치율 |
|---|---|---|---|---|
| 01_역설엔진 | 5 | 5 | 0 | 100% |
| 02_영역확장엔진 | 7 | 6 | 1 (24_전기차 — 단일 인과) | 86% |
| 03_인과추적엔진 | 17 | 12 | 5 (25·28·35·38·39 — A극↔B극 보유) | 71% |
| 04_결합형 | 2 | 2 | 0 | 100% |
| 05_다층원인수렴 | 3 | 3 | 0 | 100% |
| 06_사례비교 | 3 | 3 | 0 | 100% |
| 07_사례나열 | 1 | 1 | 0 | 100% |
| 08_다층분기수렴 | 1 | 1 | 0 | 100% |
| **전체** | **39** | **33** | **6** | **84.6%** |

### 4.3 전체 평가

- **전체 일치율 84.6%** → 임계 기준(80%) 초과
- **다만 03_인과추적엔진 일치율 71%**가 임계 미달 — 세부 점검 필요
- 02_영역확장엔진의 24_전기차(광고)는 광고 형식상 인과 깊이가 얕음 (광고 특수 케이스)

## 5. 불일치 케이스 분석

### 5.1 03_인과추적엔진 내 A극↔B극 보유 5편

| 대본 | 대립 구조 | 재분류 후보 | 비고 |
|---|---|---|---|
| 25_소주(광고) | 증류식 vs 희석식 | 인과추적 유지 | 시간순 단일 인과 골격, 대립은 부가 |
| 28_한국몽골 | 갈등·단절 vs 1990 수교 후 | 인과추적 유지 | 단절기→수교기 시기 대비, 단일 인과 |
| 35_콜로세움 | 건설·전성 vs 쇠퇴·부활 | 인과추적 유지 | 흥망성쇠 시기 대비, 단일 인과 |
| 38_강뉴부대 | 전공 vs 박해→회복 | 인과추적 유지 | 영광기→고난기, 단일 인과 |
| 39_차우셰스쿠 | 독재 인과 vs 12월 혁명 | 인과추적 유지 | 부상→몰락, 단일 인과 |

**핵심 발견:** 이 5편의 대립은 "**시기 대비** (시기 A↔시기 B)"이지 "**속성 대비** (장점↔단점·위력↔함정)"가 아니다. 역설엔진의 대립은 같은 대상의 양면(속성 대비)이지만, 인과추적엔진의 시기 대비는 시간 진행 자체가 자연스럽게 만드는 부수적 대비다.

**→ 4축 정의 정밀화 권장:** 대립 구조 축을 "**속성 대립**(양면성 강조)"과 "**시기 대립**(시간 진행 결과)"으로 세분화. 시기 대립만 보유한 경우 인과추적엔진 유지.

이 정의를 적용하면 03_인과추적엔진 일치율은 **71% → 100%**로 회복된다.

### 5.2 24_전기차(광고)

- 광고 영상으로 [전] 영역에서 인과 깊이가 얕음 (단일 인과로 판정).
- 다만 [기]·[승] 영역에서 다층 인과 일부 존재 (한반도 산업 조건·전기차 단점 다층).
- **→ 광고 특수 케이스**로 기존 영역확장엔진 분류 유지 가능.
