### 2.6 핵심 패턴

#### 패턴 1 — [승]은 빌드업, [전]은 회수가 우세

- [승] 38블록 중 충전·빌드업 17 (45%)
- [전] 25블록 중 보상 투하 12 (48%)
- **해석:** [승]은 정점(A극 정점)까지의 누적 사다리, [전]은 피벗 직후부터 회수 폭발이 연속

#### 패턴 2 — [승] 정보유형 4종(메커니즘·기원·비교·사례) 16% 동률 + [전]은 메커니즘·사례 20% 동률

- [승]은 정보유형 분산 — 메커니즘·기원·비교·사례를 골고루 활용 (단조 빌드업 방지)
- [전]은 메커니즘·사례에 집중 — 함정 작동 원리(메커니즘) + 충격 사례(아일랜드·중피종·예멘 농업) 콤보가 표준

#### 패턴 3 — [전] 입구의 피벗 = 반전형 + 호기심형 + 경고형 3종 동시 점화

- 5편 모두 피벗 블록에 "6 반전형"이 명시되어 있으며, 호기심형·경고형이 결합
- 피벗 = 빌드업 정점에서 한 문장으로 흐름 전복하는 회전축

#### 패턴 4 — A극 안에 의도적 지연(미니 굴곡) 삽입 가능

- 감자 [승]: 유럽 거부·돼지 사료 (2블록 = A극 안의 의도적 저점)
- 한반도 [승]: 4군 6진 위업·조선 위업 (A극 안의 미니 반전)
- **변주 허용 영역** — A극 단조 빌드업 길이가 6분 이상일 때 의도적 지연 1~2블록 권장

#### 패턴 5 — A극 → B극 이행 시 옮겨가는 유형

- A극에서 누적된 **메커니즘·비교**가 B극 피벗 직후 그대로 "함정 작동 원리"로 전환
- 예: 감자 — A극 "무성생식 → 척박지 강함" → B극 "무성생식 → 전염병 취약" (같은 메커니즘의 양면 노출)
- **이행 자체가 역설엔진의 핵심 장치** — 같은 정보가 위력이자 함정

### 2.7 P3 구조설계 변환 권장안 (역설형 작성 시)

#### 핵심 패턴 (강제 — 양식 본질 유지)

| 항목 | 권장 범위 | 강제 사유 |
|---|---|---|
| [승] 블록 수 | 전체의 35~50% | A극 누적이 부족하면 피벗 진폭 약화 |
| [전] 블록 수 | 전체의 25~35% | B극 회수가 부족하면 역설 미완성 |
| [승] 정보유형 분산 | 4종 이상 혼용 | 단조 빌드업 방지 |
| [전] 피벗 블록 1개 | 필수 — [전] 시작 | 반전형 6 + 호기심형 3 또는 경고형 7 결합 |
| [승] 끝 = A극 정점 | 필수 | 피벗 직전 최고점 형성 |
| [전] 끝 = 반전 회복 또는 함정 결산 | 필수 | 회수 완결 |
| A극 ↔ B극 같은 메커니즘 양면 | 권장 | 역설 핵심 장치 — 같은 정보가 위력이자 함정 |

#### 변주 허용 영역 (가변 — 주제별 자유)

| 항목 | 가변 범위 | 사유 |
|---|---|---|
| [승]·[전] 명칭 (위력/함정·장점/단점·마법/저주 등) | 주제 톤에 맞춰 자유 명명 | 5편 모두 다름 |
| [승] 첫 항목이 긍정/부정 | 자유 — 03 한반도는 [승]=단점 | 단조 → 반전 구조 자체가 핵심 |
| [승] 안의 의도적 지연 1~2블록 | A극 길이 6분 이상일 때 권장 | 짧으면 불필요 |
| 이문호공 조합 | 권장 조합이지 강제 매칭 아님 | [승] = 5+3+8 / [전] = 6+7 권장이지 새 주제는 다를 수 있음 |
| 9유형 비율 | ±5%p 가변 | 주제별 정보 가용성에 따라 |
| 결론부 블록 수 | 1~3블록 자유 | 본 채널 결론부 글자수 제거 정책 반영 |

#### P3 결정 트리 (역설형 새 주제 진입 시)

```
1. 주제의 "단조 인상" 확인 — 일면적으로 강해/약해 보이는 대상인가?
   YES → 역설형 채택. NO → 다른 양식 검토.

2. [승] 방향 결정 — A극을 긍정으로 갈지 부정으로 갈지
   주제의 통념이 긍정이면 [승]=긍정(단점 폭로 진폭 ↑)
   주제의 통념이 부정이면 [승]=부정(반전 발견 진폭 ↑)

3. [승] 길이 결정 — 6분 이상이면 의도적 지연 1~2블록 삽입
   6분 미만이면 단조 빌드업 그대로 진행

4. 피벗 1블록 — 반전형 6 + (호기심형 3 OR 경고형 7) 결합어 작성

5. [전] 사다리 설계 — 단점/회복 항목 3~5개 적층
   각 항목마다 메커니즘(원리) + 사례(증거) 1세트 권장

6. 결론부 — 양 극 결산 + 미래 전망 또는 위트 반전
```

#### 새 주제 가변성 강조

본 권장안은 **39편 중 5편 역설엔진 분석의 평균치 권장 범위**다. 새 주제가 다음 경우라면 권장안에서 벗어나도 무방:

- 주제의 핵심 정보가 **메커니즘 1종에 집중**되어 있는 경우 → [승] 정보유형 분산 강제 완화
- 주제가 **사회적 논쟁**을 다루는 경우 → 10 도덕적 공분형 가중 (예: 까트·아일랜드 영국 지배)
- 주제가 **과학·기술**인 경우 → 9 최신 정보형 + 1 이익 제시형 가중 (예: 석면 미래 발병)
- 주제가 **시청자 일상**에 직접 닿는 경우 → 4 공감형 가중 (예: 한반도·석면 삼겹살)
