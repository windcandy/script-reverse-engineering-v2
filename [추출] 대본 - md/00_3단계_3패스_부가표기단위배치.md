### 1.3 핵심 원칙 — 다양한 주제 가변성 인정

**본 산출물의 분포·패턴은 39편 분석 결과의 평균치.** 본 채널이 새 주제로 영상을 만들 때:
- 9유형 분포는 양식별 **권장 범위** (강제 비율 아님)
- 끌리는 이유(이문호공) 조합은 **권장 조합** (강제 매칭 아님)
- 영역별 정보 배치는 **핵심 패턴(강제) + 변주 허용 영역(가변)** 분리

새 주제마다 정보 유형·후킹 정보가 제각각이므로, 본 산출물은 P3 구조설계 시 **출발점** 역할만 한다. 주제 특성에 맞춰 가변 적용.

### 2.6 핵심 패턴

#### 패턴 1 — [승]은 빌드업, [전]은 회수가 우세

- [승] 38블록 중 충전·빌드업 17 (45%)

- **해석:** [승]은 정점(A극 정점)까지의 누적 사다리, [전]은 피벗 직후부터 회수 폭발이 연속

- [전]은 메커니즘·사례에 집중 — 함정 작동 원리(메커니즘) + 충격 사례(아일랜드·중피종·예멘 농업) 콤보가 표준

- 피벗 = 빌드업 정점에서 한 문장으로 흐름 전복하는 회전축

- **변주 허용 영역** — A극 단조 빌드업 길이가 6분 이상일 때 의도적 지연 1~2블록 권장

#### 패턴 5 — A극 → B극 이행 시 옮겨가는 유형

- A극에서 누적된 **메커니즘·비교**가 B극 피벗 직후 그대로 "함정 작동 원리"로 전환
- 예: 감자 — A극 "무성생식 → 척박지 강함" → B극 "무성생식 → 전염병 취약" (같은 메커니즘의 양면 노출)
- **이행 자체가 역설엔진의 핵심 장치** — 같은 정보가 위력이자 함정

### 5.6 핵심 패턴 (5종)

#### 패턴 1 — [승] 빌드업 + [전] 폭발 비대칭 — 결합형 최대 특징

- [승] 충전 55% / [전] 보상 90% — 8엔진 중 가장 극단적 비대칭
- **해석:** [승]은 인과 사다리 누적에 집중, [전]은 피벗 직후부터 회수 폭발 연속. 인과추적엔진([승] 45% / [전] 50%)이나 역설엔진([승] 45% / [전] 48%)보다 양극 분리 강함

#### 패턴 2 — 이중 Aha (역설 Aha + 인과 Aha 동시 작동)

> **이중 Aha 정의:**
> - **역설 Aha** = 속성·의미 반전 폭발 (예: "위협이 알고 보니 구원자")
> - **인과 Aha** = 시간·사상 인과 체인 도달 폭발 (예: "캘리포니아→칠레→멕시코로 환경 파괴 사슬 추적")

- **23 아보카도:** [승] 4번 "인간이 구원자" = 역설 Aha (위협→구원자 속성 반전) / [전] 2번 "캘리포니아→칠레→멕시코" = 인과 Aha (인과 추적 정점)
- **29 기독교시온주의:** [전] 1번 피벗 "세대주의 vs 성취신학" = 1차 반전 / [전] 6번 "이스라엘 = 세속국가 + 이용" = 2차 반전
- **결합형 본질** — 역설 Aha와 인과 Aha를 한 영상 안에서 둘 다 폭발시킴. 단일 엔진(역설만/인과만)은 한 종류 Aha만 작동

#### 패턴 3 — [전] 진입 = 피벗 = 보상 90%

- 2편 모두 [전] 1번 블록 = 피벗 명시 + 보상 투하 역할
- [승] 인과 정점(글로벌화 12블록 / 세대주의 8블록) 직후 = 그림자 폭발 진입
- 끌리는 이유: 6 반전형 + 5 상식 파괴형 + (7 경고형 OR 3 호기심형) 결합 표준
- **역설엔진의 피벗 표기를 그대로 차용** — 결합형은 역설 본질을 포함하므로 [전] 진입 용어 = "피벗" (인과추적엔진의 "[전] 진입"과 구분)

#### 패턴 4 — [승] 인과·기원 50% + [승] 안의 미니 반전 의도적 지연

- [승] 인과 30% + 기원 20% = 50% — 시간/사상 인과 축이 절반 차지
- [승] 안에 의도적 지연 10% 등장 (23-3 인간 등장 = 멸종 원인 / 23-9 1914 로비 = 정책 개입). 인과 단조 누적 방지용 미니 굴곡
- **변주 허용 영역** — [승] 길이가 8블록 이상일 때 의도적 지연 1~2블록 권장 (23 아보카도 12블록 [승] = 2건 지연 / 29 기독교시온주의 8블록 [승] = 0건 지연)

#### 패턴 5 — [승] 길이 가변·[전] 길이 가변 — 표본 작음 인정

- [승]: 23 = 12블록 / 29 = 8블록 — 인과 노드 수에 따라 가변
- [전]: 23 = 3블록 / 29 = 7블록 — 그림자 사례 수에 따라 가변. 23은 짧은 폭발형(3블록 압축), 29는 긴 연쇄형(7블록 — 종말 시나리오·미국 보수·이스라엘 건국·이용·시너지 누적)
- **표본 2편 한정** — 일반화는 후속 분석(또는 추가 결합형 편 발견) 누적 후 재검증

- **변주 허용 영역** — [승] 7블록 이상일 때 의도적 지연 + 오픈루프 각 1~2건 권장

#### 패턴 5 — 메인 오픈루프 = 단일 질문 (Why) / 분산 회수 N단계

- 3편 모두 메인 오픈루프 = "왜 그러한가" 단일 질문 (09 "왜 무해한 이미지?" / 26 "왜 수입하나?" / 27 "정말 한국인은 냄새 안 나나?")
- 회수는 [승] N블록 + [전] M블록에 걸쳐 분산. 각 원인 블록이 메인 오픈루프의 부분 회수
- 26편 특수 — [승]5(수요)·[승]6(설비)에서 "메인 회수 1차·2차" 명시. 다층원인수렴엔진 회수 패턴 표준
- **다층원인수렴엔진의 본질적 분산 회수** — 한 블록에서 끝나는 단발 Aha가 아니라 N블록 누적 후 1점 수렴

- **[전] 진입 표기** = "[전] 진입" 고정 (인과추적엔진과 동일 — "피벗" 사용 금지)

## § 7. 사례비교엔진 — 사례 A 깊이 / 사례 B 깊이

> **셀 숫자 정의 (§7 전체 공통):**
> - **§7.1·§7.3·§7.4 셀 = 토픽 블록 수** (가이드 §4.3 정합). 분모 = 영역별 블록 수 ([승] 18·[전] 16).
> - **§7.5 셀 = 이문호공 요인 빈도** (한 블록에 다요인 동시 매칭 가능 — 중복 카운트).
> - **비중 반올림**: 소수점 첫째 자리 반올림.
> - **본 채널 3유형 매칭:** **과정 추적형** (가이드 §4.6 정합 — 인과추적 본질 공유 + 두 사례 깊이 비교).
> - **[전] 진입 용어:** **"사례 전환점"** (가이드 §7.6.3 정합 — 사례비교엔진은 시간 단절이 아니라 **사례 A → 사례 B 전환**을 표시. 인과추적엔진·다층원인수렴엔진의 "[전] 진입"·역설엔진·결합형의 "피벗"과 구분).

### 7.2 부가 표기 영역 식별

| 편 | [승] 사례 A 깊이 명칭 | [전] 사례 B 깊이 명칭 |
|---|---|---|
| 10 종교고기금기 | 돼지 금기 사례 | 소 금기 사례 |
| 32 숟가락 | 중국·일본 사례 | 한국 사례 |
| 40 중화요리 | 동남아 화교 사례 | 19세기 이후 각국 사례 |

**관찰:**

- 3편 모두 **두 사례를 각각 깊이 있게 파고든 뒤 공통 원리로 수렴** 구조 — 사례비교엔진 본질이 양식으로 고정
- [승] 사례 A는 **시기상 먼저 발생·역사 더 긴 사례** (돼지 금기·중국·일본·동남아 송명 화교) 우세
- [전] 사례 B는 **시기상 후발·예외·특수성 강한 사례** (소 금기·한국·19세기 광동 화교)
- 3편 모두 [전] 첫 블록 = **"사례 전환점"** 명시 (보상 투하 + 사례 A·B 분기 이중 역할)
- **사례비교엔진 본질** = "사례 A 깊이 + 사례 B 깊이 + 공통 원리(보편 명제)" 3축 결합. 각 사례는 단순 나열이 아니라 환경·시기·구조 깊이까지 파고들고, 두 사례가 함께 보편 명제(10 = 종교는 환경 안전장치 / 32 = 숟가락 유무는 음식·식기 인과 / 40 = 중화요리 침투력은 화교 이주사)를 증명

### 7.6 핵심 패턴 (5종)

#### 패턴 1 — 사례 A 깊이 + 사례 B 깊이 → 공통 원리 수렴이 본질

- 10 종교고기금기: [기]에 공통 원리("종교 = 안전장치") 선제시 → [승] 돼지 금기로 1차 증명 → [전] 소 금기로 2차 증명 → [결] 양 사례 종합
- 32 숟가락: [기]에 메인 오픈루프("왜 한국만?") → [승] 중국·일본 사례로 동아시아 소멸 인과 → [전] 한국 사례로 보편 유지 인과 → [결] 만능 숟가락 해석
- 40 중화요리: [기]에 메인 오픈루프("어떻게 세계 곳곳?") → [승] 동남아 화교(송명) 사례 → [전] 19세기 광동 화교 각국 사례 → [결] 한·중·일 퓨전 종합
- **사례비교엔진과 사례나열엔진 차이** — 사례비교는 "2영역 깊이 비교 + 공통 원리 수렴" / 사례나열은 "N영역 균등 나열" (§8에서 상술 예정)
- **사례비교엔진과 다층원인수렴엔진 차이** — 사례비교는 "두 사례 깊이 → 공통 원리" / 다층원인수렴은 "N층 원인 → 1점 수렴". 본 엔진은 "사례" 단위, 다층원인수렴은 "원인 차원" 단위

#### 패턴 2 — [전] = 사례 전환점 = 보상 + 오픈루프 이중 역할

- 3편 모두 [전] 1번 블록 = "사례 전환점" 명시
- 10 [전]1 = 아리아인 한 뿌리 → 유럽·인도 분기 (사례 B 진입)
- 32 [전]1 = "어 근데 한국인들은" 즉시 반전 (사례 B 진입)
- 40 [전]1 = "19세기를 전후해서" 시기·출신지 전환 (사례 B 진입)
- 끌리는 이유: 6 반전형 + 3 호기심형 + 8 비교형 결합 표준

#### 패턴 3 — 사례 A·B 대비축은 주제별 가변

- 10 = **환경 차이축** (사막=돼지 금기 / 인도=소 금기)
- 32 = **음식·식기 인과축** (중국·일본=숟가락 소멸 / 한국=보편 유지)
- 40 = **시기·출신지축** (송명 복건 자발 / 19세기 광동 강제 모집)
- **변주 허용 영역** — 사례 A·B 대비축은 주제 특성에 맞춰 자유 선택. 환경·시기·인과·구조·지리 등 다양

#### 패턴 4 — 사례 깊이 = 다층 정보 동시 투입 (인지 부담 "높음" 등장)

- [승] "높음" 2건: 32 [승]4 (송 식문화 4팩트) · 40 [승]5 (동남아 4사례)
- [전] "높음" 4건: 32 [전]2 (한국 4원인) · 32 [전]4 (조선 유교 4팩트) · 40 [전]2 (미주 6사례) · 40 [전]5 (일본 면류)
- **사례 깊이 = 한 블록 안에 4~6개 사실 동시 투하**가 사례비교엔진의 특징
- **변주 허용 영역** — [승] 5블록 이상이면 "높음" 1~2블록 권장 (사례 깊이 정점 형성)

#### 패턴 5 — [승]·[전] 길이는 균형형 (둘 다 5~7블록)

- [승]: 10=6·32=7·40=5 (평균 6)
- [전]: 10=5·32=5·40=6 (평균 5.3)
- 인과추적엔진([승] 9.4·[전] 4.3)이나 영역확장엔진([승] 11.3·[전] 5.9)과 달리 [승]·[전] 길이 균형. 두 사례 모두 깊이 있게 다루기 때문
- **변주 허용 영역** — 사례비교엔진은 [승]·[전] 비대칭 약함. 한쪽이 다른 쪽의 2배 이상이면 사례비교 본질 약화 → 인과추적형 또는 영역확장형 재검토 권장

### 7.7 P3 구조설계 변환 권장안 (사례비교형 작성 시)

#### 핵심 패턴 (강제 — 양식 본질 유지)

| 항목 | 권장 범위 | 강제 사유 |
|---|---|---|
| [승] 블록 수 | 전체의 35~50% | 사례 A 깊이가 부족하면 공통 원리 수렴 진폭 약화 |
| [전] 블록 수 | 전체의 30~45% | 사례 B 깊이가 부족하면 비교 본질 미완성 |
| [승]·[전] 길이 균형 | 한쪽이 다른 쪽의 1.5배 이내 권장 | 한쪽 과다하면 사례비교 본질 약화 → 다른 엔진 검토 |
| [승] 인과·기원·배경 비중 | 합산 50% 이상 | 사례 A 발생 배경·인과 누적 — 단순 나열 회피 |
| [전] 첫 블록 = "사례 전환점" | 필수 — [전] 시작 | 보상+오픈루프 이중 역할, 반전형 6 + 호기심형 3 + 비교형 8 결합 |
| [승] 끝 = 사례 A 결산 또는 정점 | 필수 | 사례 전환점 직전 최고점 형성 |
| [전] 끝 = 사례 B 결산 + 공통 원리 수렴 | 필수 | 두 사례 통합 회수 완결 |
| 공통 원리(보편 명제) = [기]에 선제시 또는 [결]에 결산 | 필수 | 사례비교 본질 — 두 사례가 함께 증명할 명제 |

#### 변주 허용 영역 (가변 — 주제별 자유)

| 항목 | 가변 범위 | 사유 |
|---|---|---|
| [승]·[전] 명칭 (돼지/소·중국일본/한국·동남아/19세기각국 등) | 주제 사례에 맞춰 자유 명명 | 3편 모두 다름 |
| 사례 A·B 대비축 (환경·시기·인과·구조·지리 등) | 자유 선택 | 3편 3종 |
| 사례 깊이 정점 블록 "높음" 인지부담 | [승] 5블록 이상 시 1~2블록 권장 | 짧으면 불필요 |
| 이문호공 조합 | 권장 조합이지 강제 매칭 아님 | [승] = 5+6+1 / [전] = 6+5+3+8 권장이지 새 주제는 다를 수 있음 |
| 9유형 비율 | ±5%p 가변 | 주제별 정보 가용성에 따라 |
| 결론부 블록 수 | 1~2블록 자유 | 본 채널 결론부 글자수 제거 정책 반영 |
| 메인 오픈루프 위치 | [기]에 선제시 또는 [기]·[결]에 분산 | 10편=[기]에 공통 원리 선제시 / 32·40편=[기]에 질문 형태 |

#### P3 결정 트리 (사례비교형 새 주제 진입 시)

```
1. 주제의 "두 사례 깊이 비교 가능성" 확인 — 두 사례가 공통 원리를 함께 증명하는가?
   (환경별 종교 금기·국가별 식문화·시기별 화교 이주 등)
   YES → 사례비교형 채택. NO → 다른 양식 검토.

2. 사례 A·B 결정 — 대비축에 맞춰 두 사례 선택
   사례 A = 시기 더 길거나 친숙한 쪽 권장
   사례 B = 후발·예외·특수성 강한 쪽 권장

3. 공통 원리(보편 명제) 결정 — 두 사례가 함께 증명할 명제 명시
   [기]에 선제시 (10편형) 또는 [기]에 질문 형태 (32·40편형)

4. [승]·[전] 길이 결정 — 균형형 (5~7블록 권장)
   한쪽이 2배 이상이면 다른 엔진 검토

5. [승] 사례 A 깊이 설계 — 인과·기원·배경 50% 이상
   각 블록 = 사례 A의 한 측면 (환경·시기·구조)
   5블록 이상 시 정점 블록 "높음" 인지부담 1~2건 권장

6. 사례 전환점 1블록 — [전] 첫 블록에 보상+오픈루프
   "같은 뿌리인데 정반대 결과" 또는 "어 근데 한쪽은" 어법
   반전형 6 + 호기심형 3 + 비교형 8 결합어 작성

7. [전] 사례 B 깊이 설계 — 인과·사례·해석 결합 4~6블록
   각 블록 = 사례 B의 한 측면

8. 결론부 — 두 사례 통합 결산 + 공통 원리 봉인 + 일상 회귀 또는 위트
```

#### 새 주제 가변성 강조

본 권장안은 **39편 중 3편 사례비교엔진 분석의 평균치 권장 범위**다. **표본 작음 — 일반화 한계.** 새 주제가 다음 경우라면 권장안에서 벗어나도 무방:

- 주제가 **종교·문화·환경 결정론**(10편형)인 경우 → [기]에 보편 명제 선제시 + 4 공감형 가중 (가난한 농민·약자 시점)
- 주제가 **국가별 식문화·일상 비교**(32편형)인 경우 → 다층 원인 동시 투입 + 1 이익 제시형 가중 (도입부 의문 명시 회수)
- 주제가 **시기별 이주·전파사**(40편형)인 경우 → 사례 다층 나열(블록당 3~6사례) + 5 상식 파괴형 가중 (서양 대표 = 동양 기원 통념 정정)
- 주제의 **두 사례 깊이 차이가 큰** 경우 (한쪽만 깊이) → 사례비교 본질 약화 → 영역확장형 또는 인과추적형 검토 권장
- 주제의 **세 개 이상 사례 균등 나열** 경우 → 사례나열엔진 채택 권장
- **사례 전환점 표기** = "사례 전환점" 고정 (인과추적엔진 "[전] 진입" / 역설엔진·결합형 "피벗"과 구분 — 사례비교엔진은 시간 단절·속성 반전이 아닌 **사례 분기**를 표시)

> **§7.3~§7.5 분포 분석 기준:** 3편 [승]·[전] 34블록 수동 카운팅. 사례비교엔진은 본 분석 시점 39편 중 3편만 분류되어 표본 작음. §8·§9 후속 분석 누적 후 본 절 패턴 재검증 필요.

## § 8. 사례나열엔진 — 배경·원리 / 다중 사례 나열

> **셀 숫자 정의 (§8 전체 공통):**
> - **§8.1·§8.3·§8.4 셀 = 토픽 블록 수** (가이드 §4.3 정합). 분모 = 영역별 블록 수 ([승] 3·[전] 10).
> - **§8.5 셀 = 이문호공 요인 빈도** (한 블록에 다요인 동시 매칭 가능 — 중복 카운트).
> - **비중 반올림**: 소수점 첫째 자리 반올림.
> - **본 채널 3유형 매칭:** **과정 추적형** (가이드 §4.6 정합 — 인과추적 본질 공유 + 다중 사례 균등 나열).

### 8.6 핵심 패턴 (5종)

#### 패턴 1 — [승] 배경·원리 (무거움) → [전] 사례 N건 (가벼움 분산) 비대칭

- [승] "높음" 100% (3/3) / [전] "높음" 10% (1/10) — 8엔진 중 가장 극단적 인지 부담 비대칭
- **해석:** [승]은 무거운 기준 수치·메커니즘을 한 번에 깔고, [전]은 그 기준이 적용되는 사례를 가벼운 단위로 분산. 사례나열엔진 본질

#### 패턴 2 — [전] 사례 N건 충격 강도·친숙도 가변 정렬

- [전] 사례 배치 순서 (20 해수면상승): 작은 섬나라 → 한·중 → 일본 → 인도네시아 → 방글라데시 → 인도·파키스탄 → 유럽 → 아프리카·북미 → 미국 빅 유
- **이중 정렬** — 충격 강도순(가장 위험 먼저) + 친숙도순(시청자 가까운 쪽). 친숙 사례(한국)는 충격 정점 직후 배치
- **변주 허용 영역** — 사례 N≥5건 시 정렬축 자유 (충격순 / 친숙도순 / 시간순 / 지리순 등)

#### 패턴 3 — [전] 사례 폭발 절정 1블록 = Aha-moment

- [전] 8블록(유럽 대비책 5국 6프로젝트) = 사례 나열 정점 — 한 블록에 5국·6프로젝트 동시 투하. 메인 오픈루프 "대처?" 최대 밀도 회수
- 끌리는 이유: 1 이익 제시형 + 9 최신 정보형 결합 표준 — "이미 효과적 운용 + 진행형"
- **변주 허용 영역** — [전] N≥7블록 시 정점 1~2블록 권장 (사례 5건 이상 동시 투하)

#### 패턴 4 — [승] 결산 = 기준 수치 확정

- [승] 마지막 블록 = "16m"·"기준 수치 확정" 역할. 22cm(과거) → 1.6m(2100년) → 16m(2300년) 시간축 폭발이 [전] 모든 사례 충격의 기준점
- 인과추적엔진의 "[승] 끝 = 옛 시기 정점"·영역확장의 "[승] 끝 = 영역 A 정점"과 같은 정점 형성 기능

#### 패턴 5 — 메인 오픈루프 = N중 질문 / N단계 분산 회수

- 메인 오픈루프 = "어디?·부작용?·대처?" 3중 질문 (사례나열엔진 표준)
- 회수는 [전] N블록에 걸쳐 분산 — 각 사례 블록이 부분 회수
- 다층원인수렴엔진(단일 Why)·사례비교엔진(2사례 분기)와 달리 사례나열엔진은 **N중 질문 → N건 사례 균등 매핑**
- **표본 1편 한정** — 메인 오픈루프 N중 질문 패턴은 후속 사례나열엔진 편 발견 시 재검증

> **§8.3~§8.5 분포 분석 기준:** 1편 17블록 수동 카운팅. 사례나열엔진은 본 분석 시점 39편 중 1편만 분류되어 표본 작음. **단일 사례 가능성 명시** — §9 다층분기수렴엔진 등 후속 누적 후 본 절 패턴 재검증 필요.

   [승] 마지막 블록 = 기준 수치 확정 강제

| [승] 블록 수 | 전체의 35~50% | 분기 N≥3건 누적이 부족하면 수렴 진폭 약화 |
| [전] 블록 수 | 전체의 25~40% | 수렴 정체+양면성이 부족하면 수렴 본질 미완성 |
| [승] 사례·일화 비중 | 50% 이상 | 분기 단위(사례·국가·환경) 명료화 — 단순 메커니즘 회피 |
| [전] 첫 블록 = "수렴 진입" | 필수 — [전] 시작 | 보상+오픈루프 이중 역할, 반전형 6 + 경고형 7 결합 |
| [승] 끝 = 분기 정점 (가장 강한 분기) | 필수 | 수렴 진입 직전 최고점 형성 |
| [전] 끝 = 수렴 결정체 결산 또는 양면성 (그림자) | 필수 | 수렴 완결 |
| 분기 N ≥ 3건 | 필수 — 양식 본질 | 2건 이하는 영역확장형 또는 사례비교형 검토 |
| 수렴 1점 명확성 | 필수 | 모든 분기 변수가 한 점에 압축되는 구심점 (흑각궁 같은 단일 결정체) |
| 이중 Aha 구조 (수렴 정체 + 양면성 재해석) | 권장 | 결합형과 유사한 이중 폭발 — 표본 1편 한정이라 권장 |

#### 변주 허용 영역 (가변 — 주제별 자유)

| 항목 | 가변 범위 | 사유 |
|---|---|---|
| [승]·[전] 명칭 (세계 활 분기 / 한반도 수렴 외 가변) | 주제 분기·수렴 영역에 맞춰 자유 명명 | 표본 1편 — 가변성 검증 후속 필요 |
| 분기 단위 (사례·국가·환경·시기·기술 등) | 자유 선택 | 31 흑각궁 = 국가·환경 분기 |
| 분기 수 (N=3~5건) | 자유 — 주제 복잡도에 맞춤 | 31 흑각궁 = 4분기 |
| 이문호공 조합 | 권장 조합이지 강제 매칭 아님 | [승] = 8+6 / [전] = 6+5+7+1 권장이지 새 주제는 다를 수 있음 |
| 9유형 비율 | ±5%p 가변 | 주제별 정보 가용성에 따라 |
| 결론부 블록 수 | 1~3블록 자유 | 본 채널 결론부 글자수 제거 정책 반영 |
| Aha 횟수 | 단일 또는 이중 권장 | 표본 1편은 이중 |

#### P3 결정 트리 (다층분기수렴형 새 주제 진입 시)

```
1. 주제의 "분기 N≥3건 → 1점 수렴 가능성" 확인 — N개 분기 변수가 한 점 결정체로 압축되는가?
   (세계 N국이 한반도 흑각궁으로 수렴 / 다양한 시도가 한 산물로 압축)
   YES → 다층분기수렴형 채택 (이미지 강화형). NO → 분기 N=2는 영역확장형 / 단일 원인 다층 수렴은 다층원인수렴형 / 시간 축 단일 인과는 인과추적형 검토.

2. 분기 단위 결정 — 사례·국가·환경·시기·기술 중 1종
   31 흑각궁 표준: 환경 분기(습한/건조/유럽/유목민)

3. 수렴 1점 명확화 — 모든 분기 변수가 압축되는 한 점 결정체 정체 확정
   31 흑각궁 표준: 흑각궁 (합성궁 + 단궁 + 물소뿔)
   **수렴 1점이 모호하면 본 엔진 채택 금지** — 다른 양식 검토

4. [승] 분기 N개 설계 (3~5건) — 각 분기 1블록씩 (또는 정점 분기는 2블록)
   분기마다 사례·일화 중심 + 인과·메커니즘 보조
   [승] 마지막 = 가장 강한 분기 정점 (몽골 같은 압박 정점)

5. 수렴 진입 1블록 — [전] 첫 블록에 보상+오픈루프
   "이처럼 [모든 분기 변수]가 한 점에서 X로 수렴" 어법
   반전형 6 + 경고형 7 결합어 작성

6. [전] 수렴 사다리 설계 — 수렴 정체(1차 Aha) + 보너스 강점 + 양면성/그림자(2차 Aha)
   3~5블록 권장 — 이중 Aha 구조 권장

7. 결론부 — 수렴 결정체의 시간축 마무리 + 현대 회귀 + 메시지
```

#### 새 주제 가변성 강조

본 권장안은 **39편 중 1편 다층분기수렴엔진 분석의 잠정 패턴**이다. **표본 1편 — 일반화 큰 한계.** 새 주제가 다음 경우라면 권장안에서 벗어나도 무방:

- 주제가 **세계 N국 → 한국 수렴** 구조인 경우 → 6 반전형 + 7 경고형 가중 (한반도 압박 절박감)
- 주제가 **다양한 기술·산업 시도 → 한 산물 수렴** 구조인 경우 → 1 이익 제시형 + 9 최신 정보형 가중
- 주제가 **분기 변수가 2개**인 경우 → 영역확장엔진 채택 권장 — 본 엔진 본질 약화
- 주제가 **수렴 1점이 모호**한 경우 → 사례나열엔진(균등 N건) 또는 다층원인수렴엔진(원인 N층 → 결과 1점) 검토 권장
- 주제가 **단일 Aha 폭발**인 경우 → [전] 압축(2~3블록) + 수렴 정체만 명시
- **[전] 진입 표기** = "수렴 진입" 고정 (영역확장 "영역 전환점"·사례비교 "사례 전환점"·인과추적·다층원인수렴 "[전] 진입"·역설·결합형 "피벗"과 구분)

## § 11. 5단계 P3 구조설계 변환 권장안

본 산출물 4 전체를 P3 구조설계 5단계에서 활용하는 권장안. 본 절은 P3 진입 시 본 산출물 어디를 어떻게 참조할지 결정 흐름을 정리.

### 11.1 본 산출물 4 활용 시점

P3 구조설계 시점 — 본 채널 3유형(모순형/이미지 강화형/과정 추적형) 결정 후, 양식별 영역 분석 (§2~§9)을 참조하여 [기]·[승]·[전]·[결] 정보 배치 결정.

- P1 결과 = 주제 후보 + 본 채널 3유형 후보
- P2 결과 = 팩트 리포트
- P3 진입 = 본 채널 3유형 확정 → 매칭 엔진 결정 → 본 산출물 §2~§9 해당 절 참조 → 영역 A·B 명명 + 정보 배치

### 11.2 P3 결정 흐름 (5단계)

1. **주제 결정** → 본 채널 3유형 후보 선정 (P1 단계 결과)
2. **본 채널 3유형 확정** → 매칭 엔진 결정 (§10.3 표)
3. **양식별 영역 분석 참조** (§2~§9 해당 절)
   - 핵심 패턴 (강제) 적용
   - 변주 허용 영역 (가변) 주제 적응
4. **부가 표기 영역 결정** ([승] 영역 A 명칭·[전] 영역 B 명칭)
5. **각 영역 정보 유형 비중 결정** (양식별 권장 범위 ±5%p)

### 11.3 다양한 주제 가변성 인정 원칙 (재강조)

본 산출물 분포는 39편 평균치이며 **권장 범위**. 새 주제마다 정보 유형·후킹 정보가 다양 → 양식 본질만 유지하고 영역별 세부 비중은 주제에 따라 가변.

- 9유형 분포는 양식별 **권장 범위** (강제 비율 아님)
- 끌리는 이유(이문호공) 조합은 **권장 조합** (강제 매칭 아님)
- 영역별 정보 배치는 **핵심 패턴(강제) + 변주 허용 영역(가변)** 분리
- **표본 작은 엔진** (결합형 2편·다층원인수렴 3편·사례비교 3편·사례나열 1편·다층분기수렴 1편) — 잠정안 명시. 후속 분석 누적 후 재검증

### 11.4 8엔진 P3 변환 가이드 색인

| 본 채널 3유형 | 매칭 엔진 | 본 산출물 4 참조 절 |
|---|---|---|
| 모순형 | 역설 | §2 |
| 모순형 | 결합형 (역설+인과) | §5 (표본 2편 잠정) |
| 이미지 강화형 | 영역확장 | §3 |
