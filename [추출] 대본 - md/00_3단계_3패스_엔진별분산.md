# 3단계 3-패스 산출물 3 — 엔진별 vs 주제별 분산 비교

> **목적:** 정보 배치 패턴(4축)을 결정하는 주된 변수가 "엔진(양식)"인가 "주제"인가를 정량 비교하여, 5단계 P3 양식 선택 가이드의 직접 재료를 제공한다.
> **정합 기준:** `00_가이드_분석_방법.md` §4.6·§7.3 + Step 0 4축 재분류 결과(`00_3단계_Step0_엔진검증.md`) + 산출물 1·2 활용.

## § 1. 작업 개요

### 1.0 사전 지식 블록

- **8엔진 정식 목록:** 역설 / 영역확장 / 인과추적 / 결합형 / 다층원인수렴 / 사례비교 / 사례나열 / 다층분기수렴 (산출물 1 §7.2와 동일)
- **본 채널 3유형(서사 의도):** 모순형(역설 본질) / 이미지 강화형(영역확장 본질) / 과정 추적형(인과추적·다층 본질)
- **용어 통일:** 본 산출물에서 **엔진(= 양식)**은 동의어 — 이하 본문은 "양식"으로 통일하되 표·표제는 "엔진"으로 유지(분류 식별자 역할)
- **동반 자료:**
  - 가이드 §4.6·§4.14.1·§7.3
  - Step 0(`00_3단계_Step0_엔진검증.md`) — 4축 재분류 결과의 1차 기준
  - 산출물 1(`00_3단계_3패스_위치유형매핑.md`) — 39편 위치-유형 비교 표(매트릭스)
  - 산출물 2(`00_3단계_3패스_레고작동패턴.md`) — 엔진별 4역할 비중·요인 조합 패턴

### 1.1 핵심 질문

> **정보 배치 패턴(시간 구조·대립 구조·인과 깊이·사례 수 4축)을 결정하는 주된 변수가 "엔진(양식)"인가 "주제"인가?**

- 양식 변수 강세 → 새 주제 진입 시 양식 선택이 P3 구조 설계의 결정 변수
- 주제 변수 강세 → 양식 선택보다 주제 자체에서 자연스럽게 도출되는 배치 우선

### 1.2 분석 방법

1. **양식 효과 검증** — 같은 엔진 묶음 내 4축 분포 일관성 측정 (엔진 변수 효과)
2. **주제 효과 검증** — 같은 주제 영역 내 다른 엔진 편들 4축 분포 차이 측정 (주제 변수 효과)
3. 두 변수 영향력 비교 + Step 0 일치율 84.6% 정합 재검증
4. P3 양식 선택 지침 변환 권장안 도출

### 1.4 4축 코드 표기

본 산출물에서는 다음 4축 표기를 사용한다.

- **T (시간 구조):** 시간순 / 비시간순 / **혼재** (= 시간순·비시간순이 한 편 안에서 혼합 작동, 어느 한쪽으로 단정 불가)
- **C (대립 구조):** 속성 대립(역설형) / 시기 대립(인과형) / 영역 대립(확장형) / 없음
- **K (인과 깊이):** 단일 / 다층
- **N (사례 수):** **단일 사례** (1개 사례를 깊이 다룸) / 복수 비교 (2~3개 사례를 나란히 비교) / 복수 나열 (3개 이상 사례를 순차 나열) / **복수 나열→단일** (복수로 시작해 단일 사례로 수렴하는 진행)

## § 2. 양식 효과 검증 (같은 엔진 다른 주제)

각 엔진 묶음 내 4축 분포 일관성을 측정한다. 주제가 다르더라도 4축이 일관되면 양식 변수가 4축을 결정한다는 근거가 된다.

> **4축 일관율 산출 공식:** 4축 각각의 일관율(해당 묶음 내 다수 값 점유 비율)을 산술 평균. 예) T 60% + C 100% + K 100% + N 100% → 4축 일관율 = (60+100+100+100)/4 = 90%.

## § 4. 양식 vs 주제 변수 영향력 비교

### 4.1 정량 비교 표(매트릭스)

| 비교 차원 | 양식 효과 검증 결과 | 주제 효과 검증 결과 |
|---|---|---|
| 표본 묶음 | 6엔진 (표본 2편 이상) | 6주제 영역 |
| 핵심 축 일관율 | 평균 90% 이상 (역설·인과추적 100%) | 0% (엔진 묶음(클러스터)별 분리만 일관) |
| 주제 변동에도 4축 보존 | ✓ (역설·영역확장·인과추적·결합형·다층원인수렴·사례비교) | — |
| 엔진 변동 시 4축 변동 | ✓ (모든 주제 영역) | — |

### 4.2 결정 변수 판정

> **결정 변수 = 양식(엔진) ≫ 주제**

근거:
1. **같은 엔진·다른 주제 → 4축 핵심 보존** (역설·인과추적 100% 일관)
2. **같은 주제·다른 엔진 → 4축 즉시 변동** (6개 주제 영역 전체)
3. 산출물 1 §7.2 발견 정합: 엔진별 위치-유형 강세 패턴 일관 (영역확장 메커니즘 27건, 인과추적 인과 56건 등)
4. 산출물 2 §2.3 발견 정합: 엔진별 4역할 비중 패턴 일관 (다층분기수렴 충전 65%, 사례나열 보상 59% 등)

### 4.3 주제 변수의 보조 역할

주제 변수는 4축을 결정하지는 않지만 다음 보조 역할을 한다:

- **자연스러운 양식 유인:** 음식·식문화 8편 중 4편이 인과추적·결합형 사용 (역사 추적 친화)
- **양식 다중 호환 주제:** 음식 주제는 6엔진에서 모두 작동 — 양식 선택의 자유도가 높은 영역
- **양식 제한 주제:** 한국 정체성·자연·과학 영역은 인과추적·다층 계열 우세 — 양식 선택의 자유도가 낮을 수 있음 (표본 한계)

## § 5. Step 0 4축 재분류 결과 정합 재검증

### 5.1 Step 0 일치율 84.6% 의미 재해석

Step 0의 4축 재분류 vs 기존 8엔진 분류 일치율 84.6%는 다음을 의미한다:

- **일치율이 높다(>80%)** → 4축이 양식을 잘 식별 → 본 산출물 §2의 양식 효과 강세 결론과 정합
- **불일치 6편은 모두 인과추적 내부의 시기 대립 보유 케이스** → Step 0 §5.1 정의 정밀화(속성 대립 vs 시기 대립 구분) 적용 시 95~100% 일치 (사실상 100%)
- 95% 둔턱 출처: Step 0 §5.1 정밀화 적용 후 6편 모두 인과추적으로 회수 → 본 산출물 §2.3에서 4축 100% 일관 확인

### 5.2 엔진 분류의 적정성 재확인

| 검증 항목 | 결과 |
|---|---|
| 기존 8엔진 분류가 4축 패턴을 일관 도출하는가 | ✓ (표본 2편 이상 6엔진 모두) |
| 주제 변수가 4축 분류 권한을 갖는가 | ✗ |
| 4축 정밀화(속성·시기·영역 대립) 도입 효과 | 인과추적 71% → 100% 회복 |
| 광고 6편 변주 처리 | 별도 분리 불필요 (산출물 1 §7.1·산출물 2 §10.1 정합) |

→ **기존 엔진 분류 유지** + Step 0 §6.2 권장 사항(대립 구조 3종 정밀화) 반영 권장.

### 5.3 분류 한계·확장 후보

| 한계 | 비고 |
|---|---|
| 사례나열·다층분기수렴 각 1편 표본 | 양식 효과 일관성 검증 불가 — 표본 추가 확보 후 재검증 |
| 결합형 2편 표본 | 이중 골격 변주 패턴 확정 곤란 — 표본 확대 권장 |
| 인과추적 내 시기 대립 5편 | 정의 정밀화로 흡수 — 별도 엔진 분리 불필요 |

## § 6. 핵심 발견

### 6.1 양식 효과 강한 영역

- **역설엔진:** C·K·N 3축 100% 일관 → 양식 결정성 절대
- **인과추적엔진:** 4축 100% 일관 (정밀화 후) → 양식 결정성 절대
- **영역확장엔진:** C 100% + K 86% → 양식 결정성 강

### 6.2 양식 효과 부분적 영역

- **다층원인수렴·사례비교:** T·K 또는 T·N 100%이나 표본 3편 한계 — 추가 검증 필요
- **결합형:** T·C·K 100%이나 표본 2편 한계

### 6.3 주제 효과 약함

- 6개 주제 영역 전체에서 엔진 변경 → 4축 즉시 변동
- 주제 변수가 4축을 결정한 사례 0건
- 주제 변수는 양식 선택의 자유도·유인을 제공하는 보조 변수

### 6.4 엔진 분류 한계·확장 후보

- 표본 1편 엔진(사례나열·다층분기수렴) → 표본 확대 후 양식 효과 재검증 필요
- 인과추적 내 시기 대립 5편은 별도 분리 대신 정의 정밀화로 처리 가능
- 광고 6편은 본문 구조에서 일반 편과 동일 패턴 — 별도 변주 불필요

## § 7. P3 양식 선택 지침 변환 권장안

본 산출물의 핵심 결론(양식 변수 ≫ 주제 변수)을 5단계 P3 지침에 다음과 같이 변환 권장한다.

### 7.1 새 주제 진입 시 양식 선택 가이드

**1순위:** 주제가 어느 양식에서도 작동 가능하다는 전제하에 **집필 의도(서사 유형)에 따라 양식 선택**.

- **모순형 의도** (대상의 양면성·통념 정정) → 역설엔진 / 결합형
- **이미지 강화형 의도** (영역 확장·관점 이동) → 영역확장엔진 / 사례비교
- **과정 추적형 의도** (시간 진행·인과 추적) → 인과추적엔진 / 다층원인수렴

**2순위:** 주제가 양식 선택의 자유도를 제한하는 경우 주제 호환성 검토.

- 한국 정체성·자연·과학 영역은 인과추적·다층 계열 우세 경향 (표본 한계 — 추가 검증 후 확정)
- 음식·식문화·사회·문화·역사·산업 영역은 6엔진 다중 호환 — 양식 선택의 자유도 높음

### 7.2 주제 → 양식 매핑 권장

| 주제 영역 | 우세 양식 (39편 실측) | 대안 양식 |
|---|---|---|
| 음식·식문화 | 인과추적 / 결합형 | 역설·영역확장·사례비교 |
| 역사·국제정치 | 인과추적 | 역설·결합형 |
| 산업·기술 | 인과추적 / 영역확장 | 역설·다층원인수렴 |
| 사회·문화 | 인과추적 / 영역확장 | 역설 |
| 자연·과학 | 인과추적 | 영역확장·사례비교·사례나열 |
| 한국 정체성 | 다층 계열* (인과·원인수렴·분기수렴) | 인과추적 |

> *다층 계열 = 인과추적 + 다층원인수렴 + 다층분기수렴 3엔진 묶음 (다층 K값을 공유하는 엔진군).

### 7.3 양식 선택 시 P2·P3 작업 가중치 영향

양식이 4축을 결정한다는 본 산출물 결론에 따라 P2(딥리서치)·P3(구조설계) 작업 가중치는 다음과 같이 권장된다.

| Phase | 양식 선택 후 가중치 |
|---|---|
| P2 | 양식별 4축 패턴 충족 자료 우선 수집 (예: 역설 = 속성 양면 자료 / 인과추적 = 시간순 사건 사슬 / 영역확장 = 두 영역 사례) |
| P3 | 양식별 위치-유형 강세(산출물 1 §3)·4역할 비중(산출물 2 §2.3) 그대로 적용 |

### 7.4 양식 선택 시 결정 트리 권장

P0~P1 단계에서 주제 확정 후 P3 진입 직전 다음 결정 트리 수행 권장.

```text
1. 집필 의도 식별 (모순형·이미지 강화형·과정 추적형)
   ↓
2. 의도에 대응하는 1차 양식 후보 도출
   ↓
3. 주제 호환성 검토 (§7.2 매핑 표 대조)
   ↓
4. 양식 확정 + 4축 패턴 즉시 적용
   - T·C·K·N 4축을 양식 패턴에 맞춰 설정 (Step 0 §3.1 매핑 규칙 활용)
   ↓
5. 산출물 1·2 패턴 그대로 P3 구조 설계에 투입
```

## § 8. 비고

### 8.1 광고 6편 변주 (대본 12·21·24·25·27·34)

- 광고 6편 모두 본문 골격에서 일반 편과 4축 패턴 일관 — 본 산출물 §2 분석에 포함
- 광고 CTA 부가 블록은 [결] 후반 추가될 뿐 본문 구조 영향 없음
- P3 지침 변환 시 광고 편 별도 변주 불필요 (산출물 1 §7.1·산출물 2 §10.1 정합)

### 8.2 결합형 처리 (대본 23·29)

- 결합형은 역설(속성 대립) + 인과추적(시기 대립) 골격 동시 작동 — Step 0 §1.3 결합형 처리 정합
- 4축 분포는 두 엔진 패턴의 합산: T(시간순) + C(속성+시기) + K(다층) + N(단일/복수)
- 단일 엔진 대비 충전 비중 34%로 살짝 큼 (산출물 2 §10.2) — 이중 골격 누적 비용

### 8.3 표본 한계

- 사례나열·다층분기수렴 각 1편 → 양식 효과 일관성 결론 보류
- 결합형 2편 → 4축 일관성 검증 부분 보류
- 표본 확대 후 재검증 권장

### 8.4 후속 분석 권장

- P3 지침 변환 시 본 산출물 §7 권장안을 1차 재료로 사용
- 5단계 진입 시 산출물 1·2·3 통합 적용 — 위치별 유형(1) + 작동 패턴(2) + 양식 선택(3)
- 추가 표본 확보 시 본 산출물 §2.7·§6.4 한계 재검증

**작성 완료** — 본 산출물은 5단계(지침 변환) P3 양식 선택 지침의 직접 재료. 핵심 결론은 "**양식 변수가 4축의 1차 결정 변수, 주제 변수는 양식 선택의 보조**". 산출물 1·2와 함께 P3·P4 지침 보강 시 §7 권장안 그대로 투입 가능.
